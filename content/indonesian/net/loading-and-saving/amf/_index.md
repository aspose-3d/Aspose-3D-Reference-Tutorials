## Ekspor file AMF terkompresi

Aspose.3D memfasilitasi ekspor adegan 3D ke dalam Additive Manufacturing File Format (AMF) dengan kompresi diaktifkan. Fitur ini memastikan penyimpanan dan transmisi model 3D yang efisien, mengurangi ukuran file sekaligus menjaga fidelitas geometris dan visual dari adegan aslinya.
