## Perguntas Frequentes

**Q:** *Posso usar Aspose.3D em uma aplicação Java comercial?*  
**A:** Sim. Uma licença válida da Aspose é necessária para uso em produção; um teste gratuito está disponível para avaliação.

**Q:** *Quais formatos de arquivo 3D são suportados para carregamento e salvamento?*  
**A:** Aspose.3D suporta OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X e vários outros.

**Q:** *Como atualizo materiais para PBR sem reescrever toda a cena?*  
**A:** Use o método `MaterialConverter.convertToPbr()` fornecido na API – ele atualiza todos os materiais em uma única chamada.

**Q:** *Existe uma maneira de reduzir o tamanho do arquivo ao salvar cenas grandes?*  
**A:** Sim. Configure `SaveOptions` (por exemplo, habilite compressão, reduza a precisão da malha) para otimizar o tamanho da saída.

**Q:** *Preciso de dependências adicionais para trabalhar com Aspose.3D?*  
**A:** Apenas o JAR do Aspose.3D for Java é necessário; todas as outras dependências são incluídas.
