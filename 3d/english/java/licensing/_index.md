## Frequently Asked Questions

**Q: Can I use the same license file across multiple Java projects?**  
A: Yes, the license file can be shared across projects as long as each project references it correctly in the classpath.

**Q: What if I receive a “License not found” error?**  
A: Verify that the `.lic` file path is correct and that the file is included in your build’s resources folder.

**Q: Does the license cover all Aspose.3D file formats?**  
A: A full license enables all supported formats, including OBJ, STL, FBX, and more.

**Q: How do I switch from a trial to a full license?**  
A: Replace the trial `.lic` file with your purchased license file and redeploy the application.

**Q: Is there a way to programmatically check if the license loaded successfully?**  
A: Yes—catch any `LicenseException` when calling `setLicense`; if no exception is thrown, the license is active.
