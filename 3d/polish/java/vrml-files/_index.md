## Frequently Asked Questions

**Q: Czy mogę używać Aspose.3D w komercyjnym projekcie Java?**  
A: Tak, wymagana jest licencja komercyjna do użytku produkcyjnego. Dostępna jest darmowa wersja próbna do oceny.

**Q: Jakie wersje Java są obsługiwane?**  
A: Aspose.3D obsługuje Java 8 i nowsze, w tym wydania z długoterminowym wsparciem, takie jak Java 11 i Java 17.

**Q: Jak przekonwertować plik VRML na inny format?**  
A: Załaduj plik VRML za pomocą `Scene.load()`, a następnie wywołaj `scene.save("output.obj")` (lub inny obsługiwany format).

**Q: Co zrobić, jeśli mój plik VRML zawiera tekstury, które się nie ładują?**  
A: Upewnij się, że ścieżki do tekstur są prawidłowe i że pliki są dostępne w czasie wykonywania. Aspose.3D osadzi brakujące tekstury jako zastępniki.

**Q: Czy istnieje sposób na zmniejszenie zużycia pamięci przez duże sceny VRML?**  
A: Użyj `scene.optimize()` przed przetwarzaniem lub eksportem, aby zminimalizować zużycie zasobów.
