## Schritt‑für‑Schritt‑Anleitung (Keine Codeblöcke erforderlich)

1. **Richten Sie Ihr Projekt ein** – Fügen Sie die Aspose.3D Maven/Gradle‑Abhängigkeit hinzu und importieren Sie die erforderlichen Klassen.  
2. **Laden oder erstellen Sie eine Szene** – Verwenden Sie `Scene scene = new Scene();` oder laden Sie eine vorhandene Datei mit `scene.load("model.fbx");`.  
3. **Lokalisieren Sie den Kugelknoten** – Wenden Sie eine XPath‑ähnliche Abfrage wie `scene.selectNodes("//Sphere[@name='MySphere']")` an.  
4. **Ändern Sie den Radius** – Durchlaufen Sie die zurückgegebenen Knoten und rufen Sie `sphere.setRadius(newRadius);` auf.  
5. **Aktualisieren Sie die Ansicht** – Rufen Sie `scene.update();` auf, um sicherzustellen, dass das Viewport die Änderung widerspiegelt.  
6. **Speichern Sie die aktualisierte Szene** – Exportieren Sie in das gewünschte Format (OBJ, FBX, GLTF) mit `scene.save("updated.fbx");`.

## Fehlerbehebungstipps
- **Null‑Referenz‑Fehler:** Stellen Sie sicher, dass der Kugelknoten abgerufen wurde, bevor Sie `setRadius()` aufrufen.  
- **Szene wird nicht aktualisiert:** Rufen Sie `scene.update()` nach der Geometrieänderung auf, um das Viewport zu aktualisieren.  
- **Lizenzprobleme:** Prüfen Sie, ob die Aspose.3D‑Lizenzdatei korrekt geladen ist; andernfalls erscheint ein Test‑Wasserzeichen.
