## Häufige Probleme und Lösungen

MaterialConverter ist eine Hilfsklasse, die veraltete Materialdefinitionen in das Physically Based Rendering (PBR)-Format konvertiert.

**Problem:** *Materialien verlieren nach der Konvertierung Texturen.*  
**Lösung:** Verwenden Sie `MaterialConverter.convertToPbr()` vor dem Speichern; es bewahrt Textur‑Maps und aktualisiert Shader‑Parameter automatisch.

LoadOptions ist eine Konfigurationsklasse, die steuert, wie Aspose.3D Eingabedateien analysiert.

**Problem:** *Große OBJ‑Dateien verursachen OutOfMemoryError.*  
**Lösung:** Aktivieren Sie das Streaming über `LoadOptions.setEnableMemoryOptimization(true)` und erhöhen Sie die JVM‑Heap‑Größe (`-Xmx4g`).

SaveOptions konfiguriert Ausgabeeinstellungen wie Format, Kompression und Funktionsinklusion beim Speichern einer Szene.

**Problem:** *Exportierte FBX‑Dateien fehlen Animationsdaten.*  
**Lösung:** Setzen Sie `SaveOptions.setExportAnimations(true)` und prüfen Sie, ob die Quellszene Schlüssel‑Animationsspuren enthält.
