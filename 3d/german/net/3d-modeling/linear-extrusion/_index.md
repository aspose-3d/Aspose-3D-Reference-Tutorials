## Was ist 3D-Modellierungs-Extrusion?
3D-Modellierungs-Extrusion ist eine grundlegende Technik im computergestützten Design, die eine flache Skizze in ein festes Objekt verwandelt, indem sie entlang einer geraden Linie gezogen wird. In Aspose.3D stellt die API einfache Eigenschaften bereit, mit denen Sie die Extrusionslänge, Richtung und zusätzliche Effekte wie Drehungen oder Offsets steuern können.
