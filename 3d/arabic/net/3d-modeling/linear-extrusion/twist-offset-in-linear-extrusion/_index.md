# كيفية إضافة الالتواء في البثق الخطي باستخدام Aspose.3D لـ .NET

## المقدمة

إذا كنت تبحث عن دليل واضح خطوة بخطوة حول **كيفية إضافة الالتواء** إلى بثق خطي، فأنت في المكان المناسب. في هذا البرنامج التعليمي سنستعرض العملية بالكامل باستخدام Aspose.3D لـ .NET، موضحين لك **كيفية استخدام البثق** لإنشاء أشكال ثلاثية الأبعاد ديناميكية مثالية لسيناريوهات *asp.net 3d modeling*. في النهاية ستحصل على مثال جاهز للتنفيذ يُظهر إزاحة الالتواء، الشرائح، وحفظ النتيجة كملف OBJ.

## إجابات سريعة
- **ماذا يفعل “إزاحة الالتواء”؟** يقوم بتحريك نقطة بدء الالتواء على طول محور البثق.  
- **هل أحتاج إلى ترخيص لتشغيل العينة؟** الترخيص المؤقت يكفي للاختبار؛ الترخيص الكامل مطلوب للإنتاج.  
- **ما إصدارات .NET المدعومة؟** .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6+.  
- **هل يمكنني تغيير عدد الشرائح؟** نعم—قم بتعديل خاصية `Slices` للتحكم في سلاسة الشبكة.  
- **هل صيغة الإخراج مقتصرة على OBJ؟** لا، Aspose.3D يدعم صيغًا متعددة؛ تم اختيار OBJ هنا للبساطة.

## ما هو إزاحة الالتواء في البثق الخطي؟

إزاحة الالتواء تُعرّف التحول الانتقالي المطبق على عملية الالتواء. بدلاً من الدوران حول نقطة بدء البثق بالضبط، يبدأ الشكل الهندسي بالدوران من المتجه المحدد كإزاحة، مما يمنحك تحكمًا فنيًا أكبر في الشكل النهائي.

## لماذا نستخدم Aspose.3D لـ .NET؟

- **API كامل المميزات** – يدعم الملفات التعريفية، التحويلات، ومجموعة واسعة من صيغ الملفات.  
- **متعدد المنصات** – يعمل على Windows، Linux، و macOS مع .NET Core.  
- **محسن للأداء** – يولد شبكات نظيفة دون الحاجة إلى حسابات يدوية.  
- **توثيق ممتاز** – أمثلة كثيرة لتسريع عملية التطوير.

## استيراد المساحات الاسمية

أولاً، استورد المساحات الاسمية التي تمنحك الوصول إلى الفئات الأساسية ثلاثية الأبعاد.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

هذه العبارات تجعل الفئات `Scene`، `LinearExtrusion`، `Vector3` وغيرها من الأنواع الأساسية متاحة في الكود الخاص بك.

## دليل خطوة بخطوة

### الخطوة 1: تهيئة الملف التعريفي الأساسي

نبدأ بملف تعريف مستطيل بسيط ونضيف له نصف قطر تقويمي صغير حتى لا تكون الحواف حادة تمامًا.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

### الخطوة 2: إنشاء مشهد

كائن `Scene` يعمل كحاوية لجميع العقد، الأضواء، الكاميرات، والهياكل الهندسية.

```csharp
Scene scene = new Scene();
```

### الخطوة 3: إنشاء العقد

يتم إضافة عقدتين فرعيتين إلى جذر المشهد—واحدة للبثق العادي والأخرى للإصدار الملتوي. تُنقل العقدة اليسرى على محور X للفصل البصري.

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(18, 0, 0);
```

### الخطوة 4: البثق الخطي على العقدة اليسرى (بدون إزاحة الالتواء)

هنا نعرض بثقًا أساسيًا مع الالتواء الكامل 360° و100 شريحة للحصول على سلاسة.

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100 });
```

### الخطوة 5: البثق الخطي على العقدة اليمنى مع إزاحة الالتواء

الآن نطبق إزاحة الالتواء `(3, 0, 0)`. هذا ينقل نقطة بدء الالتواء ثلاث وحدات على محور X، مما ينتج حلزونًا مُزاحًا بوضوح.

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100, TwistOffset = new Vector3(3, 0, 0) });
```

### الخطوة 6: حفظ المشهد ثلاثي الأبعاد

أخيرًا، نكتب المشهد إلى ملف OBJ. غيّر مسار الإخراج حسب الحاجة لبيئتك.

```csharp
scene.Save("Your Output Directory" + "TwistOffsetInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```
