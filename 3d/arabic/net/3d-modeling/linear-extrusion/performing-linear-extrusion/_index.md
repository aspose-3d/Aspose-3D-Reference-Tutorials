# كيفية إجراء استخراج خطي باستخدام Aspose.3D لـ .NET

## المقدمة

مرحبًا بكم في **linear extrusion tutorial**! في هذا الدليل ستكتشف **how to linear extrude** ملف تعريف ثنائي الأبعاد وتحوّله إلى كائن ثلاثي الأبعاد كامل باستخدام Aspose.3D لـ .NET. سواء كنت تبني تطبيقًا بأسلوب CAD أو تحتاج فقط إلى **export 3d model obj** للمعالجة اللاحقة، فإن هذا الشرح خطوة بخطوة سيمنحك الثقة لإضافة إنشاء هندسي قوي إلى مشاريعك.

## إجابات سريعة
- **ما هو استخراج خطي؟** تمديد شكل ثنائي الأبعاد على طول مسار مستقيم لإنشاء صلب ثلاثي الأبعاد.  
- **أي مكتبة نستخدم؟** Aspose.3D لـ .NET.  
- **هل أحتاج إلى ترخيص؟** الترخيص المؤقت يكفي للاختبار؛ الترخيص الكامل مطلوب للإنتاج.  
- **هل يمكنني التصدير إلى OBJ؟** نعم – يتم حفظ المشهد النهائي كملف Wavefront OBJ.  
- **كم عدد أسطر الكود؟** أقل من 30 سطرًا من C# بالإضافة إلى تعليقات توضيحية.

## ما هو استخراج خطي؟

يأخذ الاستخراج الخطي ملف تعريف مسطح (مثل المستطيل أو الدائرة) ويمرره على طول خط مستقيم، مع إمكانية إضافة لفات، أو تحجيم، أو إزاحات. النتيجة هي صلب يمكن عرضه، تحريره، أو تصديره للاستخدام في أدوات ثلاثية الأبعاد أخرى.

## لماذا نستخدم Aspose.3D للاستخراج الخطي؟

* **بدون تبعيات:** لا حاجة إلى نوى CAD خارجية.  
* **دعم كامل لـ .NET:** يعمل مع .NET Framework، .NET Core، و .NET 5/6+.  
* **مرونة التصدير:** حفظ مباشر إلى OBJ، STL، FBX، والعديد من الصيغ الأخرى.  
* **API غني:** التحكم في الشرائح، اللف، والإزاحات ببضع خصائص فقط.

## استيراد المساحات الاسمية

قم بتضمين المساحات الاسمية الأساسية لفتح إمكانيات Aspose.3D:

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

هذه المساحات الاسمية تمنحك الوصول إلى `Scene`، `LinearExtrusion`، `RectangleShape`، وفئات المساعدة مثل `Vector3`.

## تنفيذ الاستخراج الخطي

فيما يلي سير العمل الكامل. يتم شرح كل خطوة بلغة بسيطة قبل كتلة الكود المقابلة، حتى تتمكن من المتابعة دون تخمين ما يفعله الكود.

### الخطوة 1: تهيئة ملف التعريف الأساسي

أولاً، أنشئ الشكل الثنائي الأبعاد الذي سيُستخرج. في هذا المثال نستخدم مستطيلًا بنصف قطر تقويس صغير.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

*لماذا هذا مهم:* يحدد ملف التعريف المقطع العرضي للكائن الثلاثي الأبعاد النهائي. عدّل `RoundingRadius` أو العرض أو الارتفاع للحصول على أشكال مختلفة.

### الخطوة 2: تطبيق الاستخراج الخطي

الآن نقوم بتمرير الملف التعريفي 10 وحدات على محور Z، مع إضافة 100 شريحة للنعومة، وتوسيط الهندسة، وتطبيق لف كامل 360° مع إزاحة.

```csharp
var extrusion = new LinearExtrusion(profile, 10) { Slices = 100, Center = true, Twist = 360, TwistOffset = new Vector3(10, 0, 0) };
```

*نصيحة احترافية:* جرب تعديل `Slices` لتحقيق التوازن بين الأداء والجودة البصرية، وجرب `Twist` للحصول على تأثيرات لولبية.

### الخطوة 3: إنشاء مشهد

`Scene` يعمل كحاوية لجميع الكيانات الثلاثية الأبعاد—فكر فيه كقماش للرسم.

```csharp
Scene scene = new Scene();
```

### الخطوة 4: إضافة الاستخراج إلى المشهد

اربط الهندسة المستخرجة بعقدة الجذر في المشهد لتصبح جزءًا من التسلسل الهرمي القابل للتصوير.

```csharp
scene.RootNode.CreateChildNode(extrusion);
```

### الخطوة 5: حفظ النموذج الثلاثي الأبعاد

أخيرًا، صدّر المشهد إلى ملف OBJ شائع الاستخدام. هذا يوضح قدرة **export 3d model obj** في Aspose.3D.

```csharp
scene.Save(RunExamples.GetOutputFilePath("LinearExtrusion.obj"), FileFormat.WavefrontOBJ);
```

عند فتح الملف الناتج `LinearExtrusion.obj` في أي عارض ثلاثي الأبعاد، سترى منشورًا مستطيلاً ملفوفًا—تمامًا كما وصفه الكود.

## المشكلات الشائعة والنصائح

| المشكلة | لماذا يحدث | كيفية الإصلاح |
|-------|----------------|------------|
| **الملف التعريفي غير مرئي** | نسيان إضافة الاستخراج إلى المشهد. | تأكد من استدعاء `CreateChildNode`. |
| **اللف يبدو مسطحًا** | `Slices` منخفض جدًا، مما يسبب هندسة خشنة. | زد `Slices` (مثلاً إلى 200) للحصول على لف أكثر سلاسة. |
| **فشل التصدير** | المجلد الهدف غير موجود أو لا توجد صلاحية كتابة. | استخدم `RunExamples.GetOutputFilePath` أو أنشئ المجلد يدويًا. |
| **تحجيم غير متوقع** | `Center` مضبوط على `false` يغيّر موضع الهندسة. | اضبط `Center = true` ما لم تحتاج إلى إزاحة. |
