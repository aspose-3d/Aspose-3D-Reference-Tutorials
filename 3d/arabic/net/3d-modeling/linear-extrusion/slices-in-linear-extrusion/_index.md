# كيفية إجراء البثق الخطي مع الشرائح باستخدام Aspose.3D لـ .NET

## المقدمة

مرحبًا بك في عالم التصميم ثلاثي الأبعاد باستخدام Aspose.3D لـ .NET! في هذا الدرس ستكتشف **كيفية إجراء البثق الخطي** مع الشرائح، وهي تقنية تسمح لك بالتحكم في مستوى التفاصيل في نماذجك. سواء كنت مطورًا متمرسًا أو مبتدئًا، سنرشدك خطوة بخطوة، نشرح لك السبب وراء كل إجراء، ونقدم لك نصائح عملية يمكنك تطبيقها فورًا.

## إجابات سريعة
- **ما هو البثق الخطي مع الشرائح؟** هو طريقة لتمديد ملف تعريف ثنائي الأبعاد إلى ثلاثي الأبعاد مع تحديد عدد المقاطع العرضية (الشرائح) المتوسطة التي يتم إنشاؤها.  
- **لماذا نستخدم الشرائح؟** المزيد من الشرائح ينتج انحناءات أكثر سلاسة؛ وعدد أقل من الشرائح يحافظ على خفة الشبكة.  
- **المتطلبات المسبقة؟** Aspose.3D لـ .NET، بيئة تطوير متوافقة مع .NET، ومعرفة أساسية بلغة C#.  
- **الوقت النموذجي للتنفيذ؟** العينة تعمل في أقل من ثانية على جهاز كمبيوتر حديث.  
- **صيغة الإخراج؟** المثال يحفظ إلى Wavefront OBJ، لكن Aspose.3D يدعم صيغًا عديدة.

## ما هو البثق الخطي مع الشرائح؟

يقوم البثق الخطي بأخذ شكل ثنائي الأبعاد (ملف تعريف) وتمديده على طول خط مستقيم لإنشاء جسم ثلاثي الأبعاد. تحدد خاصية **Slices** عدد المقاطع العرضية المتوسطة التي تُدرج بين بداية ونهاية البثق، مما يؤثر على السلاسة وعدد المضلعات.

## لماذا نستخدم الشرائح في البثق الخطي؟

- **التحكم في كثافة الشبكة:** ضبط التوازن بين جودة العرض وحجم الملف.  
- **تحسين الأداء:** استخدم عددًا أقل من الشرائح للتطبيقات الفورية، وأكثر للرسومات عالية الدقة.  
- **مرونة إبداعية:** دمج عدد مختلف من الشرائح على كائنات منفصلة لتسليط الضوء على نية التصميم.

## المتطلبات المسبقة

قبل البدء، تأكد من وجود ما يلي:

- مكتبة Aspose.3D لـ .NET – حمّلها من [هنا](https://releases.aspose.com/3d/net/).  
- بيئة تطوير تدعم C# (Visual Studio، Rider، VS Code، إلخ).  
- إلمام أساسي بصياغة C# ومفاهيم البرمجة الكائنية.  
- رغبة في تجربة الهندسة ثلاثية الأبعاد!

## استيراد المساحات الاسمية

أولاً، استورد المساحات الاسمية التي تمنحك الوصول إلى الفئات الأساسية في Aspose.3D.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

## دليل خطوة بخطوة

### الخطوة 1: تهيئة ملف التعريف الأساسي

نبدأ بشكل مستطيل بسيط ونضيف له نصف قطر تقويس صغير حتى لا تكون الحواف حادة تمامًا.

```csharp
// ExStart:InitializeBaseProfile
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
// ExEnd:InitializeBaseProfile
```

### الخطوة 2: إنشاء مشهد ثلاثي الأبعاد

كائن `Scene` يعمل كحاوية لجميع العقد، الشبكات، الأضواء، والكاميرات.

```csharp
// ExStart:Create3DScene
Scene scene = new Scene();
// ExEnd:Create3DScene
```

### الخطوة 3: إضافة عقدة يسرى وعقدة يمنى

سننشئ عقدتين شقيقتين تحت جذر المشهد. العقدة اليسرى ستحصل على عدد شرائح منخفض، والعقدة اليمنى على عدد شرائح عالي، لتتمكن من مقارنة الفرق البصري.

```csharp
// ExStart:CreateLeftRightNodes
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(15, 0, 0);
// ExEnd:CreateLeftRightNodes
```

### الخطوة 4: تنفيذ البثق الخطي على العقدة اليسرى (تفصيل منخفض)

هنا نقوم ببعث المستطيل مع **شرائحتين** فقط. ينتج ذلك شبكة خشنّة—مناسبة للمعاينات السريعة.

```csharp
// ExStart:LinearExtrusionLeftNode
left.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 2 });
// ExEnd:LinearExtrusionLeftNode
```

### الخطوة 5: تنفيذ البثق الخطي على العقدة اليمنى (تفصيل عالي)

الآن نستخدم **10 شرائح** للحصول على نتيجة أكثر سلاسة. لاحظ كيف تصبح الهندسة أدق.

```csharp
// ExStart:LinearExtrusionRightNode
right.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 10 });
// ExEnd:LinearExtrusionRightNode
```

### الخطوة 6: حفظ المشهد ثلاثي الأبعاد

أخيرًا، اكتب المشهد إلى ملف OBJ. استبدل `"Your Output Directory"` بمسار صالح على جهازك.

```csharp
// ExStart:Save3DScene
scene.Save("Your Output Directory" + "SlicesInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
// ExEnd:Save3DScene
```

## المشكلات الشائعة والحلول

| المشكلة | السبب | الحل |
|-------|--------|------|
| **لم يتم إنشاء الملف** | مسار الإخراج غير صالح أو لا توجد صلاحية كتابة. | استخدم مسارًا مطلقًا وتأكد من وجود المجلد. |
| **الكائن يبدو مسطحًا** | تم ضبط `Slices` إلى 1 أو 0. | اضبط `Slices` على الأقل إلى 2 لتظهر البثقة. |
| **الهندسة غير متوقعة** | نصف قطر التقويس كبير جدًا بالنسبة لحجم المستطيل. | قلل قيمة `RoundingRadius` لتكون أصغر من نصف أصغر جانب للمستطيل. |

## الأسئلة المتكررة

**س: هل يمكنني تغيير اتجاه البثق؟**  
ج: نعم. استخدم خاصية `Transform` على العقدة لتدوير أو إزاحة الهندسة البثقية قبل الحفظ.

**س: هل يدعم Aspose.3D أنواع بثق أخرى؟**  
ج: بالتأكيد. بجانب `LinearExtrusion`، يمكنك استخدام `RevolveExtrusion`، `SweepExtrusion`، وغيرها.

**س: ما صيغ الملفات التي يمكنني التصدير إليها؟**  
ج: يدعم Aspose.3D صيغ OBJ، STL، FBX، 3MF، Collada، والعديد غيرها. فقط غيّر قيمة تعداد `FileFormat`.

**س: هل يمكنني تعيين مادة برمجيًا؟**  
ج: نعم. بعد إنشاء العقدة، عيّن كائن `Material` إلى مجموعة `Entity` الخاصة بها.

**س: كيف يؤثر عدد الشرائح على استهلاك الذاكرة؟**  
ج: زيادة عدد الشرائح يرفع عدد الرؤوس والوجوه، مما يزيد استهلاك الذاكرة بنسبة متناسبة. استخدم أدوات التحليل للعثور على النقطة المثلى لمنصتك المستهدفة.
