# تغيير اتجاه المستوى في المشاهد ثلاثية الأبعاد

## المقدمة

في هذا الدرس الشامل ستتعلم **كيفية تغيير اتجاه المستوى** في مشهد ثلاثي الأبعاد باستخدام Aspose.3D for .NET. سواءً كنت تبني لعبة، عارض CAD، أو تصور علمي، فإن التحكم في اتجاه المستوى أمر أساسي للحصول على عرض دقيق وتصدير صحيح لملفات نموذج 3‑D بصيغة OBJ. دعنا نتبع العملية معًا، خطوة بخطوة.

## إجابات سريعة
- **ماذا يعني “تغيير اتجاه المستوى”؟** تعديل المتجه الأعلى (up‑vector) للمستوى لتدويره في الفضاء ثلاثي الأبعاد.  
- **ما هو تنسيق الملف المستخدم للتصدير؟** Wavefront OBJ (`.obj`).  
- **هل يمكنني تدوير المستوى ثلاثي الأبعاد مباشرة؟** نعم – عدل المتجه `Up` لكائن `Plane`.  
- **هل أحتاج إلى ترخيص؟** نسخة تجريبية مجانية تكفي للتطوير؛ يتطلب الترخيص التجاري للإنتاج.  
- **ما إصدارات .NET المدعومة؟** .NET Framework 4.5+، .NET Core 3.1+، .NET 5/6+.

## ما هو تغيير اتجاه المستوى؟
يشير تغيير اتجاه المستوى إلى إعادة تعريف المتجه العمودي أو المتجه الأعلى للمستوى بحيث يشير إلى اتجاه مختلف داخل نظام الإحداثيات ثلاثي الأبعاد. هذه العملية تقوم فعليًا **بتدوير كائنات المستوى ثلاثية الأبعاد** دون تغيير حجمها أو موقعها.

## لماذا نغير اتجاه المستوى؟
- **محاذاة بصرية دقيقة** – يضمن أن القوام والإضاءة تتصرف كما هو متوقع.  
- **تصدير صحيح** – بعض الأدوات اللاحقة تعتمد على اتجاه مستوى محدد عند استيراد ملفات OBJ.  
- **مرونة** – يمكنك إعادة استخدام نفس الهندسة باتجاهات مختلفة لعرض متعدد.

## المتطلبات المسبقة

- Aspose.3D for .NET: تأكد من تثبيت المكتبة. إذا لم تكن مثبتة، قم بتحميلها من [here](https://releases.aspose.com/3d/net/).  
- دليل المستندات الخاص بك: أنشئ مجلدًا سيقرأ/يكتب فيه الدرس الملفات.

الآن بعد أن غطينا الأساسيات، لننتقل إلى الشيفرة.

## استيراد المساحات الاسم (Import Namespaces)

في مشروع .NET الخاص بك، ابدأ باستيراد المساحات الاسم المطلوبة:

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Utilities;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

توفر هذه المساحات الاسم الفئات والطرق الأساسية للعمل مع المشاهد ثلاثية الأبعاد في Aspose.3D.

## الخطوة 1: تهيئة كائن المشهد (Initialize the Scene Object)

```csharp
// The path to the data directory
string dataDir = "Your Document Directory";

// Initialize scene object
Scene scene = new Scene();
```

هذا الشيفرة تُعد البيئة لمشهدك ثلاثي الأبعاد.

## الخطوة 2: تعيين المتجه لاتجاه المستوى (Rotate 3D Plane)

```csharp
// Set Vector
scene.RootNode.CreateChildNode(new Plane() { Up = new Vector3(1, 1, 3) });
```

هنا، ننشئ عقدة فرعية تمثل مستوى ونخصص اتجاهه باستخدام المتجه `Up`. تعديل قيم المتجه يدور المستوى ثلاثي الأبعاد إلى الزاوية المطلوبة.

## الخطوة 3: حفظ وتصدير نموذج 3D بصيغة OBJ

```csharp
// This will generate a plane that has customized orientation
scene.Save(dataDir + "ChangePlaneOrientation.obj", FileFormat.WavefrontOBJ);
```

حفظ المشهد يولد ملف OBJ يعكس اتجاه المستوى الجديد، مما يتيح لك **تصدير نموذج 3D بصيغة OBJ** للاستخدام في تطبيقات أخرى.

كرر هذه الخطوات حسب الحاجة لمستويات إضافية أو اتجاهات مختلفة.

## المشكلات الشائعة والحلول
- **المستوى يظهر مسطحًا أو مقلوبًا:** تأكد من أن المتجه `Up` ليس متوازيًا مع المتجه العمودي للمستوى. عدل مكونات المتجه لتحقيق الميل المطلوب.  
- **ملف OBJ المُصدّر يبدو فارغًا:** تأكد من أن مسار `dataDir` ينتهي بفاصل (`\\` أو `/`) وأن لديك صلاحيات كتابة.  
- **دوران غير متوقع:** تذكر أن المتجه `Up` يحدد المحور المحلي Y للمستوى؛ تعديلّه يدور المستوى حول محوره X.
