# تقليل حجم ملف 3D – ضغط المشاهد باستخدام Aspose.3D for Java

## المقدمة

إذا كنت تقوم بتسليم أصول 3D عبر الويب، أو عبر البريد الإلكتروني، أو تخزينها في سحابة، يمكن أن تصبح الأحجام الكبيرة للملفات عنق زجاجة سريعًا. في هذا البرنامج التعليمي ستتعلم **كيفية تقليل حجم ملف 3D** عن طريق ضغط مشاهد 3D باستخدام Aspose.3D for Java. سنستعرض إنشاء مشهد، إضافة كائنات، تعديل التحولات، وأخيرًا حفظ المشهد مع خيارات الضغط التي تحافظ على جودة العرض مع تقليل حجم الملف بشكل كبير.

## إجابات سريعة
- **ماذا يعني “تقليل حجم ملف 3d”؟** يعني تطبيق تقنيات الضغط على ملف ثلاثي الأبعاد بحيث يصبح حجمه على القرص أصغر دون فقدان دقة الهندسة أو القوام.  
- **أي صيغة تدعم الضغط في Aspose.3D؟** صيغة AMF (Additive Manufacturing File) باستخدام `AmfSaveOptions`.  
- **هل أحتاج إلى ترخيص للضغط؟** النسخة التجريبية تعمل للتطوير؛ الترخيص التجاري مطلوب للإنتاج.  
- **هل الضغط غير فقدان للبيانات؟** نعم، الضغط المدمج في Aspose.3D غير فقدان للبيانات بالنسبة للهندسة والقوام.  
- **كم نسبة تقليل الحجم التي يمكن توقعها؟** عادةً 30‑60 % حسب تعقيد المشهد وعدد القوام.

## ما هو ضغط المشهد في Aspose.3D؟
ضغط المشهد هو عملية ترميز مشهد ثلاثي الأبعاد إلى تمثيل أكثر كثافة. تستفيد Aspose.3D من ضغط gzip المدمج في صيغة AMF، مما يتيح لك نقل نماذج كبيرة بكفاءة أعلى.

## لماذا تقليل حجم ملف 3D؟
- **تنزيلات أسرع** – يحصل المستخدمون ذوو النطاق الترددي المحدود على تجربة أكثر سلاسة.  
- **تكاليف تخزين أقل** – تُحسب رسوم التخزين السحابي حسب الجيجابايت.  
- **أداء محسّن** – الملفات الأصغر تُحمّل أسرع في المتصفحات ومحركات الألعاب.  
- **مشاركة أسهل** – غالبًا ما تكون هناك حدود لحجم المرفقات في البريد الإلكتروني ومنصات التعاون.

## المتطلبات المسبقة
قبل أن تبدأ، تأكد من وجود ما يلي:

- مجموعة تطوير جافا (JDK) 8 أو أحدث مثبتة.  
- مكتبة Aspose.3D for Java محملة من الموقع الرسمي – يمكنك العثور على رابط التحميل [هنا](https://releases.aspose.com/3d/java/).  
- بيئة تطوير جافا (IntelliJ IDEA، Eclipse، أو VS Code) لإنشاء وتشغيل مشروع العينة.

## استيراد الحزم
أضف الفئات المطلوبة من Aspose.3D إلى ملف المصدر الخاص بجافا:

```java
import com.aspose.threed.AmfSaveOptions;
import com.aspose.threed.Box;
import com.aspose.threed.Scene;
import com.aspose.threed.Transform;
import com.aspose.threed.Vector3;
```

## دليل خطوة بخطوة

### الخطوة 1: إعداد مشروع جافا الخاص بك
أنشئ مشروع جافا جديد في بيئة التطوير المفضلة وأضف ملفات JAR الخاصة بـ Aspose.3D إلى مسار الفئات (classpath). يضمن ذلك أن المترجم يستطيع العثور على الفئات المستوردة.

### الخطوة 2: تهيئة مشهد 3D جديد
ابدأ بإنشاء كائن مشهد فارغ. فئة `Scene` هي الحاوية لجميع الهندسات، الأضواء، الكاميرات، والهرمية.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";

Scene scene = new Scene();
```

### الخطوة 3: إضافة هندسة صندوق بسيطة
للتوضيح، سنضيف بدائية صندوق إلى المشهد. فئة `Box` تنشئ مكعبًا يمكننا تحويله.

```java
Box box = new Box();
Transform tr = scene.getRootNode().createChildNode(box).getTransform();
tr.setScale(new Vector3(12, 12, 12));
tr.setTranslation(new Vector3(10, 0, 0));
```

### الخطوة 4: تخصيص الصندوق (التحجيم، الدوران، الموضع)
يمكنك تعديل الصندوق نفسه أو إضافة نسخ إضافية. أدناه نغيّر التحجيم ونطبق زوايا أويلر لتدوير الكائن.

```java
tr = scene.getRootNode().createChildNode(box).getTransform();
tr.setScale(new Vector3(5, 5, 5));
tr.setEulerAngles(new Vector3(50, 10, 0));
```

### الخطوة 5: حفظ المشهد مع تمكين الضغط
المفتاح لت **تقليل حجم ملف 3d** يكمن في `AmfSaveOptions`. اضبط `setEnableCompression(true)` لتفعيل ضغط gzip داخل ملف AMF.

```java
AmfSaveOptions opt = new AmfSaveOptions();
opt.setEnableCompression(true);   // Turn on compression to shrink file size
scene.save(MyDir + "compressed_scene.amf", opt);
```

> **نصيحة احترافية:** إذا كنت بحاجة إلى الاحتفاظ بنسخة غير مضغوطة لأغراض التصحيح، احفظ نسخة ثانية مع `setEnableCompression(false)`.

كرر الخطوات السابقة لأي كائنات إضافية ترغب في تضمينها في المشهد. سيُخزن كل كائن في نفس الحاوية المضغوطة، مما يحافظ على انخفاض حجم الملف الإجمالي.

## المشكلات الشائعة والحلول
| المشكلة | السبب | الحل |
|-------|-------|-----|
| **الملف المحفوظ لا يزال كبيرًا** | تم تعطيل الضغط أو استخدام صيغة لا تدعم ذلك (مثل OBJ). | تأكد من `opt.setEnableCompression(true)` واحفظ كـ **AMF**. |
| **القوام لا تظهر بعد التحميل** | القوام لم تُدمج؛ المسار خارجي. | استخدم `scene.getRootNode().getMaterial().setTexture(...).setEmbed(true)`. |
| **OutOfMemoryError في المشاهد الكبيرة** | تحميل المشهد بالكامل في الذاكرة قبل الحفظ. | فعّل وضع البث عبر `AmfSaveOptions.setEnableStreaming(true)`. |
