# تحويل النموذج إلى FBX باستخدام الكواتيرنيونات في جافا باستخدام Aspose.3D

## المقدمة

إذا كنت بحاجة إلى **تحويل النموذج إلى FBX** مع تطبيق دوران سلس، فأنت في المكان الصحيح. في هذا الدرس سنستعرض مثالًا كاملاً بلغة جافا يستخدم Aspose.3D لإنشاء مكعب، وتدويره باستخدام الكواتيرنيونات، وأخيرًا **حفظ المشهد كملف FBX**. في النهاية ستحصل على نمط قابل لإعادة الاستخدام لأي كائن ثلاثي الأبعاد تريد تصديره إلى صيغة FBX.

## إجابات سريعة
- **ما المكتبة التي تتعامل مع تصدير FBX؟** Aspose.3D for Java  
- **ما نوع التحويل المستخدم؟** دوران قائم على الكواتيرنيون للتق interpolations السلسة  
- **هل أحتاج إلى ترخيص للإنتاج؟** نعم، يلزم ترخيص تجاري (يتوفر نسخة تجريبية مجانية)  
- **هل يمكنني تصدير صيغ أخرى؟** نعم، يدعم Aspose.3D صيغ OBJ، STL، GLTF، والمزيد  
- **هل الكود متعدد المنصات؟** بالتأكيد – جافا تعمل على Windows وLinux وmacOS  

## المتطلبات المسبقة

قبل أن نغوص في الدرس، تأكد من توفر المتطلبات التالية:

- معرفة أساسية ببرمجة جافا.  
- تثبيت Aspose.3D for Java. يمكنك تنزيله [هنا](https://releases.aspose.com/3d/java/).  
- إعداد دليل مستندات لحفظ مشاهدك ثلاثية الأبعاد.

## استيراد الحزم

في هذا القسم، سنستورد الحزم اللازمة للبدء في التحولات ثلاثية الأبعاد باستخدام Aspose.3D.

```java
import com.aspose.threed.*;
```

## الخطوة 1: تهيئة كائن المشهد

للبدء، أنشئ كائن مشهد سيعمل كحاوية لعناصرك ثلاثية الأبعاد.

```java
Scene scene = new Scene();
```

## الخطوة 2: تهيئة كائن فئة العقدة

الآن، أنشئ كائن فئة العقدة، في هذه الحالة يمثل مكعبًا.

```java
Node cubeNode = new Node("cube");
```

## الخطوة 3: إنشاء شبكة باستخدام مُنشئ المضلعات

استخدم الفئة العامة لإنشاء شبكة باستخدام طريقة مُنشئ المضلعات.

```java
Mesh mesh = Common.createMeshUsingPolygonBuilder();
```

## الخطوة 4: تعيين هندسة الشبكة

قم بإسناد الشبكة التي تم إنشاؤها إلى عقدة المكعب.

```java
cubeNode.setEntity(mesh);
```

## الخطوة 5: تعيين الدوران باستخدام الكواتيرنيون

طبق الدوران على عقدة المكعب باستخدام الكواتيرنيونات. الكواتيرنيونات تتجنب قفل الجيمبال وتمنحك دورانًا سلسًا ومستمرًا.

```java
cubeNode.getTransform().setRotation(Quaternion.fromRotation(new Vector3(0, 1, 0), new Vector3(0.3, 0.5, 0.1)));
```

## الخطوة 6: تعيين الإزاحة

حدد إزاحة عقدة المكعب بحيث تظهر في الموقع المطلوب داخل المشهد.

```java
cubeNode.getTransform().setTranslation(new Vector3(0, 0, 20));
```

## الخطوة 7: إضافة المكعب إلى المشهد

أدرج عقدة المكعب في هيكلية المشهد.

```java
scene.getRootNode().getChildNodes().add(cubeNode);
```

## الخطوة 8: حفظ المشهد ثلاثي الأبعاد – تحويل النموذج إلى FBX

الآن نقوم فعليًا **بتحويل النموذج إلى FBX** عن طريق حفظ المشهد بصيغة FBX. يوضح هذا أيضًا سير عمل “حفظ المشهد كـ FBX”.

```java
String MyDir = "Your Document Directory";
MyDir = MyDir + "TransformationToNode.fbx";
scene.save(MyDir, FileFormat.FBX7500ASCII);
System.out.println("\nTransformation added successfully to node.\nFile saved at " + MyDir);
```

## لماذا نستخدم الكواتيرنيونات لتصدير FBX؟

الكواتيرنيونات توفر لك:

- **تقريب سلس** بين الاتجاهات، وهو أمر أساسي للرسوم المتحركة.  
- **عدم وجود قفل جيمبال**، الذي قد يفسد الدورات عند استخدام زوايا أويلر.  
- **تمثيل مضغوط**، مما يوفر الذاكرة في المشاهد الكبيرة.

هذه الفوائد تجعل التحولات المدفوعة بالكواتيرنيون الخيار المفضل عندما تريد **تحويل النموذج إلى FBX** لمحركات الألعاب أو خطوط أنابيب التصور.
