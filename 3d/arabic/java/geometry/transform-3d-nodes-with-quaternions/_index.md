# تحويل النموذج إلى FBX باستخدام الكواتيرنيونات في Java باستخدام Aspose.3D

## مقدمة

إذا كنت بحاجة إلى **تحويل النموذج إلى FBX** مع تطبيق سلس، مطلوب في المكان الصحيح. في هذا الدرس سنستعرض مثالا برمجيا كاملا بلغة Java يستخدم Aspose.3D الاستثمار، وتدويره باستخدام الكواتيرنيونات، وأخيرًا **حفظ المشهد باسم FBX**. في نهاية المطاف في أي مكان قابل لإعادة الاستخدام لأي كائن ثلاثي الأبعاد تريد تصديره إلى صيغة FBX، وستفهم كيف تساعد الكواتيرنيونات على **تجنب gimbal lock**.

## إجابات سريعة
- **ما المكتبة التي واخترتها مع تصدير FBX؟** Aspose.3D for Java
- **أي نوع من التغييرات؟** أثناء قائمة الكواتيرنيون للتداخل الكامل
- **هل أحتاج إلى ترخيص للإنتاج؟** نعم، يتطلب ترخيص تجاري (يتوفر نسخة مجانية مجانية)
- **هل يمكنني تصدير صيغ؟** نعم، Aspose.3D يدعم OBJ و STL و GLTF المتزايد
- **هل الكود متعدد المنصات؟** بالتأكيد – Java تعمل على Windows و Linux و macOS

## ما هو "تحويل النموذج إلى FBX" باستخدام الكواترنيونات؟

استخدام الكواتيرنيونات الأوروبية يتيح لك عقدة هندسية ثلاثية الأبعاد دون مشكلة القفل الميكانيكي (gimbal lock) المزعجة التي تعاني منها زوايا إيلر. عندما **تحويل النموذج إلى FBX**، يتم تخزين البيانات فقط في ملف FBX، مما يحافظ على عدم القدرة على القيام به في شيفرة.

## لماذا نستخدم Quaternions لتصدير FBX؟

الكواتيرنيات توفر لك:
- **تداخل سلس** بين الاتجاهات، وهو ضروري لرسوم المتحركة.
- **بدون قفل ميكانيكي**، الذي يمكن أن يفسد عند استخدام زوايا إيلر.
- **تمثيل استثنائي**، ويوفر الذاكرة في المشاهدة الجديدة.

قبل أن نغوص في الدرس، تأكد من ضرورة المتطلبات التالية:
- معرفة جوهر ببرمجة Java.

- دليل الوثائق مُعد لحفظ مشاهد 3D الخاصة بك.

## استيراد الحزم

في هذا القسم، سنستورد الحزم اللازمة للبدء في تحويلات ثلاثية الأبعاد باستخدام Aspose.3D.

```java
import com.aspose.threed.*;
```

## الخطوة 1: تهيئة كائن المشهد

للبدء، أنشئ كائن مشهد سيعمل كحاوية لعناصر 3D الخاصة بك.

```java
Scene scene = new Scene();
```

## الخطوة 2: تهيئة كائن فئة العقدة

الآن، أنشئ كائن فئة العقدة، في هذه الحالة يمثل مكعبًا.

```java
Node cubeNode = new Node("cube");
```

## الخطوة 3: إنشاء شبكة باستخدام أداة إنشاء المضلعات

استخدم الفئة العامة لإنشاء شبكة باستخدام طريقة مُنشئ المضلع.

```java
Mesh mesh = Common.createMeshUsingPolygonBuilder();
```

## الخطوة 4: ضبط هندسة الشبكة

قم بإسناد الشبكة التي تم إنشاؤها إلى عقدة المكعب.

```java
cubeNode.setEntity(mesh);
```

## الخطوة 5: ضبط الدوران باستخدام الكواتيرنيون

طبق الدوران على عقدة المكعب باستخدام الكواتيرنيونات. الكواتيرنيونات تتجنب القفل الميكانيكي وتوفر لك دورانًا سلسًا ومستمرًا.

```java
cubeNode.getTransform().setRotation(Quaternion.fromRotation(new Vector3(0, 1, 0), new Vector3(0.3, 0.5, 0.1)));
```

## الخطوة 6: ضبط الإزاحة

حدد الإزاحة لعقدة المكعب بحيث تظهر في الموضع المطلوب داخل المشهد.

```java
cubeNode.getTransform().setTranslation(new Vector3(0, 0, 20));
```

## الخطوة 7: إضافة مكعب إلى المشهد

أدرج عقدة المكعب في تسلسل المشهد الهرمي.

```java
scene.getRootNode().getChildNodes().add(cubeNode);
```

## الخطوة 8: حفظ المشهد ثلاثي الأبعاد - تحويل النموذج إلى FBX

الآن نقوم فعليًا **convert model to FBX** عن طريق حفظ المشهد بصيغة FBX. هذا أيضًا يوضح سير عمل “save scene as FBX”.

```java
String MyDir = "Your Document Directory";
MyDir = MyDir + "TransformationToNode.fbx";
scene.save(MyDir, FileFormat.FBX7500ASCII);
System.out.println("\nTransformation added successfully to node.\nFile saved at " + MyDir);
```
