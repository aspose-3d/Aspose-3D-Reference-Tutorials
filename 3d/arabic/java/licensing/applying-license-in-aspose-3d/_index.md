# كيفية تعيين الترخيص في Aspose.3D للـ Java

## المقدمة

مرحبًا! في هذا الدليل خطوة بخطوة ستكتشف **كيفية تعيين الترخيص** لـ Aspose.3D في تطبيق Java وستتعلم أيضًا **كيفية استخدام المفاتيح العامة والخاصة** للترخيص القائم على القياس. Aspose.3D هي مكتبة Java قوية تُبسّط العمل مع صيغ ملفات 3D، والترخيص الصالح يفتح كامل مجموعة الميزات. بنهاية هذا الدليل ستكون قادرًا على دمج الترخيص بسلاسة في أي مشروع Java.

## إجابات سريعة
- **ما هي الطريقة الأساسية لتعيين الترخيص؟** استخدم الفئة `License` واستدعِ `setLicense` مع مسار ملف أو تدفق.  
- **هل يمكن تحميل الترخيص من تدفق؟** نعم – `FileInputStream` يعمل بشكل مثالي.  
- **ما الغرض من المفاتيح العامة/الخاصة؟** تمكّن الترخيص القائم على القياس عبر الفئة `Metered`.  
- **هل أحتاج إلى ترخيص للتطوير؟** ترخيص مؤقت أو تجريبي يكفي للاختبار؛ الترخيص الكامل مطلوب للإنتاج.  
- **ما إصدارات Java المدعومة؟** Aspose.3D يعمل مع Java 6 وما بعده.

## المتطلبات المسبقة

قبل أن نبدأ، تأكد من وجود ما يلي:

- فهم أساسي لبرمجة Java.  
- إضافة مكتبة Aspose.3D إلى مشروعك. حمّلها من [صفحة الإصدار](https://releases.aspose.com/3d/java/).  
- ملف ترخيص `.lic` صالح أو مفاتيحك العامة والخاصة للترخيص القائم على القياس.

## استيراد الحزم

أضف الاستيرادات المطلوبة إلى ملف مصدر Java الخاص بك. تأكد من أن ملف JAR الخاص بـ Aspose.3D موجود في مسار الـ classpath.

```java
import com.aspose.threed.License;
import com.aspose.threed.Metered;

import java.io.FileInputStream;
import java.io.IOException;
```

## كيفية تعيين الترخيص باستخدام ملف

### الخطوة 1: إنشاء كائن الترخيص

أنشئ مثالًا من الفئة `License` – هذا الكائن سيحمل معلومات الترخيص.

```java
License license = new License();
```

### الخطوة 2: تعيين الترخيص من ملف

قدّم المسار النسبي أو المطلق لملف `.lic` الخاص بك وطبق الترخيص.

```java
license.setLicense("Aspose._3D.lic");
```

> **نصيحة احترافية:** احفظ ملف الترخيص خارج دليل التحكم في المصدر لتجنب كشفه عن طريق الخطأ.

## كيفية تعيين الترخيص باستخدام تدفق

### الخطوة 1: إنشاء كائن الترخيص

كما في السابق، ابدأ بمثال جديد من `License`.

```java
License license = new License();
```

### الخطوة 2: تعيين الترخيص من تدفق

اقرأ ملف الترخيص إلى `FileInputStream` ومرّر التدفق إلى `setLicense`. يضمن كتلة `try‑with‑resources` إغلاق التدفق تلقائيًا.

```java
try (FileInputStream myStream = new FileInputStream("Aspose._3D.lic")) {
    license.setLicense(myStream);
}
```

## كيفية استخدام المفاتيح العامة والخاصة للترخيص القائم على القياس

### الخطوة 1: تهيئة كائن ترخيص قياسي

أنشئ مثالًا من الفئة `Metered`، التي تدير الترخيص القائم على القياس (الدفع حسب الاستخدام).

```java
Metered metered = new Metered();
```

### الخطوة 2: تعيين المفاتيح العامة والخاصة

زوّد المفاتيح التي استلمتها من Aspose. هذه المفاتيح تمكّن المكتبة من الإبلاغ عن الاستخدام إلى خادم الترخيص.

```java
metered.setMeteredKey("your-public-key", "your-private-key");
```

> **تحذير:** لا تقم أبدًا بكتابة المفتاح الخاص مباشرة في ملف JAR موزع علنًا. فكر في تحميله من موقع آمن أو من متغيّر بيئي.

## حالات الاستخدام الشائعة

- **خطوط أنابيب التصيير ثلاثي الأبعاد للمؤسسات** – فكّ قفل واجهات برمجة التطبيقات عالية الأداء بعد تعيين الترخيص.  
- **بيئات الاختبار الآلية** – استخدم ترخيصًا مؤقتًا (انظر الأسئلة المتكررة) للتحقق من الوظائف دون شراء ترخيص كامل.  
- **حلول SaaS القابلة للقياس** – دمج المفاتيح العامة/الخاصة لفوترة العملاء بناءً على الاستخدام الفعلي.
