# درس رسومات Java 3D – المركز في البثق الخطي

## المقدمة

إذا كنت تبني تصورات ثلاثية الأبعاد في Java، فإن إتقان تقنيات البثق أمر أساسي. يوجهك هذا **java 3d graphics tutorial** خلال التحكم في مركز البثق الخطي باستخدام Aspose.3D for Java، حتى تتمكن من إنشاء نماذج دقيقة متماثلة دون الحاجة إلى حسابات إضافية. في نهاية هذا الدليل ستفهم لماذا خاصية `center` مهمة، وكيفية **تعيين نصف قطر التقريب**، وكيفية **حفظ ملف OBJ** المتوافق مع Java.

## إجابات سريعة
- **ماذا تفعل خاصية center؟** تحدد ما إذا كان البثق متماثلًا حول أصل الشكل.  
- **هل أحتاج إلى ترخيص لتشغيل الكود؟** ترخيص مؤقت يكفي للاختبار؛ الترخيص الكامل مطلوب للإنتاج.  
- **ما هو تنسيق الملف المستخدم للنتيجة؟** يتم حفظ المشهد كملف Wavefront OBJ.  
- **هل يمكنني تغيير عدد القطاعات؟** نعم، استخدم `setSlices(int)` على كائن `LinearExtrusion`.  
- **هل Aspose.3D متوافق مع Java 8+؟** بالتأكيد – يدعم جميع إصدارات Java الحديثة.

## دليل خطوة بخطوة

### الخطوة 1: إعداد الشكل الأساسي

أولاً، أنشئ الشكل الثنائي الأبعاد الذي سيُبثق. هنا نستخدم مستطيلًا ون **نحدد نصف قطر التقريب** إلى 0.3، مما يُنعّم الزوايا.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

### الخطوة 2: إنشاء مشهد ثلاثي الأبعاد

كائن `Scene` يعمل كحاوية لجميع العقد الثلاثية الأبعاد، والإضاءة، والكاميرات.

```java
Scene scene = new Scene();
```

### الخطوة 3: إضافة عقدتي اليسار واليمين

سنضع عقدتين منفصلتين جنبًا إلى جنب حتى تتمكن من مقارنة البثق مع وبدون تمركز.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

### الخطوة 4: البثق الخطي – بدون مركز (العقدة اليسرى)

أنشئ البثق على العقدة اليسرى، عطل التمركز، وحدد عدد القطاعات إلى ثلاث لتظهر نسخة منخفضة التفاصيل.

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

### الخطوة 5: إضافة سطح أرضي للمرجعية (العقدة اليسرى)

مكعب رقيق يعمل كأرضية بصرية، يساعدك على رؤية اتجاه البثق.

```java
left.createChildNode(new Box(0.01, 3, 3));
```

### الخطوة 6: البثق الخطي – متمركز (العقدة اليمنى)

الآن كرر البثق، هذه المرة مفعلاً `center`. ستصبح الهندسة متماثلة حول أصل الشكل.

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

### الخطوة 7: إضافة سطح أرضي للمرجعية (العقدة اليمنى)

نفس السطح الأرضي للجانب الأيمن، لتوضيح المقارنة.

```java
right.createChildNode(new Box(0.01, 3, 3));
```

### الخطوة 8: حفظ المشهد الثلاثي الأبعاد

أخيرًا، صدّر المشهد بالكامل إلى ملف Wavefront OBJ – تنسيق يمكن استخدامه بسهولة في معظم محررات 3‑D.

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## المشكلات الشائعة والحلول

| المشكلة | السبب | الحل |
|-------|-------|-----|
| **البثق يظهر مُزاحًا** | `setCenter(false)` يترك الشكل مثبتًا عند زاويته. | استخدم `setCenter(true)` للحصول على نتائج متماثلة. |
| **ملف OBJ فارغ** | مسار دليل الإخراج غير صحيح أو لا توجد أذونات كتابة. | تأكد من أن `MyDir` يشير إلى مجلد موجود وأن التطبيق يمتلك صلاحية الكتابة. |
| **الزوايا المدورة تبدو حادة** | نصف قطر التقريب صغير جدًا مقارنةً بحجم المستطيل. | زد قيمة نصف القطر (مثلاً `setRoundingRadius(0.5)`). |
