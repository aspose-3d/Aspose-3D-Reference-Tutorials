# التحكم في المركز في البثق الخطي باستخدام Aspose.3D للـ Java

## المقدمة

عند بناء مشاهد ثلاثية الأبعاد في Java، القدرة على **إضافة سطح أرضي** مع ضبط **خاصية المركز** بدقة في البثق الخطي يمكن أن تكون الفارق بين نموذج أولي مسطح ومرئي مصقول. في هذا الدرس سنستعرض العملية الكاملة للتحكم في مركز البثق وإضافة سطح أرضي باستخدام Aspose.3D للـ Java. ستتعرف على أهمية ذلك، كيفية الإعداد، وستحصل على عينة كود جاهزة للتنفيذ يمكنك تعديلها لتناسب مشاريعك.

## إجابات سريعة
- **ماذا يفعل “add ground plane”?** ينشئ هندسة مرجعية رقيقة تساعدك على رؤية موضع البثق بالنسبة لمحاور العالم.  
- **كيف أضبط مركز البثق الخطي؟** استخدم الطريقة `setCenter(boolean)` على كائن `LinearExtrusion`.  
- **هل أحتاج إلى ترخيص لتشغيل العينة؟** الترخيص المؤقت يكفي للاختبار؛ الترخيص الكامل مطلوب للإنتاج.  
- **ما هو تنسيق الملف المستخدم للحفظ؟** المثال يحفظ بصيغة Wavefront OBJ (`.obj`).  
- **ما نسخة Java المطلوبة؟** Java 8 أو أحدث كافية.

## ما هو “add ground plane” في مشهد ثلاثي الأبعاد؟

إضافة سطح أرضي يعني إدراج هندسة مستطيلة رقيقة (غالبًا صندوق بسمك ضئيل) تقع على مستوى X‑Z. يعمل كأرضية بصرية، مما يسهل تقدير الارتفاع ومحاذاة الكائنات الأخرى، خاصةً عندما تجرب مراكز البثق.

## لماذا ضبط خاصية المركز في البثق الخطي؟

افتراضيًا، يبدأ البثق الخطي من أصل الشكل. ضبط خاصية المركز (`setCenter(true)`) يحرّك الشكل بحيث يكون البثق متمركزًا حول الأصل، وهو مفيد للتصاميم المتناظرة أو عندما تحتاج إلى محاذاة ثابتة عبر عدة كائنات.

## المتطلبات المسبقة

قبل أن نبدأ رحلتنا في هذا الدرس، تأكد من توفر المتطلبات التالية:

1. **بيئة تطوير Java** – تأكد من إعداد بيئة تطوير Java على جهازك.  
2. **Aspose.3D للـ Java** – قم بتحميل وتثبيت مكتبة Aspose.3D. يمكنك العثور على المكتبة ووثائقها [هنا](https://reference.aspose.com/3d/java/).  
3. **دليل المستندات** – أنشئ دليلًا لتخزين مستندات Java الخاصة بك. سنسميه “Your Document Directory.”

## استيراد الحزم

في بيئة تطوير Java الخاصة بك، استورد الحزم اللازمة لـ Aspose.3D. يضمن ذلك أن يكون لكودك وصول إلى الوظائف التي توفرها المكتبة.

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## الخطوة 1: إعداد الملف الأساسي

ابدأ بتهيئة الملف الأساسي الذي سيتم بُثه. في هذا المثال، سنستخدم شكلًا مستطيلاً بنصف قطر تقويس قدره 0.3.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

## الخطوة 2: إنشاء مشهد ثلاثي الأبعاد

ابنِ أساس عالمك الثلاثي الأبعاد بإنشاء مشهد.

```java
Scene scene = new Scene();
```

## الخطوة 3: إنشاء العقد اليسرى واليمنى

أنشئ العقد اليسرى واليمنى داخل المشهد. هذه العقد تُعدّ القماش لكائناتك الثلاثية الأبعاد.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## الخطوة 4: البثق الخطي مع خاصية المركز (العقدة اليسرى)

قم بعملية البثق الخطي على العقدة اليسرى **بدون تمركز** واضبط عدد الشرائح إلى 3. لاحظ استدعاء `setCenter(false)` – هنا نضبط خاصية المركز إلى *false*.

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

## الخطوة 5: إضافة سطح أرضي للمرجعية (العقدة اليسرى)

عزز التمثيل البصري **بإضافة سطح أرضي** إلى العقدة اليسرى. الصندوق الرقيق يعمل كأرضية لتتمكن من رؤية ارتفاع البثق بوضوح.

```java
left.createChildNode(new Box(0.01, 3, 3));
```

## الخطوة 6: البثق الخطي مع خاصية المركز (العقدة اليمنى)

الآن قم بعملية البثق الخطي على العقدة اليمنى، هذه المرة **مع تمركز البثق**. استدعاء `setCenter(true)` يوضح كيفية ضبط خاصية المركز إلى *true*.

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

## الخطوة 7: إضافة سطح أرضي للمرجعية (العقدة اليمنى)

كما هو الحال في الجانب الأيسر، أضف سطحًا أرضيًا إلى العقدة اليمنى للحصول على قاعدة بصرية متسقة.

```java
right.createChildNode(new Box(0.01, 3, 3));
```

## الخطوة 8: حفظ المشهد ثلاثي الأبعاد

احفظ مشهدك الثلاثي الأبعاد بصيغة Wavefront OBJ حتى تتمكن من عرضه في أي عارض ثلاثي الأبعاد قياسي.

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## المشكلات الشائعة والحلول

| المشكلة | لماذا يحدث | الحل |
|---------|------------|------|
| سطح الأرض غير مرئي | سمك الصندوق صغير جدًا بالنسبة لمقياس المشاهد. | زيادة السمك (المعامل الأول لـ `Box`) أو تصغير التكبير في المشاهد. |
| البثق يظهر مائلًا | قيمة `setCenter` لم تُضبط كما هو مقصود. | تحقق مرة أخرى من القيمة البوليانية الممررة إلى `setCenter`. |
| الملف لم يُحفظ | مسار الدليل غير صحيح أو عدم وجود أذونات كتابة. | تأكد من أن `MyDir` يشير إلى مجلد موجود مع صلاحية كتابة. |
