# إنشاء مشهد ثلاثي الأبعاد – ضبط اتجاه البثق Aspose.3D Java

## المقدمة

في هذا البرنامج التعليمي ستتعلم **كيفية إنشاء مشهد ثلاثي الأبعاد** والتحكم في اتجاه البثق باستخدام Aspose.3D للغة Java. سواءً كنت تبني تصورات معمارية، نماذج أولية للمنتجات، أو أصول ألعاب، فإن إتقان البثق الخطي يمنحك المرونة لنمذجة أشكال معقدة بسرعة. سنمرّ بكل خطوة، من إضافة العقد في Java إلى **تصدير نموذج ثلاثي الأبعاد بصيغة obj**، حتى تتمكن من رؤية النتيجة فورًا.

## إجابات سريعة
- **ماذا يعني “إنشاء مشهد ثلاثي الأبعاد”؟** يعني تهيئة كائن Aspose.3D `Scene` سيحتوي على جميع الهندسات، الأضواء، والكاميرات.  
- **كيف أقوم بضبط اتجاه البثق؟** استخدم الطريقة `setDirection(Vector3)` على كائن `LinearExtrusion`.  
- **أي صيغة يجب أن أستخدمها للتصدير؟** صيغة OBJ (`FileFormat.WAVEFRONTOBJ`) مدعومة على نطاق واسع في سير عمل ثلاثي الأبعاد.  
- **هل أحتاج إلى ترخيص لـ Aspose.3D؟** النسخة التجريبية المجانية تكفي للتطوير؛ الترخيص التجاري مطلوب للإنتاج.  
- **هل يمكنني إضافة المزيد من العقد في Java؟** نعم—استخدم `scene.getRootNode().createChildNode()` لإضافة عدد غير محدود من الكائنات.

## ما هو سير عمل “إنشاء مشهد ثلاثي الأبعاد”؟

سير عمل **إنشاء مشهد ثلاثي الأبعاد** يبدأ بكائن `Scene` فارغ، يضيف الهندسة (مثل الملفات المستخرجة)، يضعها باستخدام التحويلات، وأخيرًا يحفظ المشهد إلى صيغة ملف مثل OBJ. هذا النمط هو العمود الفقري لمعظم تطبيقات ثلاثية الأبعاد المبنية باستخدام Aspose.3D.

## لماذا ضبط اتجاه البثق؟

ضبط اتجاه البثق يتيح لك إمالة أو تدوير أو تشويه الشكل أثناء عملية البثق، مما يمنحك تحكمًا في الهندسة النهائية دون الحاجة إلى معالجة لاحقة. يكون ذلك مفيدًا خصوصًا لـ:

- إنشاء أعمدة مدببة أو أنابيب ذات أشكال مخصصة.  
- محاذاة البثق مع محاور غير قياسية في الأجزاء الميكانيكية.  
- توليد أشكال فنية للتأثيرات البصرية.

## المتطلبات المسبقة

قبل أن نبدأ، تأكد من وجود ما يلي:

- معرفة أساسية بلغة Java.  
- مكتبة Aspose.3D مثبتة – حمّلها من [هنا](https://releases.aspose.com/3d/java/).  
- بيئة تطوير متكاملة مثل Eclipse أو IntelliJ IDEA.

## استيراد الحزم

أولًا، استورد الفئات المطلوبة من Aspose.3D:

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## الخطوة 1: تهيئة الملف الأساسي

أنشئ الملف الثنائي الأبعاد الذي سيُبثق. في هذا المثال نستخدم مستطيلًا مدور الزوايا:

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

> **نصيحة احترافية:** عدّل نصف قطر التقويس للتحكم في حدة أو نعومة زوايا المستطيل قبل البثق.

## الخطوة 2: إنشاء مشهد

الآن **ننشئ مشهدًا ثلاثيًا الأبعاد** سيستضيف كائناتنا:

```java
Scene scene = new Scene();
```

## الخطوة 3: إضافة عقد Java – تموضع الكائنات

سنضيف عقدتين فرعيتين (يسار ويمين) إلى عقدة الجذر في المشهد ونحرك العقدة اليسرى قليلًا إلى الجانب:

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## الخطوة 4: كيفية البثق – العقدة اليسرى (الاتجاه الافتراضي)

بثق الملف على العقدة اليسرى باستخدام اتجاه المحور Z الافتراضي. كما نضيف دورانًا كاملاً 360° ونزيد عدد الشرائح للحصول على سلاسة أكبر:

```java
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); }});
```

## الخطوة 5: كيفية ضبط الاتجاه – العقدة اليمنى

هنا **نوضح كيفية ضبط الاتجاه** بتوفير `Vector3` مخصص. هذا يميل البثق نحو المتجه (0.3, 0.2, 1):

```java
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); setDirection(new Vector3(0.3, 0.2, 1));}});
```

## الخطوة 6: حفظ ملف OBJ – تصدير النموذج ثلاثي الأبعاد

أخيرًا، **نحفظ ملف obj** لرؤية النتيجة في أي عارض ثلاثي الأبعاد:

```java
scene.save(MyDir + "DirectionInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

عند فتح ملف OBJ المُولد، ستلاحظ وجود مستطيلين مبثوقين: أحدهما بالاتجاه الافتراضي والآخر مائل وفق المتجه الذي حددناه.

## المشكلات الشائعة والحلول

| المشكلة | السبب | الحل |
|--------|-------|------|
| ملف OBJ يظهر فارغًا | لم يتم حفظ المشهد أو المسار غير صحيح | تحقق من أن `MyDir` يشير إلى مجلد قابل للكتابة وأن اسم الملف ينتهي بـ `.obj`. |
| البثق يبدو مسطحًا | `setSlices` منخفض جدًا | زد قيمة `setSlices` (مثلاً إلى 200) للحصول على هندسة أكثر سلاسة. |
| الاتجاه لا يتغير | المتجه غير مُعَدل (غير مُطبع) | استخدم `Vector3` مُعَدل أو عدّل القيم لتعكس الميل المطلوب. |
