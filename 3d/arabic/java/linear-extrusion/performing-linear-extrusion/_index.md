# إنشاء استخراج ثلاثي الأبعاد Java باستخدام Aspose.3D

## المقدمة

إذا كنت بحاجة إلى **إنشاء استخراج ثلاثي الأبعاد Java** بسرعة وموثوقية، فقد وصلت إلى الدليل المناسب. خلال الدقائق القليلة القادمة سنستعرض كيفية توليد استخراج خطي، تخصيص هندسته، و**تصدير ملف OBJ Java** باستخدام مكتبة Aspose.3D. سواء كنت تبني أداة شبيهة بـ CAD، أو خط أنابيب أصول ألعاب، أو أي سير عمل ثلاثي الأبعاد قائم على Java، فإن هذا الدليل يزودك بأساس صلب وجاهز للإنتاج.

## إجابات سريعة
- **ماذا يعني “استخراج خطي”؟** يقوم بتمرير ملف تعريف ثنائي الأبعاد على طول خط مستقيم لتكوين جسم ثلاثي الأبعاد.  
- **أي مكتبة تتعامل مع الاستخراج؟** Aspose.3D for Java توفر واجهة برمجة تطبيقات عالية المستوى.  
- **هل يمكنني تصدير النتيجة كملف OBJ؟** نعم – ينتهي الدليل بـ `scene.save(..., FileFormat.WAVEFRONTOBJ)`.  
- **هل أحتاج إلى ترخيص للتطوير؟** نسخة تجريبية مجانية تكفي للتعلم؛ الترخيص التجاري مطلوب للإنتاج.  
- **ما نسخة Java المدعومة؟** Java 1.6 وما بعدها.

## ما هو إنشاء استخراج ثلاثي الأبعاد Java؟
إنشاء استخراج ثلاثي الأبعاد في Java يعني أخذ شكل ثنائي الأبعاد بسيط (مثل مستطيل) وتمديده إلى البعد الثالث. يمكن حفظ الشبكة الناتجة بصيغ شائعة مثل OBJ، التي تدعمها معظم برامج التحرير ثلاثية الأبعاد.

## لماذا نستخدم Aspose.3D للاستخراج الخطي؟
- **واجهة برمجة تطبيقات Java صافية** – لا توجد تبعيات أصلية، مثالية للمشاريع متعددة المنصات.  
- **تحكم غني في الهندسة** – التقويس، الالتواء، الشرائح، والإزاحة كلها متاحة عبر خصائص بسيطة.  
- **تصدير مباشر** – احفظ إلى OBJ، STL، FBX، وأكثر دون الحاجة إلى محولات إضافية.  
- **دعم على مستوى المؤسسات** – الترخيص، الوثائق، ومنتديات المجتمع متاحة بسهولة.

## المتطلبات المسبقة

قبل أن تبدأ، تأكد من وجود ما يلي:

1. **بيئة تطوير Java** – JDK 1.6+ مثبت ومُعد.  
2. **مكتبة Aspose.3D** – حمّل أحدث ملف JAR من الموقع الرسمي [هنا](https://releases.aspose.com/3d/java/).  

## استيراد الحزم

أضف مساحة الاسم الأساسية لـ Aspose.3D إلى ملف المصدر Java الخاص بك:

```java
import com.aspose.threed.*;
```

## الخطوة 1: تحديد دليل المستند

عرّف المكان الذي ستُكتب فيه الملفات المُولدة:

```java
String MyDir = "Your Document Directory";
```

> **نصيحة محترف:** استخدم مسارًا مطلقًا أو خاصية قابلة للتكوين بحيث يعمل موقع الإخراج عبر بيئات مختلفة.

## الخطوة 2: تهيئة الشكل الأساسي

أنشئ مستطيلًا سيعمل كملف تعريف للاستخراج. نصف قطر التقويس ينعّم الزوايا:

```java
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

يمكنك تجربة `setRoundingRadius` للحصول على ملف تعريف أكثر دائرية أو حادة.

## الخطوة 3: تنفيذ الاستخراج الخطي

الآن نحول المستطيل الثنائي الأبعاد إلى كائن ثلاثي الأبعاد. يأخذ المُنشئ ملف التعريف وعمق الاستخراج (10 وحدات في هذه الحالة). الخصائص الإضافية تضبط الشبكة بدقة:

```java
LinearExtrusion extrusion = new LinearExtrusion(profile, 10) {{ setSlices(100); setCenter(true); setTwist(360); setTwistOffset(new Vector3(10, 0, 0));}};
```

- **Slices** – يتحكم في سلاسة الاستخراج.  
- **Center** – يوازن الهندسة حول الأصل.  
- **Twist** – يدور ملف التعريف على طول محور الاستخراج (هنا دورة كاملة 360°).  
- **TwistOffset** – يحرك محور الالتواء، موضحًا كيفية إنشاء الحلزونات.

## الخطوة 4: إنشاء مشهد ثلاثي الأبعاد

`Scene` هو الحاوية لجميع الكائنات ثلاثية الأبعاد. إضافة الاستخراج كعقدة فرعية يجعلها جزءًا من رسم المشهد:

```java
Scene scene = new Scene();
scene.getRootNode().createChildNode(extrusion);
```

## الخطوة 5: حفظ المشهد ثلاثي الأبعاد

أخيرًا، صدّر المشهد إلى ملف Wavefront OBJ – صيغة مدعومة على نطاق واسع من قبل محررات ثلاثية الأبعاد، محركات الألعاب، وخطوط الطباعة:

```java
scene.save(MyDir +  "LinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

بعد تشغيل الكود، ستجد **LinearExtrusion.obj** في الدليل الذي حددته، جاهزًا للفتح في Blender، Maya، أو أي عارض OBJ آخر.

## المشكلات الشائعة والحلول

| العَرَض | السبب المحتمل | الحل |
|---------|--------------|-----|
| `FileNotFoundException` عند الحفظ | `MyDir` غير موجود أو لا يملك صلاحية كتابة | أنشئ المجلد مسبقًا أو استخدم مسارًا مطلقًا مع الصلاحيات المناسبة. |
| ملف OBJ يظهر فارغًا في العارض | لم يتم إضافة أي هندسة إلى المشهد | تحقق من أن كائن `LinearExtrusion` تم إنشاؤه وربطه بعقدة الجذر بشكل صحيح. |
| الالتواء غير صحيح | قيم `TwistOffset` غير صحيحة | عدّل إحداثيات `Vector3`؛ تذكر أن الإزاحة تُطبّق قبل دوران الالتواء. |
