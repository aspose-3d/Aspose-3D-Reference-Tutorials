# كيفية إضافة المتجهات العمودية (Normals) إلى المجسمات ثلاثية الأبعاد في جافا (باستخدام Aspose.3D)

## المقدمة  

إضافة المتجهات العمودية الصحيحة إلى المجسم أمر أساسي للحصول على إضاءة وظلال وحسابات فيزياء واقعية. في هذا **دليل كيفية إضافة المتجهات العمودية** سنستعرض الخطوات الدقيقة المطلوبة لتوليد بيانات المتجهات العمودية لمجسم ثلاثي الأبعاد باستخدام مكتبة **Aspose.3D for Java**. في نهاية الدليل ستكون قادرًا على **إنشاء بيانات المتجهات العمودية**، **حساب المتجهات العمودية للمجسم**، وتصدير نموذج نظيف جاهز للتصوير.

## إجابات سريعة
- **ماذا يحقق “إضافة المتجهات العمودية”؟** يتيح إضاءة وظلال صحيحة على أسطح المجسمات ثلاثية الأبعاد.  
- **ما المكتبة المستخدمة؟** Aspose.3D for Java.  
- **هل أحتاج إلى ترخيص؟** النسخة التجريبية المجانية تكفي للتطوير؛ يلزم ترخيص تجاري للإنتاج.  
- **كم يستغرق تنفيذ العملية؟** حوالي 10‑15 دقيقة لمجسم أساسي.  
- **هل يمكن استخدامه مع صيغ أخرى؟** نعم – يدعم Aspose.3D العديد من صيغ الملفات ثلاثية الأبعاد (OBJ، FBX، STL، إلخ).

## ما معنى “إضافة المتجهات العمودية” إلى المجسم؟  
المتجهات العمودية هي متجهات عمودية على مضلعات السطح. تُخبر محرك العرض كيف يتفاعل الضوء مع كل وجه. عندما يفتقد الملف هذه المعلومات (كما هو شائع في ملفات 3DS القديمة)، يجب عليك **توليد المتجهات العمودية للمجسم** قبل أن يظهر النموذج بشكل صحيح في المشهد.

## لماذا نستخدم Aspose.3D لهذه المهمة؟  
يوفر Aspose.3D واجهة برمجة تطبيقات عالية المستوى تُجردك من حسابات الرياضيات منخفضة المستوى اللازمة لحساب المتجهات العمودية. كما يدعم مجموعات التنعيم (smoothing groups)، والمتجهات المماسية (tangents)، والمتجهات الثنائية (binormals)، ومجموعة واسعة من صيغ الملفات، مما يجعله خيارًا موثوقًا لدورة **aspose 3d tutorial** احترافية.

## المتطلبات المسبقة

## كيفية إضافة المتجهات العمودية إلى مجسماتك ثلاثية الأبعاد  

فيما يلي الدليل الكامل خطوة بخطوة. كل كتلة شفرة تبقى كما هي من الدليل الأصلي؛ النص المحيط يضيف السياق والشرح.

### استيراد الحزم  

أولاً، استورد فئات Aspose.3D وأدوات إدخال/إخراج جافا التي ستحتاجها.

```java
import com.aspose.threed.*;


import java.io.IOException;
```

*شرح:* `com.aspose.threed.*` يمنحك الوصول إلى `Scene`، `NodeVisitor`، `Mesh`، وأداة `PolygonModifier` التي ستنشئ بيانات المتجهات العمودية لنا.

### الخطوة 1: تحميل مستند 3D  

أنشئ كائن `Scene` يشير إلى ملف 3DS الخاص بك. الملف لا يحتوي على بيانات المتجهات العمودية، لكنه يحتوي على مجموعات تنعيم يمكن للمكتبة استخدامها لتوليدها.

```java
// ExStart:GenerateDataForMeshes
// The path to the documents directory.
String MyDir = "Your Document Directory";

// Load a 3ds file, 3ds file doesn't have normal data, but it has smoothing group
Scene s = new Scene(MyDir + "camera.3ds");
```

*لماذا هذا مهم:* تحميل المشهد هو الخطوة الأولى في أي خط أنابيب لمعالجة المجسمات. بمجرد أن يصبح المشهد في الذاكرة، يمكننا استعراض شجرة العقد وتطبيق التحويلات أو الحسابات مثل **generate mesh normals**.

### الخطوة 2: زيارة العقد وإنشاء بيانات المتجهات العمودية  

نمر عبر كل عقدة في رسم المشهد. لكل عقدة تحتوي على `Mesh`، نستدعي `PolygonModifier.generateNormal(mesh)` الذي يحسب ويعيد كائن `VertexElementNormal`. إضافة هذا العنصر إلى المجسم يخزن المتجهات العمودية التي تم إنشاؤها حديثًا.

```java
s.getRootNode().accept(new NodeVisitor() {
    @Override
    public boolean call(Node node) {
        Mesh mesh = (Mesh) node.getEntity();
        if (mesh != null) {
            VertexElementNormal normals = PolygonModifier.generateNormal(mesh);
            mesh.addElement(normals);
        }
        return true;
    }
});
```

*نصيحة:* طريقة `generateNormal` تحترم مجموعات التنعيم الموجودة، لذا ستظهر المتجهات العمودية ناعمة حيث يُقصد ذلك وحادة حيث تُحدّد الحواف.

### الخطوة 3: تأكيد النجاح  

بعد انتهاء الزائر، اطبع رسالة قصيرة إلى وحدة التحكم. هذا يؤكد أن بيانات المتجهات العمودية تم توليدها **لكل المجسمات** في المشهد.

```java
// ExEnd:GenerateDataForMeshes
System.out.println("\nNormal data generated successfully for all meshes.");
```

*ما المتوقع:* عند فتح المشهد الناتج في أي عارض ثلاثي الأبعاد (مثل Aspose.3D Viewer، Blender، أو Unity)، سيظهر النموذج الآن إضاءة صحيحة لأن المتجهات العمودية موجودة.

## المشكلات الشائعة & استكشاف الأخطاء وإصلاحها  

| العَرَض | السبب المحتمل | الحل |
|---------|--------------|-----|
| لا يوجد إخراج أو وحدة تحكم فارغة | مسار `MyDir` غير صحيح | تحقق من أن مسار الدليل ينتهي بشرطة مائلة وأن الملف موجود. |
| يظهر المجسم مسطحًا أو ساطعًا جدًا | لم تُضاف المتجهات العمودية | تأكد من تنفيذ `mesh.addElement(normals);` لكل مجسم. |
