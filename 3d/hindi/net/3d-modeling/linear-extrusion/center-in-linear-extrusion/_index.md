# लीनियर एक्सट्रूज़न में ग्राउंड प्लेन और सेंटर जोड़ें

## परिचय

Aspose.3D for .NET का उपयोग करके लीनियर एक्सट्रूज़न में महारत हासिल करने के इस व्यापक गाइड में आपका स्वागत है। यदि आप अपने मॉडलों में **ग्राउंड प्लेन जोड़ना** चाहते हैं और **Wavefront OBJ** फ़ाइलें निर्यात करना चाहते हैं जबकि एक्सट्रूज़न को केंद्रित रखें, तो आप सही जगह पर हैं। इस ट्यूटोरियल में हम लीनियर एक्सट्रूज़न तकनीक में गहराई से जाएंगे, विशेष रूप से सेंटरिंग पहलू पर ध्यान केंद्रित करेंगे और यह देखेंगे कि ग्राउंड प्लेन परिणाम को अधिक स्पष्ट रूप से कैसे दर्शाता है।

## त्वरित उत्तर
- **“ग्राउंड प्लेन जोड़ने” से क्या प्राप्त होता है?** यह एक दृश्य संदर्भ प्रदान करता है जिससे यह देखना आसान हो जाता है कि एक्सट्रूज़न X‑Z प्लेन पर कहाँ स्थित है।  
- **मॉडल को निर्यात करने के लिए कौन सा फ़ॉर्मेट उपयोग किया जाता है?** सीन को Wavefront OBJ फ़ाइल (`FileFormat.WavefrontOBJ`) के रूप में सहेजा जाता है।  
- **क्या कोड चलाने के लिए लाइसेंस आवश्यक है?** विकास के लिए एक फ्री ट्रायल काम करता है; उत्पादन के लिए स्थायी लाइसेंस आवश्यक है।  
- **क्या मैं एक्सट्रूज़न की लंबाई बदल सकता हूँ?** हाँ – `LinearExtrusion` के दूसरे पैरामीटर को संशोधित करें।  
- **क्या सेंटरिंग वैकल्पिक है?** `Center = true` सेट करने से प्रोफ़ाइल के चारों ओर एक्सट्रूज़न केंद्रित हो जाता है; `false` सेट करने से यह एक पक्ष पर संरेखित रहता है।

इस रोमांचक यात्रा पर निकलने से पहले सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ मौजूद हों:

- C# प्रोग्रामिंग भाषा की बुनियादी समझ।  
- आपके मशीन पर Visual Studio स्थापित हो।  
- Aspose.3D for .NET लाइब्रेरी, जिसे आप [Aspose.3D .NET Documentation](https://reference.aspose.com/3d/net/) से डाउनलोड कर सकते हैं।  
- ट्यूटोरियल के दौरान संदर्भ के लिए आपके पास [Aspose.3D .NET Documentation](https://reference.aspose.com/3d/net/) तक पहुंच हो।

## नामस्थान आयात करें

शुरुआत करने के लिए, आवश्यक नामस्थान आयात करें। ये हमारे 3D मॉडलिंग मास्टरपीस की नींव रखेंगे।

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

## चरण 1: बेस प्रोफ़ाइल को इनिशियलाइज़ करें

हम एक आयताकार प्रोफ़ाइल परिभाषित करते हैं जिसे एक्सट्रूड किया जाएगा। `RoundingRadius` कोनों में एक सूक्ष्म फ़िलेट जोड़ता है।

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

## चरण 2: 3D सीन बनाएं

एक `Scene` ऑब्जेक्ट सभी ज्योमेट्री, लाइट्स और कैमरों के लिए कंटेनर के रूप में कार्य करता है।

```csharp
Scene scene = new Scene();
```

## चरण 3: बाएँ और दाएँ नोड बनाएं

रूट के तहत दो सिब्लिंग नोड बनाए जाते हैं। एक **बिना** सेंटरिंग के एक्सट्रूज़न दिखाएगा, दूसरा **सेंटरिंग के साथ**। हम बाएँ नोड को इस प्रकार ट्रांसलेट करते हैं कि दोनों उदाहरण ओवरलैप न हों।

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(5, 0, 0);
```

## चरण 4: बाएँ नोड पर लीनियर एक्सट्रूज़न करें (कोई सेंटर नहीं)

यहाँ हम प्रोफ़ाइल को बिना सेंटरिंग के एक्सट्रूड करते हैं। `Center = false` फ़्लैग पर ध्यान दें।

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 2) { Center = false, Slices = 3 });
```

## चरण 5: बाएँ नोड के लिए रेफ़रेंस के रूप में ग्राउंड प्लेन जोड़ें

एक पतली बॉक्स को **ग्राउंड प्लेन** के रूप में जोड़ने से आप स्पष्ट रूप से देख सकते हैं कि एक्सट्रूज़न बेस पर कैसे स्थित है।

```csharp
left.CreateChildNode(new Box(0.01, 3, 3));
```

## चरण 6: दाएँ नोड पर लीनियर एक्सट्रूज़न करें (सेंटर के साथ)

अब हम वही प्रोफ़ाइल एक्सट्रूड करते हैं लेकिन सेंटरिंग सक्षम करते हैं। ज्योमेट्री प्रोफ़ाइल के मूल बिंदु के चारों ओर सममित रूप से स्थित होगी।

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 2) { Center = true, Slices = 3 });
```

## चरण 7: दाएँ नोड के लिए रेफ़रेंस के रूप में ग्राउंड प्लेन जोड़ें

दाएँ नोड में एक दूसरा ग्राउंड प्लेन जोड़ा जाता है ताकि दृश्य तुलना सुसंगत रहे।

```csharp
right.CreateChildNode(new Box(0.01, 3, 3));
```

## चरण 8: Wavefront OBJ फ़ाइल निर्यात करें

अंत में, हम **Wavefront OBJ** निर्यात करते हैं ताकि परिणाम को किसी भी मानक 3‑D व्यूअर में खोला जा सके।

```csharp
scene.Save("Your Output Directory" + "CenterInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```

## ग्राउंड प्लेन क्यों जोड़ें?

ग्राउंड प्लेन जोड़ने से आपको एक्सट्रूज़न की ऊँचाई और संरेखण के बारे में तुरंत दृश्य संकेत मिलता है। यह विशेष रूप से तब उपयोगी होता है जब आपको सेंटरड बनाम नॉन‑सेंटरड परिणामों की तुलना करनी हो, जैसा कि इस ट्यूटोरियल में दर्शाया गया है।

## सामान्य समस्याएँ और सुझाव

- **ग्राउंड प्लेन दिखाई नहीं दे रहा:** सुनिश्चित करें कि प्लेन की मोटाई (`Box` कंस्ट्रक्टर में `0.01`) इतनी छोटी हो कि एक्सट्रूज़न को ढक न सके, लेकिन रेंडर होने के लिए पर्याप्त हो।  
- **निर्यात विफल हो रहा है:** आउटपुट डायरेक्टरी मौजूद है और आपके पास लिखने की अनुमति है, यह जाँचें।  
- **सेंटरड एक्सट्रूज़न ऑफ़सेट दिख रहा है:** `Center` प्रॉपर्टी को दोबारा जांचें; `true` प्रोफ़ाइल को केंद्रित करता है, `false` एक पक्ष पर संरेखित करता है।
