# Aspose.3D for .NET का इस्तेमाल करके खोले के साथ लाइन एक्सट्रूज़न कैसे करें

## परिचय

Aspose.3D for .NET का इस्तेमाल करके 3D डिज़ाइन की दुनिया में आपका स्वागत है! इस ट्यूटोरियल में आप **रैखिक एक्सट्रूज़न** को खोले के साथ खोजेंगे, एक टेक्नीक जो आपको अपने बताए गए लेवल पर कंट्रोल करने देती है। चाहे आप एक अनुभवी डेवलपर हों या अभी शुरुआत कर रहे हों, हम आपको हर कदम पर ले चलेंगे, हर एक्शन के पीछे का कारण समझाएंगे, और आपको प्रैक्टिकल टिप्स देंगे जिन्हें आप तुरंत लागू कर सकते हैं।

## क्विक आंसर्स
- **रैखिक एक्सट्रूज़न खोले के साथ क्या है?** यह 2D प्रोफ़ाइल को 3D में जोड़ने की एक विधि है, जिसमें यह लाइन किया जाता है कि कितनी इंटरमीडिएट क्रॉस-सेक्शन (स्लाइस) उत्पन्न होती है।
- **स्लाइस का इस्तेमाल क्यों करें?** ज़्यादा खोले स्मूथ कर्वेचर देते हैं; कम खोले मेष को हल्का रखते हैं।
- **पूर्वापेक्षाएँ?** Aspose.3D for .NET, एक .NET‑संगत IDE, और आधारभूत C# ज्ञान।
- **सामान्य रनटाइम?** यह मॉडल आधुनिक PC पर एक सेकंड से कम समय में चलता है।
- **आउटपुट फ़ॉर्मेट?** उदाहरण Wavefront OBJ में सहेजता है, लेकिन Aspose.3D कई फ़ॉर्मेट्स को सपोर्ट करता है।

## स्लाइस के साथ लीनियर एक्सट्रूज़न क्या है?

लीनियर एक्सट्रूज़न एक 2‑D आकार (प्रोफ़ाइल) लेता है और उसे एक सीधी रेखा के साथ खींचकर 3‑D सॉलिड बनाता है। **स्लाइस** प्रॉपर्टी निर्धारित करती है कि एक्सट्रूज़न की शुरुआत और अंत के बीच कितनी मध्यवर्ती क्रॉस-सेक्शन डाली जाती है, जिससे रैखिकता और पॉलिगॉन की संख्या प्रभावित होती है।

## लीनियर एक्सट्रूज़न में स्लाइस का उपयोग क्यों करें?

- **मेश घनत्व नियंत्रित करें:** दृश्य गुणवत्ता और फ़ाइल आकार के बीच संतुलन को बनावट से समायोजित करें।
- **प्रदर्शन कॉन्फ़िगर करें:** रीयल-टाइम एप्लिकेशन के लिए कम खुलने का उपयोग करें, हाई-रिज़ॉल्यूशन रेंडर के लिए अधिक।
- **रचनात्मक जुड़ाव:** अलग-अलग ऑब्जेक्ट्स पर अलग-अलग खुलने वाली गिनती को मिलाकर डिज़ाइन इरादे को उजागर करें।

## नामस्थान आयात करें

सबसे पहले, उन नेमस्पेस को इम्पोर्ट करें जो आपको कोर Aspose.3D क्लासेज़ तक पहुंच प्रदान करते हैं।

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

## स्टेप-बाय-स्टेप गाइड

### स्टेप 1: बेस प्रोफ़ाइल को इनिशियलाइज़ करें

हम एक साधारण आयताकार आकार से शुरू करते हैं और इसे एक छोटा राउंडिंग रेडियस देते हैं ताकि किनारे पूरी तरह तेज़ न हों।

```csharp
// ExStart:InitializeBaseProfile
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
// ExEnd:InitializeBaseProfile
```

### स्टेप 2: एक 3D सीन बनाएँ

`Scene` सभी नोड्स, मेष, लाइट्स, और कैमरों के कंटेनर के रूप में कार्य करता है।

```csharp
// ExStart:Create3DScene
Scene scene = new Scene();
// ExEnd:Create3DScene
```

### स्टेप 3: लेफ्ट और राइट नोड जोड़ें

हम सीन की रूट के तहत दो सिब्लिंग नोड्स बनाएँगे। बायाँ नोड कम स्लाइस काउंट प्राप्त करेगा, दायाँ नोड अधिक स्लाइस काउंट, ताकि आप दृश्य अंतर की तुलना कर सकें।

```csharp
// ExStart:CreateLeftRightNodes
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(15, 0, 0);
// ExEnd:CreateLeftRightNodes
```

### स्टेप 4: लेफ्ट नोड पर लीनियर एक्सट्रूज़न करें (लो डिटेल)

यहाँ हम आयत को केवल **2 स्लाइस** के साथ एक्सट्रूड करते हैं। इससे एक मोटा मेष बनता है—तेज़ प्रीव्यू के लिए उत्तम।

```csharp
// ExStart:LinearExtrusionLeftNode
left.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 2 });
// ExEnd:LinearExtrusionLeftNode
```

### स्टेप 5: राइट नोड पर लीनियर एक्सट्रूज़न करें (हाई डिटेल)

अब हम **10 स्लाइस** का उपयोग करते हैं जिससे परिणाम बहुत स्मूथ हो जाता है। देखें कि ज्योमेट्री कितनी बारीक हो गई है।

```csharp
// ExStart:LinearExtrusionRightNode
right.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 10 });
// ExEnd:LinearExtrusionRightNode
```

### स्टेप 6: 3D सीन सेव करें

अंत में, सीन को एक OBJ फ़ाइल में लिखें। `"Your Output Directory"` को अपने मशीन पर एक वैध पाथ से बदलें।

```csharp
// ExStart:Save3DScene
scene.Save("Your Output Directory" + "SlicesInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
// ExEnd:Save3DScene
```

## आम मुद्दे और समाधान

| समस्या | कारण | समाधान |
|-------|----------------|-----|
| **फ़ाइल नहीं बनी** | आउटपुट पथ विश्लेषक है या लिखने की अनुमति नहीं है। | एक पूर्ण पथ उपयोग करें और सुनिश्चित करें कि फ़ोल्डर मौजूद है। |
| **ऑब्जेक्ट फ्लैट दिख रहा है** | `स्लाइस` को 1 या 0 पर सेट किया गया है। | इस्तेमाल करने योग्य एक्सट्रूज़न के लिए `स्लाइस` को कम से कम 2 सेट करें। |
| **अप्रत्याशित ज्योमेट्री** | आयतन के आकार के लिए राउंडिंग रेडियस बहुत बड़ा है। | `राउंडिंगरेडियस` को आयतन की सबसे छोटी साइड के आधे से छोटा मान सेट करें। |
