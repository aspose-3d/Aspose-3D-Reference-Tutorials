# सीन में कैमरा जोड़ें और 3D एनीमेशन के लिए टार्गेट सेट करें

## परिचय

यदि आप 3D एनीमेशन बना रहे हैं, तो आपको सबसे पहले एक अच्छी तरह स्थित कैमरा चाहिए। इस ट्यूटोरियल में आप सीखेंगे **सीन में कैमरा कैसे जोड़ें**, उसका टार्गेट परिभाषित करें, और अंत में Aspose.3D for .NET का उपयोग करके **सीन को FBX के रूप में निर्यात करें**। हम प्रत्येक चरण को विस्तार से बताएँगे, यह क्यों महत्वपूर्ण है समझाएँगे, और आपको व्यावहारिक टिप्स देंगे ताकि आप आत्मविश्वास के साथ प्रभावशाली 3D एनीमेशन बना सकें।

## त्वरित उत्तर
- **कैमरा जोड़ने का पहला कदम क्या है?** एक `Scene` ऑब्जेक्ट को इनिशियलाइज़ करें जो कैमरा नोड को होस्ट करेगा।  
- **इस गाइड में निर्यात के लिए कौन सा फ़ाइल फ़ॉर्मेट उपयोग किया जाता है?** FBX (`.fbx`).  
- **क्या नमूना कोड चलाने के लिए लाइसेंस चाहिए?** मूल्यांकन के लिए एक मुफ्त ट्रायल काम करता है; उत्पादन के लिए एक व्यावसायिक लाइसेंस आवश्यक है।  
- **कौन से .NET संस्करण समर्थित हैं?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **क्या निर्माण के बाद कैमरा की स्थिति बदल सकता हूँ?** हाँ – किसी भी समय `cameraNode.Transform.Translation` को संशोधित करें।

## **सीन में कैमरा जोड़ना** क्या है?
सीन में कैमरा जोड़ना मतलब एक `Camera` एंटिटी बनाना, उसे सीन ग्राफ में एक नोड से जोड़ना, और इसे इस तरह स्थित करना कि वह एक टार्गेट पॉइंट को “देखे”। यह सीन के रेंडर या निर्यात होने पर दर्शक का दृष्टिकोण निर्धारित करता है।

## कैमरा टार्गेट सेट करने और FBX के रूप में निर्यात करने का कारण क्या है?
- **दृष्टिकोण को नियंत्रित करें** – सटीक कैमरा प्लेसमेंट आपको अपनी एनीमेशन को ठीक उसी तरह फ्रेम करने देता है जैसा आप कल्पना करते हैं।  
- **इंटरऑपरेबिलिटी** – FBX गेम इंजन, Maya, Blender और अन्य 3D टूल्स द्वारा व्यापक रूप से समर्थित है, जिससे एसेट्स को साझा करना आसान हो जाता है।  
- **पुन: उपयोग योग्य एसेट्स** – एक बार कैमरा और उसका टार्गेट सहेज लेने पर, आप सीन को कई प्रोजेक्ट्स में पुन: उपयोग कर सकते हैं।

## पूर्वापेक्षाएँ

ट्यूटोरियल में डुबकी लगाने से पहले, सुनिश्चित करें कि आपके पास निम्नलिखित पूर्वापेक्षाएँ हैं:

- C# और .NET फ्रेमवर्क का बुनियादी ज्ञान।  
- Aspose.3D for .NET लाइब्रेरी स्थापित। आप इसे [यहाँ](https://releases.aspose.com/3d/net/) डाउनलोड कर सकते हैं।  
- 3D प्रोग्रामिंग के लिए तैयार विकास वातावरण।

## नेमस्पेस आयात करें

प्रक्रिया शुरू करने के लिए, आवश्यक नेमस्पेस को अपने प्रोजेक्ट में आयात करें। ये नेमस्पेस Aspose.3D for .NET की शक्ति का उपयोग करने के लिए आवश्यक हैं:

```csharp
using System;
using System.IO;
using System.Collections;
using Aspose.ThreeD;
using Aspose.ThreeD.Animation;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Utilities;
```

## चरण‑दर‑चरण गाइड

### चरण 1: सीन ऑब्जेक्ट इनिशियलाइज़ करें

सबसे पहले सीन ऑब्जेक्ट को इनिशियलाइज़ करें। यह वह कैनवास है जहाँ आपका 3D एनीमेशन जीवंत होगा।

```csharp
// ExStart:SetupTargetAndCamera
// Initialize scene object
Scene scene = new Scene();
```

### चरण 2: कैमरा नोड बनाएं

अगला, एक चाइल्ड नोड बनाएं जो कैमरा को रखेगा। नोड को एक सार्थक नाम देना सीन हायरार्की को व्यवस्थित रखने में मदद करता है।

```csharp
// Get a child node object
Node cameraNode = scene.RootNode.CreateChildNode("camera", new Camera());
```

### चरण 3: कैमरा की स्थिति निर्धारित करें

कैमरा नोड के लिए ट्रांसलेशन निर्दिष्ट करें। यह 3D स्पेस में कैमरा की प्रारंभिक स्थिति निर्धारित करता है।

```csharp
// Set camera node translation
cameraNode.Transform.Translation = new Vector3(100, 20, 0);
```

### चरण 4: कैमरा टार्गेट परिभाषित करें

कैमरा को देखने के लिए एक टार्गेट पॉइंट चाहिए। हम एक और चाइल्ड नोड बनाते हैं जो फोकल पॉइंट के रूप में कार्य करता है और इसे कैमरा की `Target` प्रॉपर्टी को असाइन करते हैं।

```csharp
cameraNode.GetEntity<Camera>().Target = scene.RootNode.CreateChildNode("target");
```

### चरण 5: कॉन्फ़िगर किए गए सीन को सहेजें

अंत में, सीन को एक FBX फ़ाइल (या किसी अन्य समर्थित फ़ॉर्मेट) में निर्यात करें। `"Your Output Directory"` को उस वास्तविक पथ से बदलें जहाँ आप फ़ाइल सहेजना चाहते हैं।

```csharp
var output = "Your Output Directory" + "camera-test.fbx";
scene.Save(output);
```

## सामान्य समस्याएँ और समाधान

| समस्या | समाधान |
|-------|----------|
| **कैमरा गलत कोण पर दिखाई देता है** | जाँचें कि टार्गेट नोड वह जगह पर स्थित है जहाँ आप अपेक्षा करते हैं। आप अभिविन्यास को नियंत्रित करने के लिए `cameraNode.GetEntity<Camera>().UpVector` भी सेट कर सकते हैं। |
| **निर्यात किया गया FBX अन्य टूल्स में नहीं खुलता** | सुनिश्चित करें कि आप Aspose.3D का नवीनतम संस्करण उपयोग कर रहे हैं (लाइब्रेरी स्वचालित रूप से संगत FBX संस्करण लिखती है)। |
| **`scene.Save` पर पाथ नहीं मिला त्रुटि** | एक पूर्ण पाथ उपयोग करें या `Save` कॉल करने से पहले सुनिश्चित करें कि आउटपुट डायरेक्टरी मौजूद है। |
