# बॉक्स पर PBR मैटेरियल लागू करने का तरीका

## परिचय

3D ग्राफिक्स की रोमांचक दुनिया में आपका स्वागत है! इस चरण‑दर‑चरण गाइड में, आप Aspose.3D for .NET का उपयोग करके बॉक्स पर **pbr लागू करने का तरीका** सीखेंगे। हम PBR मैटेरियल बनाने, उसे मेष में जोड़ने, और अंत में **STL ASCII निर्यात** करने की प्रक्रिया को समझेंगे ताकि आप मॉडल को किसी भी डाउनस्ट्रीम वर्कफ़्लो में उपयोग कर सकें। चाहे आप गेम प्रोटोटाइप बना रहे हों या प्रोडक्ट विज़ुअलाइज़ेशन, इस वर्कफ़्लो में निपुणता आपके .NET एप्लिकेशन्स में वास्तविक, फिज़िकली बेस्ड रेंडरिंग (PBR) का द्वार खोलती है।

## त्वरित उत्तर
- **What is the primary goal?** बॉक्स पर PBR मैटेरियल लागू करें और इसे STL ASCII के रूप में निर्यात करें।  
- **Which library is used?** Aspose.3D for .NET (how to use aspose).  
- **Do I need a license?** हाँ, उत्पादन के लिए एक अस्थायी या पूर्ण लाइसेंस आवश्यक है।  
- **Supported output format?** STL ASCII (export stl ascii) और कई अन्य 3D फ़ॉर्मेट।  
- **How long does it take?** बुनियादी कार्यान्वयन के लिए लगभग 10‑15 मिनट।

## PBR मैटेरियल क्या है?
Physically Based Rendering (PBR) एक शेडिंग मॉडल है जो प्रकाश के वास्तविक सतहों के साथ अंतःक्रिया को सिमुलेट करता है। धातुता (metallic) और रफ़नेस (roughness) जैसे गुणों को समायोजित करके, आप जटिल शेडर को हाथ से ट्यून किए बिना अत्यधिक वास्तविक परिणाम प्राप्त कर सकते हैं।

## Physically Based Rendering (PBR) क्यों उपयोग करें?
- **Realism:** सामग्री प्रकाश के प्रति ऐसी प्रतिक्रिया देती है जो वास्तविक भौतिकी से मेल खाती है।  
- **Consistency:** एक ही सामग्री किसी भी प्रकाश वातावरण में सही दिखती है।  
- **Efficiency:** आधुनिक GPU PBR गणनाओं के लिए अनुकूलित हैं, जिससे आपको मुफ्त में प्रदर्शन मिलता है।

## नेमस्पेस इम्पोर्ट करें

सबसे पहले, Aspose.3D for .NET की क्षमताओं का उपयोग करने के लिए आवश्यक नेमस्पेस इम्पोर्ट करना सुनिश्चित करें:

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Shading;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## चरण 1: सीन इनिशियलाइज़ करें

निम्नलिखित कोड स्निपेट का उपयोग करके 3D सीन को इनिशियलाइज़ करके शुरू करें:

```csharp
Scene scene = new Scene();
```

## चरण 2: PBR मैटेरियल बनाएं

अब हम **pbr मैटेरियल बनाते हैं** जो हमारे बॉक्स को वास्तविक लुक देगा:

```csharp
PbrMaterial mat = new PbrMaterial();
```

## चरण 3: मैटेरियल प्रॉपर्टीज़ सेट करें

मैटेरियल प्रॉपर्टीज़ को फाइन‑ट्यून करें, इसे लगभग धातु जैसा और बहुत रफ़ बनाते हुए—ब्रश्ड‑मेटल बॉक्स के लिए परफेक्ट:

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

## चरण 4: बॉक्स बनाएं

एक बॉक्स जनरेट करें जिस पर PBR मैटेरियल लागू किया जाएगा:

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

## चरण 5: बॉक्स में PBR मैटेरियल जोड़ें

पहले कॉन्फ़िगर किए गए **add pbr material** को बनाए गए बॉक्स नोड पर असाइन करें:

```csharp
boxNode.Material = mat;
```

## चरण 6: 3D सीन को STL ASCII के रूप में सहेजें

अंत में, मॉडल को किसी भी स्टैंडर्ड 3D व्यूअर या स्लाइसर में खोलने के लिए **export stl ascii** करें:

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```
