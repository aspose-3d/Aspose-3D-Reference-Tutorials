# बॉक्स पर PBR मैटेरियल कैसे लागू करें

## परिचय

3D ग्राफिक्स की आकर्षक दुनिया में आपका स्वागत है! इस चरण‑दर‑चरण ट्यूटोरियल में, **आप सीखेंगे कि कैसे pbr** मैटेरियल को एक बॉक्स पर Aspose.3D for .NET का उपयोग करके लागू किया जाए। हम एक PBR मैटेरियल बनाने, उसे मेष में जोड़ने, और अंत में **STL ASCII एक्सपोर्ट करना** समझेंगे ताकि आप मॉडल को किसी भी डाउनस्ट्रीम वर्कफ़्लो में उपयोग कर सकें। चाहे आप एक गेम प्रोटोटाइप, एक प्रोडक्ट विज़ुअलाइज़र, या 3D प्रिंटिंग के लिए तेज़ प्रोटोटाइप बना रहे हों, इस वर्कफ़्लो में महारत हासिल करने से आपके .NET एप्लिकेशन में वास्तविक, फिज़िकली बेस्ड रेंडरिंग (PBR) का द्वार खुलता है।

## त्वरित उत्तर
- **मुख्य लक्ष्य क्या है?** बॉक्स पर PBR मैटेरियल लागू करना और उसे STL ASCII के रूप में एक्सपोर्ट करना।  
- **कौन लाइब्रेरी उपयोग की जाती है?** Aspose.3D for .NET (how to use aspose)।  
- **क्या मुझे लाइसेंस चाहिए?** हाँ, उत्पादन के लिए एक अस्थायी या पूर्ण लाइसेंस आवश्यक है।  
- **समर्थित आउटपुट फ़ॉर्मेट?** STL ASCII (export stl ascii) और कई अन्य 3D फ़ॉर्मेट।  
- **यह कितना समय लेता है?** बुनियादी कार्यान्वयन के लिए लगभग 10‑15 मिनट।

## PBR मैटेरियल क्या है?
फ़िज़िकली बेस्ड रेंडरिंग (PBR) एक शेडिंग मॉडल है जो वास्तविक दुनिया की सतहों के साथ प्रकाश के इंटरैक्शन का अनुकरण करता है। धातुता (metallic) और रफ़नेस (roughness) जैसे गुणों को समायोजित करके, आप जटिल शेडर को हाथ से ट्यून किए बिना अत्यधिक वास्तविक परिणाम प्राप्त कर सकते हैं।

## फिज़िकली बेस्ड रेंडरिंग (PBR) क्यों उपयोग करें?
- **वास्तविकता:** सामग्री प्रकाश के प्रति ऐसे प्रतिक्रिया देती है जो वास्तविक भौतिकी से मेल खाती है।  
- **संगतता:** समान सामग्री किसी भी प्रकाश वातावरण में सही दिखती है।  
- **कुशलता:** आधुनिक GPU PBR गणनाओं के लिए अनुकूलित हैं, जिससे आपको मुफ्त में प्रदर्शन मिलता है।

## चरण‑दर‑चरण गाइड

### चरण 1: सीन इनिशियलाइज़ करें
एक खाली 3D सीन बनाकर शुरू करें। यह कंटेनर बाद में आप द्वारा जोड़े गए सभी ज्योमेट्री, मैटेरियल और लाइट्स को रखेगा।

```csharp
Scene scene = new Scene();
```

### चरण 2: PBR मैटेरियल बनाएं
अब हम **pbr मैटेरियल बनाते हैं** जो हमारे बॉक्स को वास्तविक दिखावट देगा। `PbrMaterial` क्लास फिज़िकली बेस्ड रेंडरिंग के लिए आवश्यक सभी पैरामीटर प्रदान करता है।

```csharp
PbrMaterial mat = new PbrMaterial();
```

### चरण 3: मैटेरियल प्रॉपर्टीज़ सेट करें
मैटेरियल प्रॉपर्टीज़ को फाइन‑ट्यून करें। इस उदाहरण में हम सतह को लगभग धातु जैसा और बहुत रफ़ बनाते हैं—ब्रश्ड‑मेटल बॉक्स के लिए उपयुक्त।

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

### चरण 4: बॉक्स मेष बनाएं
एक सरल बॉक्स ज्योमेट्री जनरेट करें। यह **create box mesh** चरण है जिसका मुख्य कीवर्ड संदर्भ देता है।

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

### चरण 5: बॉक्स पर PBR मैटेरियल लागू करें
पहले कॉन्फ़िगर किए गए **add pbr material** को उस बॉक्स नोड पर असाइन करें जिसे हमने अभी बनाया है।

```csharp
boxNode.Material = mat;
```

### चरण 6: STL ASCII एक्सपोर्ट करें (How to Export STL)
अंत में, **export stl ascii** करके मॉडल को किसी भी मानक 3D व्यूअर या स्लाइसर में खोला जा सकता है। `FileFormat.STLASCII` का उपयोग करने से एक मानव‑पठनीय फ़ाइल सुनिश्चित होती है।

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```

- **प्रदर्शन टिप:** यदि आपको कई मेष पर एक ही मैटेरियल लागू करना है तो एक ही `PbrMaterial` इंस्टेंस को पुन: उपयोग करें; इससे मेमोरी ओवरहेड कम होता है।
