# Aspose.3D for Java के साथ रैखिक एक्सट्रूज़न में सेंटर को नियंत्रित करना

## परिचय

जब आप Java में 3D सीन बना रहे होते हैं, तो **ground plane जोड़ने** की क्षमता और रैखिक एक्सट्रूज़न पर सटीक **center प्रॉपर्टी सेट करने** की सुविधा एक साधारण प्रोटोटाइप और एक पॉलिश्ड विज़ुअल के बीच अंतर बना सकती है। इस ट्यूटोरियल में हम Aspose.3D for Java का उपयोग करके एक्सट्रूज़न सेंटर को नियंत्रित करने और ground plane जोड़ने की पूरी प्रक्रिया को चरण-दर-चरण देखेंगे। आप समझेंगे कि यह क्यों महत्वपूर्ण है, इसे कैसे सेटअप करें, और एक तैयार‑चलाने‑योग्य कोड सैंपल प्राप्त करेंगे जिसे आप अपने प्रोजेक्ट्स में अनुकूलित कर सकते हैं।

## त्वरित उत्तर
- **“add ground plane” क्या करता है?** यह एक पतली रेफ़रेंस ज्योमेट्री बनाता है जो आपको एक्सट्रूज़न की स्थिति को विश्व अक्षों के सापेक्ष देखने में मदद करती है।  
- **रैखिक एक्सट्रूज़न का सेंटर कैसे सेट करें?** `LinearExtrusion` ऑब्जेक्ट पर `setCenter(boolean)` मेथड का उपयोग करें।  
- **क्या सैंपल चलाने के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक अस्थायी लाइसेंस काम करता है; प्रोडक्शन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **सहेजने के लिए कौन सा फ़ाइल फ़ॉर्मेट उपयोग किया जाता है?** उदाहरण Wavefront OBJ (`.obj`) में सहेजता है।  
- **कौन सा Java संस्करण आवश्यक है?** Java 8 या उसके बाद का संस्करण पर्याप्त है।

## 3D सीन में “add ground plane” क्या है?

ground plane जोड़ने का मतलब है X‑Z प्लेन पर स्थित एक पतली आयताकार ज्योमेट्री (अक्सर न्यूनतम मोटाई वाला बॉक्स) डालना। यह एक दृश्य फ़्लोर के रूप में कार्य करता है, जिससे अन्य वस्तुओं की ऊँचाई और संरेखण को आंकना आसान हो जाता है, विशेषकर जब आप एक्सट्रूज़न सेंटर के साथ प्रयोग कर रहे हों।

## रैखिक एक्सट्रूज़न पर सेंटर प्रॉपर्टी सेट क्यों करें?

डिफ़ॉल्ट रूप से, रैखिक एक्सट्रूज़न प्रोफ़ाइल के मूल बिंदु से शुरू होता है। सेंटर प्रॉपर्टी (`setCenter(true)`) सेट करने से प्रोफ़ाइल इस तरह शिफ्ट हो जाती है कि एक्सट्रूज़न मूल बिंदु के चारों ओर केंद्रित हो, जो सममित डिज़ाइनों या कई वस्तुओं में निरंतर संरेखण की आवश्यकता होने पर उपयोगी होता है।

## आवश्यकताएँ

1. **Java Development Environment** – सुनिश्चित करें कि आपके मशीन पर Java विकास पर्यावरण स्थापित है।  
2. **Aspose.3D for Java** – Aspose.3D लाइब्रेरी डाउनलोड और इंस्टॉल करें। आप लाइब्रेरी और उसकी डाक्यूमेंटेशन [here](https://reference.aspose.com/3d/java/) पर पा सकते हैं।  
3. **Document Directory** – अपने Java दस्तावेज़ों को संग्रहीत करने के लिए एक डायरेक्टरी बनाएं। इसे “Your Document Directory” कहें।

## पैकेज इम्पोर्ट करें

अपने Java विकास पर्यावरण में, Aspose.3D के आवश्यक पैकेज इम्पोर्ट करें। इससे आपका कोड लाइब्रेरी द्वारा प्रदान की गई कार्यक्षमताओं तक पहुँच सकेगा।

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## चरण 1: बेस प्रोफ़ाइल सेट करें

एक्सट्रूड करने के लिए बेस प्रोफ़ाइल को इनिशियलाइज़ करें। इस उदाहरण में, हम 0.3 के राउंडिंग रेडियस वाले आयत आकार का उपयोग करेंगे।

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

## चरण 2: 3D सीन बनाएं

एक सीन बनाकर अपने 3D विश्व की नींव तैयार करें।

```java
Scene scene = new Scene();
```

## चरण 3: बाएँ और दाएँ नोड बनाएं

अपने सीन में बाएँ और दाएँ नोड स्थापित करें। ये नोड आपके 3D ऑब्जेक्ट्स के लिए कैनवास के रूप में कार्य करेंगे।

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## चरण 4: सेंटर प्रॉपर्टी के साथ रैखिक एक्सट्रूज़न (बाएँ नोड)

बाएँ नोड पर **बिना सेंटरिंग** के रैखिक एक्सट्रूज़न करें और स्लाइस की संख्या को 3 सेट करें। `setCenter(false)` कॉल पर ध्यान दें – यहाँ हम **center प्रॉपर्टी** को *false* सेट कर रहे हैं।

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

## चरण 5: रेफ़रेंस के लिए ग्राउंड प्लेन जोड़ें (बाएँ नोड)

बाएँ नोड में **ground plane जोड़कर** दृश्य प्रतिनिधित्व को बेहतर बनाएं। पतला बॉक्स फ़्लोर के रूप में कार्य करता है जिससे आप एक्सट्रूज़न की ऊँचाई को स्पष्ट रूप से देख सकते हैं।

```java
left.createChildNode(new Box(0.01, 3, 3));
```

## चरण 6: सेंटर प्रॉपर्टी के साथ रैखिक एक्सट्रूज़न (दाएँ नोड)

अब दाएँ नोड पर रैखिक एक्सट्रूज़न करें, इस बार **एक्सट्रूज़न को सेंटर** करते हुए। `setCenter(true)` कॉल दिखाता है कि कैसे **center प्रॉपर्टी** को *true* सेट किया जाता है।

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

## चरण 7: रेफ़रेंस के लिए ग्राउंड प्लेन जोड़ें (दाएँ नोड)

बाएँ पक्ष की तरह, दाएँ नोड में भी एक ग्राउंड प्लेन जोड़ें ताकि एक समान दृश्य बेसलाइन मिल सके।

```java
right.createChildNode(new Box(0.01, 3, 3));
```

## चरण 8: 3D सीन सहेजें

अपने 3D सीन को Wavefront OBJ फ़ॉर्मेट में सहेजें ताकि आप इसे किसी भी मानक 3D व्यूअर में देख सकें।

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```
