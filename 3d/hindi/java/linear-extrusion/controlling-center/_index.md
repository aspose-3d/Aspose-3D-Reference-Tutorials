# Java 3D ग्राफिक्स ट्यूटोरियल – लीनियर एक्सट्रूज़न में सेंटर

## परिचय

यदि आप Java में 3D विज़ुअलाइज़ेशन बना रहे हैं, तो एक्सट्रूज़न तकनीकों में निपुण होना आवश्यक है। यह **java 3d graphics tutorial** आपको Aspose.3D for Java का उपयोग करके लीनियर एक्सट्रूज़न के सेंटर को नियंत्रित करने की प्रक्रिया दिखाता है, ताकि आप अतिरिक्त गणना के बिना सटीक, सममित मॉडल बना सकें। इस गाइड के अंत तक आप समझेंगे कि `center` प्रॉपर्टी क्यों महत्वपूर्ण है, **set rounding radius** कैसे सेट करें, और **save OBJ file java**‑संगत आउटपुट कैसे बनाएं।

## त्वरित उत्तर
- **center प्रॉपर्टी क्या करती है?** यह निर्धारित करती है कि एक्सट्रूज़न प्रोफ़ाइल के मूल बिंदु के चारों ओर सममित है या नहीं।  
- **कोड चलाने के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक अस्थायी लाइसेंस काम करता है; उत्पादन के लिए पूर्ण लाइसेंस आवश्यक है।  
- **परिणाम के लिए कौन सा फ़ाइल फ़ॉर्मेट उपयोग होता है?** सीन को Wavefront OBJ फ़ाइल के रूप में सहेजा जाता है।  
- **क्या मैं स्लाइस की संख्या बदल सकता हूँ?** हाँ, `LinearExtrusion` ऑब्जेक्ट पर `setSlices(int)` का उपयोग करें।  
- **क्या Aspose.3D Java 8+ के साथ संगत है?** बिल्कुल – यह सभी आधुनिक Java संस्करणों को सपोर्ट करता है।

## जावा 3D ग्राफिक्स ट्यूटोरियल क्या है?

एक **java 3d graphics tutorial** चरण‑दर‑चरण बताता है कि Java लाइब्रेरीज़ का उपयोग करके त्रि‑आयामी वस्तुओं को कैसे बनाया, संशोधित और रेंडर किया जाए। इस मामले में हम Aspose.3D की एक्सट्रूज़न API पर ध्यान देते हैं, जो 2‑D प्रोफ़ाइल को पूर्ण 3‑D मेष में बदलती है।

अपने Java IDE में आवश्यक Aspose.3D क्लासेज़ को इम्पोर्ट करें। इससे कंपाइलर को एक्सट्रूज़न और सीन‑बिल्डिंग सुविधाओं तक पहुंच मिलती है।

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## चरण‑दर‑चरण गाइड

### चरण 1: बेस प्रोफ़ाइल सेट करें

पहले 2‑D आकार को बनाएं जिसे एक्सट्रूड किया जाएगा। यहाँ हम एक आयत का उपयोग करते हैं और **set rounding radius** को 0.3 पर सेट करते हैं, जिससे कोने स्मूद हो जाते हैं।

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

### चरण 2: 3D सीन बनाएं

एक `Scene` ऑब्जेक्ट सभी 3‑D नोड्स, लाइट्स और कैमरों का कंटेनर होता है।

```java
Scene scene = new Scene();
```

### चरण 3: बाएँ और दाएँ नोड जोड़ें

हम दो अलग‑अलग नोड्स को साइड‑बाय‑साइड रखेंगे ताकि आप सेंटरिंग के साथ और बिना एक्सट्रूज़न की तुलना कर सकें।

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

### चरण 4: लीनियर एक्सट्रूज़न – बिना सेंटर (बायाँ नोड)

बाएँ नोड पर एक्सट्रूज़न बनाएं, सेंटरिंग को डिसेबल करें, और लो‑पॉली प्रीव्यू के लिए मेष को तीन स्लाइस तक सीमित रखें।

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

### चरण 5: रेफ़रेंस के लिए ग्राउंड प्लेन जोड़ें (बायाँ नोड)

एक पतला बॉक्स विज़ुअल फ़्लोर के रूप में कार्य करता है, जिससे आप एक्सट्रूज़न की ओरिएंटेशन देख सकें।

```java
left.createChildNode(new Box(0.01, 3, 3));
```

### चरण 6: लीनियर एक्सट्रूज़न – सेंटरड (दायाँ नोड)

अब वही एक्सट्रूज़न दोहराएँ, इस बार `center` को एनेबल करें। जियोमेट्री प्रोफ़ाइल के मूल बिंदु के चारों ओर सममित होगी।

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

### चरण 7: रेफ़रेंस के लिए ग्राउंड प्लेन जोड़ें (दायाँ नोड)

दाएँ पक्ष के लिए भी वही ग्राउंड प्लेन जोड़ें, जिससे तुलना स्पष्ट हो।

```java
right.createChildNode(new Box(0.01, 3, 3));
```

### चरण 8: 3D सीन सहेजें

अंत में पूरे सीन को Wavefront OBJ फ़ाइल में एक्सपोर्ट करें – यह फ़ॉर्मेट अधिकांश 3‑D एडिटर्स में आसानी से उपयोग किया जा सकता है।

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## सामान्य समस्याएँ और समाधान

| समस्या | क्यों होता है | समाधान |
|-------|----------------|-----|
| **Extrusion appears offset** | `setCenter(false)` प्रोफ़ाइल को उसके कोने पर एंकर कर देता है। | सममित परिणामों के लिए `setCenter(true)` उपयोग करें। |
| **OBJ file is empty** | आउटपुट डायरेक्टरी पाथ गलत है या लिखने की अनुमति नहीं है। | सुनिश्चित करें कि `MyDir` मौजूदा फ़ोल्डर की ओर इशारा करता है और एप्लिकेशन के पास लिखने की अनुमति है। |
| **Rounded corners look sharp** | राउंडिंग रेडियस आयत के आकार की तुलना में बहुत छोटा है। | रेडियस मान बढ़ाएँ (उदाहरण: `setRoundingRadius(0.5)`)। |
