# 3D सीन बनाएं – एक्सट्रूज़न दिशा सेट करें Aspose.3D Java

## परिचय

इस ट्यूटोरियल में आप **3D सीन कैसे बनाएं** ऑब्जेक्ट्स सीखेंगे और Aspose.3D for Java के साथ एक्सट्रूज़न दिशा को नियंत्रित करेंगे। चाहे आप आर्किटेक्चरल विज़ुअलाइज़ेशन, प्रोडक्ट प्रोटोटाइप, या गेम एसेट बना रहे हों, लीनियर एक्सट्रूज़न में महारत हासिल करने से आप जटिल आकारों को जल्दी मॉडल कर सकते हैं। हम हर कदम को दिखाएंगे, Java में नोड्स जोड़ने से लेकर **3D मॉडल OBJ निर्यात करें** फ़ाइलों तक, ताकि आप तुरंत परिणाम देख सकें।

## त्वरित उत्तर
- **create 3d scene** का क्या अर्थ है? यह Aspose.3D `Scene` ऑब्जेक्ट को इनिशियलाइज़ करने को कहते हैं जो सभी ज्योमेट्री, लाइट्स, और कैमरों को रखेगा।  
- **मैं एक्सट्रूज़न दिशा कैसे सेट करूँ?** `LinearExtrusion` इंस्टेंस पर `setDirection(Vector3)` मेथड का उपयोग करें।  
- **मैं निर्यात के लिए कौन सा फ़ॉर्मेट उपयोग करूँ?** OBJ फ़ॉर्मेट (`FileFormat.WAVEFRONTOBJ`) 3‑D वर्कफ़्लो के लिए व्यापक रूप से समर्थित है।  
- **क्या मुझे Aspose.3D के लिए लाइसेंस चाहिए?** विकास के लिए फ्री ट्रायल काम करता है; उत्पादन के लिए एक कमर्शियल लाइसेंस आवश्यक है।  
- **क्या मैं Java में अधिक नोड्स जोड़ सकता हूँ?** हाँ—`scene.getRootNode().createChildNode()` का उपयोग करके जितने भी ऑब्जेक्ट्स चाहिए उतने जोड़ें।

## “create 3d scene” वर्कफ़्लो क्या है?
एक **create 3d scene** वर्कफ़्लो एक खाली `Scene` ऑब्जेक्ट से शुरू होता है, ज्योमेट्री (जैसे एक्सट्रूडेड प्रोफ़ाइल) जोड़ता है, ट्रांसफ़ॉर्म्स के साथ उसकी स्थिति निर्धारित करता है, और अंत में सीन को OBJ जैसे फ़ाइल फ़ॉर्मेट में सहेजता है। यह पैटर्न Aspose.3D के साथ निर्मित अधिकांश 3‑D एप्लिकेशनों की रीढ़ है।

## एक्सट्रूज़न दिशा क्यों सेट करें?
एक्सट्रूज़न दिशा सेट करने से आप आकार को एक्सट्रूड होते समय झुका, घुमा या तिरछा कर सकते हैं, जिससे अतिरिक्त पोस्ट‑प्रोसेसिंग के बिना अंतिम ज्योमेट्री पर नियंत्रण मिलता है। यह विशेष रूप से उपयोगी है:

- टेपर कॉलम या कस्टम‑शेप्ड पाइप बनाने के लिए।  
- मैकेनिकल पार्ट्स में गैर‑मानक अक्षों के साथ एक्सट्रूज़न को संरेखित करने के लिए।  
- विज़ुअल इफ़ेक्ट्स के लिए कलात्मक आकार उत्पन्न करने के लिए।

## पैकेज इम्पोर्ट करें
, आवश्यक Aspose.3D क्लासेज़ इम्पोर्ट करें:

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## चरण 1: बेस प्रोफ़ाइल इनिशियलाइज़ करें
एक 2‑D प्रोफ़ाइल बनाएं जिसे एक्सट्रूड किया जाएगा। इस उदाहरण में हम एक गोल किनारा वाला आयत उपयोग करते हैं:

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

> **Pro tip:** गोलाई त्रिज्या को समायोजित करें ताकि एक्सट्रूज़न से पहले आयत के कोनों की तीक्ष्णता या स्मूदनेस को नियंत्रित किया जा सके।

## चरण 2: सीन बनाएं
अब हम **create 3d scene** बनाते हैं जो हमारे ऑब्जेक्ट्स को होस्ट करेगा:

```java
Scene scene = new Scene();
```

## चरण 3: नोड्स जोड़ें Java – ऑब्जेक्ट्स की पोजिशनिंग
हम सीन के रूट नोड में दो चाइल्ड नोड्स (बायाँ और दायाँ) जोड़ेंगे और बाएँ नोड को थोड़ा साइड में ले जाएंगे:

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## चरण 4: एक्सट्रूड कैसे करें – बायाँ नोड (डिफ़ॉल्ट दिशा)
डिफ़ॉल्ट Z‑axis दिशा का उपयोग करके बाएँ नोड पर प्रोफ़ाइल को एक्सट्रूड करें। हम एक पूर्ण 360° ट्विस्ट भी सेट करते हैं और स्मूदनेस के लिए स्लाइस काउंट बढ़ाते हैं:

```java
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); }});
```

## चरण 5: दिशा कैसे सेट करें – दायाँ नोड
यहाँ हम कस्टम `Vector3` प्रदान करके **direction कैसे सेट करें** दिखाते हैं। यह एक्सट्रूज़न को वेक्टर (0.3, 0.2, 1) की ओर झुकाता है:

```java
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); setDirection(new Vector3(0.3, 0.2, 1));}});
```

## चरण 6: OBJ फ़ाइल सहेजें – 3D मॉडल निर्यात करें
अंत में, हम **obj फ़ाइल सहेजते** हैं ताकि परिणाम किसी भी 3‑D व्यूअर में देखा जा सके:

```java
scene.save(MyDir + "DirectionInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

जब आप जेनरेट की गई OBJ फ़ाइल खोलेंगे, तो आपको दो एक्सट्रूडेड आयतें दिखेंगी: एक डिफ़ॉल्ट दिशा के साथ और एक वह वेक्टर जिसके अनुसार हमने झुका दिया है, उसके अनुसार झुकी हुई।

## सामान्य समस्याएँ और समाधान
| समस्या | कारण | समाधान |
|-------|--------|-----|
| OBJ फ़ाइल खाली दिखाई देती है | सीन सहेजा नहीं गया या पाथ गलत है | `MyDir` एक लिखने योग्य फ़ोल्डर की ओर इशारा करता है और फ़ाइल नाम `.obj` पर समाप्त होता है, यह सुनिश्चित करें। |
| एक्सट्रूज़न सपाट दिखता है | `setSlices` बहुत कम है | स्मूथर ज्योमेट्री के लिए `setSlices` बढ़ाएँ (उदा., 200)। |
| दिशा अपरिवर्तित लगती है | वेक्टर नॉर्मलाइज़्ड नहीं है | नॉर्मलाइज़्ड `Vector3` का उपयोग करें या मानों को समायोजित करके इच्छित झुकाव को दर्शाएँ। |
