# स्लाइस के साथ 3D एक्सट्रूज़न बनाएं – Aspose.3D for Java

## परिचय

यदि आपको **3d एक्सट्रूज़न** ऑब्जेक्ट्स बनाने की आवश्यकता है जो स्मूथ और सटीक दिखें, तो स्लाइस की संख्या को नियंत्रित करना ही कुंजी है। इस ट्यूटोरियल में हम Aspose.3D for Java के साथ लीनियर एक्सट्रूज़न करते समय स्लाइस कैसे निर्दिष्ट करें, इसे चरण‑दर‑चरण देखेंगे। आप समझेंगे कि स्लाइस काउंट क्यों महत्वपूर्ण है, राउंडिंग रेडियस कैसे सेट करें, और परिणाम को OBJ फ़ाइल के रूप में कैसे एक्सपोर्ट करें जिसे किसी भी 3D पाइपलाइन में उपयोग किया जा सकता है।

## त्वरित उत्तर
- **“स्लाइस” क्या नियंत्रित करता है?** एक्सट्रूज़न सतह को अनुमानित करने के लिए उपयोग किए जाने वाले मध्यवर्ती क्रॉस‑सेक्शन की संख्या।  
- **कौन सा मेथड स्लाइस काउंट सेट करता है?** `LinearExtrusion.setSlices(int)`  
- **क्या मैं प्रोफ़ाइल का राउंडिंग रेडियस बदल सकता हूँ?** हाँ, `RectangleShape.setRoundingRadius(double)` के माध्यम से  
- **इस उदाहरण में कौन सा फ़ाइल फ़ॉर्मेट उपयोग किया गया है?** Wavefront OBJ (`FileFormat.WAVEFRONTOBJ`)  
- **कोड चलाने के लिए क्या लाइसेंस चाहिए?** मूल्यांकन के लिए एक फ्री ट्रायल काम करता है; प्रोडक्शन के लिए एक कमर्शियल लाइसेंस आवश्यक है।

## स्लाइस के साथ लीनियर एक्सट्रूज़न क्या है?

लीनियर एक्सट्रूज़न 2‑D प्रोफ़ाइल (जैसे एक रेक्टैंगल) को एक सीधी रेखा के साथ खींचकर 3‑D ठोस बनाता है। **स्लाइस** निर्दिष्ट करके आप Aspose.3D को बताते हैं कि कितने मध्यवर्ती चरण उत्पन्न करने हैं, जो सीधे गोल कोनों जैसे राउंडेड रेक्टैंगल की स्मूदनेस को प्रभावित करता है।

## Aspose.3D for Java के साथ 3d एक्सट्रूज़न क्यों बनाएं?

* **पूर्ण नियंत्रण** – आप प्रोग्रामेटिक रूप से स्लाइस काउंट, राउंडिंग रेडियस, और एक्सपोर्ट फ़ॉर्मेट को समायोजित कर सकते हैं।  
* **क्रॉस‑प्लेटफ़ॉर्म** – किसी भी Java‑सक्षम वातावरण में बिना नेटिव डिपेंडेंसी के काम करता है।  
* **एक्सपोर्ट लचीलापन** – सीधे OBJ, FBX, STL, और कई अन्य फ़ॉर्मेट में सहेजें।

## चरण‑दर‑चरण गाइड

### चरण 1: सीन सेट करें और प्रोफ़ाइल परिभाषित करें

पहले हम एक `RectangleShape` बनाते हैं और उसे **राउंडिंग रेडियस** देते हैं ताकि कोने स्मूथ रहें। फिर हम एक नया `Scene` इनिशियलाइज़ करते हैं जो सभी जियोमेट्री को रखेगा।

```java
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
Scene scene = new Scene();
```

### चरण 2: प्रत्येक एक्सट्रूज़न के लिए **चाइल्ड नोड** ऑब्जेक्ट बनाएं

हर जियोमेट्री टुकड़ा एक `Node` के अंतर्गत रहता है। यहाँ हम दो सिब्लिंग नोड्स जनरेट करते हैं – एक लो‑स्लाइस एक्सट्रूज़न के लिए और दूसरा हाई‑स्लाइस एक्सट्रूज़न के लिए – और बाएँ नोड को थोड़ा साइड में ले जाते हैं ताकि परिणाम आसानी से तुलना किए जा सकें।

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

### चरण 3: लीनियर एक्सट्रूज़न करें और **स्लाइस सेट** करें

अब हम वास्तव में **3d एक्सट्रूज़न** ऑब्जेक्ट बनाते हैं। `LinearExtrusion` कंस्ट्रक्टर प्रोफ़ाइल और एक्सट्रूज़न डेप्थ लेता है। एक **एनॉनिमस इनर क्लास** का उपयोग करके हम `setSlices` को कॉल करते हैं ताकि स्मूदनेस नियंत्रित हो सके। बाएँ नोड को केवल 2 स्लाइस (खुरदरा) मिलते हैं, जबकि दाएँ नोड को 10 स्लाइस (स्मूथ) मिलते हैं।

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{setSlices(2);}});
right.createChildNode(new LinearExtrusion(profile, 2) {{setSlices(10);}});
```

### चरण 4: **OBJ एक्सपोर्ट** – सीन सहेजें

अंत में हम सीन को Wavefront OBJ फ़ाइल में लिखते हैं, जो 3‑D एडिटर्स और गेम इंजनों द्वारा व्यापक रूप से समर्थित फ़ॉर्मेट है। यह Aspose.3D की **export obj java** क्षमता को दर्शाता है।

```java
scene.save(MyDir + "SlicesInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```
