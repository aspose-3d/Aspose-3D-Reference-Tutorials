# Aspose.3D for Java के साथ प्रिमिटिव 3D मॉडल बनाना

## परिचय

यदि आपको कभी सीधे Java कोड से **3D सिलिंडर** ऑब्जेक्ट (या कोई अन्य बुनियादी आकार) बनाने की आवश्यकता पड़ी हो, तो Aspose.3D इस कार्य को बेहद आसान बनाता है। इस ट्यूटोरियल में हम पूरे वर्कफ़्लो को चरण‑दर‑चरण देखेंगे—पर्यावरण सेटअप से लेकर अंतिम सीन को FBX फ़ाइल के रूप में सहेजने तक—ताकि आप तुरंत प्रोग्रामेटिक रूप से 3D ज्योमेट्री जनरेट करना शुरू कर सकें।

## त्वरित उत्तर
- **Java में 3D सिलिंडर बनाने के लिए कौन सी लाइब्रेरी है?** Aspose.3D for Java.  
- **मैं सीन को किस फॉर्मेट में एक्सपोर्ट कर सकता हूँ?** FBX (ASCII 7500) using `FileFormat.FBX7500ASCII`.  
- **क्या विकास के लिए लाइसेंस चाहिए?** परीक्षण के लिए एक फ्री ट्रायल काम करता है; प्रोडक्शन के लिए स्थायी लाइसेंस आवश्यक है।  
- **समर्थित मुख्य प्रिमिटिव कौन‑से हैं?** बॉक्स, सिलिंडर, स्फीयर, कोन, और अधिक।  
- **क्या कोड Java 8 और उसके बाद के संस्करणों के साथ संगत है?** हाँ, Aspose.3D JDK 8+ को टारगेट करता है।

## 3D सिलिंडर प्रिमिटिव क्या है?

एक सिलिंडर प्रिमिटिव मूलभूत ज्यामितीय आकार है जो त्रिज्या और ऊँचाई द्वारा परिभाषित होता है। कई 3D पाइपलाइन में यह पाइप, पहिया, या वास्तु स्तंभ जैसे जटिल मॉडलों के निर्माण ब्लॉक के रूप में कार्य करता है। Aspose.3D एक तैयार `Cylinder` क्लास प्रदान करता है, जिससे आपको वर्टिसेज़ मैन्युअली गणना करने की ज़रूरत नहीं पड़ती।

## Aspose.3D for Java क्यों उपयोग करें?

- **पूर्ण‑फ़ीचर वाला 3D इंजन** – लोकप्रिय फॉर्मेट्स (FBX, OBJ, STL, आदि) का इम्पोर्ट/एक्सपोर्ट सपोर्ट करता है।  
- **शुद्ध Java API** – कोई नेटिव डिपेंडेंसी नहीं, क्रॉस‑प्लेटफ़ॉर्म प्रोजेक्ट्स के लिए आदर्श।  
- **मज़बूत सीन ग्राफ** – आपको ऑब्जेक्ट्स को पदानुक्रमित रूप से व्यवस्थित करने देता है।  
- **आसान लाइसेंसिंग** – फ्री ट्रायल से शुरू करें, फिर स्थायी लाइसेंस में अपग्रेड करें।

## पूर्वापेक्षाएँ

1. **Java Development Kit (JDK)** – आपके मशीन पर JDK 8 या नया स्थापित होना चाहिए।  
2. **Aspose.3D for Java लाइब्रेरी** – इसे [download page](https://releases.aspose.com/3d/java/) से डाउनलोड और इंस्टॉल करें।  

## पैकेज इम्पोर्ट करें

सबसे पहले कोर Aspose.3D नेमस्पेस को इम्पोर्ट करें। इससे आपको `Scene`, `Box`, `Cylinder`, और फ़ाइल‑फ़ॉर्मेट कॉन्स्टेंट्स तक पहुंच मिलती है।

```java
import com.aspose.threed.*;
```

अब लाइब्रेरी रेफ़रेंस हो गई है, चलिए एक सीन बनाते हैं और कुछ प्रिमिटिव ज्योमेट्री जोड़ते हैं।

## सीन में 3D सिलिंडर और अन्य प्रिमिटिव कैसे बनाएं

### चरण 1: Scene ऑब्जेक्ट को इनिशियलाइज़ करें

सबसे पहले, हमें एक `Scene` कंटेनर चाहिए जो हमारे सभी 3D नोड्स को रखेगा।

```java
// The path to the documents directory.
String myDir = "Your Document Directory";

// Initialize a Scene object
Scene scene = new Scene();
```

### चरण 2: 3D बॉक्स मॉडल बनाएं

बॉक्स प्रिमिटिव कोऑर्डिनेट सिस्टम को टेस्ट करने में उपयोगी है। यहाँ हम इसे सीन के रूट नोड के चाइल्ड के रूप में जोड़ते हैं।

```java
// Create a Box model
scene.getRootNode().createChildNode("box", new Box());
```

### चरण 3: 3D सिलिंडर मॉडल बनाएं

अब हम वास्तव में **3D सिलिंडर** ज्योमेट्री बनाते हैं। `Cylinder` क्लास डिफ़ॉल्ट आकारों के साथ आती है, लेकिन आवश्यकता पड़ने पर आप इसके कंस्ट्रक्टर के माध्यम से रेडियस और ऊँचाई कस्टमाइज़ कर सकते हैं।

```java
// Create a Cylinder model
scene.getRootNode().createChildNode("cylinder", new Cylinder());
```

### चरण 4: ड्रॉइंग को FBX फॉर्मेट में सहेजें

सीन को असेंबल करने के बाद, इसे एक्सपोर्ट करें ताकि अन्य टूल्स (जैसे Unity, Blender) इसे पढ़ सकें। हम `FBX7500ASCII` फॉर्मेट का उपयोग करते हैं, जो व्यापक रूप से समर्थित है।

```java
// Save drawing in the FBX format
myDir = myDir + "test.fbx";
scene.save(myDir, FileFormat.FBX7500ASCII);
```

## सामान्य समस्याएँ और समाधान

| समस्या | क्यों होता है | समाधान |
|-------|----------------|-----|
| **फ़ाइल नहीं मिली** सहेजते समय | `myDir` एक गैर‑मौजूद फ़ोल्डर की ओर इशारा कर रहा है | डायरेक्टरी मौजूद है यह सुनिश्चित करें या इसे `new File(myDir).mkdirs();` से बनाएं |
| **खाली सीन** एक्सपोर्ट के बाद | `save` कॉल करने से पहले कोई नोड नहीं जोड़ा गया था | सुनिश्चित करें कि `createChildNode` कॉल्स निष्पादित हो रहे हैं (डिबग के लिए `scene.getRootNode().getChildCount()` देखें) |
| **लाइसेंस अपवाद** | प्रोडक्शन में वैध लाइसेंस के बिना चल रहा है | एक अस्थायी या स्थायी लाइसेंस लागू करें इस तरह: `License license = new License(); license.setLicense("Aspose.3D.Java.lic");` |
