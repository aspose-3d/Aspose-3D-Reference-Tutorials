## Frequently Asked Questions

**Q: Can I use Aspose.3D to **build 3d cylinder** in a real‑time game engine?**  
A: はい。生成したシリンダーメッシュを FBX や OBJ などの形式でエクスポートすれば、Unity や Unreal などのゲームエンジンで広くサポートされています。

**Q: How do I **change plane orientation** for a mesh that’s already been created?**  
A: `Node.Transform` プロパティを使用して、メッシュを含むノードに回転行列またはオイラー角を適用します。

**Q: What is the best way to **generate 3d mesh** data for large scenes?**  
A: ジオメトリをバッチ作成し、マテリアルを再利用し、エクスポート前に `Scene.Optimize` を呼び出してメモリ使用量を抑えます。

**Q: Is there a limit to the number of segments when I **create cylinder mesh**?**  
A: 実質的にはメモリとパフォーマンスが唯一の制限です。滑らかな外観を得るために、一般的なシリンダーは 16‑32 のラジアルセグメントを使用します。

**Q: Do I need a separate license for each .NET platform?**  
A: いいえ。単一の Aspose.3D ライセンスで、サポートされているすべての .NET ランタイムをカバーします。
