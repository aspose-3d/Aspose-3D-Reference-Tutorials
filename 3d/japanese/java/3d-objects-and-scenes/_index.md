## よくある質問

**Q: 複数の球体の半径を同時に変更できますか？**  
A: はい。Aspose.3D の XPath ライクなクエリを使用してすべての球体ノードを選択し、イテレートして各半径を設定します。

**Q: 半径を変更すると球体のテクスチャ座標に影響しますか？**  
A: テクスチャマッピングは半径に合わせて自動的にスケールし、UV の一貫性が保たれます。

**Q: 時間経過に伴う半径のアニメーションは可能ですか？**  
A: もちろんです。`setRadius()` とタイマーまたはアニメーションループを組み合わせてスムーズな遷移を作成します。

**Q: 必要な Aspose.3D のバージョンは？**

**Q: 変更したシーンを他のフォーマットにエクスポートできますか？**
