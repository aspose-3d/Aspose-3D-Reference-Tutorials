## よくある質問

**Q: このチュートリアルを受講するのに事前の 3D グラフィック経験は必要ですか？**  
A: いいえ。チュートリアルは基本概念から始まり、徐々に高度なトピックを紹介するため、初心者から経験豊富な開発者まで幅広く対応しています。

**Q: Java 以外のプラットフォームでも Aspose.3D を使用できますか？**  
A: はい。Aspose.3D は .NET、C++、Python でも利用可能ですが、このチュートリアルのコード例は Java に特化しています。

**Q: 本番で使用するライセンスはどうやって取得しますか？**  
A: 無料トライアルを評価した後、Aspose のウェブサイトからライセンスを購入できます。取得したライセンス ファイルは、実行時に Java アプリケーションにロードされます。

**Q: このライブラリは Android 開発と互換性がありますか？**  
A: Aspose.3D は Android API レベル 21 以上をサポートしており、これらのチュートリアルで示された API 呼び出しを Android プロジェクト内でも使用できます。

**Q: 完全な API リファレンスはどこで見つけられますか？**  
A: 公式 API リファレンスは Aspose のドキュメントポータルで提供されており、これらのチュートリアルで使用されているすべてのクラスとメソッドの詳細な説明が含まれています。
