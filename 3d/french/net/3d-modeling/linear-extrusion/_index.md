## Réponses rapides
- **Qu'est‑ce que l'extrusion ?** Une opération de modélisation 3D qui étend un profil 2D le long d'un chemin pour créer du volume.
- **Pourquoi choisir Aspose.3D ?** Support complet .NET, aucune dépendance externe, et un traitement géométrique haute performance.
- **Combien de temps prend une extrusion linéaire de base ?** Typiquement moins d'une seconde pour des profils simples sur du matériel moderne.
- **Ai‑je besoin d'une licence ?** Une version d'essai est disponible, mais une licence commerciale est requise pour une utilisation en production.
- **Quelles versions de .NET sont prises en charge ?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Questions fréquentes

**Q : Puis‑je combiner plusieurs techniques d'extrusion dans un même modèle ?**  
R : Oui, Aspose.3D vous permet d'enchaîner les opérations de centrage, de direction, de torsion et de décalage pour créer des géométries complexes.

**Q : Existe‑t‑il une limite au nombre de tranches ou de torsions que je peux appliquer ?**  
R : Pratiquement, la limite est la mémoire et la puissance de traitement ; l'API gère efficacement les grands ensembles de données.

**Q : Dois‑je calculer manuellement le chemin d'extrusion ?**  
R : Non, la bibliothèque fournit des méthodes d'aide pour définir les chemins et générer automatiquement la géométrie.

**Q : Comment prévisualiser l'extrusion avant l'exportation ?**  
R : Utilisez le visualiseur de scène intégré ou exportez vers un format courant comme OBJ/GLTF pour une inspection rapide.

**Q : Existe‑t‑il des conseils de performance pour les grosses extrusions ?**  
R : Réutilisez les objets géométriques, limitez le nombre de sommets par tranche, et activez le traitement multi‑thread lorsque c'est possible.
