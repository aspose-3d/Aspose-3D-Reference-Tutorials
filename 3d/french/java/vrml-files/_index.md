## Questions fréquentes

**Q : Puis‑je utiliser Aspose.3D dans un projet Java commercial ?**  
R : Oui, une licence commerciale est requise pour une utilisation en production. Un essai gratuit est disponible pour évaluation.

**Q : Quelles versions de Java sont prises en charge ?**  
R : Aspose.3D prend en charge Java 8 et les versions ultérieures, y compris les versions à support à long terme comme Java 11 et Java 17.

**Q : Comment convertir un fichier VRML vers un autre format ?**  
R : Chargez le fichier VRML avec `Scene.load()`, puis appelez `scene.save("output.obj")` (ou tout autre format pris en charge).

**Q : Que faire si mon fichier VRML contient des textures qui ne se chargent pas ?**  
R : Assurez‑vous que les chemins des textures sont corrects et que les fichiers sont accessibles à l’exécution. Aspose.3D intégrera les textures manquantes comme espaces réservés.

**Q : Existe‑t‑il un moyen de réduire l’empreinte mémoire des scènes VRML volumineuses ?**  
R : Utilisez `scene.optimize()` avant le traitement ou l’exportation pour minimiser l’utilisation des ressources.
