## Frequently Asked Questions

**Q:** *Can I use Aspose.3D in a commercial Java application?*  
**A:** Oui. Une licence Aspose valide est requise pour une utilisation en production ; un essai gratuit est disponible pour l’évaluation.

**Q:** *Which 3D file formats are supported for loading and saving?*  
**A:** Aspose.3D prend en charge OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X, et plusieurs autres.

**Q:** *How do I upgrade materials to PBR without rewriting the whole scene?*  
**A:** Utilisez la méthode `MaterialConverter.convertToPbr()` fournie dans l’API – elle met à jour tous les matériaux en un seul appel.

**Q:** *Is there a way to reduce file size when saving large scenes?*  
**A:** Oui. Configurez `SaveOptions` (par ex., activez la compression, réduisez la précision des maillages) pour optimiser la taille du fichier de sortie.

**Q:** *Do I need any additional dependencies to work with Aspose.3D?*  
**A:** Seul le JAR Aspose.3D for Java est requis ; toutes les autres dépendances sont incluses.
