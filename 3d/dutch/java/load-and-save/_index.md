## Veelgestelde Vragen

**Q:** *Kan ik Aspose.3D gebruiken in een commerciële Java‑applicatie?*  
**A:** Ja. Een geldige Aspose‑licentie is vereist voor productie; een gratis proefversie is beschikbaar voor evaluatie.

**Q:** *Welke 3D‑bestandsformaten worden ondersteund voor laden en opslaan?*  
**A:** Aspose.3D ondersteunt OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X en verschillende andere.

**Q:** *Hoe upgrade ik materialen naar PBR zonder de hele scène opnieuw te schrijven?*  
**A:** Gebruik de `MaterialConverter.convertToPbr()`‑methode die in de API wordt geleverd – deze werkt alle materialen in één oproep bij.

**Q:** *Is er een manier om de bestandsgrootte te verkleinen bij het opslaan van grote scènes?*  
**A:** Ja. Configureer `SaveOptions` (bijv. compressie inschakelen, mesh‑precisie verlagen) om de uitvoergrootte te optimaliseren.

**Q:** *Heb ik extra afhankelijkheden nodig om met Aspose.3D te werken?*  
**A:** Alleen de Aspose.3D for Java JAR is vereist; alle andere afhankelijkheden zijn inbegrepen.
