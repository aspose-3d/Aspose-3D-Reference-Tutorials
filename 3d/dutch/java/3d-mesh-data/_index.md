## Waarom deze onderwerpen belangrijk zijn

* **Prestaties:** Het genereren van tangents en het splitsen van meshes vermindert draw calls en verbetert het GPU‑gebruik.  
* **Bestandsgrootte:** Draco‑compressie kan assets tot wel 90 % verkleinen, waardoor downloads sneller zijn voor eindgebruikers.  
* **Visuele getrouwheid:** Correcte tangents zorgen ervoor dat normal maps correct worden gerenderd, waardoor je modellen er realistisch uitzien.
