## Snelle Antwoorden
- **Wat is extrusie?** Een 3‑D-modelleringsoperatie die een 2‑D-profiel langs een pad uitstrekt om volume te creëren.
- **Waarom Aspose.3D kiezen?** Volledige .NET-ondersteuning, geen externe afhankelijkheden, en high‑performance geometrieverwerking.
- **Hoe lang duurt een basis lineaire extrusie?** Meestal minder dan een seconde voor eenvoudige profielen op moderne hardware.
- **Heb ik een licentie nodig?** Er is een proefversie beschikbaar, maar een commerciële licentie is vereist voor productiegebruik.
- **Welke .NET-versies worden ondersteund?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Veelgestelde vragen

**Q: Kan ik meerdere extrusietechnieken combineren in één model?**  
A: Ja, Aspose.3D stelt je in staat om centering-, direction-, twist- en offset‑operaties te combineren om complexe geometrieën te creëren.

**Q: Is er een limiet aan het aantal slices of twists dat ik kan toepassen?**  
A: In de praktijk is de limiet geheugen en verwerkingskracht; de API verwerkt grote datasets efficiënt.

**Q: Moet ik het extrusiepad handmatig berekenen?**  
A: Nee, de bibliotheek biedt hulpfuncties om paden te definiëren en automatisch de geometrie te genereren.

**Q: Hoe kan ik de extrusie bekijken voordat ik exporteer?**  
A: Gebruik de ingebouwde scene‑viewer of exporteer naar een gangbaar formaat zoals OBJ/GLTF voor snelle inspectie.

**Q: Zijn er prestatietips voor grote extrusies?**  
A: Herbruik geometrie‑objecten, beperk het aantal vertices per slice, en schakel waar mogelijk multi‑threaded verwerking in.
