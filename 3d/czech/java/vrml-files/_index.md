## Často kladené otázky

**Q: Mohu použít Aspose.3D v komerčním Java projektu?**  
A: Ano, pro produkční použití je vyžadována komerční licence. K dispozici je bezplatná zkušební verze pro vyhodnocení.

**Q: Které verze Javy jsou podporovány?**  
A: Aspose.3D podporuje Java 8 a novější, včetně dlouhodobě podporovaných verzí jako Java 11 a Java 17.

**Q: Jak převést VRML soubor do jiného formátu?**  
A: Načtěte VRML soubor pomocí `Scene.load()`, poté zavolejte `scene.save("output.obj")` (nebo jakýkoli jiný podporovaný formát).

**Q: Co když můj VRML soubor obsahuje textury, které se nenačtou?**  
A: Ujistěte se, že cesty k texturám jsou správné a že soubory jsou přístupné během běhu. Aspose.3D vloží chybějící textury jako zástupné objekty.

**Q: Existuje způsob, jak snížit paměťovou stopu velkých VRML scén?**  
A: Použijte `scene.optimize()` před zpracováním nebo exportem, abyste minimalizovali využití zdrojů.
