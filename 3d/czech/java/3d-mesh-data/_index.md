## Proč je důležité snižovat velikost 3D souboru

* **Výkon:** Generování tangents a rozdělení mesh snižuje počet draw callů a zlepšuje využití GPU.  
* **Velikost souboru:** Komprese Draco může zmenšit assety až o 90 %, což urychluje stahování pro koncové uživatele.  
* **Vizuální věrnost:** Správné tangents zajišťují, že normální mapy se renderují správně, což vašim modelům dodává realistický vzhled.

## Často kladené otázky

**Q: Mohu kombinovat kompresi Draco s generováním mesh‑dat v jedné pipeline?**  
A: Ano. Typický workflow je nejprve vygenerovat chybějící data (normály, tangents, binormály) a poté aplikovat kompresi Draco na obohacený mesh.

**Q: Ovlivňuje snižování velikosti 3D souboru výkon za běhu?**  
A: Snížení velikosti souboru obvykle zlepšuje časy načítání a snižuje využití paměti. V kombinaci se správnou organizací mesh (např. rozdělení podle materiálu) také snižuje počet draw callů, čímž zvyšuje výkon za běhu.

**Q: Existují nějaká omezení velikosti mesh, které lze komprimovat pomocí Draco?**  
A: Draco zvládá velmi velké mesh, ale extrémně vysokopolygónové modely mohou vyžadovat úpravu nastavení komprese (např. počet kvantizačních bitů) pro vyvážení velikosti a kvality.

**Q: Musím po dekompresi Draco mesh znovu generovat tangents?**  
A: Ne. Draco zachovává atributy vrcholů, včetně tangents, pokud byly vygenerovány před kompresí.

**Q: Je pro produkční použití vyžadována komerční licence?**  
A: Ano. Zatímco bezplatná zkušební verze vám umožní prozkoumat funkce, pro produkční nasazení je potřeba platná licence Aspose.3D.
