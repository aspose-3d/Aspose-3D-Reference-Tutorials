# Pokročilá podpora výrobních formátů souborů

## Úvod

## Exportujte komprimovaný soubor AMF

Aspose.3D usnadňuje export 3D scén do formátu Additive Manufacturing File Format (AMF) s povolenou kompresí. Tato funkce zajišťuje efektivní ukládání a přenos 3D modelů, snižuje velikost souboru při zachování geometrické a vizuální věrnosti původních scén.
