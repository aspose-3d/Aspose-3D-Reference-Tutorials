## Rychlé odpovědi
- **Co je extruze?** Operace 3‑D modelování, která prodlužuje 2‑D profil podél cesty a vytváří objem.
- **Proč zvolit Aspose.3D?** Kompletní podpora .NET, žádné externí závislosti a vysoce výkonný proces geometrie.
- **Jak dlouho trvá základní lineární extruze?** Obvykle méně než sekunda pro jednoduché profily na moderním hardwaru.
- **Potřebuji licenci?** Je k dispozici zkušební verze, ale pro produkční použití je vyžadována komerční licence.
- **Které verze .NET jsou podporovány?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Často kladené otázky

**Q: Mohu kombinovat více technik extruze v jednom modelu?**  
A: Ano, Aspose.3D vám umožňuje řetězit operace centrování, směru, točení a posunu k vytvoření složitých geometrií.

**Q: Existuje limit na počet řezů nebo točení, které mohu použít?**  
A: Prakticky je limit omezen pamětí a výpočetním výkonem; API efektivně zpracovává velké datové sady.

**Q: Musím ručně vypočítávat cestu extruze?**  
A: Ne, knihovna poskytuje pomocné metody pro definování cest a automatické generování geometrie.

**Q: Jak si mohu prohlédnout extruzi před exportem?**  
A: Použijte vestavěný prohlížeč scény nebo exportujte do běžného formátu jako OBJ/GLTF pro rychlou kontrolu.

**Q: Existují tipy na výkon pro velké extruze?**  
A: Znovu používejte objekty geometrie, omezte počet vrcholů na řez a kde je to možné, povolte vícevláknové zpracování.
