## Masalah Umum dan Solusinya

MaterialConverter adalah kelas utilitas yang mengonversi definisi material lama ke format Physically Based Rendering (PBR).

**Issue:** *Materials lose texture after conversion.*  
**Solution:** Use `MaterialConverter.convertToPbr()` before saving; it preserves texture maps and updates shader parameters automatically.

**Issue:** *Material kehilangan tekstur setelah konversi.*  
**Solution:** Gunakan `MaterialConverter.convertToPbr()` sebelum menyimpan; itu mempertahankan peta tekstur dan memperbarui parameter shader secara otomatis.

LoadOptions adalah kelas konfigurasi yang mengontrol bagaimana Aspose.3D mem-parsing file masukan.

**Issue:** *Large OBJ files cause OutOfMemoryError.*  
**Solution:** Enable streaming via `LoadOptions.setEnableMemoryOptimization(true)` and increase the JVM heap size (`-Xmx4g`).

**Issue:** *File OBJ besar menyebabkan OutOfMemoryError.*  
**Solution:** Aktifkan streaming melalui `LoadOptions.setEnableMemoryOptimization(true)` dan tingkatkan ukuran heap JVM (`-Xmx4g`).

SaveOptions mengonfigurasi pengaturan output seperti format, kompresi, dan inklusi fitur saat menyimpan adegan.

**Issue:** *Exported FBX files are missing animation data.*  
**Solution:** Set `SaveOptions.setExportAnimations(true)` and verify that the source scene contains keyed animation tracks.

**Issue:** *File FBX yang diekspor tidak memiliki data animasi.*  
**Solution:** Atur `SaveOptions.setExportAnimations(true)` dan pastikan adegan sumber berisi trek animasi yang diberi kunci.

## Pertanyaan yang Sering Diajukan

**Q:** *Can I use Aspose.3D in a commercial Java application?*  
**A:** Yes. A valid Aspose license is required for production use; a free trial is available for evaluation.

**Q:** *Bisakah saya menggunakan Aspose.3D dalam aplikasi Java komersial?*  
**A:** Ya. Lisensi Aspose yang valid diperlukan untuk penggunaan produksi; versi percobaan gratis tersedia untuk evaluasi.

**Q:** *Which 3D file formats are supported for loading and saving?*  
**A:** Aspose.3D supports OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X, and several others—over 30 formats in total.

**Q:** *Format file 3D apa yang didukung untuk memuat dan menyimpan?*  
**A:** Aspose.3D mendukung OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X, dan beberapa lainnya—lebih dari 30 format secara total.

**Q:** *How do I upgrade materials to PBR without rewriting the whole scene?*  
**A:** Use the `MaterialConverter.convertToPbr()` method provided in the API – it updates all materials in one call.

**Q:** *Bagaimana cara meng-upgrade material ke PBR tanpa menulis ulang seluruh adegan?*  
**A:** Gunakan metode `MaterialConverter.convertToPbr()` yang disediakan dalam API – ia memperbarui semua material dalam satu panggilan.

**Q:** *Is there a way to reduce file size when saving large scenes?*  
**A:** Yes. Configure `SaveOptions` (e.g., enable compression, reduce mesh precision) to optimize output size.

**Q:** *Apakah ada cara mengurangi ukuran file saat menyimpan adegan besar?*  
**A:** Ya. Konfigurasikan `SaveOptions` (misalnya, aktifkan kompresi, kurangi presisi mesh) untuk mengoptimalkan ukuran output.

**Q:** *Do I need any additional dependencies to work with Aspose.3D?*  
**A:** Only the Aspose.3D for Java JAR is required; all other dependencies are bundled.

**Q:** *Apakah saya memerlukan dependensi tambahan untuk bekerja dengan Aspose.3D?*  
**A:** Hanya JAR Aspose.3D untuk Java yang diperlukan; semua dependensi lain sudah disertakan.
