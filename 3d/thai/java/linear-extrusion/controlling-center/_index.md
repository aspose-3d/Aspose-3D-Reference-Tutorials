# การควบคุมจุดศูนย์กลางใน Linear Extrusion ด้วย Aspose.3D for Java

## บทนำ

เมื่อคุณสร้างฉาก 3 มิติใน Java ความสามารถในการ **เพิ่มพื้นราบ (ground plane)** พร้อมกับ **ตั้งค่าคุณสมบัติศูนย์กลาง** อย่างแม่นยำบน Linear Extrusion สามารถทำให้ผลลัพธ์แตกต่างจากต้นแบบแบน ๆ ไปสู่ภาพที่ดูเป็นมืออาชีพ ในบทแนะนำนี้เราจะเดินผ่านกระบวนการทั้งหมดของการควบคุมศูนย์กลางของ extrusion และการเพิ่มพื้นราบโดยใช้ Aspose.3D for Java คุณจะได้เห็นว่าทำไมสิ่งนี้ถึงสำคัญ วิธีการตั้งค่า และตัวอย่างโค้ดที่พร้อมรันซึ่งคุณสามารถปรับใช้กับโปรเจกต์ของคุณได้

## คำตอบสั้น ๆ
- **“add ground plane” ทำอะไร?** มันสร้างเรขาคณิตอ้างอิงบาง ๆ ที่ช่วยให้คุณมองเห็นตำแหน่งของ extrusion เทียบกับแกนโลก  
- **ฉันจะตั้งศูนย์กลางของ Linear Extrusion ได้อย่างไร?** ใช้เมธอด `setCenter(boolean)` บนวัตถุ `LinearExtrusion`  
- **ต้องมีลิขสิทธิ์เพื่อรันตัวอย่างหรือไม่?** ลิขสิทธิ์ชั่วคราวใช้ได้สำหรับการทดสอบ; ต้องมีลิขสิทธิ์เต็มสำหรับการใช้งานจริง  
- **บันทึกเป็นรูปแบบไฟล์ใด?** ตัวอย่างบันทึกเป็น Wavefront OBJ (`.obj`)  
- **ต้องใช้ Java เวอร์ชันใด?** Java 8 หรือใหม่กว่าเพียงพอ

## “add ground plane” คืออะไรในฉาก 3 มิติ?

การเพิ่มพื้นราบหมายถึงการแทรกเรขาคณิตสี่เหลี่ยมผืนผ้าบาง (มักเป็นกล่องที่มีความหนาน้อย) ที่วางบนระนาบ X‑Z ทำหน้าที่เป็นพื้นมองเห็น ช่วยให้ประเมินความสูงและการจัดแนวของวัตถุอื่น ๆ ได้ง่ายขึ้น โดยเฉพาะเมื่อคุณกำลังทดลองกับศูนย์กลางของ extrusion

## ทำไมต้องตั้งค่าคุณสมบัติศูนย์กลางบน Linear Extrusion?

โดยค่าเริ่มต้น Linear Extrusion จะเริ่มจากจุดกำเนิดของโปรไฟล์ การตั้งค่าคุณสมบัติศูนย์กลาง (`setCenter(true)`) จะทำให้โปรไฟล์ถูกย้ายเพื่อให้ extrusion อยู่กึ่งกลางจุดกำเนิด ซึ่งมีประโยชน์สำหรับการออกแบบสมมาตรหรือเมื่อคุณต้องการการจัดแนวที่สอดคล้องกันในหลายวัตถุ

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่มการเดินทางในบทแนะนำนี้ โปรดตรวจสอบว่าคุณมีข้อกำหนดต่อไปนี้พร้อมใช้งานแล้ว:

1. **สภาพแวดล้อมการพัฒนา Java** – ตรวจสอบว่าคุณได้ตั้งค่าสภาพแวดล้อมการพัฒนา Java บนเครื่องของคุณแล้ว  
2. **Aspose.3D for Java** – ดาวน์โหลดและติดตั้งไลบรารี Aspose.3D คุณสามารถค้นหาไลบรารีและเอกสารได้ [ที่นี่](https://reference.aspose.com/3d/java/)  
3. **โฟลเดอร์เอกสาร** – สร้างโฟลเดอร์เพื่อเก็บไฟล์ Java ของคุณ สมมติว่าเรียกว่า “Your Document Directory”

## นำเข้าแพ็กเกจ

ในสภาพแวดล้อมการพัฒนา Java ของคุณ ให้นำเข้าแพ็กเกจที่จำเป็นสำหรับ Aspose.3D เพื่อให้โค้ดของคุณเข้าถึงฟังก์ชันต่าง ๆ ของไลบรารี

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## ขั้นตอนที่ 1: ตั้งค่าโปรไฟล์พื้นฐาน

กำหนดค่าโปรไฟล์พื้นฐานที่จะทำการ extrusion ในตัวอย่างนี้ เราจะใช้รูปสี่เหลี่ยมที่มีรัศมีการโค้ง 0.3

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

## ขั้นตอนที่ 2: สร้างฉาก 3 มิติ

สร้างพื้นฐานของโลก 3 มิติของคุณโดยการสร้าง Scene

```java
Scene scene = new Scene();
```

## ขั้นตอนที่ 3: สร้าง Node ซ้ายและขวา

สร้าง Node ซ้ายและขวาใน Scene ของคุณ Node เหล่านี้ทำหน้าที่เป็นผังงานสำหรับวัตถุ 3 มิติของคุณ

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## ขั้นตอนที่ 4: Linear Extrusion พร้อมคุณสมบัติศูนย์กลาง (Node ซ้าย)

ทำ Linear Extrusion บน Node ซ้าย **โดยไม่ตั้งศูนย์กลาง** และตั้งจำนวน slice เป็น 3 โปรดสังเกตการเรียก `setCenter(false)` – นี่คือการ **ตั้งค่าคุณสมบัติศูนย์กลาง** เป็น *false*

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

## ขั้นตอนที่ 5: เพิ่มพื้นราบเพื่ออ้างอิง (Node ซ้าย)

เพิ่มการแสดงผลด้วยการ **เพิ่มพื้นราบ** ไปยัง Node ซ้าย กล่องบาง ๆ ทำหน้าที่เป็นพื้นเพื่อให้คุณมองเห็นความสูงของ extrusion ได้ชัดเจน

```java
left.createChildNode(new Box(0.01, 3, 3));
```

## ขั้นตอนที่ 6: Linear Extrusion พร้อมคุณสมบัติศูนย์กลาง (Node ขวา)

ต่อไปทำ Linear Extrusion บน Node ขวา โดย **ตั้งศูนย์กลาง extrusion** การเรียก `setCenter(true)` แสดงวิธี **ตั้งค่าคุณสมบัติศูนย์กลาง** เป็น *true*

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

## ขั้นตอนที่ 7: เพิ่มพื้นราบเพื่ออ้างอิง (Node ขวา)

เช่นเดียวกับด้านซ้าย ให้เพิ่มพื้นราบไปยัง Node ขวาเพื่อให้มีฐานการมองเห็นที่สอดคล้องกัน

```java
right.createChildNode(new Box(0.01, 3, 3));
```

## ขั้นตอนที่ 8: บันทึกฉาก 3 มิติ

บันทึกฉาก 3 มิติของคุณในรูปแบบ Wavefront OBJ เพื่อให้สามารถดูได้ในโปรแกรมดู 3 มิติทั่วไป

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## ปัญหาที่พบบ่อยและวิธีแก้

| ปัญหา | สาเหตุ | วิธีแก้ |
|-------|--------|--------|
| ไม่เห็นพื้นราบ | ความหนาของกล่องน้อยเกินกว่ามาตราส่วนของ viewer | เพิ่มความหนา (พารามิเตอร์แรกของ `Box`) หรือซูมออกใน viewer |
| Extrusion แสดงตำแหน่งผิด | ค่า `setCenter` ไม่ได้ตั้งตามที่ต้องการ | ตรวจสอบค่า boolean ที่ส่งให้ `setCenter` อีกครั้ง |
| ไม่สามารถบันทึกไฟล์ | เส้นทางโฟลเดอร์ไม่ถูกต้องหรือไม่มีสิทธิ์เขียน | ยืนยันว่า `MyDir` ชี้ไปยังโฟลเดอร์ที่มีอยู่และมีสิทธิ์เขียน |
