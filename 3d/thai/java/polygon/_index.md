## Frequently Asked Questions

**Q: Can I use Aspose.3D to create UV mapping for real‑time engines like Unity?**  
A: ใช่ คุณสามารถส่งออกเมชพร้อม UV ไปยังฟอร์แมตที่ Unity รองรับ (เช่น FBX หรือ glTF) แล้วนำเข้าโดยตรง  

**Q: Does triangle conversion affect my original model hierarchy?**  
A: การแปลงจะสร้างเมชใหม่ที่เป็นสามเหลี่ยมโดยคงลำดับชั้นของโหนดเดิมไว้ ดังนั้นการแปลงจึงไม่กระทบต่อการเปลี่ยนแปลงที่มีอยู่  

**Q: What if my model already contains UVs?**  
A: Aspose.3D จะเขียนทับช่อง UV ที่มีอยู่เฉพาะเมื่อคุณเรียกเมธอดการสร้าง UV อย่างชัดเจน มิฉะนั้นจะปล่อยไว้ตามเดิม  

**Q: Is there a performance penalty for generating UVs at runtime?**  
A: แนะนำให้สร้าง UV เพียงครั้งเดียวในขั้นตอนการเตรียมทรัพยากร การสร้าง UV ขณะรันไทม์ทำได้แต่อาจเพิ่มภาระสำหรับเมชขนาดใหญ่  

**Q: Which file formats retain the generated UV coordinates?**  
A: OBJ, FBX, glTF, และ STL (เมื่อใช้รูปแบบ STL ขยาย) ทั้งหมดจะคงข้อมูล UV ที่เขียนโดย Aspose.3D
