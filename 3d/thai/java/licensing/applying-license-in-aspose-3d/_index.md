# วิธีตั้งค่าไลเซนส์ใน Aspose.3D สำหรับ Java

## บทนำ

ยินดีต้อนรับ! ในบทแนะนำแบบขั้นตอนนี้คุณจะได้เรียนรู้ **วิธีตั้งค่าไลเซนส์** สำหรับ Aspose.3D ในแอปพลิเคชัน Java และยังเรียนรู้ **วิธีใช้คีย์สาธารณะและส่วนตัว** สำหรับการให้ไลเซนส์แบบมิเตอร์ Aspose.3D เป็นไลบรารี Java ที่ทรงพลังซึ่งทำให้การทำงานกับรูปแบบไฟล์ 3D ง่ายขึ้น และไลเซนส์ที่ถูกต้องจะเปิดใช้งานคุณสมบัติทั้งหมดของมัน เมื่ออ่านคู่มือนี้จนจบแล้วคุณจะสามารถผสานการจัดการไลเซนส์เข้าไปในโครงการ Java ใด ๆ ได้อย่างราบรื่น

## คำตอบอย่างรวดเร็ว
- **วิธีหลักในการตั้งค่าไลเซนส์คืออะไร?** ใช้คลาส `License` แล้วเรียก `setLicense` พร้อมกับเส้นทางไฟล์หรือสตรีม  
- **ฉันสามารถโหลดไลเซนส์จากสตรีมได้หรือไม่?** ได้ – `FileInputStream` ทำงานได้อย่างสมบูรณ์  
- **คีย์สาธารณะ/ส่วนตัวใช้ทำอะไร?** คีย์เหล่านี้เปิดใช้งานการให้ไลเซนส์แบบมิเตอร์ผ่านคลาส `Metered`  
- **ต้องการไลเซนส์สำหรับการพัฒนาหรือไม่?** ไลเซนส์ชั่วคราวหรือไลเซนส์ทดลองเพียงพอสำหรับการทดสอบ; ไลเซนส์เต็มจำเป็นสำหรับการใช้งานจริง  
- **เวอร์ชัน Java ที่รองรับคืออะไร?** Aspose.3D ทำงานกับ Java 6 ขึ้นไป

## ข้อกำหนดเบื้องต้น

ก่อนเริ่มทำตามขั้นตอน โปรดตรวจสอบว่าคุณมี:

- ความเข้าใจพื้นฐานเกี่ยวกับการเขียนโปรแกรม Java  
- ไลบรารี Aspose.3D ที่เพิ่มเข้าในโครงการของคุณ ดาวน์โหลดได้จาก [release page](https://releases.aspose.com/3d/java/)  
- ไฟล์ `.lic` ที่ถูกต้องหรือคีย์สาธารณะและส่วนตัวสำหรับการให้ไลเซนส์แบบมิเตอร์

## นำเข้าแพ็กเกจ

เพิ่มการนำเข้าที่จำเป็นในไฟล์ซอร์ส Java ของคุณ ตรวจสอบให้แน่ใจว่า JAR ของ Aspose.3D อยู่ใน classpath

```java
import com.aspose.threed.License;
import com.aspose.threed.Metered;

import java.io.FileInputStream;
import java.io.IOException;
```

## วิธีตั้งค่าไลเซนส์โดยใช้ไฟล์

### ขั้นตอน 1: สร้างอ็อบเจกต์ License

สร้างอินสแตนซ์ของคลาส `License` – อ็อบเจกต์นี้จะเก็บข้อมูลไลเซนส์

```java
License license = new License();
```

### ขั้นตอน 2: ตั้งค่าไลเซนส์จากไฟล์

ระบุเส้นทางแบบสัมพันธ์หรือแบบเต็มไปยังไฟล์ `.lic` ของคุณและเรียกใช้

```java
license.setLicense("Aspose._3D.lic");
```

> **เคล็ดลับ:** เก็บไฟล์ไลเซนส์ไว้ไกลจากไดเรกทอรีที่ควบคุมเวอร์ชันเพื่อป้องกันการเปิดเผยโดยบังเอิญ

## วิธีตั้งค่าไลเซนส์โดยใช้สตรีม

### ขั้นตอน 1: สร้างอ็อบเจกต์ License

เช่นเดียวกับก่อนหน้า เริ่มด้วยอินสแตนซ์ `License` ใหม่

```java
License license = new License();
```

### ขั้นตอน 2: ตั้งค่าไลเซนส์จากสตรีม

อ่านไฟล์ไลเซนส์เข้า `FileInputStream` แล้วส่งสตรีมนี้ให้กับ `setLicense` บล็อก `try‑with‑resources` จะรับประกันว่าสตรีมจะถูกปิดโดยอัตโนมัติ

```java
try (FileInputStream myStream = new FileInputStream("Aspose._3D.lic")) {
    license.setLicense(myStream);
}
```

## วิธีใช้คีย์สาธารณะและส่วนตัวสำหรับการให้ไลเซนส์แบบมิเตอร์

### ขั้นตอน 1: เริ่มต้นอ็อบเจกต์ Metered

สร้างอินสแตนซ์ของคลาส `Metered` ซึ่งจัดการการให้ไลเซนส์แบบมิเตอร์ (จ่ายตามการใช้)

```java
Metered metered = new Metered();
```

### ขั้นตอน 2: ตั้งค่าคีย์สาธารณะและส่วนตัว

ใส่คีย์ที่คุณได้รับจาก Aspose คีย์เหล่านี้ทำให้ไลบรารีสามารถรายงานการใช้งานกลับไปยังเซิร์ฟเวอร์ไลเซนส์ได้

```java
metered.setMeteredKey("your-public-key", "your-private-key");
```

> **คำเตือน:** อย่าเขียนคีย์ส่วนตัวของคุณลงใน JAR ที่เผยแพร่ต่อสาธารณะ พิจารณาโหลดจากตำแหน่งที่ปลอดภัยหรือจากตัวแปรสภาพแวดล้อม

## กรณีการใช้งานทั่วไป

- **สายการเรนเดอร์ 3D ระดับองค์กร** – ปลดล็อก API ประสิทธิภาพสูงหลังจากตั้งค่าไลเซนส์  
- **สภาพแวดล้อมการทดสอบอัตโนมัติ** – ใช้ไลเซนส์ชั่วคราว (ดู FAQ) เพื่อตรวจสอบฟังก์ชันโดยไม่ต้องซื้อไลเซนส์เต็ม  
- **โซลูชัน SaaS แบบมิเตอร์** – ผสานคีย์สาธารณะ/ส่วนตัวเพื่อเรียกเก็บค่าบริการจากลูกค้าตามการใช้งานจริง

## สรุป

ขอแสดงความยินดี! ตอนนี้คุณรู้ **วิธีตั้งค่าไลเซนส์** สำหรับ Aspose.3D ใน Java ด้วยไฟล์หรือสตรีม และ **วิธีใช้คีย์สาธารณะและส่วนตัว** สำหรับการให้ไลเซนส์แบบมิเตอร์ ด้วยขั้นตอนเหล่านี้คุณสามารถผสาน Aspose.3D เข้ากับแอปพลิเคชัน Java ใด ๆ ได้อย่างมั่นใจและใช้ประโยชน์จากความสามารถการประมวลผล 3D อย่างเต็มที่
