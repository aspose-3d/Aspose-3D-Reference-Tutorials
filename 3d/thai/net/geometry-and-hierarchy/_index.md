# Geometry and Hierarchy

## Introduction

ยินดีต้อนรับสู่คู่มือขั้นสุดท้ายสำหรับ **geometric transformation 3d** ด้วย Aspose.3D for .NET ไม่ว่าคุณจะเพิ่งเริ่มต้นหรือสร้างแอปพลิเคชัน 3D มานานหลายปี คอลเลกชันบทเรียนเชิงปฏิบัตินี้จะช่วยให้คุณเชี่ยวชาญทุกอย่างตั้งแต่การใช้วัสดุ PBR ไปจนถึงการจัดการโหนดด้วยเมทริกซ์การแปลง มาดำดิ่งเข้าไปและดูว่าคุณจะเปลี่ยนแนวคิดให้เป็นฉาก 3D ในโลกจริงได้อย่างไร

## Geometric Transformation 3D Overview

ในส่วนนี้เราจะให้ภาพรวมอย่างรวดเร็วของหัวข้อหลักที่ครอบคลุมต่อไป คุณจะได้เรียนรู้วิธี:

* ใช้วัสดุ **physically‑based rendering (PBR)** กับอ็อบเจ็กต์เพื่อให้แสงสว่างเสมือนจริง  
* นำทางและสืบค้นโครงสร้างฉากด้วยไวยากรณ์แบบ **XPath‑like**  
* ต่อและใช้ **quaternions**, **Euler angles**, และ **transformation matrices** กับโหนด  

แต่ละบทเรียนเป็นอิสระ มีโค้ดตัวอย่างและแสดงผลลัพธ์ภาพที่คาดหวัง เพื่อให้คุณทำตามได้ขั้นตอนต่อขั้นตอน

## Applying PBR Material to Box in 3D Scenes

สำรวจพลังของการเรนเดอร์แบบฟิสิกส์โดยเรียนรู้วิธี **apply PBR material** ให้กับกล่องง่าย ๆ บทเรียนนี้จะพาคุณผ่านการสร้างวัสดุ การกำหนดให้กับเรขาคณิต และการเรนเดอร์ฉากด้วยการสะท้อนที่สมจริง  
[Read more](./apply-pbr-material-to-box/)


## XPath-Like Object Queries

นำทางกราฟฉากที่ซับซ้อนได้อย่างง่ายดายด้วยการคิวรีแบบ **XPath‑like** คู่มือนี้จะแสดงวิธีค้นหาโหนด กรองตามประเภท และจัดการอ็อบเจ็กต์โดยไม่ต้องเขียนโค้ดการเดินทางที่ยาวเยื้อ  
[XPath-Like Object Queries Tutorial](./xpath-like-object-queries/)

ปลดล็อกศักยภาพของ Aspose.3D for .NET! ใช้ประโยชน์จากการคิวรีแบบ XPath‑like เพื่อการนำทางและจัดการอ็อบเจ็กต์ในโครงสร้างฉากอย่างง่ายดาย ดาวน์โหลดตอนนี้เพื่อประสบการณ์ที่ปฏิวัติการจัดการกราฟิก 3D อย่างที่ไม่เคยมีมาก่อน

## Transforming Node by Euler Angles in 3D Scenes

## Transforming Node by Quaternion in 3D Scenes
