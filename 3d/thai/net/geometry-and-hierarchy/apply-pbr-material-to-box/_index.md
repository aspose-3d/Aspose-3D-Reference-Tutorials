# วิธีการใช้วัสดุ PBR กับกล่อง

## คำนำ

ยินดีต้อนรับสู่โลกที่น่าตื่นตาตื่นใจของกราฟิก 3 มิติ! ในคู่มือขั้นตอนนี้ คุณจะได้เรียนรู้ **วิธีการใช้วัสดุ pbr** กับกล่องโดยใช้ Aspose.3D for .NET เราจะอธิบายการสร้างวัสดุ PBR การเพิ่มวัสดุลงในเมช และสุดท้าย **การส่งออก STL ASCII** เพื่อให้คุณสามารถใช้โมเดลนี้ในกระบวนการทำงานต่อไป ไม่ว่าคุณจะสร้างต้นแบบเกมหรือการแสดงผลผลิตภัณฑ์ การเชี่ยวชาญกระบวนการนี้จะเปิดประตูสู่การเรนเดอร์แบบฟิสิกส์จริง (PBR) ที่สมจริงในแอปพลิเคชัน .NET ของคุณ

## คำตอบสั้น ๆ
- **เป้าหมายหลักคืออะไร?** ใช้วัสดุ PBR กับกล่องและส่งออกเป็น STL ASCII  
- **ใช้ไลบรารีใด?** Aspose.3D for .NET (how to use aspose)  
- **ต้องมีลิขสิทธิ์หรือไม่?** ใช่ จำเป็นต้องมีลิขสิทธิ์ชั่วคราวหรือเต็มสำหรับการใช้งานจริง  
- **รูปแบบผลลัพธ์ที่รองรับ?** STL ASCII (export stl ascii) และรูปแบบ 3D อื่น ๆ อีกหลายรูปแบบ  
- **ใช้เวลานานแค่ไหน?** ประมาณ 10‑15 นาทีสำหรับการทำงานพื้นฐาน

## วัสดุ PBR คืออะไร?
Physically Based Rendering (PBR) เป็นโมเดลการเชดดิ้งที่จำลองการทำงานของแสงกับพื้นผิวในโลกจริง โดยการปรับค่าคุณสมบัติต่าง ๆ เช่น ปัจจัย metallic และ roughness คุณสามารถสร้างผลลัพธ์ที่สมจริงอย่างมากโดยไม่ต้องปรับเชดเดอร์ที่ซับซ้อนด้วยตนเอง

## ทำไมต้องใช้ Physically Based Rendering (PBR)?
- **ความสมจริง:** วัสดุตอบสนองต่อแสงในลักษณะที่สอดคล้องกับฟิสิกส์จริง  
- **ความสอดคล้อง:** วัสดุเดียวกันจะดูถูกต้องภายใต้สภาพแสงใด ๆ ก็ตาม  
- **ประสิทธิภาพ:** GPU สมัยใหม่ได้รับการปรับให้ทำงานกับการคำนวณ PBR อย่างมีประสิทธิภาพ ทำให้คุณได้ประสิทธิภาพโดยไม่ต้องเสียค่าใช้จ่ายเพิ่มเติม

## นำเข้า Namespaces
ก่อนอื่น ให้แน่ใจว่าได้นำเข้า namespaces ที่จำเป็นเพื่อใช้ความสามารถของ Aspose.3D for .NET:

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Shading;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
```

## ขั้นตอนที่ 1: เริ่มต้น Scene
เริ่มต้นด้วยการสร้างฉาก 3 มิติโดยใช้โค้ดตัวอย่างต่อไปนี้:

```csharp
Scene scene = new Scene();
```

## ขั้นตอนที่ 2: สร้างวัสดุ PBR
ต่อไปเราจะ **สร้างวัสดุ pbr** ที่จะทำให้กล่องของเราดูสมจริง:

```csharp
PbrMaterial mat = new PbrMaterial();
```

## ขั้นตอนที่ 3: ตั้งค่าคุณสมบัติวัสดุ
ปรับแต่งคุณสมบัติวัสดุให้เป็นแบบเกือบเป็นโลหะและมีความหยาบสูง—เหมาะสำหรับกล่องแบบ brushed‑metal:

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

## ขั้นตอนที่ 4: สร้างกล่อง
สร้างกล่องที่จะนำวัสดุ PBR ไปใช้:

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

## ขั้นตอนที่ 5: เพิ่มวัสดุ PBR ให้กับกล่อง
กำหนด **add pbr material** ที่ได้ตั้งค่าไว้ก่อนหน้านี้ให้กับโหนดกล่องที่สร้างขึ้น:

```csharp
boxNode.Material = mat;
```

## ขั้นตอนที่ 6: บันทึก Scene 3D เป็น STL ASCII
สุดท้าย **export stl ascii** เพื่อให้โมเดลสามารถเปิดได้ในโปรแกรมดู 3D หรือ slicer มาตรฐานใด ๆ:

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```

ขอแสดงความยินดี! คุณได้ทำการใช้วัสดุ PBR กับกล่องในฉาก 3 มิติสำเร็จโดยใช้ Aspose.3D for .NET

## ข้อผิดพลาดที่พบบ่อยและเคล็ดลับ
- **ไม่พบลิขสิทธิ์:** ตรวจสอบให้แน่ใจว่าไฟล์ลิขสิทธิ์ถูกโหลดก่อนการเรียกใช้ Aspose ใด ๆ มิฉะนั้นไลบรารีจะทำงานในโหมดประเมินผล  
- **เส้นทางไฟล์ไม่ถูกต้อง:** ใช้ `Path.Combine` เพื่อหลีกเลี่ยงการขาดเครื่องหมายแยกเส้นทางบนระบบปฏิบัติการต่าง ๆ  
- **Roughness กับ Metallic:** การตั้งค่าปัจจัยทั้งสองสูงเกินไปอาจทำให้พื้นผิวดูแบนราบ ทดลองปรับค่าระหว่าง 0.5‑0.9 เพื่อให้ได้ลุคที่สมดุล
