# วิธีใช้วัสดุ PBR กับกล่อง

## บทนำ

ยินดีต้อนรับสู่โลกที่น่าตื่นเต้นของกราฟิก 3 มิติ! ในบทแนะนำแบบขั้นตอนนี้, **คุณจะได้เรียนรู้วิธีใช้วัสดุ pbr** กับกล่องโดยใช้ Aspose.3D for .NET. เราจะอธิบายการสร้างวัสดุ PBR, การเพิ่มลงในเมช, และสุดท้าย **การส่งออก STL ASCII** เพื่อให้คุณสามารถใช้โมเดลในกระบวนการต่อไปได้. ไม่ว่าคุณจะกำลังสร้างต้นแบบเกม, ตัวแสดงผลผลิตภัณฑ์, หรือต้นแบบเร็วสำหรับการพิมพ์ 3 มิติ, การเชี่ยวชาญกระบวนการนี้จะเปิดประตูสู่การเรนเดอร์ที่สมจริงและอิงฟิสิกส์ (PBR) ในแอปพลิเคชัน .NET ของคุณ.

## คำตอบด่วน
- **เป้าหมายหลักคืออะไร?** Apply a PBR material to a box and export it as STL ASCII.  
- **ไลบรารีที่ใช้คืออะไร?** Aspose.3D for .NET (วิธีใช้ aspose).  
- **ต้องการใบอนุญาตหรือไม่?** ใช่, จำเป็นต้องมีใบอนุญาตชั่วคราวหรือเต็มสำหรับการใช้งานจริง.  
- **รูปแบบเอาต์พุตที่รองรับ?** STL ASCII (export stl ascii) และรูปแบบ 3 มิติอื่น ๆ อีกหลายรูปแบบ.  
- **ใช้เวลานานเท่าไหร่?** ประมาณ 10‑15 นาทีสำหรับการทำงานพื้นฐาน.

## วัสดุ PBR คืออะไร?
Physically Based Rendering (PBR) คือโมเดลการเชดดิ้งที่จำลองการทำงานของแสงกับพื้นผิวในโลกจริง โดยการปรับค่าต่าง ๆ เช่น ปัจจัยโลหะและความหยาบ คุณสามารถสร้างผลลัพธ์ที่สมจริงโดยไม่ต้องปรับเชดเดอร์ซับซ้อนด้วยตนเอง.

## ทำไมต้องใช้ Physically Based Rendering (PBR)?
- **ความสมจริง:** วัสดุตอบสนองต่อแสงในลักษณะที่สอดคล้องกับฟิสิกส์จริง.  
- **ความสอดคล้อง:** วัสดุเดียวกันดูถูกต้องภายใต้สภาพแสงใด ๆ ก็ตาม.  
- **ประสิทธิภาพ:** GPU สมัยใหม่ได้รับการปรับให้เหมาะกับการคำนวณ PBR ทำให้คุณได้ประสิทธิภาพโดยไม่ต้องเพิ่มค่าใช้จ่าย.

## คู่มือขั้นตอนต่อขั้นตอน

### ขั้นตอน 1: เริ่มต้น Scene
เริ่มต้นด้วยการสร้าง Scene 3 มิติเปล่า คอนเทนเนอร์นี้จะเก็บเรขาคณิต, วัสดุ, และแสงทั้งหมดที่คุณจะเพิ่มในภายหลัง.

```csharp
Scene scene = new Scene();
```

### ขั้นตอน 2: สร้างวัสดุ PBR
ตอนนี้เราจะ **สร้างวัสดุ pbr** ที่จะทำให้กล่องของเราดูสมจริง คลาส `PbrMaterial` เปิดเผยพารามิเตอร์ทั้งหมดที่คุณต้องการสำหรับ Physically Based Rendering.

```csharp
PbrMaterial mat = new PbrMaterial();
```

### ขั้นตอน 3: ตั้งค่าคุณสมบัติวัสดุ
ปรับจูนคุณสมบัติวัสดุอย่างละเอียด ในตัวอย่างนี้เราทำให้พื้นผิวเกือบเป็นโลหะและมีความหยาบสูง—เหมาะสำหรับกล่องแบบ brushed‑metal.

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

### ขั้นตอน 4: สร้างเมชกล่อง
สร้างเรขาคณิตกล่องแบบง่าย นี่คือขั้นตอน **create box mesh** ที่คีย์เวิร์ดหลักอ้างอิง.

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

### ขั้นตอน 5: ใช้วัสดุ PBR กับกล่อง
กำหนด **add pbr material** ที่กำหนดค่าก่อนหน้านี้ให้กับโหนดกล่องที่เราสร้างขึ้น.

```csharp
boxNode.Material = mat;
```

### ขั้นตอน 6: ส่งออก STL ASCII (วิธีส่งออก STL)
สุดท้าย, **export stl ascii** เพื่อให้โมเดลสามารถเปิดได้ในโปรแกรมดู 3 มิติหรือ slicer มาตรฐานใด ๆ การใช้ `FileFormat.STLASCII` รับประกันไฟล์ที่มนุษย์อ่านได้.

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```

## ข้อผิดพลาดทั่วไปและเคล็ดลับ
- **ไม่พบใบอนุญาต:** ตรวจสอบให้แน่ใจว่าไฟล์ใบอนุญาตถูกโหลด *ก่อน* การเรียกใช้ Aspose ใด ๆ; หากไม่เป็นเช่นนั้น ไลบรารีจะทำงานในโหมดประเมินผล.  
- **เส้นทางไฟล์ไม่ถูกต้อง:** ใช้ `Path.Combine` เพื่อหลีกเลี่ยงการขาดเครื่องหมายแยกเส้นทางบนระบบปฏิบัติการต่าง ๆ.  
- **ความสมดุลระหว่าง Roughness กับ Metallic:** การตั้งค่าปัจจัยทั้งสองสูงเกินไปอาจทำให้พื้นผิวดูแบน; ทดลองค่าระหว่าง `0.5‑0.9` เพื่อให้ได้ลักษณะที่สมดุล.  
- **เคล็ดลับประสิทธิภาพ:** ใช้ `PbrMaterial` ตัวเดียวซ้ำหากต้องการใช้วัสดุเดียวกันกับหลายเมช; นี้จะลดภาระหน่วยความจำ.
