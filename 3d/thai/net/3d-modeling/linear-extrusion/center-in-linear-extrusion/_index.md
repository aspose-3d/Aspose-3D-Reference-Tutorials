# เพิ่มพื้นราบและศูนย์กลางในการดันเชิงเส้น

## บทนำ

ยินดีต้อนรับสู่คู่มือฉบับสมบูรณ์นี้สำหรับการเชี่ยวชาญการดันเชิงเส้นโดยใช้ Aspose.3D for .NET หากคุณกำลังมองหา **การเพิ่มพื้นราบ** ให้กับโมเดลของคุณและ **การส่งออกไฟล์ Wavefront OBJ** พร้อมกับการทำให้การดันอยู่ตรงกลาง คุณมาถูกที่แล้ว ในบทเรียนนี้เราจะเจาะลึกเทคนิคการดันเชิงเส้น โดยเน้นที่การตั้งศูนย์กลางและวิธีที่พื้นราบช่วยให้คุณมองเห็นผลลัพธ์ได้ชัดเจนยิ่งขึ้น

## คำตอบสั้น ๆ
- **การ “เพิ่มพื้นราบ” ทำอะไรได้บ้าง?** มันให้จุดอ้างอิงภาพที่ทำให้คุณเห็นได้ง่ายว่าการดันอยู่บนระนาบ X‑Z ที่ไหน  
- **ใช้รูปแบบใดในการส่งออกโมเดล?** ฉากจะถูกบันทึกเป็นไฟล์ Wavefront OBJ (`FileFormat.WavefrontOBJ`)  
- **ต้องมีลิขสิทธิ์เพื่อรันโค้ดหรือไม่?** เวอร์ชันทดลองฟรีใช้ได้สำหรับการพัฒนา; ต้องมีลิขสิทธิ์ถาวรสำหรับการใช้งานในผลิตภัณฑ์  
- **สามารถเปลี่ยนความยาวของการดันได้หรือไม่?** ได้ – ปรับพารามิเตอร์ที่สองของ `LinearExtrusion`  
- **การตั้งศูนย์กลางเป็นตัวเลือกหรือไม่?** การตั้งค่า `Center = true` จะทำให้การดันอยู่ตรงกลางของโปรไฟล์; `false` จะจัดตำแหน่งไปทางด้านหนึ่ง

## ข้อกำหนดเบื้องต้น

ก่อนที่เราจะเริ่มการเดินทางที่น่าตื่นเต้นนี้ โปรดตรวจสอบว่าคุณมีข้อกำหนดต่อไปนี้พร้อมใช้งานแล้ว:

- ความเข้าใจพื้นฐานเกี่ยวกับภาษาโปรแกรม C#  
- มี Visual Studio ติดตั้งบนเครื่องของคุณแล้ว  
- ไลบรารี Aspose.3D for .NET ซึ่งคุณสามารถดาวน์โหลดได้จาก [Aspose.3D .NET Documentation](https://reference.aspose.com/3d/net/)  
- ตรวจสอบให้แน่ใจว่าคุณสามารถเข้าถึง [Aspose.3D .NET Documentation](https://reference.aspose.com/3d/net/) เพื่ออ้างอิงตลอดการทำบทเรียน

## นำเข้า Namespaces

เพื่อเริ่มต้น เรามานำเข้า namespaces ที่จำเป็นกันเถอะ ซึ่งจะเป็นพื้นฐานสำหรับผลงานการสร้างโมเดล 3 มิติของเรา

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

## ขั้นตอนที่ 1: เริ่มต้นโปรไฟล์ฐาน

เราจะกำหนดโปรไฟล์สี่เหลี่ยมผืนผ้าที่จะถูกดันออก `RoundingRadius` จะเพิ่มฟิลเลตเล็ก ๆ ที่มุม

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

## ขั้นตอนที่ 2: สร้าง Scene 3 มิติ

อ็อบเจกต์ `Scene` ทำหน้าที่เป็นคอนเทนเนอร์สำหรับเรขาคณิต, แสง, และกล้องทั้งหมด

```csharp
Scene scene = new Scene();
```

## ขั้นตอนที่ 3: สร้างโหนดซ้ายและขวา

เราจะสร้างโหนดพี่น้องสองโหนดภายใต้รูท โหนดหนึ่งจะแสดงการดัน **โดยไม่มี** การตั้งศูนย์กลาง อีกโหนดหนึ่ง **มี** การตั้งศูนย์กลาง เราจะย้ายโหนดซ้ายเพื่อไม่ให้ตัวอย่างสองตัวทับกัน

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(5, 0, 0);
```

## ขั้นตอนที่ 4: ทำ Linear Extrusion บนโหนดซ้าย (ไม่มีศูนย์กลาง)

ที่นี่เราจะดันโปรไฟล์โดยไม่ตั้งศูนย์กลาง โปรดสังเกตแฟล็ก `Center = false`

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 2) { Center = false, Slices = 3 });
```

## ขั้นตอนที่ 5: เพิ่มพื้นราบเพื่ออ้างอิง (โหนดซ้าย)

การเพิ่มกล่องบาง ๆ ทำหน้าที่เป็น **พื้นราบ** เพื่อให้คุณเห็นได้ชัดว่าการดันอยู่บนฐานอย่างไร

```csharp
left.CreateChildNode(new Box(0.01, 3, 3));
```

## ขั้นตอนที่ 6: ทำ Linear Extrusion บนโหนดขวา (มีศูนย์กลาง)

ตอนนี้เราจะดันโปรไฟล์เดียวกันแต่เปิดการตั้งศูนย์กลาง เราจะได้เรขาคณิตที่วางสมมาตรรอบจุดกำเนิดของโปรไฟล์

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 2) { Center = true, Slices = 3 });
```

## ขั้นตอนที่ 7: เพิ่มพื้นราบเพื่ออ้างอิง (โหนดขวา)

เราจะเพิ่มพื้นราบที่สองในโหนดขวาเพื่อให้การเปรียบเทียบภาพเป็นไปอย่างสอดคล้อง

```csharp
right.CreateChildNode(new Box(0.01, 3, 3));
```

## ขั้นตอนที่ 8: ส่งออกไฟล์ Wavefront OBJ

สุดท้าย เราจะ **ส่งออก Wavefront OBJ** เพื่อให้ผลลัพธ์สามารถเปิดได้ในโปรแกรมดู 3‑D มาตรฐานใด ๆ

```csharp
scene.Save("Your Output Directory" + "CenterInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```

## ทำไมต้องเพิ่มพื้นราบ?

การเพิ่มพื้นราบให้สัญญาณภาพที่ชัดเจนเกี่ยวกับความสูงและการจัดแนวของการดัน มันเป็นประโยชน์โดยเฉพาะเมื่อคุณต้องการเปรียบเทียบผลลัพธ์ที่ตั้งศูนย์กลางกับที่ไม่ได้ตั้งศูนย์กลางตามที่แสดงในบทเรียนนี้

## ปัญหาที่พบบ่อยและเคล็ดลับ

- **พื้นราบไม่แสดง:** ตรวจสอบให้แน่ใจว่าความหนาของพื้น (`0.01` ในคอนสตรัคเตอร์ `Box`) เล็กพอที่จะไม่บังการดัน แต่ใหญ่พอที่จะถูกเรนเดอร์  
- **การส่งออกล้มเหลว:** ตรวจสอบว่าไดเรกทอรีปลายทางมีอยู่และคุณมีสิทธิ์เขียนไฟล์  
- **การดันที่ตั้งศูนย์กลางดูเหมือนเลื่อน:** ตรวจสอบคุณสมบัติ `Center` อีกครั้ง; `true` จะตั้งศูนย์โปรไฟล์, `false` จะจัดตำแหน่งไปทางด้านหนึ่ง
