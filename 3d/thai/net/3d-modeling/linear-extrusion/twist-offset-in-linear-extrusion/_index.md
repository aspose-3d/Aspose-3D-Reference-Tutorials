# วิธีเพิ่มการบิด (Twist) ใน Linear Extrusion ด้วย Aspose.3D สำหรับ .NET

## การแนะนำ

เป็นหลักที่เห็นได้ชัดในขั้นตอน ** วิธีเพิ่มการบิด** ให้กับ linear extrusion คุณมาถูกที่แล้วในบทแนะนำนี้เราจะพาคุณผ่านเฟิร์มแวร์ทั้งหมดด้วย Aspose.3D สำหรับ .NET แสดงให้คุณเห็น ** วิธีใช้ extrusion** การสร้างโครงสร้าง 3D วิจารณ์สภาพการณ์ *asp.net 3d modeling* ก่อนอื่นคุณจะต้องมีตัวอย่างที่พร้อมรันซึ่งสาธิต twist offset, Slices และผลลัพธ์เป็นไฟล์ OBJ

## คำตอบด่วน
- **“twist offset” ส่งอะไร?** HD เลื่อนไปที่การบิดตามแกน extrusion.
- **ต้องมีลิขสิทธิ์เพื่อรันตัวอย่างหรือไม่?** ลิขสิทธิ์ชั่วคราวสำหรับการทดสอบได้; ต้องลิขสิทธิ์เต็มๆเลยจริง.
- ** รองรับการทำงานของ .NET ในเวลา?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+
- **สามารถนับจำนวนสไลซ์ได้หรือไม่** ได้ — ค่า `Slices` สามารถควบคุมความเรียบของเมชได้
- ** รูปแบบผลลัพธ์จำกัดที่ OBJ ในเวลา?** ไม่, Aspose.3D รองรับหลายรูปแบบ; ใช้ OBJ ตรงนี้เพื่อความง่าย

## Twist Offset ในการอัดขึ้นรูปเชิงเส้นคืออะไร?

Twist offset การตั้งค่าปฏิทินเลื่อนเชิงแปลที่อธิบายการบิดและหมุนรอบทิศทางของการอัดขึ้นรูป, เพื่อเป็นแกนหมุนจากออฟเซ็ตที่กำหนด อย่าลืมควบคุมศิลปะของรูปทรงสุดท้ายอีกครั้ง...

## นำเข้าเนมสเปซ

ก่อนอื่นให้ import namespaces ที่ให้คุณเข้าถึงคลาส 3D หลัก.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

คำสั่งเหล่านี้ทำให้ `Scene`, `LinearExtrusion`, `Vector3` และประเภทสำคัญอื่น ๆ พร้อมใช้งานในโค้ดของคุณ.

## คู่มือทีละขั้นตอน

### ขั้นตอนที่ 1: กำหนดค่าเริ่มต้นของโปรไฟล์พื้นฐาน

เราจะเริ่มด้วยโปรไฟล์สี่เหลี่ยมพื้นฐานและกำหนดรัศมีโค้งเล็กน้อยเพื่อให้ขอบไม่คมเกินไป.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

### ขั้นตอนที่ 2: สร้างฉาก

`Scene` ทำหน้าที่เป็นคอนเทนเนอร์สำหรับโหนด, แสง, กล้อง, และเรขาคณิตทั้งหมด.

```csharp
Scene scene = new Scene();
```

### ขั้นตอนที่ 3: สร้างโหนด

เพิ่มโหนดลูกสองโหนดลงในรูทของซีน — หนึ่งโหนดสำหรับ extrusion ปกติและอีกหนึ่งโหนดสำหรับเวอร์ชันที่บิด. โหนดด้านซ้ายจะถูกเลื่อนบนแกน X เพื่อแยกการมองเห็น.

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(18, 0, 0);
```

### ขั้นตอนที่ 4: การอัดขึ้นรูปเชิงเส้นบนโหนดด้านซ้าย (ไม่มีการชดเชยการบิด)

ที่นี่เราจะแสดงการ extrusion พื้นฐานพร้อมการบิด 360° เต็มรอบและ 100 slices เพื่อความเรียบ.

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100 });
```

### ขั้นตอนที่ 5: การอัดขึ้นรูปเชิงเส้นบนโหนดด้านขวาโดยมีการชดเชยการบิด

ตอนนี้เราจะใช้ twist offset ที่ `(3, 0, 0)`. ค่าดังกล่าวจะย้ายจุดเริ่มต้นของการบิดสามหน่วยตามแกน X, ทำให้เกิดเฮลิกซ์ที่เลื่อนตำแหน่งชัดเจน.

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100, TwistOffset = new Vector3(3, 0, 0) });
```

### ขั้นตอนที่ 6: บันทึกฉาก 3 มิติ

สุดท้ายเราจะบันทึกซีนเป็นไฟล์ OBJ. ปรับเส้นทางออกตามความต้องการของสภาพแวดล้อมของคุณ.

```csharp
scene.Save("Your Output Directory" + "TwistOffsetInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```
