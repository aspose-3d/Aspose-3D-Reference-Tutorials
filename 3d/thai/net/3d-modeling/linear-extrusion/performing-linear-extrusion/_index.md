# การทำ Linear Extrusion

## บทนำ:

เริ่มต้นการเดินทางที่น่าตื่นเต้นสู่โลกของกราฟิก 3 มิติด้วย Aspose.3D for .NET ซึ่งเป็นเครื่องมือที่ยกระดับการพัฒนาของคุณ ในบทเรียนนี้ **คุณจะได้เรียนรู้วิธีสร้าง extrusion** – เทคนิคที่น่าสนใจซึ่งเปลี่ยนโปรไฟล์ 2‑D ให้กลายเป็นเมช 3‑D เต็มรูปแบบ เราจะเดินผ่านทุกขั้นตอน ตั้งแต่การติดตั้ง Aspose.3D จนถึงการส่งออกผลลัพธ์เป็นไฟล์ Wavefront OBJ เพื่อให้คุณ **สร้าง 3D จาก 2D** ได้อย่างมั่นใจ

## คำตอบอย่างรวดเร็ว
- **Linear extrusion คืออะไร?** การขยายรูปทรง 2‑D ไปตามเส้นทางตรงเพื่อสร้างวัตถุ 3‑D  
- **ไลบรารีที่ใช้ในบทเรียนนี้คืออะไร?** Aspose.3D for .NET  
- **จะบันทึกเป็น OBJ อย่างไร?** ใช้ `scene.Save(..., FileFormat.WavefrontOBJ)`  
- **สามารถส่งออก Wavefront OBJ ได้หรือไม่?** ได้ – ฟอร์แมตนี้รองรับเต็มรูปแบบ  
- **ต้องมีลิขสิทธิ์หรือไม่?** ใบอนุญาตชั่วคราวเพียงพอสำหรับการทดสอบ; ต้องมีใบอนุญาตเชิงพาณิชย์สำหรับการใช้งานจริง

## การทำ Linear Extrusion:

เราจะสร้างวัตถุ 3D ที่น่าตื่นตาตื่นใจผ่าน Linear Extrusion ด้วย Aspose.3D ทำตามขั้นตอนต่อไปนี้:

### วิธีสร้าง Extrusion – ขั้นตอนที่ 1: เริ่มต้น Base Profile
```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

ขั้นตอนนี้ตั้งค่าโปรไฟล์ 2‑D ที่จะเป็นฐานสำหรับผลงาน 3‑D ของเรา ปรับพารามิเตอร์ตามต้องการเพื่อให้ได้รูปทรงและลักษณะที่ต้องการ

### วิธีสร้าง Extrusion – ขั้นตอนที่ 2: Linear Extrusion
```csharp
var extrusion = new LinearExtrusion(profile, 10) { Slices = 100, Center = true, Twist = 360, TwistOffset = new Vector3(10, 0, 0) };
```

ที่นี่ทำการ Linear Extrusion โดยนำโปรไฟล์ 2‑D ไปขยายในมิติที่สาม ทดลองกับพารามิเตอร์เช่น **Slices**, **Twist**, และ **TwistOffset** เพื่อ **generate 3D mesh** ที่ตรงกับความตั้งใจออกแบบของคุณ

### วิธีสร้าง Extrusion – ขั้นตอนที่ 3: สร้าง Scene
```csharp
Scene scene = new Scene();
```

สร้างผืนผ้าใบเปล่า – ฉากที่วัตถุ 3‑D ของคุณจะมีชีวิตขึ้นมา

### วิธีสร้าง Extrusion – ขั้นตอนที่ 4: เพิ่ม Extrusion ลงใน Scene
```csharp
scene.RootNode.CreateChildNode(extrusion);
```

ผลงานที่ถูก Extrude ของคุณถูกเพิ่มเป็นโหนดลูกใน Scene

### วิธีสร้าง Extrusion – ขั้นตอนที่ 5: บันทึก Scene 3D
```csharp
scene.Save(RunExamples.GetOutputFilePath("LinearExtrusion.obj"), FileFormat.WavefrontOBJ);
```

สุดท้าย **how to save OBJ** – เราเก็บผลลัพธ์ในฟอร์แมต Wavefront OBJ ที่เป็นที่นิยม ซึ่งสามารถเปิดได้โดยโปรแกรมแก้ไข 3‑D ส่วนใหญ่

ตอนนี้จงชมผลงาน 3D ของคุณ!

## ทำไมเรื่องนี้ถึงสำคัญ

Linear extrusion เป็นวิธีที่รวดเร็วในการ **สร้าง 3D จาก 2D** ด้วยการสเก็ตช์ เหมาะสำหรับการทำต้นแบบอย่างรวดเร็ว, การสร้างโมเดลสถาปัตยกรรม, หรือการสร้างเมชที่พิมพ์ได้ ด้วยการเชี่ยวชาญเทคนิคนี้ คุณจะเปิดใช้งานเวิร์กโฟลว์ที่ยืดหยุ่น ประหยัดเวลาและลดความจำเป็นในการใช้เครื่องมือโมเดลที่ซับซ้อน

## ข้อผิดพลาดทั่วไปและเคล็ดลับ

- **ค่าการ Twist สูงเกินไป** อาจทำให้เกิดการตัดกันของเมชเอง ควรจำกัด Twist ไว้ต่ำกว่า 720° สำหรับรูปทรงง่ายส่วนใหญ่  
- **Slices ไม่เพียงพอ** อาจทำให้เมชดูเป็นเหลี่ยมเพิ่ม `Slices` เพื่อให้ได้ผลลัพธ์ที่เรียบเนียนขึ้น  
- **อย่าลืมตั้งค่า `Center = true`** หากต้องการให้ extrusion อยู่กึ่งกลางจุดกำเนิดของโปรไฟล์ – จะทำให้การจัดตำแหน่งในภายหลังง่ายขึ้น
