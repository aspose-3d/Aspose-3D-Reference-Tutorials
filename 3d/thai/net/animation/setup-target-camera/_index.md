# เพิ่มกล้องไปยังฉากและตั้งค่าเป้าหมายสำหรับการแอนิเมชัน 3 มิติ

## บทนำ

หากคุณกำลังสร้างแอนิเมชัน 3 มิติ สิ่งแรกที่คุณต้องการคือกล้องที่วางตำแหน่งอย่างเหมาะสม ในบทเรียนนี้คุณจะได้เรียนรู้ **วิธีเพิ่มกล้องไปยังฉาก**, กำหนดเป้าหมายของมัน, และสุดท้าย **ส่งออกฉากเป็น FBX** ด้วย Aspose.3D for .NET เราจะเดินผ่านแต่ละขั้นตอน, อธิบายว่ามันสำคัญอย่างไร, และให้เคล็ดลับที่เป็นประโยชน์เพื่อให้คุณสร้างแอนิเมชัน 3 มิติที่น่าสนใจด้วยความมั่นใจ เมื่อเสร็จสิ้นคุณจะเข้าใจวิธี **วางตำแหน่งกล้องใน 3d** เพื่อการจัดกรอบที่ดีที่สุด

## คำตอบอย่างรวดเร็ว
- **ขั้นตอนแรกในการเพิ่มกล้องคืออะไร?** สร้างอ็อบเจ็กต์ `Scene` ที่จะเป็นโฮสต์ให้กับโหนดกล้อง  
- **รูปแบบไฟล์ใดที่ใช้สำหรับการส่งออกในคู่มือนี้?** FBX (`.fbx`)  
- **ฉันต้องมีลิขสิทธิ์เพื่อรันโค้ดตัวอย่างหรือไม่?** การทดลองใช้ฟรีทำงานสำหรับการประเมิน; จำเป็นต้องมีลิขสิทธิ์เชิงพาณิชย์สำหรับการใช้งานจริง  
- **เวอร์ชัน .NET ใดที่รองรับ?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+  
- **ฉันสามารถเปลี่ยนตำแหน่งกล้องหลังจากสร้างได้หรือไม่?** ได้ – แก้ไข `cameraNode.Transform.Translation` ได้ทุกเวลา  

## **add camera to scene** คืออะไร?
การเพิ่มกล้องไปยังฉากหมายถึงการสร้างเอนทิตี `Camera`, แนบมันเข้ากับโหนดในกราฟของฉาก, และวางตำแหน่งเพื่อให้ “มองไปที่” จุดเป้าหมาย ซึ่งกำหนดมุมมองของผู้ชมเมื่อฉากถูกเรนเดอร์หรือส่งออก

## ทำไมต้องตั้งค่าเป้าหมายของกล้องและส่งออกเป็น FBX?
- **ควบคุมมุมมอง** – การวางตำแหน่งกล้องอย่างแม่นยำช่วยให้คุณจัดกรอบแอนิเมชันได้ตามที่ต้องการ  
- **ความเข้ากันได้** – FBX ได้รับการสนับสนุนอย่างกว้างขวางโดยเอนจินเกม, Maya, Blender และเครื่องมือ 3D อื่น ๆ ทำให้การแชร์ทรัพยากรเป็นเรื่องง่าย  
- **ทรัพยากรที่ใช้ซ้ำได้** – เมื่อบันทึกกล้องและเป้าหมายแล้ว คุณสามารถใช้ฉากนี้ซ้ำในหลายโครงการ  

## กรณีการใช้งานทั่วไป
- **ฉากตัดภาพแบบภาพยนตร์** ในเกมที่กล้องคงที่ตามตัวละคร  
- **การแสดงผลผลิตภัณฑ์** ที่ต้องการมุมมองคงที่เพื่อแสดงโมเดลจากหลายมุม  
- **การพรีวิว** สำหรับภาพยนตร์หรือโครงการ AR/VR ช่วยให้ผู้ออกแบบปรับตำแหน่งกล้องก่อนการเรนเดอร์ขั้นสุดท้าย  

## ข้อกำหนดเบื้องต้น

ก่อนเริ่มทำบทเรียนนี้ โปรดตรวจสอบว่าคุณมีข้อกำหนดต่อไปนี้ครบถ้วน:

- ความรู้พื้นฐานเกี่ยวกับ C# และ .NET framework  
- ไลบรารี Aspose.3D for .NET ติดตั้งแล้ว คุณสามารถดาวน์โหลดได้ [ที่นี่](https://releases.aspose.com/3d/net/)  
- สภาพแวดล้อมการพัฒนาที่พร้อมสำหรับการเขียนโปรแกรม 3D  

## นำเข้า Namespaces

เพื่อเริ่มต้นกระบวนการ ให้นำเข้า Namespaces ที่จำเป็นเข้าสู่โปรเจกต์ของคุณ Namespaces เหล่านี้เป็นสิ่งสำคัญสำหรับการใช้พลังของ Aspose.3D for .NET:

```csharp
using System;
using System.IO;
using System.Collections;
using Aspose.ThreeD;
using Aspose.ThreeD.Animation;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Utilities;
```

## คู่มือขั้นตอนโดยละเอียด

### ขั้นตอนที่ 1: เริ่มต้นอ็อบเจ็กต์ Scene

เริ่มต้นด้วยการสร้างอ็อบเจ็กต์ Scene ซึ่งทำหน้าที่เป็นผืนผ้าใบที่แอนิเมชัน 3 มิติของคุณจะเกิดขึ้น

```csharp
// ExStart:SetupTargetAndCamera
// Initialize scene object
Scene scene = new Scene();
```

### ขั้นตอนที่ 2: สร้างโหนดกล้อง

ต่อไป สร้างโหนดลูกที่จะเก็บกล้อง การตั้งชื่อโหนดให้มีความหมายช่วยให้การจัดระเบียบโครงสร้างฉากเป็นระเบียบง่ายขึ้น

```csharp
// Get a child node object
Node cameraNode = scene.RootNode.CreateChildNode("camera", new Camera());
```

### ขั้นตอนที่ 3: กำหนดตำแหน่งกล้อง

ระบุการแปลงตำแหน่ง (translation) สำหรับโหนดกล้อง ซึ่งกำหนดตำแหน่งเริ่มต้นของกล้องในพื้นที่ 3 มิติ ปรับค่า `Vector3` เพื่อ **วางตำแหน่งกล้องใน 3d** ตามที่ต้องการสำหรับช็อตของคุณ

```csharp
// Set camera node translation
cameraNode.Transform.Translation = new Vector3(100, 20, 0);
```

### ขั้นตอนที่ 4: กำหนดเป้าหมายของกล้อง

กล้องต้องมีจุดเป้าหมายเพื่อมองไปที่ เราจะสร้างโหนดลูกอีกอันหนึ่งที่ทำหน้าที่เป็นจุดโฟกัสและกำหนดให้กับคุณสมบัติ `Target` ของกล้อง นี่คือวิธี **ตั้งค่าเป้าหมายกล้อง** เพื่อให้ได้มุมมองที่คงที่

```csharp
cameraNode.GetEntity<Camera>().Target = scene.RootNode.CreateChildNode("target");
```

### ขั้นตอนที่ 5: บันทึกฉากที่กำหนดค่าแล้ว

สุดท้าย ส่งออกฉากเป็นไฟล์ FBX (หรือรูปแบบที่รองรับอื่น) แทนที่ `"Your Output Directory"` ด้วยเส้นทางจริงที่คุณต้องการบันทึกไฟล์

```csharp
var output = "Your Output Directory" + "camera-test.fbx";
scene.Save(output);
```

## ปัญหาทั่วไปและวิธีแก้

| ปัญหา | วิธีแก้ |
|-------|----------|
| **กล้องปรากฏที่มุมผิด** | ตรวจสอบว่าโหนดเป้าหมายตั้งอยู่ในตำแหน่งที่คุณคาดหวัง คุณยังสามารถตั้งค่า `cameraNode.GetEntity<Camera>().UpVector` เพื่อควบคุมการวางแนวได้ |
| **FBX ที่ส่งออกไม่เปิดในเครื่องมืออื่น** | ตรวจสอบว่าคุณใช้เวอร์ชันล่าสุดของ Aspose.3D (ไลบรารีจะเขียนไฟล์ FBX ที่เข้ากันได้โดยอัตโนมัติ) |
| **เกิดข้อผิดพลาด Path not found บน `scene.Save`** | ใช้เส้นทางแบบ absolute หรือให้แน่ใจว่าไดเรกทอรีปลายทางมีอยู่ก่อนเรียก `Save` |
