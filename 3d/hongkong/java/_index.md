## 常見問題

**Q: 我可以在商業 Java 應用程式中使用 Aspose.3D 嗎？**  
A: 可以。商業授權移除所有評估限制，並授予完整的部署權限。

**Q: Aspose.3D 支援哪些檔案格式的匯入/匯出？**  
A: OBJ、STL、FBX、GLTF/GLB、3MF、PLY、VRML 等。此函式庫亦支援這些格式之間的相互轉換。

**Q: 如何啟用硬體加速渲染？**  
A: Aspose.3D 內部處理渲染；若需即時 GPU 加速，可將模型匯出為相容於外部引擎的格式（例如 GLTF），再以 OpenGL/DirectX 進行渲染。

**Q: 減少網格大小以供網路傳輸的最佳方法是什麼？**  
A: 使用「Optimizing and Working with 3D Mesh Data」教學中提供的 Draco 壓縮工具，壓縮網格而不會有視覺損失。

**Q: 有關於沿樣條路徑動畫化相機的範例嗎？**  
A: 有。「Working with Animations in Java」教學包含完整的關鍵影格相機動畫範例。
