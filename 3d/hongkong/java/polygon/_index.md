## 常見問題

**Q: 我可以使用 Aspose.3D 為 Unity 等即時引擎建立 UV 映射嗎？**  
A: 可以。將含 UV 的網格匯出為 Unity 支援的格式（例如 FBX 或 glTF），然後直接匯入。

**Q: 三角形轉換會影響原始模型的層級結構嗎？**  
A: 轉換會產生一個包含三角形的新網格，同時保留原始節點層級，因而變換保持不變。

**Q: 如果我的模型已經有 UV 呢？**  
A: Aspose.3D 只會在您明確呼叫 UV 產生方法時覆寫既有的 UV 通道；否則保持不變。

**Q: 在執行時產生 UV 會有效能損失嗎？**  
A: 建議在資產前處理階段一次性產生 UV。雖然執行時產生是可行的，但對大型網格可能會增加額外負擔。

**Q: 哪些檔案格式會保留產生的 UV 座標？**  
A: OBJ、FBX、glTF 以及 STL（使用擴充 STL 格式時）皆會保留 Aspose.3D 所寫入的 UV 資料。
