**問：使用 Aspose.3D for Java 是否需要顯示卡？**  
答：不需要。Aspose.3D 所有計算皆在 CPU 上執行，因而可在任何能執行 Java 的機器上運作。

**問：我可以將 PBR 材質與自訂著色器結合嗎？**  
答：可以。您可以在使用 Aspose.3D 的 PBR 工作流程時，將自訂著色器程式附加至 mesh。

**問：「how to concatenate quaternions」如何提升動畫？**
