## 常見問題

**Q: 使用 Aspose.3D for Java 是否需要顯示卡？**  
A: 不需要。Aspose.3D 所有計算皆在 CPU 上執行，因而可在任何能執行 Java 的機器上運行。

**Q: 我可以將 PBR 材質與自訂著色器結合嗎？**  
A: 可以。您可以將自訂著色器程式附加到網格，同時使用 Aspose.3D 的 PBR 工作流程。

**Q: 「如何串接四元數」如何提升動畫？**  
A: 串接四元數可將多個旋轉合併為單一平滑的變換，避免萬向鎖問題。

**Q: 是否支援匯出為 glTF 或 OBJ？**  
A: Aspose.3D 可將場景匯出為 glTF、OBJ、FBX 以及其他多種常見 3D 格式。

**Q: 我可以在哪裡找到範例專案？**  
A: Aspose.3D 的 GitHub 倉庫與官方文件網站提供即時可執行的範例，涵蓋上述所有教學。
