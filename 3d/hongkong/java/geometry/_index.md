## 常見問題

**Q: 使用 Aspose.3D for Java 是否需要顯示卡？**  
A: 不需要。Aspose.3D 所有計算皆在 CPU 上執行，任何能執行 Java 的機器皆可使用。

**Q: 我可以將 PBR 材質與自訂 shader 結合嗎？**  
A: 可以。你可以在保持 Aspose.3D PBR 工作流程的同時，為網格附加自訂 shader 程式。

**Q: 「how to concatenate quaternions」如何提升動畫效果？**  
A: 串接四元數可將多個旋轉合併為單一平滑的變換，避免萬向鎖的問題。

**Q: 是否支援匯出至 glTF 或 OBJ？**  
A: Aspose.3D 可將場景匯出為 glTF、OBJ、FBX 以及其他多種常見 3D 格式。

**Q: 我可以在哪裡找到範例專案？**  
A: Aspose.3D 的 GitHub 倉庫與官方文件網站提供所有上述教學的即用範例。
