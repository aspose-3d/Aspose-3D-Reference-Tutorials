## 常見問題

**Q: 我可以在同一個流程中結合 Draco 壓縮與 mesh‑data 產生嗎？**  
A: 可以。一般的工作流程是先產生缺失的資料（normals、tangents、binormals），然後對已豐富的 mesh 套用 Draco 壓縮。

**Q: 縮減 3D 檔案大小會影響執行時效能嗎？**  
A: 縮減檔案大小通常會提升載入速度並降低記憶體使用量。若再搭配適當的 mesh 組織（例如材質分割），亦可減少 draw call 數量，提升執行時效能。

**Q: 使用 Draco 壓縮 mesh 的大小有任何限制嗎？**  
A: Draco 能處理非常大的 mesh，但極高多邊形模型可能需要調整壓縮設定（例如量化位元）以在大小與品質之間取得平衡。

**Q: 解壓縮 Draco mesh 後需要重新產生 tangents 嗎？**  
A: 不需要。如果在壓縮前已產生 tangents，Draco 會保留這些頂點屬性。

**Q: 正式上線需要商業授權嗎？**  
A: 需要。雖然免費試用可讓你探索功能，但正式部署時必須擁有有效的 Aspose.3D 授權。
