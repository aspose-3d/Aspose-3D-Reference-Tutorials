## 常見問題

**問：Aspose.3D 底層使用哪個渲染引擎？**  
A：Aspose.3D 採用高效能、硬體加速的引擎，跨平台運作且不需額外的原生相依性。

**問：我可以在同一場景中結合多個渲染目標嗎？**  
A：可以，透過設定不同的 `RenderOptions`，即可將同一場景渲染至多個目標（例如視窗與 `BufferedImage`）。

**問：能否將渲染的場景匯出為 PNG/JPEG 以外的格式？**  
A：當然可以。Aspose.3D 支援 BMP、TIFF，甚至原始像素緩衝區，以供自訂後處理。

**問：如何達到即時的幀率？**  
A：使用 SWT 或 JavaFX 整合，啟用 GPU 加速，並保持場景複雜度低。每幀僅更新變更的物件也有助於提升效能。

**問：渲染至影像時需要手動管理記憶體嗎？**  
A：API 會自動釋放資源，但及時釋放大型 `BufferedImage` 物件可提升長時間執行應用程式的效能。
