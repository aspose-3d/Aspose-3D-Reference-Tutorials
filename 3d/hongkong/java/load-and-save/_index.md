## 常見問題

**Q:** *我可以在商業 Java 應用程式中使用 Aspose.3D 嗎？*  
**A:** 可以。正式上線需擁有有效的 Aspose 授權；可使用免費試用版進行評估。

**Q:** *支援哪些 3D 檔案格式的載入與儲存？*  
**A:** Aspose.3D 支援 OBJ、STL、FBX、3DS、glTF、PLY、U3D、X 等多種格式。

**Q:** *如何在不重寫整個場景的情況下將材質升級為 PBR？*  
**A:** 使用 API 中提供的 `MaterialConverter.convertToPbr()` 方法 – 一次呼叫即可更新所有材質。

**Q:** *儲存大型場景時，有沒有方法減少檔案大小？*  
**A:** 有。設定 `SaveOptions`（例如啟用壓縮、降低網格精度）即可最佳化輸出大小。

**Q:** *使用 Aspose.3D 是否需要額外的相依套件？*  
**A:** 只需要 Aspose.3D for Java 的 JAR，其他相依套件皆已內建。
