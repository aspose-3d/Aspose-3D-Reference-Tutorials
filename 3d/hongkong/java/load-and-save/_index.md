## 常見問題

**Q: 如何使用 Aspose.3D 開始一個新的 3D 專案？**  
A: 加入 Aspose.3D 的 Maven/Gradle 相依性，建立 `Scene` 物件，即可開始建構幾何、光源與相機。

**Q: 我可以為特定格式自訂載入流程嗎？**  
A: 可以，使用 `LoadOptions` 指定單位轉換、材質處理，以及是否匯入動畫。

**Q: 升級材質為 PBR 的最佳方法是什麼？**  
A: 在載入後對每個材質呼叫 `Material.convertToPbr()`；API 會自動將貼圖對應至 PBR 工作流程。

**Q: 儲存大型場景時，如何減少檔案大小？**  
A: 啟用網格壓縮、移除未使用的節點，並透過 `SaveOptions` 選擇二進位格式。

**Q: 是否有辦法以程式方式批次轉換多個檔案？**  
A: 遍歷來源目錄，使用 `Scene.load()` 載入每個檔案，然後以目標格式呼叫 `save()`。
