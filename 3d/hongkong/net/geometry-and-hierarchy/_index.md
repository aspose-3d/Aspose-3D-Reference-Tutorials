# 幾何與層級結構

## 介紹

歡迎閱讀使用 Aspose.3D for .NET 的 **geometric transformation 3d** 終極指南。無論您是剛起步，還是已經開發 3D 應用多年，本系列實作教學將協助您從套用 PBR 材質到使用變換矩陣操作節點，全面掌握相關技術。**您將學會如何套用 PBR——具體而言，如何將 pbr 材質套用至您的 3D 模型**。讓我們一起深入，看看如何將概念轉化為真實的 3D 場景。

## 快速解答
- **什麼是 PBR？** 物理基礎渲染 (Physically‑Based Rendering, PBR) 模擬光線對材質的真實回應。  
- **為什麼使用 Aspose.3D 進行 PBR？** 它提供簡潔的 .NET API 來建立材質與渲染。  
- **需要授權嗎？** 提供免費試用版；正式生產環境需購買商業授權。  
- **支援哪些 .NET 版本？** .NET Framework 4.5 以上、.NET Core 3.1 以上、.NET 5/6 以上。  
- **可以將 PBR 與自訂 shader 結合嗎？** 可以，您可以使用自訂 shader 程式碼擴充材質。

## 如何在 Aspose.3D 中套用 PBR 材質
套用 PBR 材質是建立寫實 3D 場景的基礎。本節將逐步說明關鍵步驟——建立材質、指派至幾何體、以及渲染結果。相同的作法在以下多個教學中皆會重複使用，掌握後能加速您探索其他範例。

### 為什麼使用 PBR？
- **寫實度：** PBR 模擬光線在現實世界中與表面的互動。  
- **一致性：** 材質在任何光照條件下皆能正確呈現。  
- **效能：** 經過最佳化的 shader 使 PBR 適用於互動式應用。

### 前置條件
- 已安裝 Aspose.3D for .NET（最新版本）。  
- .NET 開發環境（如 Visual Studio 2022 或更新版本）。

### 與其他主題的關聯
一旦您了解 **how to apply pbr**，即可輕鬆結合其他變換，如 **euler angle rotation**、四元數混合或直接矩陣操作——這些皆在後續教學中說明。

## 常見問題

**Q: 如何開始使用 PBR 材質的新專案？**  
A: 安裝 Aspose.3D NuGet 套件，建立 `Scene`，並依循「Applying PBR Material to Box」教學加入第一個 PBR 材質。

**Q: 這些教學能在 .NET Core 上使用嗎？**  
A: 可以，所有範例皆支援 .NET Core 3.1 及以上，亦支援 .NET 5/6。

**Q: 查詢節點層級的最佳方式是什麼？**  
A: 使用「XPath‑Like Object Queries」教學中示範的類 XPath 查詢語法——這是最簡潔的 **how to query hierarchy** 方法。

**Q: 我的立方體光照不正確——應該檢查什麼？**  
A: 確認法線已正確設定（參見「Setting Up Normals on Cube」），且 PBR 材質包含正確的金屬度與粗糙度值。

**Q: 使用大量變換矩陣時有性能考量嗎？**  
A: 盡可能批次處理變換並重複使用矩陣物件以減少分配；「Transforming Node by Transformation Matrix」指南提供最佳實踐建議。
