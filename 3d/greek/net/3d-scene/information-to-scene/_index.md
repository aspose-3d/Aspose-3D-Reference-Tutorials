# Πώς να Προσθέσετε Πληροφορίες Προμηθευτή και να Αποθηκεύσετε Σκηνή FBX Χρησιμοποιώντας το Aspose.3D

## Εισαγωγή

Καλώς ήρθατε σε αυτό το ολοκληρωμένο tutorial που δείχνει **πώς να προσθέσετε προμηθευτή** σε μια 3D σκηνή και στη συνέχεια **πώς να αποθηκεύσετε αρχεία FBX** με το Aspose.3D για .NET. Είτε δημιουργείτε αρχιτεκτονικές απεικονίσεις, περιουσιακά στοιχεία παιχνιδιών ή μοντέλα μηχανικής, η ενσωμάτωση μεταδεδομένων προμηθευτή και εφαρμογής κάνει τις σκηνές σας πιο ενημερωτικές και πιο εύκολες στη διαχείριση σε επόμενα στάδια. Ας περάσουμε βήμα-βήμα τη διαδικασία.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “add vendor”;** Αποθηκεύει τα ονόματα της εφαρμογής και του προμηθευτή μέσα στο μπλοκ AssetInfo της σκηνής.  
- **Ποια μορφή υποστηρίζει πληροφορίες προμηθευτή;** Το FBX (ASCII ή binary) διατηρεί τα μεταδεδομένα όταν αποθηκεύεται.  
- **Πώς να αποθηκεύσετε FBX;** Χρησιμοποιήστε `scene.Save(path, FileFormat.FBX7500ASCII)` ή το ισοδύναμο binary.  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να αλλάξω μονάδες μέτρησης;** Ναι, ορίστε `AssetInfo.UnitName` και `AssetInfo.UnitScaleFactor`.

## Τι σημαίνει “how to add vendor” σε σκηνή 3D;
Η προσθήκη πληροφοριών προμηθευτή σημαίνει τη συμπλήρωση των ιδιοτήτων `AssetInfo` ενός αντικειμένου `Scene`. Αυτές οι ιδιότητες μεταφέρονται μαζί με το αρχείο, επιτρέποντας σε οποιονδήποτε καταναλωτή του αρχείου FBX να δει ποια εφαρμογή το δημιούργησε και ποιος είναι ο προμηθευτής.

## Γιατί να προσθέσετε πληροφορίες προμηθευτή;
- **Ιχνηλασιμότητα:** Ταυτοποιήστε γρήγορα την πηγή ενός μοντέλου σε μεγάλες γραμμές παραγωγής.  
- **Συμμόρφωση:** Κάποιες βιομηχανίες απαιτούν ρητή σήμανση προμηθευτή για τη διαχείριση περιουσιακών στοιχείων.  
- **Αυτοματοποίηση:** Τα σενάρια μπορούν να φιλτράρουν ή να επεξεργαστούν αρχεία βάσει των μεταδεδομένων προμηθευτή.

## Πώς να Προσθέσετε Πληροφορίες Προμηθευτή

### Βήμα 1: Αρχικοποίηση μιας 3D Σκηνής

```csharp
Scene scene = new Scene();
```

Η δημιουργία μιας νέας `Scene` σας παρέχει έναν καθαρό καμβά για εργασία.

### Βήμα 2: Ορισμός Πληροφοριών Εφαρμογής και Προμηθευτή

```csharp
scene.AssetInfo.ApplicationName = "Egypt";
scene.AssetInfo.ApplicationVendor = "Manualdesk";
```

Εδώ δείχνουμε **πώς να προσθέσετε προμηθευτή** αναθέτοντας κατάλληλες συμβολοσειρές στα `ApplicationName` και `ApplicationVendor`.

### Βήμα 3: Ορισμός Μονάδων Μέτρησης

```csharp
scene.AssetInfo.UnitName = "pole";
scene.AssetInfo.UnitScaleFactor = 0.6;
```

Ο καθορισμός του συστήματος μονάδων εξασφαλίζει ότι όποιος ανοίγει το αρχείο FBX θα ερμηνεύσει σωστά τις διαστάσεις. Σε αυτό το παράδειγμα, ένα “pole” ισούται με 60 cm.

## Πώς να Αποθηκεύσετε Σκηνή FBX

### Βήμα 4: Αποθήκευση της Σκηνής (πώς να αποθηκεύσετε fbx)

```csharp
var output = "Your Output Directory" + "InformationToScene.fbx";
scene.Save(output, FileFormat.FBX7500ASCII);
```

Αυτή η γραμμή δείχνει **πώς να αποθηκεύσετε fbx** χρησιμοποιώντας την ASCII έκδοση του FBX 7.5.0. Αν προτιμάτε binary, αντικαταστήστε το `FBX7500ASCII` με `FBX7500Binary`.

> **Pro tip:** Διατηρήστε την επέκταση αρχείου `.fbx` συνεπή με τη μορφή που επιλέγετε· διαφορετικά ορισμένοι προβολείς μπορεί να ερμηνεύσουν λανθασμένα το περιεχόμενο.

### Βήμα 5: Εμφάνιση Μηνύματος Επιτυχίας

```csharp
Console.WriteLine("\nAsset information added successfully to Scene.\nFile saved at " + output);
```

Ένα φιλικό μήνυμα στην κονσόλα επιβεβαιώνει ότι η σκηνή, με πλήρη μεταδεδομένα προμηθευτή, έχει γραφτεί στο δίσκο.

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Λύση |
|----------|------|
| **Οι πληροφορίες προμηθευτή δεν εμφανίζονται στον προβολέα** | Βεβαιωθείτε ότι αποθηκεύσατε το αρχείο ως **FBX ASCII** ή **Binary**· ορισμένοι παλαιότεροι προβολείς διαβάζουν μόνο μία μορφή. |
| **Η διαδρομή περιέχει κενά** | Τυλίξτε τη διαδρομή σε εισαγωγικά ή χρησιμοποιήστε `Path.Combine` για ασφαλή δημιουργία διαδρομής. |
| **Η κλίμακα μονάδας φαίνεται λανθασμένη** | Ελέγξτε ξανά το `UnitScaleFactor`; είναι ένας πολλαπλασιαστής σε σχέση με τα μέτρα. |
| **Απόρριψη άδειας** | Χρησιμοποιήστε τη δωρεάν δοκιμή για δοκιμές· αποκτήστε πλήρη άδεια για παραγωγικές εκδόσεις. |
