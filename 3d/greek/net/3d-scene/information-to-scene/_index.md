# Πώς να Προσθέσετε Μεταδεδομένα: Εξαγωγή Πληροφοριών σε Πόρους Σκηνής

## Εισαγωγή

Σε αυτό το ολοκληρωμένο tutorial θα ανακαλύψετε **πώς να προσθέσετε μεταδεδομένα** στις 3D σκηνές σας με το Aspose.3D for .NET. Η προσθήκη μεταδεδομένων όπως στοιχεία προμηθευτή, προσαρμοσμένες μονάδες μέτρησης και άλλες πληροφορίες πόρων κάνει τα μοντέλα σας πιο ενημερωτικά, αναζητήσιμα και έτοιμα για downstream pipelines όπως μηχανές παιχνιδιών ή πλατφόρμες AR/VR.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος σκοπός;** Να ενσωματώσετε μεταδεδομένα (πληροφορίες προμηθευτή, μονάδες, προσαρμοσμένες ετικέτες) απευθείας σε μια 3D σκηνή.  
- **Ποια βιβλιοθήκη χρησιμοποιείται;** Aspose.3D for .NET.  
- **Μπορώ να εξάγω το μοντέλο μετά την προσθήκη μεταδεδομένων;** Ναι – μπορείτε **να εξάγετε 3D μοντέλο** αρχεία, π.χ. αποθήκευση ως FBX.  
- **Χρειάζομαι άδεια;** Διατίθεται δωρεάν δοκιμαστική έκδοση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6.

## Τι σημαίνει “πώς να προσθέσετε μεταδεδομένα” σε μια 3D σκηνή;

Η προσθήκη μεταδεδομένων σημαίνει την προσάρτηση περιγραφικών πληροφοριών—όπως το όνομα της εφαρμογής, ο προμηθευτής, οι μονάδες μέτρησης ή προσαρμοσμένες ετικέτες—στο ίδιο το αρχείο σκηνής. Αυτά τα δεδομένα μεταφέρονται μαζί με το μοντέλο όταν **αποθηκεύετε τη σκηνή ως FBX** ή άλλες υποστηριζόμενες μορφές, επιτρέποντας στα downstream εργαλεία να κατανοήσουν το πλαίσιο χωρίς εξωτερικά αρχεία.

## Γιατί να ενσωματώσετε προσαρμοσμένα μεταδεδομένα και πληροφορίες προμηθευτή;

- **Αναζητησιμότητα:** Τα συστήματα διαχείρισης πόρων μπορούν να φιλτράρουν μοντέλα κατά προμηθευτή ή τύπο μονάδας.  
- **Διαλειτουργικότητα:** Οι μηχανές που διαβάζουν FBX μπορούν αυτόματα να εφαρμόσουν τη σωστή κλίμακα εάν **ορίσετε μονάδες μέτρησης**.  
- **Branding:** Η συμπερίληψη του ονόματος της εφαρμογής και του προμηθευτή ενισχύει την ιδιοκτησία και τη συμμόρφωση με την άδεια.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

- Το Aspose.3D for .NET εγκατεστημένο. Μπορείτε να το κατεβάσετε από τη σελίδα [Aspose.3D for .NET page](https://releases.aspose.com/3d/net/).

using System;

## Βήμα 1: Αρχικοποίηση μιας 3D Σκηνής

```csharp
Scene scene = new Scene();
```

Δημιουργήστε ένα νέο αντικείμενο `Scene` που θα περιέχει όλη τη γεωμετρία και τις πληροφορίες πόρων.

## Βήμα 2: Ορισμός Εφαρμογής και **Προσθήκη Πληροφοριών Προμηθευτή**

```csharp
scene.AssetInfo.ApplicationName = "Egypt";
scene.AssetInfo.ApplicationVendor = "Manualdesk";
```

Εδώ ενσωματώνουμε το **όνομα της εφαρμογής** και τις **πληροφορίες προμηθευτή**. Αυτό αποτελεί βασικό μέρος του **πώς να προσθέσετε μεταδεδομένα** που ταυτοποιεί την πηγή του μοντέλου.

## Βήμα 3: **Ορισμός Μονάδων Μέτρησης** για Ακριβή Κλιμάκωση

```csharp
scene.AssetInfo.UnitName = "pole";
scene.AssetInfo.UnitScaleFactor = 0.6;
```

Καθορίζοντας `UnitName` και `UnitScaleFactor`, ενημερώνετε τα downstream εργαλεία ότι ένα “pole” ισούται με 0,6 μέτρα (60 cm). Αυτό το βήμα είναι απαραίτητο όταν αργότερα **εξάγετε 3D μοντέλο** αρχεία ώστε να διασφαλιστούν σωστές πραγματικές διαστάσεις.

## Βήμα 4: **Αποθήκευση Σκηνής ως FBX** – **Εξαγωγή 3D Μοντέλου** με Μεταδεδομένα

```csharp
var output = "Your Output Directory" + "InformationToScene.fbx";
scene.Save(output, FileFormat.FBX7500ASCII);
```

Η κλήση `Save` γράφει τη σκηνή σε αρχείο FBX, ενσωματώνοντας όλα τα μεταδεδομένα που προσθέσαμε. Αυτό δείχνει **πώς να προσθέσετε μεταδεδομένα** και στη συνέχεια **να αποθηκεύσετε τη σκηνή ως FBX**, εξάγοντας ουσιαστικά **3D μοντέλο** με πλήρεις πληροφορίες πόρων.

## Βήμα 5: Εμφάνιση Μηνύματος Επιτυχίας

```csharp
Console.WriteLine("\nAsset information added successfully to Scene.\nFile saved at " + output);
```

Ένα φιλικό μήνυμα στην κονσόλα επιβεβαιώνει ότι τα μεταδεδομένα έχουν εγγραφεί και εμφανίζει τη θέση του αρχείου.

## Συχνά Προβλήματα & Συμβουλές

- **Λανθασμένη κλίμακα μονάδας:** Ελέγξτε ξανά ότι το `UnitScaleFactor` ταιριάζει με τη μετατροπή στον πραγματικό κόσμο· διαφορετικά τα εξαγόμενα μοντέλα μπορεί να εμφανιστούν πολύ μεγάλα ή μικρά.  
- **Απουσία πληροφοριών προμηθευτή:** Εάν το `ApplicationVendor` μείνει κενό, ορισμένοι προβολείς θα εμφανίσουν “Unknown”. Θέστε το πάντα όταν είναι δυνατόν.  
- **Σφάλματα διαδρομής αρχείου:** Βεβαιωθείτε ότι ο φάκελος εξόδου υπάρχει· διαφορετικά το `scene.Save` θα ρίξει εξαίρεση.
