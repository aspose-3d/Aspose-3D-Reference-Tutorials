# Εκτέλεση Γραμμικής Εξώθησης

## Εισαγωγή:

Ξεκινήστε ένα συναρπαστικό ταξίδι στον κόσμο των 3D γραφικών με το Aspose.3D for .NET, μια ισχυρή πλατφόρμα που ανεβάζει το επίπεδο της ανάπτυξής σας. Σε αυτό το tutorial, **θα μάθετε πώς να δημιουργήσετε εξώθηση** – μια συναρπαστική τεχνική που μετατρέπει ένα 2‑D προφίλ σε ένα πλήρες 3‑D πλέγμα. Θα περάσουμε από κάθε βήμα, από την εγκατάσταση του Aspose.3D μέχρι την εξαγωγή του αποτελέσματος ως αρχείο Wavefront OBJ, ώστε να **δημιουργήσετε 3D από 2D** σχήματα με σιγουριά.

## Γρήγορες Απαντήσεις
- **Τι είναι η γραμμική εξώθηση;** Επέκταση ενός 2‑D σχήματος κατά μήκος μιας ευθείας διαδρομής για να δημιουργηθεί ένα 3‑D αντικείμενο.  
- **Ποια βιβλιοθήκη χρησιμοποιεί αυτό το tutorial;** Aspose.3D for .NET.  
- **Πώς να αποθηκεύσετε OBJ;** Χρησιμοποιήστε `scene.Save(..., FileFormat.WavefrontOBJ)`.  
- **Μπορώ να εξάγω Wavefront OBJ;** Ναι – η μορφή υποστηρίζεται πλήρως.  
- **Χρειάζομαι άδεια;** Μια προσωρινή άδεια αρκεί για δοκιμές· απαιτείται εμπορική άδεια για παραγωγή.

Τώρα που είστε έτοιμοι, ας βουτήξουμε στη μαγεία του Aspose.3D!

## Εισαγωγή Ονομάτων Χώρων:

Συμπεριλάβετε τα απαραίτητα ονόματα χώρων για να ξεκινήσετε την 3D περιπέτειά σας:

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;

```

Αυτά τα ονόματα χώρων θέτουν τη βάση για το 3D προγραμματιστικό σας ταξίδι, παρέχοντας πρόσβαση στα εργαλεία που χρειάζονται για αδιάλειπτη ενσωμάτωση των λειτουργιών του Aspose.3D.

## Εκτέλεση Γραμμικής Εξώθησης:

Ας δημιουργήσουμε ένα μαγευτικό 3D αντικείμενο μέσω Γραμμικής Εξώθησης χρησιμοποιώντας το Aspose.3D. Ακολουθήστε αυτά τα βήματα:

### Πώς να Δημιουργήσετε Εξώθηση – Βήμα 1: Αρχικοποίηση της Βάσης Προφίλ
```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

Αυτό το βήμα δημιουργεί το 2‑D προφίλ που θα λειτουργήσει ως βάση για το 3‑D αριστούργημά μας. Προσαρμόστε τις παραμέτρους όπως χρειάζεται για να επιτύχετε το επιθυμητό σχήμα και μορφή.

### Πώς να Δημιουργήσετε Εξώθηση – Βήμα 2: Γραμμική Εξώθηση
```csharp
var extrusion = new LinearExtrusion(profile, 10) { Slices = 100, Center = true, Twist = 360, TwistOffset = new Vector3(10, 0, 0) };
```

Εδώ, εκτελείται η Γραμμική Εξώθηση, λαμβάνοντας το 2‑D προφίλ και επεκτείνοντάς το στην τρίτη διάσταση. Πειραματιστείτε με παραμέτρους όπως **Slices**, **Twist**, και **TwistOffset** για να **δημιουργήσετε 3D πλέγμα** παραλλαγές που ταιριάζουν με το σχεδιαστικό σας σκεπτικό.

### Πώς να Δημιουργήσετε Εξώθηση – Βήμα 3: Δημιουργία Σκηνής
```csharp
Scene scene = new Scene();
```

Δημιουργείται ένας κενός καμβάς – μια σκηνή όπου το 3‑D αντικείμενό σας θα ζωντανέψει.

### Πώς να Δημιουργήσετε Εξώθηση – Βήμα 4: Προσθήκη Εξώθησης στη Σκηνή
```csharp
scene.RootNode.CreateChildNode(extrusion);
```

Το εξωθημένο αριστούργημά σας προστίθεται ως παιδικός κόμβος στη σκηνή.

### Πώς να Δημιουργήσετε Εξώθηση – Βήμα 5: Αποθήκευση της 3D Σκηνής
```csharp
scene.Save(RunExamples.GetOutputFilePath("LinearExtrusion.obj"), FileFormat.WavefrontOBJ);
```

Τέλος, **πώς να αποθηκεύσετε OBJ** – αποθηκεύουμε το αποτέλεσμα στη δημοφιλή μορφή Wavefront OBJ, η οποία μπορεί να ανοιχθεί από τους περισσότερους 3‑D επεξεργαστές.

Τώρα, θαυμάστε το 3D θαύμα σας!

## Κοινά προβλήματα & συμβουλές

- **Οι τιμές Twist πολύ υψηλές** μπορούν να προκαλέσουν αυτοδιασταυρώσεις. Κρατήστε το twist κάτω από 720° για τις περισσότερες απλές μορφές.  
- **Ανεπαρκείς slices** μπορεί να δημιουργήσουν εμφάνιση με επίπεδα. Αυξήστε την ιδιότητα `Slices` για πιο ομαλά αποτελέσματα.  
- **Θυμηθείτε να ορίσετε `Center = true`** αν θέλετε η εξώθηση να κεντράρεται γύρω από το αρχικό σημείο του προφίλ – αυτό συχνά απλοποιεί τη θέση αργότερα.
