# Προσθήκη Επιπέδου Εδάφους και Κεντράρισμα στην Γραμμική Εξώθηση

## Εισαγωγή

Καλώς ήρθατε σε αυτόν τον ολοκληρωμένο οδηγό για την εξοικείωση με τη γραμμική εξώθηση χρησιμοποιώντας το Aspose.3D for .NET. Εάν θέλετε να **προσθέσετε επίπεδο εδάφους** στα μοντέλα σας και να **εξάγετε αρχεία Wavefront OBJ** διατηρώντας το εξώθημα κεντραρισμένο, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial, θα εμβαθύνουμε στην τεχνική της γραμμικής εξώθησης, εστιάζοντας ειδικά στο θέμα του κεντραρίσματος και στο πώς ένα επίπεδο εδάφους σας βοηθά να οπτικοποιήσετε το αποτέλεσμα πιο καθαρά.

## Γρήγορες Απαντήσεις
- **Τι επιτυγχάνει η “προσθήκη επιπέδου εδάφους”;** Παρέχει ένα οπτικό σημείο αναφοράς που κάνει εύκολο να δείτε πού βρίσκεται η εξώθηση στο επίπεδο X‑Z.  
- **Ποια μορφή χρησιμοποιείται για την εξαγωγή του μοντέλου;** Η σκηνή αποθηκεύεται ως αρχείο Wavefront OBJ (`FileFormat.WavefrontOBJ`).  
- **Χρειάζεται άδεια για την εκτέλεση του κώδικα;** Μια δωρεάν δοκιμαστική έκδοση λειτουργεί για ανάπτυξη· απαιτείται μόνιμη άδεια για παραγωγή.  
- **Μπορώ να αλλάξω το μήκος της εξώθησης;** Ναι – τροποποιήστε τη δεύτερη παράμετρο του `LinearExtrusion`.  
- **Είναι το κεντράρισμα προαιρετικό;** Ορίζοντας `Center = true` κεντράρει την εξώθηση γύρω από το προφίλ· `false` την ευθυγραμμίζει σε μία πλευρά.

## Προαπαιτούμενα

Πριν ξεκινήσουμε αυτό το συναρπαστικό ταξίδι, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:

- Βασική κατανόηση της γλώσσας προγραμματισμού C#.  
- Εγκατεστημένο το Visual Studio στον υπολογιστή σας.  
- Βιβλιοθήκη Aspose.3D for .NET, την οποία μπορείτε να κατεβάσετε από την [Aspose.3D .NET Documentation](https://reference.aspose.com/3d/net/).  
- Βεβαιωθείτε ότι έχετε πρόσβαση στην [Aspose.3D .NET Documentation](https://reference.aspose.com/3d/net/) για αναφορά καθ' όλη τη διάρκεια του tutorial.

## Εισαγωγή Ονομάτων Χώρου

Για να ξεκινήσουμε, ας εισάγουμε τα απαραίτητα namespaces. Αυτά θα θέσουν τη βάση για το αριστούργημά μας στη 3D μοντελοποίηση.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

## Βήμα 1: Αρχικοποίηση του Βασικού Προφίλ

Ξεκινάμε ορίζοντας ένα ορθογώνιο προφίλ που θα εξωθηθεί. Η `RoundingRadius` προσθέτει μια ήπια στρογγυλοποίηση στις γωνίες.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

## Βήμα 2: Δημιουργία 3D Σκηνής

Ένα αντικείμενο `Scene` λειτουργεί ως δοχείο για όλα τα γεωμετρικά στοιχεία, τα φώτα και τις κάμερες.

```csharp
Scene scene = new Scene();
```

## Βήμα 3: Δημιουργία Αριστερού και Δεξιού Κόμβου

Δύο αδερφικοί κόμβοι δημιουργούνται κάτω από τη ρίζα. Ο ένας θα δείξει την εξώθηση **χωρίς** κεντράρισμα, ο άλλος **με** κεντράρισμα. Επίσης μετατοπίζουμε τον αριστερό κόμβο ώστε τα δύο παραδείγματα να μην επικαλύπτονται.

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(5, 0, 0);
```

## Βήμα 4: Εκτέλεση Γραμμικής Εξώθησης στον Αριστερό Κόμβο (Χωρίς Κέντρο)

Εδώ εξωθούμε το προφίλ χωρίς κεντράρισμα. Παρατηρήστε τη σημαία `Center = false`.

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 2) { Center = false, Slices = 3 });
```

## Βήμα 5: Προσθήκη Επιπέδου Εδάφους για Αναφορά (Αριστερός Κόμβος)

Η προσθήκη ενός λεπτού κουτιού λειτουργεί ως **επίπεδο εδάφους** ώστε να μπορείτε να δείτε καθαρά πώς η εξώθηση τοποθετείται στη βάση.

```csharp
left.CreateChildNode(new Box(0.01, 3, 3));
```

## Βήμα 6: Εκτέλεση Γραμμικής Εξώθησης στον Δεξιό Κόμβο (Με Κέντρο)

Τώρα εξωθούμε το ίδιο προφίλ αλλά ενεργοποιούμε το κεντράρισμα. Η γεωμετρία θα τοποθετηθεί συμμετρικά γύρω από το αρχικό σημείο του προφίλ.

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 2) { Center = true, Slices = 3 });
```

## Βήμα 7: Προσθήκη Επιπέδου Εδάφους για Αναφορά (Δεξιός Κόμβος)

Ένα δεύτερο επίπεδο εδάφους προστίθεται στον δεξιό κόμβο για να διατηρηθεί η οπτική σύγκριση συνεπής.

```csharp
right.CreateChildNode(new Box(0.01, 3, 3));
```

## Βήμα 8: Εξαγωγή Αρχείου Wavefront OBJ

Τέλος, **εξάγουμε Wavefront OBJ** ώστε το αποτέλεσμα να μπορεί να ανοιχθεί σε οποιονδήποτε τυπικό 3‑D προβολέα.

```csharp
scene.Save("Your Output Directory" + "CenterInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```

## Γιατί να Προσθέσετε Επίπεδο Εδάφους;

Η προσθήκη ενός επιπέδου εδάφους σας δίνει άμεσο οπτικό σήμα για το ύψος και την ευθυγράμμιση της εξώθησης. Είναι ιδιαίτερα χρήσιμη όταν χρειάζεται να συγκρίνετε κεντραρισμένα και μη‑κεντραρισμένα αποτελέσματα, όπως φαίνεται σε αυτό το tutorial.

## Συχνά Προβλήματα & Συμβουλές

- **Το επίπεδο εδάφους δεν είναι ορατό:** Βεβαιωθείτε ότι το πάχος του επιπέδου (`0.01` στον κατασκευαστή `Box`) είναι αρκετά μικρό ώστε να μην κρύβει την εξώθηση, αλλά και αρκετά μεγάλο για να αποδοθεί.  
- **Αποτυχία εξαγωγής:** Ελέγξτε ότι ο φάκελος εξόδου υπάρχει και έχετε δικαιώματα εγγραφής.  
- **Η κεντραρισμένη εξώθηση φαίνεται μετατοπισμένη:** Επαληθεύστε την ιδιότητα `Center`; `true` κεντράρει το προφίλ, `false` το ευθυγραμμίζει σε μία πλευρά.
