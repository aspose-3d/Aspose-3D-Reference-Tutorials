# Πώς να προσθέσετε στρίψιμο σε γραμμική εξώθηση χρησιμοποιώντας το Aspose.3D για .NET

## Εισαγωγή

Αν ψάχνετε για έναν σαφή, βήμα‑βήμα οδηγό σχετικά με **πώς να προσθέσετε στρίψιμο** σε μια γραμμική εξώθηση, βρίσκεστε στο σωστό μέρος. Σε αυτό το tutorial θα περάσουμε από τη διαδικασία με το Aspose.3D για .NET, δείχνοντάς σας **πώς να χρησιμοποιήσετε την εξώθηση** για να δημιουργήσετε δυναμικά 3D σχήματα που είναι ιδανικά για σενάρια *asp.net 3d modeling*. Στο τέλος θα έχετε ένα έτοιμο παράδειγμα που επιδεικνύει το offset του στριψίματος, τις φέτες και την αποθήκευση του αποτελέσματος ως αρχείο OBJ.

## Γρήγορες Απαντήσεις
- **Τι κάνει το “twist offset”;** Μετατοπίζει το σημείο εκκίνησης του στριψίματος κατά τον άξονα της εξώθησης.  
- **Χρειάζομαι άδεια για να εκτελέσω το δείγμα;** Μια προσωρινή άδεια λειτουργεί για δοκιμές· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις του .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Μπορώ να αλλάξω τον αριθμό των φετών;** Ναι—προσαρμόστε την ιδιότητα `Slices` για να ελέγξετε την ομαλότητα του πλέγματος.  
- **Είναι η μορφή εξόδου περιορισμένη σε OBJ;** Όχι, το Aspose.3D υποστηρίζει πολλές μορφές· το OBJ χρησιμοποιείται εδώ για απλότητα.

## Τι είναι το Twist Offset σε Γραμμική Εξώθηση;

Ένα twist offset ορίζει μια μεταφορική μετατόπιση που εφαρμόζεται στη λειτουργία στριψίματος. Αντί να περιστρέφεται γύρω από το ακριβές σημείο εκκίνησης της εξώθησης, η γεωμετρία αρχίζει να περιστρέφεται από το καθορισμένο διάνυσμα offset, προσφέροντάς σας μεγαλύτερο καλλιτεχνικό έλεγχο πάνω στο τελικό σχήμα.

## Εισαγωγή Namespaces

Πρώτα, εισάγετε τα namespaces που σας δίνουν πρόσβαση στις βασικές κλάσεις 3D.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

Αυτές οι δηλώσεις καθιστούν διαθέσιμα στον κώδικά σας τα `Scene`, `LinearExtrusion`, `Vector3` και άλλους απαραίτητους τύπους.

## Οδηγός βήμα‑βήμα

### Βήμα 1: Αρχικοποίηση του Βασικού Προφίλ

Ξεκινάμε με ένα απλό ορθογώνιο προφίλ και του δίνουμε μια μικρή ακτίνα στρογγυλοποίησης ώστε οι άκρες να μην είναι απόλυτα αιχμηρές.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

### Βήμα 2: Δημιουργία Scene

Ένα `Scene` λειτουργεί ως δοχείο για όλους τους κόμβους, τα φώτα, τις κάμερες και τη γεωμετρία.

```csharp
Scene scene = new Scene();
```

### Βήμα 3: Δημιουργία Κόμβων

Δύο κόμβοι-παιδιά προστίθενται στη ρίζα του scene—ένας για την απλή εξώθηση και ένας για την στριμμένη έκδοση. Ο αριστερός κόμβος μετατοπίζεται στον άξονα X για οπτικό διαχωρισμό.

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(18, 0, 0);
```

### Βήμα 4: Γραμμική Εξώθηση στον Αριστερό Κόμβο (Χωρίς Twist Offset)

Εδώ δείχνουμε μια βασική εξώθηση με πλήρες στρίψιμο 360° και 100 φέτες για ομαλότητα.

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100 });
```

### Βήμα 5: Γραμμική Εξώθηση στον Δεξιό Κόμβο με Twist Offset

Τώρα εφαρμόζουμε ένα twist offset του `(3, 0, 0)`. Αυτό μετακινεί την αρχή του στριψίματος τρεις μονάδες κατά τον άξονα X, δημιουργώντας μια οπτικά μετατοπισμένη έλικα.

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100, TwistOffset = new Vector3(3, 0, 0) });
```

### Βήμα 6: Αποθήκευση του 3D Scene

Τέλος, γράφουμε το scene σε αρχείο OBJ. Αλλάξτε τη διαδρομή εξόδου όπως απαιτείται για το περιβάλλον σας.

```csharp
scene.Save("Your Output Directory" + "TwistOffsetInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **Το στρίψιμο φαίνεται επίπεδο** | Το `Slices` είναι ορισμένο πολύ χαμηλά, με αποτέλεσμα ένα χονδροειδές πλέγμα. | Αυξήστε το `Slices` (π.χ., 200) για πιο ομαλή περιστροφή. |
| **Το αντικείμενο είναι εκτός κέντρου** | Το `TwistOffset` χρησιμοποιεί συντεταγμένες κόσμου· ο κόμβος μπορεί ήδη να έχει μετατοπιστεί. | Εφαρμόστε το offset σχετικά με τον τοπικό μετασχηματισμό του κόμβου ή προσαρμόστε τη μετάφραση του κόμβου αναλόγως. |
| **Το αρχείο δεν αποθηκεύεται** | Λανθασμένη διαδρομή εξόδου ή έλλειψη δικαιωμάτων εγγραφής. | Επαληθεύστε ότι ο φάκελος υπάρχει και η εφαρμογή έχει δικαιώματα εγγραφής. |
| **Εξαίρεση άδειας** | Εκτέλεση χωρίς έγκυρη άδεια σε μη‑δοκιμαστική έκδοση. | Φορτώστε μια προσωρινή ή μόνιμη άδεια πριν δημιουργήσετε το scene. |
