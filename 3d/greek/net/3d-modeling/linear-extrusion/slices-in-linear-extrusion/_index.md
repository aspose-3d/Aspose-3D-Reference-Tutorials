# Δημιουργία 3Δ Σκηνής με Γραμμική Εξώθηση σε Φέτες

## Εισαγωγή

Καλώς ήρθατε στον κόσμο του 3Δ σχεδιασμού με το Aspose.3D για .NET! Σε αυτό το tutorial θα **δημιουργήσετε 3Δ σκηνή** αντικειμένων, θα εφαρμόσετε γραμμική εξώθηση με προσαρμοσμένο αριθμό φετών και τελικά θα **αποθηκεύσετε 3Δ μοντέλο** ως αρχείο Wavefront OBJ. Είτε δημιουργείτε ένα γρήγορο πρωτότυπο είτε μια παραγωγική οπτικοποίηση, τα παρακάτω βήματα θα σας δείξουν **πώς να χρησιμοποιήσετε το Aspose** για να παράγετε γεωμετρία υψηλής ποιότητας απευθείας από C#.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “δημιουργία 3Δ σκηνής”;** Σημαίνει τη δημιουργία ενός αντικειμένου Scene που περιέχει όλα τα 3Δ στοιχεία (πλέγματα, φώτα, κάμερες).  
- **Ποια μορφή χρησιμοποιείται για εξαγωγή;** Το παράδειγμα εξάγει σε **Wavefront OBJ** (`export wavefront obj`).  
- **Πόσες φέτες μπορώ να ορίσω;** Μπορείτε να ορίσετε οποιονδήποτε ακέραιο αριθμό· η demo δείχνει 2 και 10 φέτες.  
- **Χρειάζομαι άδεια;** Απαιτείται προσωρινή ή πλήρης άδεια για παραγωγική χρήση.  
- **Μπορώ να το τρέξω σε .NET Core;** Ναι, το Aspose.3D υποστηρίζει .NET Framework και .NET Core.

## Εισαγωγή Namespaces

Για να ξεκινήσετε το ταξίδι σας στο 3Δ σχεδιασμό με το Aspose.3D, θα χρειαστεί να εισάγετε τα απαραίτητα namespaces. Αυτό εξασφαλίζει ότι ο κώδικάς σας μπορεί να έχει πρόσβαση στις απαιτούμενες κλάσεις και λειτουργίες.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

## Πώς να δημιουργήσετε 3Δ σκηνή με Γραμμική Εξώθηση

Παρακάτω περπατάμε βήμα‑βήμα τη διαδικασία δημιουργίας της σκηνής, εφαρμογής της εξώθησης και **αποθήκευσης 3Δ μοντέλου** ως αρχείο OBJ. Οι εξηγήσεις είναι γραμμένες με φιλικό τόνο ώστε να μπορείτε να τις ακολουθήσετε εύκολα.

### Βήμα 1: Αρχικοποίηση του Βασικού Προφίλ

Πρώτα, ορίζουμε το 2‑Δ σχήμα που θα εξωθηθεί. Σε αυτήν την περίπτωση ένα ορθογώνιο με στρογγυλεμένες γωνίες.

```csharp
// ExStart:InitializeBaseProfile
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
// ExEnd:InitializeBaseProfile
```

### Βήμα 2: Δημιουργία 3Δ Σκηνής

Ένα αντικείμενο `Scene` είναι ο container για όλη τη γεωμετρία, τα φώτα και τις κάμερες – ο πυρήνας της **δημιουργίας 3Δ σκηνής**.

```csharp
// ExStart:Create3DScene
Scene scene = new Scene();
// ExEnd:Create3DScene
```

### Βήμα 3: Δημιουργία Αριστερού και Δεξιού Κόμβου

Προσθέτουμε δύο παιδικούς κόμβους στη ρίζα της σκηνής. Ο ένας θα χρησιμοποιήσει μικρό αριθμό φετών, ο άλλος υψηλότερο, ώστε να δείτε τη διαφορά στην εμφάνιση.

```csharp
// ExStart:CreateLeftRightNodes
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(15, 0, 0);
// ExEnd:CreateLeftRightNodes
```

### Βήμα 4: Εφαρμογή Γραμμικής Εξώθησης στον Αριστερό Κόμβο

Εδώ εξωθούμε το ορθογώνιο με **2 φέτες**. Λιγότερες φέτες δίνουν πιο αδρή εμφάνιση.

```csharp
// ExStart:LinearExtrusionLeftNode
left.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 2 });
// ExEnd:LinearExtrusionLeftNode
```

### Βήμα 5: Εφαρμογή Γραμμικής Εξώθησης στον Δεξιό Κόμβο

Τώρα εξωθούμε το ίδιο προφίλ αλλά με **10 φέτες**, παράγοντας μια πιο ομαλή επιφάνεια.

```csharp
// ExStart:LinearExtrusionRightNode
right.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 10 });
// ExEnd:LinearExtrusionRightNode
```

### Βήμα 6: Αποθήκευση 3Δ Σκηνής

Τέλος, εξάγουμε τη σκηνή σε αρχείο Wavefront OBJ. Αυτό δείχνει **πώς να αποθηκεύσετε obj** και **export wavefront obj** χρησιμοποιώντας το Aspose.3D.

```csharp
// ExStart:Save3DScene
scene.Save("Your Output Directory" + "SlicesInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
// ExEnd:Save3DScene
```
