# Δημιουργία 3D Σκηνής .NET – Περιστροφή σε Γραμμική Εξώθηση

## Εισαγωγή

Στον συνεχώς εξελισσόμενο κόσμο της ανάπτυξης .NET, η αξιοποίηση της δύναμης των 3D γραφικών είναι μια συναρπαστική προσπάθεια. **Aspose.3D for .NET** εμφανίζεται ως ένα πολύτιμο σύνολο εργαλείων, δίνοντας τη δυνατότητα στους προγραμματιστές να **create 3D scene .NET** εφαρμογές που είναι τόσο εμβληματικές όσο και οπτικά εντυπωσιακές. Σε αυτόν τον ολοκληρωμένο οδηγό, θα εξερευνήσουμε το ενδιαφέρον χαρακτηριστικό — Linear Extrusion with a Twist — και θα σας καθοδηγήσουμε βήμα προς βήμα ώστε να προσθέσετε δυναμικές περιστροφές στα μοντέλα σας με σιγουριά.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει το “create 3d scene .net”**; Αναφέρεται στην κατασκευή μιας 3‑Δ σκηνής προγραμματιστικά χρησιμοποιώντας τη βιβλιοθήκη Aspose.3D σε περιβάλλον .NET.  
- **Πώς προσθέτω μια περιστροφή σε μια εξώθηση;** Ορίστε την ιδιότητα `Twist` σε ένα αντικείμενο `LinearExtrusion`; η τιμή είναι η γωνία περιστροφής σε μοίρες.  
- **Χρειάζομαι άδεια για το Aspose.3D;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για χρήση σε παραγωγή.  
- **Ποιες εκδόσεις του .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6 και μεταγενέστερες.  
- **Τι επίδραση έχει η τιμή `Slices`;** Περισσότερα slices προσφέρουν πιο ομαλή περιστροφή αλλά αυξάνουν τη μνήμη και το χρόνο επεξεργασίας.

## Τι είναι η Γραμμική Εξώθηση με Περιστροφή;

Η γραμμική εξώθηση διασχίζει ένα 2‑Δ προφίλ κατά μήκος μιας ευθείας διαδρομής, ενώ η ιδιότητα **twist** περιστρέφει το προφίλ σταδιακά, παράγοντας ένα ελικοειδές αποτέλεσμα. Αυτή η τεχνική είναι ιδανική για τη δημιουργία ελατηρίων, στριμωγμένων κολόνων ή διακοσμητικών στοιχείων.

## Γιατί να χρησιμοποιήσετε το Aspose.3D για αυτήν την εργασία;

- **Straightforward API** – ενστικτώδεις κλάσεις όπως `LinearExtrusion` και `RectangleShape`.

## Πώς να δημιουργήσετε 3d scene .net με Linear Extrusion Twist

Παρακάτω υπάρχει ένας βήμα‑βήμα οδηγός που δείχνει ακριβώς πώς να **create a 3D scene .NET** και να εφαρμόσετε μια περιστροφή σε μια γραμμική εξώθηση.

### Βήμα 1: Αρχικοποίηση του Βασικού Προφίλ

```csharp
// Initialize the base profile to be extruded
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

Ξεκινάμε ορίζοντας ένα προφίλ ορθογωνίου. Προσαρμόστε το `RoundingRadius` για να μαλακώσετε τις γωνίες αν το επιθυμείτε.

### Βήμα 2: Δημιουργία 3D Σκηνής

```csharp
// Create a scene 
Scene scene = new Scene();
```

Το αντικείμενο `Scene` λειτουργεί ως καμβάς όπου θα ζουν όλα τα 3‑Δ αντικείμενα.

### Βήμα 3: Δημιουργία Αριστερών και Δεξιών Κόμβων

```csharp
// Create left node
var left = scene.RootNode.CreateChildNode();
// Create right node
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(15, 0, 0);
```

Οι κόμβοι είναι δοχεία για γεωμετρία. Δημιουργούμε δύο κόμβους ώστε να συγκρίνουμε μια μη‑στριμωγμένη εξώθηση (αριστερά) με μια στριμωγμένη (δεξιά). Ο αριστερός κόμβος μετακινείται 15 μονάδες στον άξονα X για οπτικό διαχωρισμό.

### Βήμα 4: Εκτέλεση Γραμμικής Εξώθησης με Περιστροφή στον Αριστερό Κόμβο

```csharp
// Twist property defines the degree of the rotation while extruding the profile
// Perform linear extrusion on the left node using twist and slices property
left.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 0, Slices = 100 });
```

Εδώ το `Twist` ορίζεται σε **0°**, παράγοντας μια ευθεία εξώθηση. Η τιμή `Slices` των **100** δίνει μια ομαλή επιφάνεια.

### Βήμα 5: Εκτέλεση Γραμμικής Εξώθησης με Περιστροφή στον Δεξιό Κόμβο

```csharp
// Perform linear extrusion on the right node using twist and slices property
right.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 90, Slices = 100 });
```

Ορίζοντας `Twist = 90` περιστρέφει το προφίλ κατά πλήρη 90 μοίρες κατά τη διάρκεια του μήκους της εξώθησης, δημιουργώντας ένα εμφανές έλικα.

### Βήμα 6: Αποθήκευση της 3D Σκηνής

```csharp
// Save 3D scene
scene.Save("Your Output Directory" + "TwistInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```

Η σκηνή εξάγεται ως αρχείο OBJ, το οποίο μπορείτε να ανοίξετε στα περισσότερα 3‑D προγράμματα προβολής ή να το εισάγετε σε άλλες αλυσίδες επεξεργασίας.

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Πώς να διορθώσετε |
|---|---|---|
| **Twist φαίνεται επίπεδο** | `Slices` είναι πολύ χαμηλό, προκαλώντας χονδροειδή γεωμετρία. | Αυξήστε το `Slices` (π.χ., 200) για πιο ομαλές περιστροφές. |
| **Performance μειώνεται με υψηλό `Slices`** | Περισσότερα πολύγωνα απαιτούν περισσότερη μνήμη/CPU. | Χρησιμοποιήστε το χαμηλότερο `Slices` που εξακολουθεί να ικανοποιεί την οπτική ποιότητα, ή ενεργοποιήστε την απλοποίηση γεωμετρίας μετά την εξώθηση. |
| **File not found on save** | Η διαδρομή του φακέλου εξόδου είναι μη έγκυρη. | Παρέχετε πλήρη απόλυτη διαδρομή ή βεβαιωθείτε ότι ο φάκελος υπάρχει πριν καλέσετε το `Save`. |
