# Προσθήκη κάμερας στη σκηνή και ρύθμιση στόχων για 3D Animation

## Εισαγωγή

Αν δημιουργείτε μια 3D animation, το πρώτο που χρειάζεστε είναι μια καλά τοποθετημένη κάμερα. Σε αυτό το tutorial θα μάθετε **how to add camera to scene**, ορίστε τον στόχο της, και τελικά **export scene as FBX** χρησιμοποιώντας το Aspose.3D for .NET. Θα περάσουμε από κάθε βήμα, θα εξηγήσουμε γιατί είναι σημαντικό, και θα σας δώσουμε πρακτικές συμβουλές ώστε να δημιουργήσετε εντυπωσιακές 3D animations με σιγουριά. Στο τέλος θα κατανοήσετε επίσης πώς να **position camera in 3d** χώρο για βέλτιστη σύνθεση.

## Γρήγορες Απαντήσεις
- **Ποιο είναι το πρώτο βήμα για την προσθήκη κάμερας;** Initialize a `Scene` object that will host the camera node.  
- **Ποια μορφή αρχείου χρησιμοποιείται για εξαγωγή σε αυτόν τον οδηγό;** FBX (`.fbx`).  
- **Χρειάζομαι άδεια για να εκτελέσω το δείγμα κώδικα;** A free trial works for evaluation; a commercial license is required for production.  
- **Ποιες εκδόσεις .NET υποστηρίζονται;** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Μπορώ να αλλάξω τη θέση της κάμερας μετά τη δημιουργία;** Yes – modify `cameraNode.Transform.Translation` at any time.

## Τι είναι **add camera to scene**?
Η προσθήκη μιας κάμερας σε μια σκηνή σημαίνει δημιουργία μιας οντότητας `Camera`, σύνδεσή της με έναν κόμβο στο γράφημα σκηνής, και τοποθέτησή της ώστε να «κοιτάζει» ένα σημείο-στόχο. Αυτό ορίζει την προοπτική του θεατή όταν η σκηνή αποδίδεται ή εξάγεται.

## Γιατί να ρυθμίσετε στόχους κάμερας και να εξάγετε ως FBX;
- **Control the viewpoint** – η ακριβής τοποθέτηση της κάμερας σας επιτρέπει να πλαισιώσετε την animation σας ακριβώς όπως το φαντάζεστε.  
- **Interoperability** – το FBX υποστηρίζεται ευρέως από game engines, Maya, Blender και άλλα 3D εργαλεία, καθιστώντας εύκολη την κοινή χρήση πόρων.  
- **Reusable assets** – μόλις η κάμερα και ο στόχος της αποθηκευτούν, μπορείτε να επαναχρησιμοποιήσετε τη σκηνή σε πολλαπλά έργα.

## Συνηθισμένες περιπτώσεις χρήσης
- **Cinematic cut‑scenes** σε παιχνίδια όπου μια σταθερή κάμερα ακολουθεί έναν χαρακτήρα.  
- **Product visualizations** όπου χρειάζεστε μια σταθερή προοπτική για να παρουσιάσετε ένα μοντέλο από διαφορετικές γωνίες.  
- **Pre‑visualization** για ταινίες ή έργα AR/VR, επιτρέποντας στους σχεδιαστές να δοκιμάσουν την τοποθέτηση της κάμερας πριν την τελική απόδοση.

## Οδηγός βήμα‑βήμα

### Βήμα 1: Αρχικοποίηση αντικειμένου Scene

Ξεκινήστε με την αρχικοποίηση του αντικειμένου σκηνής. Αυτό λειτουργεί ως καμβάς όπου η 3D animation σας θα ζωντανέψει.

```csharp
// ExStart:SetupTargetAndCamera
// Initialize scene object
Scene scene = new Scene();
```

### Βήμα 2: Δημιουργία κόμβου Camera

Στη συνέχεια, δημιουργήστε έναν υποκόμβο που θα κρατά την κάμερα. Η ανάθεση ενός περιγραφικού ονόματος στον κόμβο βοηθά στη διατήρηση της ιεραρχίας της σκηνής οργανωμένης.

```csharp
// Get a child node object
Node cameraNode = scene.RootNode.CreateChildNode("camera", new Camera());
```

### Βήμα 3: Τοποθέτηση της κάμερας

Καθορίστε τη μετάφραση (translation) για τον κόμβο της κάμερας. Αυτό καθορίζει την αρχική θέση της κάμερας στο 3D χώρο. Ρυθμίστε τις τιμές `Vector3` ώστε να **position camera in 3d** όπως απαιτείται για το πλάνο σας.

```csharp
// Set camera node translation
cameraNode.Transform.Translation = new Vector3(100, 20, 0);
```

### Βήμα 4: Ορισμός στόχου της κάμερας

Μια κάμερα χρειάζεται ένα σημείο-στόχο για να κοιτάξει. Δημιουργούμε έναν ακόμη υποκόμβο που λειτουργεί ως σημείο εστίασης και τον αναθέτουμε στην ιδιότητα `Target` της κάμερας. Έτσι **set camera target** για μια σταθερή προβολή.

```csharp
cameraNode.GetEntity<Camera>().Target = scene.RootNode.CreateChildNode("target");
```

### Βήμα 5: Αποθήκευση της διαμορφωμένης σκηνής

Τέλος, εξάγετε τη σκηνή σε αρχείο FBX (ή οποιαδήποτε άλλη υποστηριζόμενη μορφή). Αντικαταστήστε το `"Your Output Directory"` με την πραγματική διαδρομή όπου θέλετε να αποθηκευτεί το αρχείο.

```csharp
var output = "Your Output Directory" + "camera-test.fbx";
scene.Save(output);
```

## Συχνά προβλήματα και λύσεις

| Πρόβλημα | Λύση |
|-------|----------|
| **Η κάμερα εμφανίζεται σε λάθος γωνία** | Επαληθεύστε ότι ο κόμβος-στόχος είναι τοποθετημένος όπου το αναμένετε. Μπορείτε επίσης να ορίσετε `cameraNode.GetEntity<Camera>().UpVector` για να ελέγξετε τον προσανατολισμό. |
| **Το εξαγόμενο FBX δεν ανοίγει σε άλλα εργαλεία** | Βεβαιωθείτε ότι χρησιμοποιείτε μια πρόσφατη έκδοση του Aspose.3D (η βιβλιοθήκη γράφει αυτόματα μια συμβατή έκδοση FBX). |
| **Σφάλμα «Διαδρομή δεν βρέθηκε» στο `scene.Save`** | Χρησιμοποιήστε απόλυτη διαδρομή ή βεβαιωθείτε ότι ο φάκελος εξόδου υπάρχει πριν καλέσετε το `Save`. |
