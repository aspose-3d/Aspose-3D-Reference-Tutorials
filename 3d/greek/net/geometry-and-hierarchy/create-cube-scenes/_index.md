## Τι είναι μια σκηνή 3D κύβου;

Μια σκηνή 3D κύβου είναι ένα ελάχιστο τρισδιάστατο μοντέλο που αποτελείται από μια μόνο γεωμετρία κύβου τοποθετημένη μέσα σε ένα γράφημα σκηνής. Λειτουργεί ως το “Hello World” των 3D γραφικών, επιτρέποντάς σας να επαληθεύσετε ότι η αλυσίδα επεξεργασίας σας – από τη δημιουργία mesh έως την εξαγωγή αρχείου – λειτουργεί σωστά.

## Εισαγωγή Namespaces

Πρώτα, προσθέστε τις απαιτούμενες δηλώσεις `using` ώστε ο μεταγλωττιστής να γνωρίζει πού βρίσκονται οι τύποι Aspose.3D.

```csharp
using System;
using System.Collections.Generic;
using System.IO;
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
```

## Οδηγός βήμα προς βήμα

### Βήμα 1: Αρχικοποίηση της Σκηνής

Δημιουργήστε ένα κενό αντικείμενο `Scene` που θα περιέχει όλα τα nodes, meshes, φώτα και κάμερες.

```csharp
// ExStart:CreateCubeScene
// Initialize scene object
Scene scene = new Scene();
```

### Βήμα 2: Δημιουργία Node για τον Κύβο

Ένα `Node` λειτουργεί ως δοχείο για γεωμετρία. Δώστε του ένα φιλικό όνομα ώστε να μπορείτε να το βρείτε αργότερα στην ιεραρχία.

```csharp
// Initialize Node class object
Node cubeNode = new Node("cube");
```

### Βήμα 3: Δημιουργία του Mesh

Το Aspose.3D παρέχει ένα βοηθητικό εργαλείο που ονομάζεται `Common` και μπορεί να δημιουργήσει ένα mesh κύβου χρησιμοποιώντας έναν polygon builder. Αυτό σας εξοικονομεί τον ορισμό των κορυφών και των προσώπων με το χέρι.

```csharp
// Call Common class create mesh using polygon builder method to set mesh instance 
Mesh mesh = Common.CreateMeshUsingPolygonBuilder();
```

### Βήμα 4: Προσθήκη mesh στο node

Αναθέστε το mesh που μόλις δημιουργήσατε στην ιδιότητα `Entity` του node. Αυτό συνδέει τη γεωμετρία με το γράφημα σκηνής.

```csharp
// Point node to the Mesh geometry
cubeNode.Entity = mesh;
```

### Βήμα 5: Προσθήκη Node στη Σκηνή

Εισάγετε το node του κύβου στη ρίζα της σκηνής ώστε να γίνει μέρος του τελικού αποτελέσματος.

```csharp
// Add Node to a scene
scene.RootNode.ChildNodes.Add(cubeNode);
```

### Βήμα 6: Πώς να εξάγετε FBX (αποθήκευση σκηνής ως FBX)

Επιλέξτε μια διαδρομή εξόδου και εξάγετε τη σκηνή. Εδώ χρησιμοποιούμε τη μορφή FBX 7400 ASCII, η οποία υποστηρίζεται ευρέως από επεξεργαστές 3D.

```csharp
// The path to the documents directory.
var output = "Your Output Directory" + "CubeScene.fbx";

// Save 3D scene in the supported file formats
scene.Save(output, FileFormat.FBX7400ASCII);
```

### Βήμα 7: Εμφάνιση Μηνύματος Επιτυχίας

Δώστε στον χρήστη μια σαφή επιβεβαίωση ότι το αρχείο γράφτηκε επιτυχώς.

```csharp
Console.WriteLine("\nCube Scene created successfully.\nFile saved at " + output);
```

## Συνηθισμένα προβλήματα και λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **Σφάλμα “File not found”** κατά την εκτέλεση του `scene.Save` | Ο φάκελος εξόδου δεν υπάρχει ή δεν έχετε δικαίωμα εγγραφής. | Δημιουργήστε πρώτα τον φάκελο (`Directory.CreateDirectory`) ή χρησιμοποιήστε απόλυτη διαδρομή που έχετε δικαίωμα. |
| **Κενό αρχείο** μετά την εξαγωγή | Το mesh δεν ήταν προσαρτημένο στο node ή το node δεν προστέθηκε στη σκηνή. | Βεβαιωθείτε ότι εκτελούνται `cubeNode.Entity = mesh;` και `scene.RootNode.ChildNodes.Add(cubeNode);`. |
| **Λάθος μορφή** κατά το άνοιγμα σε viewer | Χρήση λανθασμένης τιμής του enum `FileFormat`. | Χρησιμοποιήστε `FileFormat.FBX7400ASCII` για ASCII FBX ή `FileFormat.FBX7400Binary` για binary. |
