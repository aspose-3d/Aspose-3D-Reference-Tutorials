# Πώς να Εφαρμόσετε Υλικό PBR σε Ένα Κουτί

## Εισαγωγή

Καλώς ήρθατε στον συναρπαστικό κόσμο των 3D γραφικών! Σε αυτόν τον οδηγό βήμα‑βήμα, θα μάθετε **πώς να εφαρμόσετε pbr** υλικό σε ένα κουτί χρησιμοποιώντας το Aspose.3D for .NET. Θα περάσουμε από τη δημιουργία ενός υλικού PBR, την προσθήκη του σε ένα mesh, και τελικά **εξαγωγή STL ASCII** ώστε να μπορείτε να χρησιμοποιήσετε το μοντέλο σε οποιαδήποτε επόμενη ροή εργασίας. Είτε δημιουργείτε ένα πρωτότυπο παιχνιδιού είτε μια οπτικοποίηση προϊόντος, η κατανόηση αυτής της διαδικασίας ανοίγει την πόρτα σε ρεαλιστικό, φυσικά βασισμένο rendering (PBR) στις .NET εφαρμογές σας.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος στόχος;** Εφαρμογή υλικού PBR σε ένα κουτί και εξαγωγή του ως STL ASCII.  
- **Ποια βιβλιοθήκη χρησιμοποιείται;** Aspose.3D for .NET (how to use aspose).  
- **Χρειάζομαι άδεια;** Ναι, απαιτείται προσωρινή ή πλήρης άδεια για παραγωγή.  
- **Υποστηριζόμενη μορφή εξόδου;** STL ASCII (export stl ascii) και πολλές άλλες μορφές 3D.  
- **Πόσο χρόνο παίρνει;** Περίπου 10‑15 λεπτά για μια βασική υλοποίηση.

## Τι είναι το Υλικό PBR;
Το Physically Based Rendering (PBR) είναι ένα μοντέλο σκίασης που προσομοιώνει πώς το φως αλληλεπιδρά με πραγματικές επιφάνειες. Με την προσαρμογή ιδιοτήτων όπως οι παράγοντες metallic και roughness, μπορείτε να πετύχετε εξαιρετικά ρεαλιστικά αποτελέσματα χωρίς να ρυθμίζετε χειροκίνητα σύνθετους shaders.

## Γιατί να Χρησιμοποιήσετε Physically Based Rendering (PBR);
- **Ρεαλισμός:** Τα υλικά αντιδρούν στον φωτισμό με τρόπο που ταιριάζει με την πραγματική φυσική.  
- **Συνέπεια:** Το ίδιο υλικό φαίνεται σωστό σε οποιοδήποτε περιβάλλον φωτισμού.  
- **Αποδοτικότητα:** Οι σύγχρονοι GPU είναι βελτιστοποιημένοι για υπολογισμούς PBR, παρέχοντας απόδοση δωρεάν.

## Βήμα 1: Αρχικοποίηση Σκηνής
Ξεκινήστε με την αρχικοποίηση μιας 3D σκηνής χρησιμοποιώντας το παρακάτω απόσπασμα κώδικα:

```csharp
Scene scene = new Scene();
```

## Βήμα 2: Δημιουργία Υλικού PBR
Τώρα **δημιουργούμε υλικό pbr** που θα δώσει στο κουτί μας μια ρεαλιστική εμφάνιση:

```csharp
PbrMaterial mat = new PbrMaterial();
```

## Βήμα 3: Ορισμός Ιδιοτήτων Υλικού
Ρυθμίστε προσεκτικά τις ιδιότητες του υλικού, κάνοντάς το σχεδόν metallic και πολύ rough—ιδανικό για ένα κουτί με βουρτσισμένο μέταλλο:

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

## Βήμα 4: Δημιουργία Κουτιού
Δημιουργήστε ένα κουτί στο οποίο θα εφαρμοστεί το υλικό PBR:

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

## Βήμα 5: Προσθήκη Υλικού PBR στο Κουτί
Αναθέστε το προηγουμένως διαμορφωμένο **add pbr material** στον κόμβο του δημιουργημένου κουτιού:

```csharp
boxNode.Material = mat;
```

## Βήμα 6: Αποθήκευση της 3D Σκηνής ως STL ASCII
Τέλος, **export stl ascii** ώστε το μοντέλο να μπορεί να ανοιχτεί σε οποιονδήποτε τυπικό 3D viewer ή slicer:

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```
