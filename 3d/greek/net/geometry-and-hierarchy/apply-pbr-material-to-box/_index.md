# Πώς να Εφαρμόσετε Υλικό PBR σε Ένα Κουτί

## Εισαγωγή

Καλώς ήρθατε στον συναρπαστικό κόσμο των 3D γραφικών! Σε αυτό το βήμα‑βήμα tutorial, **θα μάθετε πώς να εφαρμόσετε pbr** υλικό σε ένα κουτί χρησιμοποιώντας Aspose.3D for .NET. Θα περάσουμε από τη δημιουργία ενός υλικού PBR, την προσθήκη του σε ένα mesh, και τελικά **εξαγωγή STL ASCII** ώστε να μπορείτε να χρησιμοποιήσετε το μοντέλο σε οποιαδήποτε επόμενη ροή εργασίας. Είτε δημιουργείτε ένα πρωτότυπο παιχνιδιού, έναν οπτικοποιητή προϊόντος, ή ένα γρήγορο πρωτότυπο για 3D εκτύπωση, η κατανόηση αυτής της διαδικασίας ανοίγει την πόρτα σε ρεαλιστική, φυσικά βασισμένη απόδοση (PBR) στις .NET εφαρμογές σας.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος στόχος;** Εφαρμογή υλικού PBR σε ένα κουτί και εξαγωγή του ως STL ASCII.  
- **Ποια βιβλιοθήκη χρησιμοποιείται;** Aspose.3D for .NET (how to use aspose).  
- **Χρειάζεται άδεια;** Ναι, απαιτείται προσωρινή ή πλήρης άδεια για παραγωγή.  
- **Υποστηριζόμενη μορφή εξόδου;** STL ASCII (export stl ascii) και πολλές άλλες μορφές 3D.  
- **Πόσο χρόνο διαρκεί;** Περίπου 10‑15 λεπτά για μια βασική υλοποίηση.

## Τι είναι το Υλικό PBR;
Η Φυσικά Βασισμένη Απόδοση (PBR) είναι ένα μοντέλο σκίασης που προσομοιώνει πώς το φως αλληλεπιδρά με πραγματικές επιφάνειες. Ρυθμίζοντας ιδιότητες όπως οι παράγοντες μεταλλικότητας και τραχύτητας, μπορείτε να πετύχετε εξαιρετικά ρεαλιστικά αποτελέσματα χωρίς χειροκίνητη ρύθμιση πολύπλοκων shaders.

## Γιατί να Χρησιμοποιήσετε Φυσικά Βασισμένη Απόδοση (PBR);
- **Ρεαλισμός:** Τα υλικά αντιδρούν στο φωτισμό με τρόπο που ταιριάζει στη φυσική πραγματικότητα.  
- **Συνέπεια:** Το ίδιο υλικό φαίνεται σωστό υπό οποιοδήποτε φωτιστικό περιβάλλον.  
- **Αποδοτικότητα:** Οι σύγχρονοι GPU είναι βελτιστοποιημένοι για υπολογισμούς PBR, παρέχοντας απόδοση δωρεάν.

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Αρχικοποίηση Σκηνής
Ξεκινήστε δημιουργώντας μια κενή 3D σκηνή. Αυτό το container θα κρατήσει όλα τα γεωμετρικά στοιχεία, τα υλικά και τα φώτα που θα προσθέσετε αργότερα.

```csharp
Scene scene = new Scene();
```

### Βήμα 2: Δημιουργία Υλικού PBR
Τώρα **δημιουργούμε pbr material** που θα δώσει στο κουτί μας ρεαλιστική εμφάνιση. Η κλάση `PbrMaterial` εκθέτει όλες τις παραμέτρους που χρειάζεστε για φυσικά βασισμένη απόδοση.

```csharp
PbrMaterial mat = new PbrMaterial();
```

### Βήμα 3: Ορισμός Ιδιοτήτων Υλικού
Ρυθμίστε προσεκτικά τις ιδιότητες του υλικού. Σε αυτό το παράδειγμα κάνουμε την επιφάνεια σχεδόν μεταλλική και πολύ τραχιά—τέλεια για ένα κουτί με βουρτσισμένο μέταλλο.

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

### Βήμα 4: Δημιουργία Πλέγματος Κουτιού
Δημιουργήστε μια απλή γεωμετρία κουτιού. Αυτό είναι το **create box mesh** βήμα στο οποίο αναφέρεται η κύρια λέξη-κλειδί.

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

### Βήμα 5: Εφαρμογή του Υλικού PBR στο Κουτί
Αναθέστε το προηγουμένως διαμορφωμένο **add pbr material** στο node του κουτιού που μόλις δημιουργήσαμε.

```csharp
boxNode.Material = mat;
```

### Βήμα 6: Εξαγωγή STL ASCII (Πώς να Εξάγετε STL)
Τέλος, **export stl ascii** ώστε το μοντέλο να μπορεί να ανοιχτεί σε οποιονδήποτε τυπικό 3D viewer ή slicer. Η χρήση του `FileFormat.STLASCII` εγγυάται ένα αρχείο αναγνώσιμο από άνθρωπο.

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```

## Κοινά Πιθανά Σφάλματα & Συμβουλές
- **Άδεια δεν βρέθηκε:** Βεβαιωθείτε ότι το αρχείο άδειας φορτώνεται *πριν* από οποιαδήποτε κλήση Aspose· διαφορετικά η βιβλιοθήκη λειτουργεί σε λειτουργία αξιολόγησης.  
- **Λανθασμένη διαδρομή αρχείου:** Χρησιμοποιήστε `Path.Combine` για να αποφύγετε ελλείψεις διαχωριστών διαδρομής σε διαφορετικά λειτουργικά συστήματα.  
- **Ισορροπία Roughness vs. Metallic:** Η ταυτόχρονη υψηλή τιμή και των δύο παραγόντων μπορεί να κάνει την επιφάνεια να φαίνεται επίπεδη· πειραματιστείτε με τιμές μεταξύ `0.5‑0.9` για ισορροπημένο αποτέλεσμα.  
- **Συμβουλή απόδοσης:** Επαναχρησιμοποιήστε ένα μόνο αντικείμενο `PbrMaterial` αν χρειάζεται να εφαρμόσετε το ίδιο υλικό σε πολλά meshes· αυτό μειώνει το φορτίο μνήμης.
