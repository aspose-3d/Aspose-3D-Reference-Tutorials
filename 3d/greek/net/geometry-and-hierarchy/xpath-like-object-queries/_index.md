## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Άνοιγμα Visual Studio
Δημιουργήστε ένα νέο έργο C# ή ανοίξτε ένα υπάρχον όπου θέλετε να εργαστείτε με 3D σκηνές.

### Βήμα 2: Δημιουργία Νέας Σκηνής (Προσθήκη Κάμερας σε Σκηνή)
Δημιουργήστε ένα φρέσκο αντικείμενο `Scene` που θα λειτουργήσει ως καμβάς για όλα τα επόμενα αντικείμενα.

```csharp
Scene s = new Scene();
```

### Βήμα 3: Συμπλήρωση της Σκηνής – Προσθήκη Κόμβων, Κάμερας και Φωτός
Δομήστε μια απλή ιεραρχία, στη συνέχεια **προσθέστε μια κάμερα** και **προσθέστε φως στη σκηνή** για να δείξετε το ερώτημα αργότερα.

```csharp
var a = s.RootNode.CreateChildNode("a");
a.CreateChildNode("a1");
a.CreateChildNode("a2");
s.RootNode.CreateChildNode("b");
var c = s.RootNode.CreateChildNode("c");
c.CreateChildNode("c1").AddEntity(new Camera("cam"));
c.CreateChildNode("c2").AddEntity(new Light("light"));
```

Η προκύπτουσα ιεραρχία φαίνεται ως εξής:

```
- Root
    - a
        - a1
        - a2
    - b
    - c
        - c1
            - cam
        - c2
            - light
```

### Βήμα 4: Επιλογή Αντικειμένων – Πώς να ερωτήσετε 3D αντικείμενα
Χρησιμοποιήστε μια έκφραση τύπου XPath για να λάβετε όλες τις κάμερες **ή** οποιονδήποτε κόμβο με όνομα “light”.

```csharp
var objects = s.RootNode.SelectObjects("//*[(@Type = 'Camera') or (@Name = 'light')]");
```

### Βήμα 5: Επιλογή Μονού Αντικειμένου – Επιλογή ενός αντικειμένου με μονοπάτι
Ανακτήστε άμεσα τον πρώτο κόμβο κάμερας με ένα σύντομο μονοπάτι.

```csharp
var c1 = s.RootNode.SelectSingleObject("/c/*/<Camera>");
```

### Βήμα 6: Επιλογή Κόμβου με Όνομα – Γρήγορος τρόπος εντοπισμού κόμβου
Αν γνωρίζετε το όνομα του κόμβου, μπορείτε να τον ανακτήσετε χωρίς να σας ενδιαφέρει η θέση του στην ιεραρχία.

```csharp
var obj = s.RootNode.SelectSingleObject("a1");
```

### Βήμα 7: Επιλογή Ριζικού Κόμβου – Χρήσιμο για παγκόσμιες λειτουργίες
Μερικές φορές χρειάζεστε μια αναφορά στον ριζικό κόμβο της σκηνής για μαζικές μετασχηματίσεις.

```csharp
obj = s.RootNode.SelectSingleObject("/");
```

## Συνηθισμένα Προβλήματα και Λύσεις
| Πρόβλημα | Λύση |
|----------|------|
| **Η κάμερα δεν εμφανίζεται στα αποτελέσματα του ερωτήματος** | Βεβαιωθείτε ότι το `Entity` του κόμβου είναι `Camera` και ότι το όνομα ταιριάζει ακριβώς με το ερώτημα (διάκριση πεζών‑κεφαλαίων). |
| **Το SelectSingleObject επιστρέφει null** | Ελέγξτε τη σύνταξη της έκφρασης XPath· χρησιμοποιήστε αρχικό `/` για απόλυτα μονοπάτια. |
| **Το φως δεν επηρεάζει την απόδοση** | Θυμηθείτε ότι οι υπολογισμοί φωτισμού απαιτούν μηχανή απόδοσης· η οντότητα Light από μόνη της δεν αποδίδει κάτι. |
| **Μείωση απόδοσης σε μεγάλες σκηνές** | Περιορίστε τα ερωτήματα σε υποδέντρα (`RootNode.SelectObjects("//c/*")`) ή αποθηκεύστε τα αποτελέσματα στην μνήμη όταν είναι δυνατόν. |
