# Δημιουργία 3D Εξώθησης με Φέτες – Aspose.3D for Java

## Εισαγωγή

Αν χρειάζεστε **create 3d extrusion** αντικείμενα που φαίνονται ομαλά και ακριβή, ο έλεγχος του αριθμού των φετών είναι το κλειδί. Σε αυτό το tutorial θα δούμε πώς να ορίζετε τις φέτες κατά την εκτέλεση μιας γραμμικής εξώθησης με Aspose.3D for Java. Θα καταλάβετε γιατί η ποσότητα των φετών είναι σημαντική, πώς να ορίσετε μια ακτίνα στρογγυλοποίησης και πώς να εξάγετε το αποτέλεσμα ως αρχείο OBJ που μπορεί να χρησιμοποιηθεί σε οποιοδήποτε pipeline 3D.

## Γρήγορες Απαντήσεις
- **Τι ελέγχει η “slices”;** Ο αριθμός των ενδιάμεσων διατομών που χρησιμοποιούνται για την προσέγγιση της επιφάνειας εξώθησης.  
- **Ποια μέθοδος ορίζει τον αριθμό των φετών;** `LinearExtrusion.setSlices(int)`  
- **Μπορώ να αλλάξω την ακτίνα στρογγυλοποίησης του προφίλ;** Ναι, μέσω `RectangleShape.setRoundingRadius(double)`  
- **Ποια μορφή αρχείου χρησιμοποιείται σε αυτό το παράδειγμα;** Wavefront OBJ (`FileFormat.WAVEFRONTOBJ`)  
- **Χρειάζομαι άδεια για την εκτέλεση του κώδικα;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.

## Τι είναι η γραμμική εξώθηση με φέτες;

Η γραμμική εξώθηση παίρνει ένα 2‑Δ προφίλ (όπως ένα ορθογώνιο) και το τεντώνει κατά μήκος μιας ευθείας γραμμής για να δημιουργήσει ένα 3‑Δ στερεό. Με τον καθορισμό **slices**, λέτε στο Aspose.3D πόσα ενδιάμεσα βήματα να δημιουργήσει, κάτι που επηρεάζει άμεσα την ομαλότητα των καμπυλωτών άκρων, όπως ενός στρογγυλεμένου ορθογωνίου.

## Γιατί να χρησιμοποιήσετε το Aspose.3D for Java για τη δημιουργία 3d extrusion;

* **Full control** – Μπορείτε να ρυθμίσετε τον αριθμό των φετών, την ακτίνα στρογγυλοποίησης και τη μορφή εξαγωγής προγραμματιστικά.  
* **Cross‑platform** – Λειτουργεί σε οποιοδήποτε περιβάλλον με Java χωρίς εγγενείς εξαρτήσεις.  
* **Export flexibility** – Αποθήκευση απευθείας σε OBJ, FBX, STL και πολλές άλλες μορφές.

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Ρύθμιση της σκηνής και ορισμός του προφίλ

Πρώτα δημιουργούμε ένα `RectangleShape` και του δίνουμε **rounding radius** ώστε οι γωνίες να είναι ομαλές. Στη συνέχεια αρχικοποιούμε ένα νέο `Scene` που θα περιέχει όλη τη γεωμετρία.

```java
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
Scene scene = new Scene();
```

### Βήμα 2: **Δημιουργία αντικειμένων child node** για κάθε εξώθηση

Κάθε κομμάτι γεωμετρίας ζει κάτω από ένα `Node`. Εδώ δημιουργούμε δύο αδερφικά nodes – ένα για εξώθηση με χαμηλό αριθμό φετών και ένα για εξώθηση με υψηλό αριθμό φετών – και μετακινούμε το αριστερό node λίγο προς τα πλάγια ώστε τα αποτελέσματα να είναι εύκολα συγκρίσιμα.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

### Βήμα 3: Εκτέλεση γραμμικής εξώθησης και **ορισμός φετών**

Τώρα δημιουργούμε πραγματικά **create 3d extrusion** αντικείμενα. Ο κατασκευαστής `LinearExtrusion` δέχεται το προφίλ και το βάθος εξώθησης. Χρησιμοποιώντας μια **anonymous inner class** καλούμε `setSlices` για να ελέγξουμε την ομαλότητα. Το αριστερό node παίρνει μόνο 2 φέτες (τραχύ), ενώ το δεξιό node παίρνει 10 φέτες (ομαλό).

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{setSlices(2);}});
right.createChildNode(new LinearExtrusion(profile, 2) {{setSlices(10);}});
```

### Βήμα 4: **Εξαγωγή OBJ** – αποθήκευση της σκηνής

Τέλος, γράφουμε τη σκηνή σε αρχείο Wavefront OBJ, μια μορφή που υποστηρίζεται ευρέως από επεξεργαστές 3‑D και μηχανές παιχνιδιών. Αυτό επιδεικνύει τη δυνατότητα **export obj java** του Aspose.3D.

```java
scene.save(MyDir + "SlicesInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## Κοινά Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **Η εξώθηση φαίνεται επίπεδη** | Ο αριθμός φετών ορίστηκε σε 1 ή 0 | Βεβαιωθείτε ότι το `setSlices` καλείται με τιμή ≥ 2. |
| **Οι στρογγυλεμένες γωνίες φαίνονται σκαλιστές** | Η ακτίνα στρογγυλοποίησης είναι πολύ μικρή σε σχέση με τον αριθμό φετών | Αυξήστε είτε την ακτίνα είτε τον αριθμό φετών για πιο ομαλές καμπύλες. |
| **Το αρχείο δεν βρέθηκε κατά την αποθήκευση** | Το `MyDir` δείχνει σε φάκελο που δεν υπάρχει | Δημιουργήστε τον φάκελο εκ των προτέρων ή χρησιμοποιήστε απόλυτη διαδρομή. |
