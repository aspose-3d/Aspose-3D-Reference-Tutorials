# Έλεγχος του κέντρου σε γραμμική εξώθηση με Aspose.3D για Java

## Εισαγωγή

Όταν δημιουργείτε 3D σκηνές σε Java, η δυνατότητα **add ground plane** ενώ ορίζετε με ακρίβεια την **set center property** σε μια γραμμική εξώθηση μπορεί να κάνει τη διαφορά μεταξύ ενός επίπεδου πρωτοτύπου και ενός επαγγελματικού οπτικού αποτελέσματος. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τη διαδικασία ελέγχου του κέντρου της εξώθησης και προσθήκης ενός ground plane χρησιμοποιώντας το Aspose.3D για Java. Θα δείτε γιατί είναι σημαντικό, πώς να το ρυθμίσετε και θα λάβετε ένα έτοιμο παράδειγμα κώδικα που μπορείτε να προσαρμόσετε στα δικά σας έργα.

## Γρήγορες Απαντήσεις
- **What does “add ground plane” do?** Δημιουργεί μια λεπτή γεωμετρία αναφοράς που σας βοηθά να δείτε τη θέση της εξώθησης σε σχέση με τους άξονες του κόσμου.  
- **How do I set the center of a linear extrusion?** Χρησιμοποιήστε τη μέθοδο `setCenter(boolean)` στο αντικείμενο `LinearExtrusion`.  
- **Do I need a license to run the sample?** Μια προσωρινή άδεια λειτουργεί για δοκιμές· απαιτείται πλήρης άδεια για παραγωγή.  
- **Which file format is used for saving?** Το παράδειγμα αποθηκεύει σε Wavefront OBJ (`.obj`).  
- **What Java version is required?** Η Java 8 ή νεότερη είναι επαρκής.

## Τι είναι το “add ground plane” σε μια 3D σκηνή;

Η προσθήκη ενός ground plane σημαίνει την εισαγωγή μιας λεπτής ορθογώνιας γεωμετρίας (συχνά ένα κουτί με ελάχιστο πάχος) που βρίσκεται στο επίπεδο X‑Z. Λειτουργεί ως οπτικό δάπεδο, καθιστώντας πιο εύκολο τον προσδιορισμό του ύψους και της ευθυγράμμισης άλλων αντικειμένων, ειδικά όταν πειραματίζεστε με κέντρα εξώθησης.

## Γιατί να ορίσετε την ιδιότητα center σε μια γραμμική εξώθηση;

Από προεπιλογή, μια γραμμική εξώθηση ξεκινά από το αρχικό σημείο του προφίλ. Ορίζοντας την ιδιότητα center (`setCenter(true)`) μετατοπίζει το προφίλ ώστε η εξώθηση να είναι κεντραρισμένη γύρω από το αρχικό σημείο, κάτι χρήσιμο για συμμετρικά σχέδια ή όταν χρειάζεστε συνεπή ευθυγράμμιση σε πολλαπλά αντικείμενα.

3. **Document Directory** – Δημιουργήστε έναν φάκελο για την αποθήκευση των Java εγγράφων σας. Ας τον ονομάσουμε «Your Document Directory».

## Εισαγωγή Πακέτων

Στο περιβάλλον ανάπτυξης Java, εισάγετε τα απαραίτητα πακέτα για το Aspose.3D. Αυτό εξασφαλίζει ότι ο κώδικάς σας έχει πρόσβαση στις λειτουργίες που παρέχει η βιβλιοθήκη.

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## Βήμα 1: Ρύθμιση του Βασικού Προφίλ

Αρχικοποιήστε το βασικό προφίλ που θα εξωθηθεί. Σε αυτό το παράδειγμα, θα χρησιμοποιήσουμε ένα ορθογώνιο σχήμα με ακτίνα στρογγυλοποίησης 0.3.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

## Βήμα 2: Δημιουργία 3D Σκηνής

Δημιουργήστε τη βάση του 3D κόσμου σας δημιουργώντας μια σκηνή.

```java
Scene scene = new Scene();
```

## Βήμα 3: Δημιουργία Αριστερών και Δεξιών Κόμβων

Καθορίστε αριστερούς και δεξιούς κόμβους μέσα στη σκηνή. Αυτοί οι κόμβοι λειτουργούν ως καμβάς για τα 3D αντικείμενά σας.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## Βήμα 4: Γραμμική Εξώθηση με Ιδιότητα Center (Αριστερός Κόμβος)

Εκτελέστε γραμμική εξώθηση στον αριστερό κόμβο **χωρίς κεντράρισμα** και ορίστε τον αριθμό των φέτες σε 3. Παρατηρήστε την κλήση `setCenter(false)` – εδώ **ορίζουμε την ιδιότητα center** σε *false*.

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

## Βήμα 5: Προσθήκη Ground Plane για Αναφορά (Αριστερός Κόμβος)

Βελτιώστε την οπτική παρουσίαση **προσθέτοντας ένα ground plane** στον αριστερό κόμβο. Το λεπτό κουτί λειτουργεί ως δάπεδο ώστε να μπορείτε να δείτε καθαρά το ύψος της εξώθησης.

```java
left.createChildNode(new Box(0.01, 3, 3));
```

## Βήμα 6: Γραμμική Εξώθηση με Ιδιότητα Center (Δεξιός Κόμβος)

Τώρα εκτελέστε γραμμική εξώθηση στον δεξιό κόμβο, αυτή τη φορά **κεντράροντας την εξώθηση**. Η κλήση `setCenter(true)` δείχνει πώς **ορίζουμε την ιδιότητα center** σε *true*.

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

## Βήμα 7: Προσθήκη Ground Plane για Αναφορά (Δεξιός Κόμβος)

Όπως και στην αριστερή πλευρά, προσθέστε ένα ground plane στον δεξιό κόμβο για μια συνεπή οπτική βάση.

```java
right.createChildNode(new Box(0.01, 3, 3));
```

## Βήμα 8: Αποθήκευση της 3D Σκηνής

Αποθηκεύστε τη 3D σκηνή σας σε μορφή Wavefront OBJ ώστε να μπορείτε να τη δείτε σε οποιονδήποτε τυπικό 3D προβολέα.

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## Κοινά Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί Συμβαίνει | Λύση |
|----------|----------------|------|
| Το ground plane δεν είναι ορατό | Το πάχος του κουτιού είναι πολύ μικρό για την κλίμακα του προβολέα. | Αυξήστε το πάχος (πρώτη παράμετρος του `Box`) ή απομακρύνετε την προβολή. |
| Η εξώθηση εμφανίζεται μετατοπισμένη | Η τιμή του `setCenter` δεν έχει οριστεί όπως προοριζόταν. | Ελέγξτε ξανά το boolean που περνάτε στο `setCenter`. |
| Το αρχείο δεν αποθηκεύεται | Λανθασμένη διαδρομή φακέλου ή έλλειψη δικαιωμάτων εγγραφής. | Βεβαιωθείτε ότι το `MyDir` δείχνει σε υπάρχον φάκελο με δικαιώματα εγγραφής. |
