# Εφαρμογή Περιστροφής σε Γραμμική Εξώθηση με Aspose.3D για Java

## Εισαγωγή

Καλώς ήρθατε σε αυτό το βήμα‑βήμα tutorial σχετικά με **πώς να δημιουργήσετε ένα στριφτό 3D μοντέλο** εφαρμόζοντας μια περιστροφή κατά τη διάρκεια της γραμμικής εξώθησης χρησιμοποιώντας το Aspose.3D για Java. Είτε δημιουργείτε αρχιτεκτονικές απεικονίσεις, πόρους παιχνιδιών ή πρωτότυπα μηχανικής, η προσθήκη μιας περιστροφής μπορεί να δώσει στη γεωμετρία σας μια δυναμική, σπειροειδή εμφάνιση με λίγες μόνο γραμμές κώδικα.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει «περιστροφή» στην εξώθηση;** Περιστρέφει το προφίλ γύρω από τον άξονα εξώθησης καθώς το σχήμα επεκτείνεται.  
- **Ποια κλάση API διαχειρίζεται την περιστροφή;** `LinearExtrusion` παρέχει τη μέθοδο `setTwist`.  
- **Χρειάζομαι άδεια για να εκτελέσω το παράδειγμα;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να εξάγω το αποτέλεσμα ως OBJ;** Ναι, χρησιμοποιήστε `scene.save(..., FileFormat.WAVEFRONTOBJ)`.  
- **Ποια έκδοση Java απαιτείται;** Η Java 8 ή νεότερη υποστηρίζεται πλήρως.

## Προαπαιτούμενα

Πριν βουτήξετε στο tutorial, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:

- **Περιβάλλον Ανάπτυξης Java:** Βεβαιωθείτε ότι η Java είναι εγκατεστημένη στο σύστημά σας.  
- **Βιβλιοθήκη Aspose.3D:** Κατεβάστε και εγκαταστήστε τη βιβλιοθήκη Aspose.3D για Java από το [download link](https://releases.aspose.com/3d/java/).  
- **Τεκμηρίωση:** Ανατρέξτε στην [Aspose.3D documentation](https://reference.aspose.com/3d/java/) για ολοκληρωμένη καθοδήγηση.

## Εισαγωγή Πακέτων

Πριν ξεκινήσετε τη διαδικασία κωδικοποίησης, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας. Ακολουθεί ένα παράδειγμα:

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## Ορισμός Καταλόγου Εγγράφου

Πρώτα, ορίστε πού θα αποθηκευτεί το παραγόμενο αρχείο 3D.

```java
// ExStart:SetDocumentDirectory
String MyDir = "Your Document Directory";
// ExEnd:SetDocumentDirectory
```

## Αρχικοποίηση Βασικού Προφίλ

Στη συνέχεια, δημιουργήστε το σχήμα που θα εξαχθεί. Σε αυτό το παράδειγμα χρησιμοποιούμε ένα ορθογώνιο με μικρή ακτίνα στρογγυλοποίησης.

```java
// ExStart:InitializeBaseProfile
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
// ExEnd:InitializeBaseProfile
```

## Δημιουργία Σκηνής

Ένα αντικείμενο `Scene` λειτουργεί ως ο κοντέινερ για όλους τους 3D κόμβους.

```java
// ExStart:CreateScene
Scene scene = new Scene();
// ExEnd:CreateScene
```

## Δημιουργία Κόμβων

Προσθέστε δύο θυγατρικούς κόμβους στη σκηνή – ο ένας θα παραμείνει ευθύς, ο άλλος θα λάβει την περιστροφή.

```java
// ExStart:CreateNodes
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
// ExEnd:CreateNodes
```

## Γραμμική Εξώθηση με Περιστροφή

Τώρα εκτελούμε **γραμμική εξώθηση με περιστροφή** και στους δύο κόμβους. Ο αριστερός κόμβος λαμβάνει περιστροφή 0° (ευθύς), ενώ ο δεξιός λαμβάνει περιστροφή 90°, δημιουργώντας σπειροειδές σχήμα. Ορίζουμε επίσης τον αριθμό των φετών για ομαλή γεωμετρία.

```java
// ExStart:LinearExtrusionWithTwist
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(0); setSlices(100); }});
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(90); setSlices(100); }});
// ExEnd:LinearExtrusionWithTwist
```

## Εξαγωγή Αρχείου OBJ Java

Τέλος, αποθηκεύστε τη σκηνή σε μορφή OBJ, η οποία υποστηρίζεται ευρέως. Αυτό δείχνει τη δυνατότητα **export OBJ file Java** του Aspose.3D.

```java
// ExStart:Save3DScene
scene.save(MyDir + "TwistInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
// ExEnd:Save3DScene
```
