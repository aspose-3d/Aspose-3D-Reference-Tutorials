# Εκτέλεση Γραμμικής Εξώθησης στο Aspose.3D για Java

## Εισαγωγή

Καλώς ήρθατε σε αυτό το ολοκληρωμένο tutorial σχετικά με **how to extrude shape** στο Aspose.3D για Java! Εάν θέλετε να ενισχύσετε τις δεξιότητές σας στη 3D μοντελοποίηση χρησιμοποιώντας Java, βρίσκεστε στο σωστό μέρος. Θα σας καθοδηγήσουμε στη δημιουργία μιας 3D σκηνής, στην εκτέλεση γραμμικής εξώθησης και στην εξαγωγή του αποτελέσματος ως αρχείο Wavefront OBJ — όλα με σαφή, βήμα‑βήμα παραδείγματα κώδικα.

## Γρήγορες Απαντήσεις
- **Τι είναι η γραμμική εξώθηση;** Επέκταση ενός 2D προφίλ κατά μήκος μιας ευθείας διαδρομής για δημιουργία ενός 3D στερεού.  
- **Ποια βιβλιοθήκη το διαχειρίζεται σε Java;** Aspose.3D για Java.  
- **Μπορώ να εξάγω σε OBJ;** Ναι, χρησιμοποιώντας τη δυνατότητα εξαγωγής Wavefront OBJ.  
- **Χρειάζεται άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται άδεια για παραγωγή.  
- **Ποια έκδοση Java απαιτείται;** Java 1.6 ή νεότερη.

## Τι είναι το “how to extrude shape”;
Η γραμμική εξώθηση είναι μια βασική τεχνική στο **3d modeling java** που μετατρέπει ένα επίπεδο προφίλ — όπως ένα ορθογώνιο — σε ένα όγκο αντικειμένου, τραβώντας το κατά μήκος μιας καθορισμένης απόστασης. Αυτή η μέθοδος χρησιμοποιείται ευρέως για τη δημιουργία μηχανικών εξαρτημάτων, αρχιτεκτονικών στοιχείων και διακοσμητικών μοντέλων.

## Γιατί να χρησιμοποιήσετε το Aspose.3D για γραμμική εξώθηση;
- **Πλήρης έλεγχος** πάνω στη γεωμετρία (σπάις, στρίψιμο, μετατόπιση).  
- **Απρόσκοπτη ενσωμάτωση** με έργα Java — χωρίς εγγενείς εξαρτήσεις.  
- **Ενσωματωμένοι εξαγωγείς** για δημοφιλείς μορφές, συμπεριλαμβανομένου του Wavefront OBJ, καθιστώντας εύκολη τη **generate 3d model** δημιουργία αρχείων για επόμενες διαδικασίες.

## Προαπαιτούμενα

Πριν ξεκινήσετε το tutorial, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:

1. **Περιβάλλον Ανάπτυξης Java** – ένα JDK (1.6 ή νεότερο) και το αγαπημένο σας IDE.  
2. **Βιβλιοθήκη Aspose.3D** – κατεβάστε και εγκαταστήστε τη βιβλιοθήκη από την επίσημη ιστοσελίδα **[εδώ](https://releases.aspose.com/3d/java/)**.

## Εισαγωγή Πακέτων

Αφού ρυθμίσετε το περιβάλλον ανάπτυξης και εγκαταστήσετε τη βιβλιοθήκη Aspose.3D, εισάγετε το απαιτούμενο πακέτο:

```java
import com.aspose.threed.*;
```

### Βήμα 1: Ορισμός Καταλόγου Εγγράφου

Ορίστε το φάκελο όπου θα αποθηκευτούν τα παραγόμενα αρχεία:

```java
String MyDir = "Your Document Directory";
```

> Αυτό εξασφαλίζει ότι η λειτουργία **generate 3d model** γράφει το αρχείο OBJ σε γνωστή τοποθεσία.

### Βήμα 2: Αρχικοποίηση Βασικού Σχήματος

Δημιουργήστε ένα ορθογώνιο που θα χρησιμεύσει ως προφίλ εξώθησης:

```java
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

Μπορείτε να προσαρμόσετε την ακτίνα στρογγυλοποίησης για στρογγυλεμένες γωνίες ή να την ορίσετε σε `0` για τέλειο ορθογώνιο.

### Βήμα 3: Εκτέλεση Γραμμικής Εξώθησης

Τώρα εξωθούμε το ορθογώνιο κατά τον άξονα Z, προσθέτουμε σπάις, ενεργοποιούμε το κεντράρισμα και εφαρμόζουμε στρίψιμο με μετατόπιση:

```java
LinearExtrusion extrusion = new LinearExtrusion(profile, 10) {{ setSlices(100); setCenter(true); setTwist(360); setTwistOffset(new Vector3(10, 0, 0));}};
```

- **Μήκος εξώθησης** – `10` μονάδες.  
- **Σπάις** – `100` για ομαλή γεωμετρία.  
- **Στρίψιμο** – `360°` δημιουργεί πλήρη περιστροφή.  
- **Μετατόπιση στρίψιμου** – μετακινεί το σημείο προέλευσης του στρίψιμου στο `(10, 0, 0)`.

### Βήμα 4: Δημιουργία 3D Σκηνής

Δημιουργήστε ένα κοντέινερ σκηνής και προσθέστε την εξώθηση ως παιδί κόμβου. Αυτό το βήμα **creates 3d scene** που μπορεί να περιέχει πολλαπλά αντικείμενα:

```java
Scene scene = new Scene();
scene.getRootNode().createChildNode(extrusion);
```

### Βήμα 5: Αποθήκευση 3D Σκηνής

Εξάγετε τη σκηνή σε αρχείο Wavefront OBJ. Αυτό επιδεικνύει τις δυνατότητες **wavefront obj export** και **save 3d obj**:

```java
scene.save(MyDir +  "LinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

Μετά την εκτέλεση του κώδικα, θα βρείτε το `LinearExtrusion.obj` στον φάκελο που καθορίσατε, έτοιμο να ανοιχθεί σε οποιονδήποτε 3D προβολέα ή να υποβληθεί σε περαιτέρω επεξεργασία.

## Συχνά Προβ και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Το αρχείο OBJ είναι κενό | Η διαδρομή εξόδου είναι λανθασμένη ή δεν είναι εγγράψιμη | Επαληθεύστε ότι το `MyDir` δείχνει σε υπάρχον φάκελο με δικαιώματα εγγραφής. |
| Το στρίψιμο δεν εφαρμόζεται | Παραλείπεται το `setCenter(true)` | Βεβαιωθείτε ότι το κεντράρισμα είναι ενεργοποιημένο ή προσαρμόστε τις τιμές του `setTwistOffset`. |
| Σφάλμα μεταγλώττισης στο `LinearExtrusion` | Χρήση παλαιότερης έκδοσης Aspose.3D | Αναβαθμίστε στην πιο πρόσφατη έκδοση του Aspose.3D. |
