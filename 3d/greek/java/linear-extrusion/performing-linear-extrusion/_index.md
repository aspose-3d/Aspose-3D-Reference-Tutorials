# Δημιουργία 3D Εξώθησης Java με Aspose.3D

## Εισαγωγή

Αν χρειάζεστε **create 3d extrusion java** γρήγορα και αξιόπιστα, βρεθήκατε στο σωστό tutorial. Στα επόμενα λεπτά θα δούμε πώς να δημιουργήσετε μια γραμμική εξώθηση, να προσαρμόσετε τη γεωμετρία της και να **export obj file java** χρησιμοποιώντας τη βιβλιοθήκη Aspose.3D. Είτε χτίζετε ένα εργαλείο τύπου CAD, μια αλυσίδα παραγωγής assets για παιχνίδια, ή οποιοδήποτε Java‑based 3‑D workflow, αυτός ο οδηγός σας παρέχει μια σταθερή, έτοιμη για παραγωγή βάση.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει «γραμμική εξώθηση»;** Σαρώνει ένα 2‑D προφίλ κατά μήκος μιας ευθείας γραμμής για να δημιουργήσει ένα 3‑D στερεό.  
- **Ποια βιβλιοθήκη διαχειρίζεται την εξώθηση;** Το Aspose.3D for Java παρέχει ένα υψηλού επιπέδου API.  
- **Μπορώ να εξάγω το αποτέλεσμα ως OBJ;** Ναι – το tutorial ολοκληρώνεται με `scene.save(..., FileFormat.WAVEFRONTOBJ)`.  
- **Χρειάζεται άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για εκμάθηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποια έκδοση Java υποστηρίζεται;** Java 1.6 και νεότερες.

## Τι είναι το Create 3D Extrusion Java;
Η δημιουργία 3D εξώθησης σε Java σημαίνει ότι παίρνετε ένα απλό 2‑D σχήμα (π.χ. ένα ορθογώνιο) και το επεκτείνετε στην τρίτη διάσταση. Το προκύπτον mesh μπορεί να αποθηκευτεί σε κοινές μορφές όπως OBJ, τις οποίες κατανοούν πολλοί 3‑D επεξεργαστές.

## Γιατί να χρησιμοποιήσετε το Aspose.3D για Γραμμική Εξώθηση;
- **Pure Java API** – χωρίς εγγενείς εξαρτήσεις, ιδανικό για cross‑platform έργα.  
- **Πλούσιο έλεγχο γεωμετρίας** – στρογγυλοποίηση, στρίψιμο, slices και offset εκτίθενται μέσω απλών ιδιοτήτων.  
- **Άμεση εξαγωγή** – αποθήκευση σε OBJ, STL, FBX και άλλα χωρίς επιπλέον μετατροπείς.  
- **Υποστήριξη επιχειρηματικού επιπέδου** – άδειες, τεκμηρίωση και φόρουμ κοινότητας είναι άμεσα διαθέσιμα.

## Προαπαιτούμενα

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε:

1. **Java Development Environment** – εγκατεστημένο και ρυθμισμένο JDK 1.6+.  
2. **Aspose.3D Library** – Κατεβάστε το τελευταίο JAR από την επίσημη σελίδα [here](https://releases.aspose.com/3d/java/).  

## Εισαγωγή Πακέτων

Προσθέστε τον κύριο χώρο ονομάτων Aspose.3D στο αρχείο πηγαίου κώδικα Java:

```java
import com.aspose.threed.*;
```

## Βήμα 1: Ορισμός Καταλόγου Εγγράφου

Καθορίστε πού θα γραφτούν τα παραγόμενα αρχεία:

```java
String MyDir = "Your Document Directory";
```

> **Συμβουλή:** Χρησιμοποιήστε απόλυτη διαδρομή ή μια ρυθμιζόμενη ιδιότητα ώστε η τοποθεσία εξόδου να λειτουργεί σε διαφορετικά περιβάλλοντα.

## Βήμα 2: Αρχικοποίηση Βασικού Σχήματος

Δημιουργήστε ένα ορθογώνιο που θα λειτουργήσει ως προφίλ εξώθησης. Η ακτίνα στρογγυλοποίησης μαλακώνει τις γωνίες:

```java
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

Μπορείτε να πειραματιστείτε με το `setRoundingRadius` για πιο κυκλικό ή πιο αιχμηρό προφίλ.

## Βήμα 3: Εκτέλεση Γραμμικής Εξώθησης

Τώρα μετατρέπουμε το 2‑D ορθογώνιο σε 3‑D αντικείμενο. Ο κατασκευαστής δέχεται το προφίλ και το βάθος εξώθησης (10 μονάδες σε αυτήν την περίπτωση). Πρόσθετες ιδιότητες ρυθμίζουν λεπτομερώς το mesh:

```java
LinearExtrusion extrusion = new LinearExtrusion(profile, 10) {{ setSlices(100); setCenter(true); setTwist(360); setTwistOffset(new Vector3(10, 0, 0));}};
```

- **Slices** – ελέγχει την ομαλότητα της εξώθησης.  
- **Center** – ευθυγραμμίζει τη γεωμετρία γύρω από το αρχικό σημείο.  
- **Twist** – περιστρέφει το προφίλ κατά τον άξονα εξώθησης (εδώ πλήρες 360°).  
- **TwistOffset** – μετακινεί τον άξονα στρίψιμου, δείχνοντας πώς δημιουργούνται σπείρες.

## Βήμα 4: Δημιουργία 3D Σκηνής

Ένα `Scene` είναι το δοχείο για όλα τα 3‑D αντικείμενα. Η προσθήκη της εξώθησης ως παιδικού κόμβου την ενσωματώνει στο γράφημα σκηνής:

```java
Scene scene = new Scene();
scene.getRootNode().createChildNode(extrusion);
```

## Βήμα 5: Αποθήκευση 3D Σκηνής

Τέλος, εξάγετε τη σκηνή σε αρχείο Wavefront OBJ – μορφή που υποστηρίζεται ευρέως από 3‑D επεξεργαστές, μηχανές παιχνιδιών και pipelines εκτύπωσης:

```java
scene.save(MyDir +  "LinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

Μετά την εκτέλεση του κώδικα, θα βρείτε το **LinearExtrusion.obj** στον καθορισμένο κατάλογο, έτοιμο να ανοιχθεί στο Blender, Maya ή οποιονδήποτε άλλο προβολέα συμβατό με OBJ.

## Συχνά Προβλήματα και Λύσεις

| Σύμπτωμα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| `FileNotFoundException` κατά την αποθήκευση | `MyDir` δεν υπάρχει ή δεν έχει δικαίωμα εγγραφής | Δημιουργήστε το φάκελο εκ των προτέρων ή χρησιμοποιήστε απόλυτη διαδρομή με τις κατάλληλες άδειες. |
| Το OBJ εμφανίζεται κενό στον προβολέα | Δεν προστέθηκε γεωμετρία στη σκηνή | Επαληθεύστε ότι το αντικείμενο `LinearExtrusion` έχει δημιουργηθεί σωστά και έχει προσαρτηθεί στον ριζικό κόμβο. |
| Το Twist φαίνεται λανθασμένο | Λανθασμένες τιμές `TwistOffset` | Προσαρμόστε τις συντεταγμένες του `Vector3`; θυμηθείτε ότι το offset εφαρμόζεται πριν από την περιστροφή twist. |
