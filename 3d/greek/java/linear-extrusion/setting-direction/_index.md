# Δημιουργία 3D Σκηνής – Ορισμός Κατεύθυνσης Εξώθησης Aspose.3D Java

## Εισαγωγή

Σε αυτό το tutorial θα μάθετε **πώς να δημιουργήσετε 3d σκηνή** αντικείμενα και να ελέγξετε την κατεύθυνση εξώθησης με το Aspose.3D για Java. Είτε δημιουργείτε αρχιτεκτονικές απεικονίσεις, πρωτότυπα προϊόντων ή στοιχεία παιχνιδιών, η εξοικείωση με τη γραμμική εξώθηση σας δίνει την ευελιξία να μοντελοποιείτε σύνθετα σχήματα γρήγορα. Θα περάσουμε από κάθε βήμα, από την προσθήκη κόμβων σε Java μέχρι την **εξαγωγή 3d μοντέλου obj** αρχείων, ώστε να δείτε το αποτέλεσμα άμεσα.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “create 3d scene”;** Σημαίνει την αρχικοποίηση ενός αντικειμένου Aspose.3D `Scene` που θα περιέχει όλα τα γεωμετρικά στοιχεία, τα φώτα και τις κάμερες.  
- **Πώς ορίζω την κατεύθυνση εξώθησης;** Χρησιμοποιήστε τη μέθοδο `setDirection(Vector3)` σε ένα αντικείμενο `LinearExtrusion`.  
- **Ποια μορφή πρέπει να χρησιμοποιήσω για εξαγωγή;** Η μορφή OBJ (`FileFormat.WAVEFRONTOBJ`) υποστηρίζεται ευρέως για 3‑D ροές εργασίας.  
- **Χρειάζομαι άδεια για το Aspose.3D;** Η δωρεάν δοκιμή λειτουργεί για ανάπτυξη· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να προσθέσω περισσότερους κόμβους σε Java;** Ναι—χρησιμοποιήστε `scene.getRootNode().createChildNode()` για να προσθέσετε όσους αντικειμενικούς κόμβους χρειάζεστε.

## Τι είναι η ροή εργασίας “create 3d scene”;

Μια ροή εργασίας **create 3d scene** ξεκινά με ένα κενό αντικείμενο `Scene`, προσθέτει γεωμετρία (όπως εξωθημένα προφίλ), την τοποθετεί με μετασχηματισμούς και τελικά αποθηκεύει τη σκηνή σε μορφή αρχείου όπως OBJ. Αυτό το μοτίβο αποτελεί τη ραχοκοκαλιά των περισσότερων 3‑D εφαρμογών που δημιουργούνται με το Aspose.3D.

## Γιατί να ορίσετε την κατεύθυνση εξώθησης;

Ορίζοντας την κατεύθυνση εξώθησης μπορείτε να κλίνετε, περιστρέψετε ή παραμορφώσετε το σχήμα κατά την εξώση, παρέχοντας έλεγχο πάνω στη τελική γεωμετρία χωρίς πρόσθετη μετα‑επεξεργασία. Είναι ιδιαίτερα χρήσιμο για:

- Δημιουργία στενών στηλών ή σωλήνων προσαρμοσμένου σχήματος.  
- Στοίχιση εξωθήσεων με μη τυπικούς άξονες σε μηχανικά μέρη.  
- Δημιουργία καλλιτεχνικών σχημάτων για οπτικά εφέ.

## Προαπαιτούμενα

- Βασικές γνώσεις Java.  
- Βιβλιοθήκη Aspose.3D εγκατεστημένη – κατεβάστε την από [εδώ](https://releases.aspose.com/3d/java/).  
- Ένα IDE όπως το Eclipse ή το IntelliJ IDEA.

## Εισαγωγή Πακέτων

Πρώτα, εισάγετε τις απαιτούμενες κλάσεις του Aspose.3D:

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## Βήμα 1: Αρχικοποίηση Βασικού Προφίλ

Δημιουργήστε το 2‑D προφίλ που θα εξωθηθεί. Σε αυτό το παράδειγμα χρησιμοποιούμε ένα στρογγυλεμένο ορθογώνιο:

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

> **Συμβουλή:** Ρυθμίστε την ακτίνα στρογγυλοποίησης για να ελέγξετε πόσο αιχμηρές ή ομαλές εμφανίζονται οι γωνίες του ορθογωνίου πριν την εξώθηση.

## Βήμα 2: Δημιουργία Σκηνής

Τώρα **δημιουργούμε 3d σκηνή** που θα φιλοξενήσει τα αντικείμενά μας:

```java
Scene scene = new Scene();
```

## Βήμα 3: Προσθήκη Κόμβων Java – Τοποθέτηση των Αντικειμένων

Θα προσθέσουμε δύο παιδικούς κόμβους (αριστερό και δεξιό) στον ριζικό κόμβο της σκηνής και θα μετακινήσουμε τον αριστερό λίγο προς τα πλάγια:

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## Βήμα 4: Πώς να εξώσετε – Αριστερός Κόμβος (προεπιλεγμένη κατεύθυνση)

Εξωθήστε το προφίλ στον αριστερό κόμβο χρησιμοποιώντας την προεπιλεγμένη κατεύθυνση του άξονα Z. Ορίζουμε επίσης μια πλήρη περιστροφή 360° και αυξάνουμε τον αριθμό των φέτες για ομαλότητα:

```java
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); }});
```

## Βήμα 5: Πώς να ορίσετε την κατεύθυνση – Δεξιός Κόμβος

Εδώ **πώς να ορίσετε την κατεύθυνση** παρέχοντας ένα προσαρμένο `Vector3`. Αυτό κλίνει την εξώθηση προς το διάνυσμα (0.3, 0.2, 1):

```java
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); setDirection(new Vector3(0.3, 0.2, 1));}});
```

## Βήμα 6: Αποθήκευση αρχείου OBJ – Εξαγωγή 3D μοντέλου

Τέλος, **αποθηκεύουμε το αρχείο obj** για να δείτε το αποτέλεσμα σε οποιονδήποτε 3‑D προβολέα:

```java
scene.save(MyDir + "DirectionInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

Όταν ανοίξετε το παραγόμενο αρχείο OBJ, θα δείτε δύο εξωθημένα ορθογώνια: ένα με την προεπιλεγμένη κατεύθυνση και ένα κλινόμενο σύμφωνα με το διάνυσμα που ορίσαμε.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| Το αρχείο OBJ φαίνεται κενό | Η σκηνή δεν αποθηκεύτηκε ή το μονοπάτι είναιθασμένο | Επαληθεύστε ότι το `MyDir` δείχνει σε φάκελο με δικαιώματα εγγραφής και ότι το όνομα αρχείου λήγει σε `.obj`. |
| Η εξώθηση φαίνεται επίπεδη | Το `setSlices` είναι πολύ χαμηλό | Αυξήστε το `setSlices` (π.χ., 200) για πιο ομαλή γεωμετρία. |
| Η κατεύθυνση φαίνεται αμετάβλητη | Το διάνυσμα δεν είναι κανονικοποιημένο | Χρησιμοποιήστε ένα κανονικοποιημένο `Vector3` ή προσαρμόστε τις τιμές ώστε να αντανακλούν την επιθυμητή κλίση. |
