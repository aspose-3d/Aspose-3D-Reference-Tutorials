# Αλλαγή χρώματος υφής και διαχείριση 3D ιδιοτήτων σε σκηνές Java χρησιμοποιώντας Aspose.3D

## Εισαγωγή

Σε αυτό το **Aspose 3D tutorial** θα ανακαλύψετε πώς να **αλλάξετε το χρώμα της υφής** και να εργαστείτε με 3D ιδιότητες και προσαρμοσμένα δεδομένα μέσα σε σκηνές Java. Είτε δημιουργείτε ένα παιχνίδι, έναν οπτικοποιητή προϊόντων ή έναν επιστημονικό προβολέα, η δυνατότητα τροποποίησης των χαρακτηριστικών του υλικού σε χρόνο εκτέλεσης σας δίνει πλήρη καλλιτεχνικό έλεγχο. Ας περάσουμε βήμα‑βήμα από τη φόρτωση μιας σκηνής μέχρι τη ρύθμιση του χρώματος *Diffuse* χρησιμοποιώντας μια τιμή `Vector3`.

## Γρήγορες Απαντήσεις
- **Τι μπορώ να τροποποιήσω;** Μπορείτε να αλλάξετε το χρώμα της υφής, τη διαφάνεια, τη λάμψη και οποιαδήποτε προσαρμοσμένη ιδιότητα συνδεδεμένη με ένα υλικό.  
- **Ποια κλάση κρατά τα δεδομένα;** `Material` και η `PropertyCollection` της.  
- **Πώς ορίζω νέο χρώμα;** Χρησιμοποιήστε `props.set("Diffuse", new Vector3(r, g, b))`.  
- **Χρειάζομαι άδεια;** Μια προσωρινή άδεια λειτουργεί για αξιολόγηση· απαιτείται πλήρης άδεια για παραγωγή.  
- **Υποστηριζόμενες μορφές;** FBX, OBJ, STL, GLTF και πολλές άλλες.

## Προαπαιτούμενα

- Java Development Kit (JDK) 8 ή νεότερο εγκατεστημένο.  
- Βιβλιοθήκη Aspose.3D for Java (κατεβάστε από την [Ιστοσελίδα Aspose](https://releases.aspose.com/3d/java/)).  
- Βασική εξοικείωση με τη σύνταξη της Java και τις αντικειμενοστραφείς έννοιες.

## Εισαγωγή Πακέτων

Πριν γράψετε οποιαδήποτε λογική, εισάγετε τις κλάσεις που σας δίνουν πρόσβαση στις ιδιότητες του υλικού και στη διαχείριση διανυσμάτων.

```java
import java.io.IOException;

import com.aspose.threed.Material;
import com.aspose.threed.Property;
import com.aspose.threed.PropertyCollection;
import com.aspose.threed.Scene;
import com.aspose.threed.Vector3;
```

### Γιατί να εισάγετε αυτές τις κλάσεις;

- `Scene` φορτώνει και αντιπροσωπεύει το 3D αρχείο.  
- `Material` σας παρέχει τον ορισμό της επιφάνειας (υφές, χρώματα κ.λπ.).  
- `PropertyCollection` είναι ένας τύπος δομής τύπου λεξικού που σας επιτρέπει να **προσπελάσετε τις ιδιότητες του υλικού** με όνομα.  
- `Vector3` είναι ο τύπος δεδομένων που χρησιμοποιείται για χρώματα και άλλα διανύσματα τριών συνιστωσών.

## Βήμα 1: Αρχικοποίηση της Σκηνής

```java
String dataDir = "Your Document Directory";
Scene scene = new Scene(dataDir + "EmbeddedTexture.fbx");
```

Δημιουργούμε ένα αντικείμενο `Scene` φορτώνοντας ένα αρχείο FBX που ήδη περιέχει υφή. Αυτό είναι το καμβάς πάνω στον οποίο θα **αλλάξουμε το χρώμα της υφής**.

## Βήμα 2: Πρόσβαση στις ιδιότητες υλικού

```java
Material material = scene.getRootNode().getChildNodes().get(0).getMaterial();
PropertyCollection props = material.getProperties();
```

Εδώ **προσπελάζουμε τις ιδιότητες του υλικού** του πρώτου πλέγματος στη σκηνή. Το αντικείμενο `Material` περιέχει μια `PropertyCollection` που αποθηκεύει κάθε ρυθμιζόμενο χαρακτηριστικό, όπως *Diffuse*, *Specular* και προσαρμοσμένα δεδομένα χρήστη.

## Βήμα 3: Λίστα όλων των ιδιοτήτων

```java
for (Property prop : props) {
    System.out.println("Name" + prop.getName() + " Value = " + prop.getValue());
}
```

Η επανάληψη πάνω στο `props` εκτυπώνει κάθε όνομα ιδιότητας και την τρέχουσα τιμή της. Αυτό το γρήγορο απόθεμα σας βοηθά να ανακαλύψετε ποια κλειδιά μπορείτε αργότερα να τροποποιήσετε, π.χ. `"Diffuse"` για το βασικό χρώμα.

## Βήμα 4: Ορισμός τιμής Vector3 για αλλαγή χρώματος υφής

```java
props.set("Diffuse", new Vector3(1, 0, 1));
```

**Pro tip:** Ο κατασκευαστής `Vector3` δέχεται τρεις αριθμούς κινητής υποδιαστολής που αντιπροσωπεύουν τις συνιστώσες **κόκκινο, πράσινο και μπλε** (εύρος 0‑1). Ορίζοντας `(1, 0, 1)` αλλάζει το βασικό χρώμα της υφής σε ματζέντα, αλλάζοντας ουσιαστικά **το χρώμα της υφής** του μοντέλου.

## Βήμα 5: Ανάκτηση ιδιότητας με όνομα

```java
Object diffuse = (Vector3) props.get("Diffuse");
System.out.println(diffuse);
```

Αυτό δείχνει **ανάκτηση ιδιότητας με όνομα**. Κάνουμε cast το επιστρεφόμενο `Object` σε `Vector3` για να εργαστούμε προγραμματιστικά με το χρώμα.

## Βήμα 6: Πρόσβαση στο αντικείμενο ιδιότητας

```java
Property pdiffuse = props.findProperty("Diffuse");
System.out.println(pdiffuse);
```

`findProperty` επιστρέφει το πλήρες αντικείμενο `Property`, δίνοντάς σας πρόσβαση σε μεταδεδομένα όπως ο τύπος της ιδιότητας, η ετικέτα και τυχόν προσαρμοσμένα δεδομένα.

## Βήμα 7: Διάσχιση υπο‑ιδιοτήτων της ιδιότητας

```java
for (Property pp : pdiffuse.getProperties()) {
    System.out.println("Diffuse. " + pp.getName() + " = " + pp.getValue());
}
```

Κάποιες ιδιότητες είναι ιεραρχικές. Η διάσχιση του `pdiffuse.getProperties()` σας δείχνει τυχόν ένθετες ιδιότητες (π.χ. συντεταγμένες υφής, κλειδιά animation) που ανήκουν στην καταχώρηση *Diffuse*.

## Συνηθισμένα Προβλήματα & Λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **`NullPointerException` on `material`** | Ο κόμβος μπορεί να μην έχει εκχωρημένο υλικό. | Κλήση `node.setMaterial(new Material())` πριν την πρόσβαση στις ιδιότητες. |
| **Color does not change** | Το μοντέλο χρησιμοποιεί υφή που παρακάμπτει το χρώμα *Diffuse*. | Απενεργοποιήστε την υφή ή τροποποιήστε άμεσα την εικόνα της υφής. |
| **`ClassCastException` when retrieving** | Προσπάθεια cast μιας ιδιότητας που δεν είναι Vector3. | Επαληθεύστε τον τύπο της ιδιότητας με `pdiffuse.getValue().getClass()` πριν το cast. |
