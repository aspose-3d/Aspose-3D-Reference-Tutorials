# Δημιουργία Πρωτότυπων 3Δ Μοντέλων με Aspose.3D για Java

## Εισαγωγή

Αν ποτέ χρειάστηκε να **δημιουργήσετε αντικείμενα 3Δ κυλίνδρου** (ή οποιοδήποτε άλλο βασικό σχήμα) απευθείας από κώδικα Java, το Aspose.3D κάνει την εργασία αυτή άνετη. Σε αυτό το tutorial θα περάσουμε από όλη τη ροή εργασίας — από τη ρύθμιση του περιβάλλοντος μέχρι την αποθήκευση της τελικής σκηνής ως αρχείο FBX — ώστε να μπορείτε να αρχίσετε να παράγετε 3Δ γεωμετρία προγραμματιστικά αμέσως.

## Γρήγορες Απαντήσεις
- **Ποια βιβλιοθήκη μου επιτρέπει να δημιουργήσω 3Δ κύλινδρο σε Java;** Aspose.3D for Java.  
- **Σε ποια μορφή μπορώ να εξάγω τη σκηνή;** FBX (ASCII 7500) χρησιμοποιώντας `FileFormat.FBX7500ASCII`.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια δωρεάν δοκιμή λειτουργεί για δοκιμές· απαιτείται μόνιμη άδεια για παραγωγή.  
- **Ποια είναι τα κύρια primitives που υποστηρίζονται;** Box, Cylinder, Sphere, Cone και άλλα.  
- **Είναι ο κώδικας συμβατός με Java 8 και νεότερες εκδόσεις;** Ναι, το Aspose.3D στοχεύει σε JDK 8+.  

## Τι είναι ένα 3Δ primitive κυλίνδρου;

Ένα primitive κυλίνδρου είναι ένα βασικό γεωμετρικό σχήμα που ορίζεται από ακτίνα και ύψος. Σε πολλές 3Δ αλυσίδες επεξεργασίας χρησιμεύει ως δομικό στοιχείο για πιο σύνθετα μοντέλα όπως σωλήνες, τροχοί ή αρχιτεκτονικές κολώνες. Το Aspose.3D παρέχει μια έτοιμη κλάση `Cylinder`, ώστε να μην χρειάζεται να υπολογίζετε χειροκίνητα τις κορυφές.

## Γιατί να χρησιμοποιήσετε Aspose.3D για Java;

- **Πλήρης μηχανή 3Δ** – υποστηρίζει εισαγωγή/εξαγωγή δημοφιλών μορφών (FBX, OBJ, STL κ.λπ.).  
- **Καθαρό Java API** – χωρίς εγγενείς εξαρτήσεις, ιδανικό για跨‑πλατφόρμα έργα.  
- **Ανθεκτικό γράφημα σκηνής** – σας επιτρέπει να οργανώνετε αντικείμενα ιεραρχικά.  
- **Εύκολη αδειοδότηση** – ξεκινήστε με δωρεάν δοκιμή, στη συνέχεια αναβαθμίστε σε μόνιμη άδεια.  

## Προαπαιτούμενα

Πριν βυθιστείτε στον κώδικα, βεβαιωθείτε ότι έχετε:

1. **Java Development Kit (JDK)** – εγκατεστημένο JDK 8 ή νεότερο στη μηχανή σας.  
2. **Aspose.3D for Java library** – κατεβάστε και εγκαταστήστε το από τη [σελίδα λήψης](https://releases.aspose.com/3d/java/).  

## Εισαγωγή Πακέτων

Ξεκινήστε εισάγοντας τον πυρήνα του χώρου ονομάτων Aspose.3D. Αυτό σας δίνει πρόσβαση στα `Scene`, `Box`, `Cylinder` και στα σταθερά μορφής αρχείου.

```java
import com.aspose.threed.*;
```

Τώρα που η βιβλιοθήκη έχει αναφερθεί, ας δημιουργήσουμε μια σκηνή και να προσθέσουμε κάποια primitive γεωμετρία.

## Πώς να δημιουργήσετε 3Δ κύλινδρο και άλλα primitives σε μια σκηνή

### Βήμα 1: Αρχικοποίηση Αντικειμένου Scene

Πρώτα, χρειαζόμαστε ένα κοντέινερ `Scene` που θα κρατά όλους τους 3Δ κόμβους μας.

```java
// The path to the documents directory.
String myDir = "Your Document Directory";

// Initialize a Scene object
Scene scene = new Scene();
```

### Βήμα 2: Κατασκευή μοντέλου 3Δ κουτιού

Το primitive του κουτιού είναι χρήσιμο για δοκιμή του συστήματος συντεταγμένων. Εδώ το προσθέτουμε ως παιδί του ριζικού κόμβου της σκηνής.

```java
// Create a Box model
scene.getRootNode().createChildNode("box", new Box());
```

### Βήμα 3: Δημιουργία μοντέλου 3Δ κυλίνδρου

Τώρα δημιουργούμε πραγματικά τη **γεωμετρία 3Δ κυλίνδρου**. Η κλάση `Cylinder` έρχεται με λογικές προεπιλεγμένες διαστάσεις, αλλά μπορείτε να προσαρμόσετε την ακτίνα και το ύψος μέσω του κατασκευαστή της αν χρειαστεί.

```java
// Create a Cylinder model
scene.getRootNode().createChildNode("cylinder", new Cylinder());
```

### Βήμα 4: Αποθήκευση του σχεδίου σε μορφή FBX

Αφού συναρμολογήσετε τη σκηνή, εξάγετε την ώστε άλλα εργαλεία (π.χ., Unity, Blender) να μπορούν να την διαβάσουν. Χρησιμοποιούμε τη μορφή `FBX7500ASCII`, η οποία υποστηρίζεται ευρέως.

```java
// Save drawing in the FBX format
myDir = myDir + "test.fbx";
scene.save(myDir, FileFormat.FBX7500ASCII);
```

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Γιατί συμβαίνει | Διόρθωση |
|----------|----------------|----------|
| **File not found** κατά την αποθήκευση | `myDir` δείχνει σε φάκελο που δεν υπάρχει | Βεβαιωθείτε ότι ο φάκελος υπάρχει ή δημιουργήστε τον με `new File(myDir).mkdirs();` |
| **Empty scene** μετά την εξαγωγή | Δεν προστέθηκαν κόμβοι πριν κληθεί το `save` | Επαληθεύστε ότι οι κλήσεις `createChildNode` εκτελούνται (αποσφαλματώστε με `scene.getRootNode().getChildCount()` ) |
| **License exception** | Εκτέλεση χωρίς έγκυρη άδεια σε παραγωγή | Εφαρμόστε προσωρινή ή μόνιμη άδεια χρησιμοποιώντας `License license = new License(); license.setLicense("Aspose.3D.Java.lic");` |
