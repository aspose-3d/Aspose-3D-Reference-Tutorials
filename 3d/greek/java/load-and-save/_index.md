## Συχνά Προβλήματα και Λύσεις

MaterialConverter είναι μια βοηθητική κλάση που μετατρέπει παλαιότερους ορισμούς υλικών σε μορφή Physically Based Rendering (PBR).

**Πρόβλημα:** *Τα υλικά χάνουν υφή μετά τη μετατροπή.*  
**Λύση:** Χρησιμοποιήστε `MaterialConverter.convertToPbr()` πριν από την αποθήκευση· διατηρεί τους χάρτες υφής και ενημερώνει αυτόματα τις παραμέτρους shader.

LoadOptions είναι μια κλάση διαμόρφωσης που ελέγχει πώς το Aspose.3D αναλύει τα αρχεία εισόδου.

**Πρόβλημα:** *Μεγάλα αρχεία OBJ προκαλούν OutOfMemoryError.*  
**Λύση:** Ενεργοποιήστε τη ροή μέσω `LoadOptions.setEnableMemoryOptimization(true)` και αυξήστε το μέγεθος της μνήμης heap της JVM (`-Xmx4g`).

SaveOptions διαμορφώνει τις ρυθμίσεις εξόδου όπως μορφή, συμπίεση και ένταξη χαρακτηριστικών κατά την αποθήκευση μιας σκηνής.

**Πρόβλημα:** *Τα εξαγόμενα αρχεία FBX λείπουν τα δεδομένα κίνησης.*  
**Λύση:** Ορίστε `SaveOptions.setExportAnimations(true)` και βεβαιωθείτε ότι η πηγή σκηνής περιέχει κλειδωμένα animation tracks.
