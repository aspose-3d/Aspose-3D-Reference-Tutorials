# Δημιουργία 3D Σκηνής Java – Εφαρμογή Υλικών PBR με Aspose.3D

## Εισαγωγή

Σε αυτό το πρακτικό tutorial θα μάθετε **πώς να δημιουργήσετε μια 3D σκηνή σε Java** και να την εμπλουτίσετε με υλικά Physically Based Rendering (PBR) χρησιμοποιώντας τη βιβλιοθήκη Aspose.3D. Στο τέλος του οδηγού θα μπορείτε να αποδώσετε ρεαλιστικές επιφάνειες και **να αποθηκεύσετε το 3D μοντέλο STL** για περαιτέρω χρήση σε οποιοδήποτε 3D pipeline.

## Γρήγορες Απαντήσεις
- **Τι σημαίνει “create 3d scene java”;** Αυτή είναι η διαδικασία δημιουργίας ενός αντικειμένου Scene που περιέχει γεωμετρία, φωτισμούς και κάμερες σε μια εφαρμογή Java.  
- **Ποια βιβλιοθήκη διαχειρίζεται τα υλικά PBR;** Η Aspose.3D παρέχει μια έτοιμη κλάση `PbrMaterial`.  
- **Μπορώ να εξάγω το αποτέλεσμα ως STL;** Ναι – η μέθοδος `Scene.save` υποστηρίζει STL (ASCII και binary).  
- **Χρειάζομαι άδεια για παραγωγή;** Απαιτείται μόνιμη άδεια Aspose.3D για εμπορική χρήση· μια προσωρινή άδεια λειτουργεί για δοκιμές.  
- **Ποια έκδοση Java απαιτείται;** Οποιοδήποτε runtime Java 8+ υποστηρίζεται.

## Τι είναι μια 3D σκηνή σε Java;

Μια *σκηνή* είναι ο κοντέινερ που περιέχει όλα τα αντικείμενα (κόμβοι), τη γεωμετρία τους, τα υλικά, τους φωτισμούς και τις κάμερες. Σκεφτείτε το ως το εικονικό σκηνικό στο οποίο τοποθετείτε τα 3D μοντέλα σας. Η κλάση `Scene` της Aspose.3D σας παρέχει έναν καθαρό, αντικειμενοστραφή τρόπο να χτίσετε αυτό το σκηνικό προγραμματιστικά.

## Γιατί να χρησιμοποιήσετε υλικά PBR για απόδοση 3D αντικειμένων σε Java;

Το PBR (Physically Based Rendering) μιμείται την αλληλεπίδραση του φωτός στον πραγματικό κόσμο χρησιμοποιώντας παραμέτρους όπως οι παράγοντες metallic και roughness. Το αποτέλεσμα είναι μια πιο πειστική εμφάνιση κάτω από διαφορετικές συνθήκες φωτισμού, κάτι που είναι ιδιαίτερα πολύτιμο για οπτικοποίηση προϊόντων, παιχνίδια ή εμπειρίες AR/VR.

## Εισαγωγή Πακέτων

Προσθέστε το namespace της Aspose.3D στο αρχείο πηγαίου κώδικα Java σας:

```java
import com.aspose.threed.*;
```

## Βήμα 1: Αρχικοποίηση Σκηνής

Δημιουργήστε τη σκηνή που θα λειτουργήσει ως καμβάς για τη γεωμετρία και τα υλικά σας.

```java
// ExStart:InitializeScene
String MyDir = "Your Document Directory";
Scene scene = new Scene();
// ExEnd:InitializeScene
```

> **Συμβουλή:** Κρατήστε το `MyDir` να δείχνει σε φάκελο με δικαιώματα εγγραφής· διαφορετικά η κλήση `save` θα αποτύχει.

## Βήμα 2: Αρχικοποίηση Υλικού PBR

Διαμορφώστε ένα υλικό PBR με τιμές metallic και roughness που δίνουν μια σχεδόν μεταλλική εμφάνιση.

```java
// ExStart:InitializePBRMaterial
PbrMaterial mat = new PbrMaterial();
mat.setMetallicFactor(0.9);
mat.setRoughnessFactor(0.9);
// ExEnd:InitializePBRMaterial
```

> **Γιατί αυτές οι τιμές;** Ένας υψηλός παράγοντας metallic (≈ 0.9) κάνει την επιφάνεια να συμπεριφέρεται σαν μέταλλο, ενώ μια υψηλή roughness (≈ 0.9) προσθέτει ήπια διάχυση, αποτρέποντας ένα τέλειο καθρέφτη.

## Βήμα 3: Δημιουργία 3D Αντικειμένου και Εφαρμογή του Υλικού

Εδώ δημιουργούμε μια απλή γεωμετρία κουτιού, την συνδέουμε με τον ριζικό κόμβο της σκηνής και αναθέτουμε το υλικό PBR που μόλις δημιουργήσαμε.

```java
// ExStart:Create3DObject
Node boxNode = scene.getRootNode().createChildNode("box", new Box());
boxNode.setMaterial(mat);
// ExEnd:Create3DObject
```

> **Κοινό λάθος:** Η παράλειψη ορισμού του υλικού στον κόμβο θα αφήσει το αντικείμενο με την προεπιλεγμένη (μη‑PBR) εμφάνιση.

## Βήμα 4: Αποθήκευση 3D Σκηνής (Εξαγωγή STL)

Εξάγετε ολόκληρη τη σκηνή—συμπεριλαμβανομένου του κουτιού με βελτιωμένο PBR—σε αρχείο STL. Το STL είναι μια ευρέως υποστηριζόμενη μορφή για 3‑D εκτύπωση και γρήγορους οπτικούς ελέγχους.

```java
// ExStart:Save3DScene
scene.save(MyDir + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
// ExEnd:Save3DScene
```

Μπορείτε επίσης να επιλέξετε `FileFormat.STLBINARY` εάν απαιτείται μικρότερο μέγεθος αρχείου.

## Συνηθισμένα Προβλήματα και Λύσεις

| Πρόβλημα | Πιθανή Αιτία | Διόρθωση |
|----------|--------------|----------|
| **Αρχείο δεν αποθηκεύτηκε** | `MyDir` δείχνει σε φάκελο που δεν υπάρχει ή δεν έχει δικαίωμα εγγραφής | Επαληθεύστε ότι ο φάκελος υπάρχει και η διαδικασία Java έχει δικαίωμα εγγραφής |
| **Το υλικό φαίνεται επίπεδο** | Οι τιμές Metallic/Roughness είναι 0 | Αυξήστε το `setMetallicFactor` και/ή το `setRoughnessFactor` |
| **Το αρχείο STL δεν μπορεί να ανοίξει** | Λάθος σημαία μορφής αρχείου (ASCII vs) για το πρόγραμμα προβολής | Χρησιμοποιήστε το αντίστοιχο enum `FileFormat` για το πρόγραμμα προβολής σας |
