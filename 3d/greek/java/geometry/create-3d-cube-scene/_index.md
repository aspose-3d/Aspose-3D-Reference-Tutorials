## Πώς να δημιουργήσετε σκηνή 3d με το Aspose.3D

Φορτώστε, διαμορφώστε και αποθηκεύστε μια βασική σκηνή κύβου σε τέσσερα σύντομα βήματα: αρχικοποιήστε το `Scene`, δημιουργήστε ένα `Node` με ένα `Mesh` κύβου, συνδέστε το node στο γράφημα σκηνής και τέλος γράψτε το αρχείο στο δίσκο. Αυτή η ροή από άκρη σε άκρη σας επιτρέπει να δημιουργήσετε ένα πλήρες αρχείο 3‑D σε λιγότερο από ένα λεπτό, ακόμη και σε μέτριο υλικό.

### Βήμα 1: Αρχικοποίηση της Σκηνής

Δημιουργήστε ένα νέο αντικείμενο Scene που θα λειτουργήσει ως ρίζα της ιεραρχίας 3‑D.

```java
// Initialize scene object
Scene scene = new Scene();
```

### Βήμα 2: Αρχικοποίηση Node και Mesh

Δημιουργήστε ένα Node και ένα Mesh που αντιπροσωπεύει έναν κύβο για προσθήκη στη σκηνή.

```java
// Initialize Node class object
Node cubeNode = new Node("box");

// Call Common class create mesh using polygon builder method to set mesh instance
Mesh mesh = Common.createMeshUsingPolygonBuilder();

// Point node to the Mesh geometry
cubeNode.setEntity(mesh);
```

### Βήμα 3: Προσθήκη Node στη Σκηνή

Συνδέστε το node του κύβου στο γράφημα σκηνής ώστε να γίνει μέρος του μοντέλου.

```java
// Add Node to a scene
scene.getRootNode().getChildNodes().add(cubeNode);
```

### Βήμα 4: Αποθήκευση της Σκηνής 3D

Αποθηκεύστε τη σκηνή σε αρχείο στην επιθυμητή μορφή χρησιμοποιώντας τη μέθοδο save.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
MyDir = MyDir + "CubeScene.fbx";

// Save 3D scene in the supported file formats
scene.save(MyDir, FileFormat.FBX7400ASCII);
```

### Βήμα 5: Εκτύπωση Μηνύματος Επιτυχίας

Εκτυπώστε ένα μήνυμα επιβεβαίωσης που υποδεικνύει ότι το αρχείο δημιουργήθηκε επιτυχώς.

```java
System.out.println("\nCube Scene created successfully.\nFile saved at " + MyDir);
```

| Πρόβλημα | Αιτία | Διόρθωση |

| **`FileFormat.FBX7400ASCII` not recognized** | Χρήση παλαιότερης έκδοσης Aspose.3D. | Αναβαθμίστε στην τελευταία JAR του Aspose.3D όπου το enum είναι διαθέσιμο. |
| **Output file is empty** | Ο φάκελος προορισμού δεν υπάρχει. | Βεβαιωθείτε ότι το `MyDir` δείχνει σε έγκυρο φάκελο ή δημιουργήστε το προγραμματιστικά. |
