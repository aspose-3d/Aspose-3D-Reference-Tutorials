# Πώς να ορίσετε άδεια στο Aspose.3D για Java

## Εισαγωγή

Καλώς ήρθατε! Σε αυτό το βήμα‑βήμα tutorial θα ανακαλύψετε **πώς να ορίσετε άδεια** για το Aspose.3D σε μια εφαρμογή Java και επίσης θα μάθετε **πώς να χρησιμοποιείτε δημόσια/ιδιωτικά κλειδιά** για μετρημένη άδεια. Το Aspose.3D είναι μια ισχυρή βιβλιοθήκη Java που απλοποιεί την εργασία με μορφές αρχείων 3D, και μια έγκυρη άδεια ξεκλειδώνει το πλήρες σύνολο λειτουργιών της. Στο τέλος αυτού του οδηγού θα μπορείτε να ενσωματώσετε την άδεια άψογα σε οποιοδήποτε έργο Java.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος τρόπος για να ορίσετε μια άδεια;** Χρησιμοποιήστε την κλάση `License` και καλέστε το `setLicense` με διαδρομή αρχείου ή ροή.  
- **Μπορώ να φορτώσω την άδεια από ροή;** Ναι – ένα `FileInputStream` λειτουργεί τέλεια.  
- **Για τι χρησιμεύουν τα δημόσια/ιδιωτικά κλειδιά;** Επιτρέπουν τη μετρημένη άδεια μέσω της κλάσης `Metered`.  
- **Χρειάζομαι άδεια για ανάπτυξη;** Μια προσωρινή ή δοκιμαστική άδεια είναι επαρκής για δοκιμές· απαιτείται πλήρης άδεια για παραγωγή.  
- **Ποιες εκδόσεις Java υποστηρίζονται;** Το Aspose.3D λειτουργεί με Java 6 και νεότερες.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε:

- Βασική κατανόηση του προγραμματισμού Java.  
- Τη βιβλιοθήκη Aspose.3D προστιθέμενη στο έργο σας. Κατεβάστε την από τη [σελίδα κυκλοφορίας](https://releases.aspose.com/3d/java/).  
- Ένα έγκυρο αρχείο `.lic` ή τα δημόσια και ιδιωτικά κλειδιά μετρημένης άδειας.

## Εισαγωγή Πακέτων

Προσθέστε τις απαιτούμενες εισαγωγές στο αρχείο πηγαίου κώδικα Java. Βεβαιωθείτε ότι το JAR του Aspose.3D βρίσκεται στο classpath.

```java
import com.aspose.threed.License;
import com.aspose.threed.Metered;

import java.io.FileInputStream;
import java.io.IOException;
```

## Πώς να ορίσετε άδεια χρησιμοποιώντας αρχείο

### Βήμα 1: Δημιουργία αντικειμένου License

Δημιουργήστε μια παρουσία της κλάσης `License` – αυτό το αντικείμενο θα κρατά τις πληροφορίες άδειας.

```java
License license = new License();
```

### Βήμα 2: Ορισμός άδειας από αρχείο

Δώστε τη σχετική ή απόλυτη διαδρομή προς το αρχείο `.lic` σας και εφαρμόστε την.

```java
license.setLicense("Aspose._3D.lic");
```

> **Συμβουλή:** Κρατήστε το αρχείο άδειας εκτός του καταλόγου ελέγχου εκδόσεων για να αποφύγετε τυχαία αποκάλυψη.

## Πώς να ορίσετε άδεια χρησιμοποιώντας ροή

### Βήμα 1: Δημιουργία αντικειμένου License

Όπως και πριν, ξεκινήστε με μια νέα παρουσία `License`.

```java
License license = new License();
```

### Βήμα 2: Ορισμός άδειας από ροή

Διαβάστε το αρχείο άδειας σε ένα `FileInputStream` και περάστε τη ροή στο `setLicense`. Το μπλοκ try‑with‑resources εγγυάται ότι η ροή κλείνει αυτόματα.

```java
try (FileInputStream myStream = new FileInputStream("Aspose._3D.lic")) {
    license.setLicense(myStream);
}
```

## Πώς να χρησιμοποιήσετε δημόσια/ιδιωτικά κλειδιά για μετρημένη άδεια

### Βήμα 1: Αρχικοποίηση αντικειμένου Metered License

Δημιουργήστε μια παρουσία της κλάσης `Metered`, η οποία διαχειρίζεται τη μετρημένη (pay‑as‑you‑go) άδεια.

```java
Metered metered = new Metered();
```

### Βήμα 2: Ορισμός δημόσιου και ιδιωτικού κλειδιού

Παρέχετε τα κλειδιά που λάβατε από το Aspose. Αυτά τα κλειδιά επιτρέπουν στη βιβλιοθήκη να αναφέρει τη χρήση στον διακομιστή αδειών.

```java
metered.setMeteredKey("your-public-key", "your-private-key");
```

> **Προειδοποίηση:** Ποτέ μην κωδικοποιείτε σκληρά το ιδιωτικό σας κλειδί σε ένα JAR που διανέμεται δημόσια. Σκεφτείτε να το φορτώνετε από ασφαλή τοποθεσία ή μεταβλητή περιβάλλοντος.

## Συνηθισμένες περιπτώσεις χρήσης

- **Συστήματα απόδοσης 3D επιχείρησης** – ξεκλειδώστε APIs υψηλής απόδοσης μετά τον ορισμό της άδειας.  
- **Περιβάλλοντα αυτοματοποιημένων δοκιμών** – χρησιμοποιήστε προσωρινή άδεια (δείτε τις Συχνές Ερωτήσεις) για να επαληθεύσετε τη λειτουργικότητα χωρίς αγορά πλήρους άδειας.  
- **Λύσεις SaaS με μετρημένη χρέωση** – ενσωματώστε δημόσια/ιδιωτικά κλειδιά για να χρεώνετε τους πελάτες βάσει πραγματικής χρήσης.
