# Πώς να ορίσετε την άδεια Aspose στο Aspose.3D για Java

## Εισαγωγή

Σε αυτό το ολοκληρωμένο tutorial θα ανακαλύψετε **πώς να ορίσετε την άδεια Aspose** για το Aspose.3D σε περιβάλλον Java. Είτε προτιμάτε τη φόρτωση ενός αρχείου άδειας, τη ροή του, είτε τη χρήση μετρημένης άδειας με δημόσια και ιδιωτικά κλειδιά, θα περάσουμε από κάθε προσέγγιση βήμα‑βήμα ώστε να μπορείτε να ξεκλειδώσετε πλήρως τις δυνατότητες του Aspose.3D γρήγορα και με σιγουριά.

## Γρήγορες Απαντήσεις
- **Ποιος είναι ο κύριος τρόπος για να ορίσετε μια άδεια Aspose.3D;** Χρησιμοποιήστε την κλάση `License` και καλέστε `setLicense` με διαδρομή αρχείου ή ροή.  
- **Μπορώ να φορτώσω την άδεια από ροή;** Ναι – τυλίξτε το αρχείο `.lic` σε ένα `FileInputStream` και περάστε το στο `setLicense`.  
- **Τι κάνω αν χρειάζομαι μετρημένη άδεια;** Αρχικοποιήστε ένα αντικείμενο `Metered` και καλέστε `setMeteredKey` με τα δημόσια και ιδιωτικά κλειδιά σας.  
- **Χρειάζομαι άδεια για εκδόσεις ανάπτυξης;** Απαιτείται δοκιμαστική ή προσωρινή άδεια για οποιοδήποτε σενάριο εκτός αξιολόγησης.  
- **Ποιες εκδόσεις Java υποστηρίζονται;** Το Aspose.3D λειτουργεί με Java 6 και μεταγενέστερες.

## Προαπαιτούμενα

Πριν ξεκινήσουμε, βεβαιωθείτε ότι έχετε τα παρακάτω προαπαιτούμενα:

- Βασική κατανόηση του προγραμματισμού Java.  
- Η βιβλιοθήκη Aspose.3D εγκατεστημένη. Μπορείτε να τη κατεβάσετε από τη [σελίδα κυκλοφορίας](https://releases.aspose.com/3d/java/).  

## Εισαγωγή Πακέτων

Για να ξεκινήσετε, εισάγετε τα απαραίτητα πακέτα στο έργο Java σας. Βεβαιωθείτε ότι το Aspose.3D έχει προστεθεί στο classpath. Ακολουθεί ένα παράδειγμα:

```java
import com.aspose.threed.License;
import com.aspose.threed.Metered;

import java.io.FileInputStream;
import java.io.IOException;
```

## Εφαρμογή Άδειας Χρησιμοποιώντας Αρχείο

### Βήμα 1: Δημιουργία Αντικειμένου License

Αρχικά, δημιουργήστε ένα αντικείμενο `License` στον κώδικα Java.

```java
License license = new License();
```

### Βήμα 2: Εφαρμογή Άδειας από Αρχείο

Καθορίστε τη διαδρομή προς το αρχείο άδειας και ορίστε την άδεια χρησιμοποιώντας τον παρακάτω κώδικα. Αυτό δείχνει την τεχνική **apply license from file**.

```java
license.setLicense("Aspose._3D.lic");
```

## Εφαρμογή Άδειας Χρησιμοποιώντας Αντικείμενο Ροής

### Βήμα 1: Δημιουργία Αντικειμένου License

Αναλόγως, δημιουργήστε ένα αντικείμενο `License` στον κώδικα Java.

```java
License license = new License();
```

### Βήμα 2: Ορισμός Άδειας από Αντικείμενο Ροής

Χρησιμοποιήστε ένα `FileInputStream` για να δημιουργήσετε ροή και ορίστε την άδεια:

```java
try (FileInputStream myStream = new FileInputStream("Aspose._3D.lic")) {
    license.setLicense(myStream);
}
```

## Χρήση Δημόσιων και Ιδιωτικών Κλειδιών για Μετρημένη Άδεια

### Βήμα 1: Αρχικοποίηση Αντικειμένου Metered License

Αρχικοποιήστε ένα αντικείμενο `Metered` license:

```java
Metered metered = new Metered();
```

### Βήμα 2: Ορισμός Δημόσιων και Ιδιωτικών Κλειδιών

Ορίστε τα δημόσια και ιδιωτικά σας κλειδιά για να ενεργοποιήσετε τη μετρημένη άδεια. Αυτό καλύπτει το σενάριο **set public private keys**.

```java
metered.setMeteredKey("your-public-key", "your-private-key");
```

## Γιατί η Ρύθμιση της Άδειας Είναι Σημαντική

Η σωστή εφαρμογή της άδειας αφαιρεί τα υδατογράμματα αξιολόγησης, ξεκλειδώνει premium μορφές αρχείων και εξασφαλίζει τη συμμόρφωση με το μοντέλο αδειοδότησης της Aspose. Η χρήση της κατάλληλης μεθόδου (αρχείο, ροή ή μετρημένη) σας επιτρέπει να ενσωματώσετε την αδειοδότηση αβίαστα σε CI/CD pipelines, cloud deployments ή desktop εφαρμογές.

## Συχνά Προβλήματα & Συμβουλές

- **Αρχείο δεν βρέθηκε** – Επαληθεύστε ότι η διαδρομή του αρχείου `.lic` είναι σωστή σε σχέση με τον τρέχοντα φάκελο ή χρησιμοποιήστε απόλυτη διαδρομή.  
- **Η ροή κλείνει πρόωρα** – Όταν χρησιμοποιείτε ροή, διατηρήστε το αντικείμενο `License` ενεργό για όλη τη διάρκεια της εφαρμογής· η άδεια αποθηκεύεται στην κρυφή μνήμη μετά την πρώτη επιτυχημένη κλήση.  
- **Ασυμφωνία κλειδιού Metered** – Ελέγξτε ξανά ότι τα δημόσια και ιδιωτικά κλειδιά αντιστοιχούν στην ίδια μετρημένη άδεια· ένα τυπογραφικό λάθος θα προκαλέσει εξαίρεση χρόνου εκτέλεσης.  
- **Συμβουλή επαγγελματία:** Αποθηκεύστε το αρχείο άδειας σε ασφαλή τοποθεσία εκτός του δέντρου πηγαίου κώδικα και φορτώστε το μέσω μεταβλητής περιβάλλοντος για να αποφύγετε την καταχώρηση του σε σύστημα ελέγχου εκδόσεων.
