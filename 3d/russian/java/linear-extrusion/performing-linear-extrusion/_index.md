# Выполнение линейной экструзии в Aspose.3D для Java

## Введение

Добро пожаловать в этот всесторонний учебник по **how to extrude shape** в Aspose.3D для Java! Если вы хотите улучшить свои навыки 3D‑моделирования с помощью Java, вы попали по адресу. Мы пошагово покажем, как создать 3D‑сцену, выполнить линейную экструзию и экспортировать результат в файл Wavefront OBJ — всё с понятными примерами кода.

## Быстрые ответы
- **Что такое линейная экструзия?** Расширение 2D‑профиля вдоль прямой траектории для создания 3D‑твердого тела.  
- **Какая библиотека обрабатывает это в Java?** Aspose.3D for Java.  
- **Можно ли экспортировать в OBJ?** Да, с помощью функции экспорта Wavefront OBJ.  
- **Нужна ли лицензия для разработки?** Бесплатная пробная версия подходит для тестирования; для продакшна требуется лицензия.  
- **Какая версия Java требуется?** Java 1.6 или новее.

## Что такое “how to extrude shape”?
Линейная экструзия — фундаментальная техника в **3d modeling java**, которая превращает плоский профиль, например прямоугольник, в объёмный объект, вытягивая его на заданное расстояние. Этот метод широко используется для создания механических деталей, архитектурных элементов и декоративных моделей.

## Почему использовать Aspose.3D для линейной экструзии?
- **Полный контроль** над геометрией (срезы, скручивание, смещение).  
- **Бесшовная интеграция** с Java‑проектами — без нативных зависимостей.  
- **Встроенные экспортеры** популярных форматов, включая Wavefront OBJ, что упрощает **generate 3d model** файлы для последующих конвейеров.

## Импорт пакетов

После настройки среды разработки и установки библиотеки Aspose.3D импортируйте необходимые пакеты:

```java
import com.aspose.threed.*;
```

### Шаг 1: Установить каталог документа

Определите папку, в которую будут сохраняться сгенерированные файлы:

```java
String MyDir = "Your Document Directory";
```

> Это гарантирует, что операция **generate 3d model** запишет файл OBJ в известное место.

### Шаг 2: Инициализировать базовую форму

Создайте прямоугольник, который будет служить профилем для экструзии:

```java
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

Вы можете изменить радиус скругления, чтобы получить закруглённые углы, или установить его в `0` для идеального прямоугольника.

### Шаг 3: Выполнить линейную экструзию

Теперь мы экструдируем прямоугольник вдоль оси Z, добавляем срезы, включаем центрирование и применяем скручивание со смещением:

```java
LinearExtrusion extrusion = new LinearExtrusion(profile, 10) {{ setSlices(100); setCenter(true); setTwist(360); setTwistOffset(new Vector3(10, 0, 0));}};
```

- **Длина экструзии** — `10` единиц.  
- **Срезы** — `100` для плавной геометрии.  
- **Скручивание** — `360°` создаёт полный оборот.  
- **Смещение скручивания** — перемещает начало скручивания в точку `(10, 0, 0)`.

### Шаг 4: Создать 3D‑сцену

Создайте контейнер сцены и добавьте экструзию как дочерний узел. Этот шаг **creates 3d scene**, который может содержать несколько объектов:

```java
Scene scene = new Scene();
scene.getRootNode().createChildNode(extrusion);
```

### Шаг 5: Сохранить 3D‑сцену

Экспортируйте сцену в файл Wavefront OBJ. Это демонстрирует возможности **wavefront obj export** и **save 3d obj**:

```java
scene.save(MyDir +  "LinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

После выполнения кода вы найдёте `LinearExtrusion.obj` в указанном каталоге, готовый к открытию в любом 3D‑просмотрщике или дальнейшей обработке.

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| OBJ‑файл пустой | Неправильный путь к выходному каталогу или отсутствие прав записи | Убедитесь, что `MyDir` указывает на существующую папку с правами записи. |
| Скручивание не применилось | Пропущен вызов `setCenter(true)` | Включите центрирование или скорректируйте значения `setTwistOffset`. |
| Ошибка компиляции на `LinearExtrusion` | Используется более старая версия Aspose.3D | Обновите до последней версии Aspose.3D. |
