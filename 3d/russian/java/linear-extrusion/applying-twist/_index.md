# Создание 3D‑сцены с кручением при линейной экструзии – Aspose.3D for Java

## Введение

В этом практическом **java 3d tutorial** вы узнаете, как **создавать 3d‑сцену**, применять *кручение при линейной экструзии* и в конце **экспортировать obj java** файлы с помощью Aspose.3D for Java. Независимо от того, создаёте ли вы игровой объект, прототип CAD или визуальный эффект, добавление кручения во время экструзии придаёт вашим моделям динамический, спиральный вид, который трудно достичь обычной экструзией.

## Быстрые ответы

- **Можно ли экспортировать результат в OBJ?** Да – используйте `FileFormat.WAVEFRONTOBJ`.

- **Какая версия Java требуется?** Java 8 или выше.

## Что такое «кручение» при линейной экструзии?

Кручение – это преобразование, которое вращает каждый срез экструдированной формы на заданный угол. Управляя углом, вы можете создавать спирали, винтовые формы или лёгкие скручивания, добавляющие визуальный интерес к иначе плоским экструзиям.

```java
import com.aspose.threed.*;

```

## Шаг 1: Установите каталог документа

Определите, куда будет сохраняться сгенерированный OBJ‑файл. Замените заполнители реальным путём к папке на вашей системе.

```java
// ExStart:SetDocumentDirectory
String MyDir = "Your Document Directory";
// ExEnd:SetDocumentDirectory
```

## Шаг 2: Инициализируйте базовый профиль

Создайте форму, которую будете экструдировать. Здесь используется прямоугольник с небольшим радиусом скругления, чтобы края выглядели мягче.

```java
// ExStart:InitializeBaseProfile
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
// ExEnd:InitializeBaseProfile
```

## Шаг 3: Создайте сцену для размещения узлов

Объект `Scene` служит контейнером для всех 3‑D‑сущностей (мэш, свет, камера и т.д.).  

```java
// ExStart:CreateScene
Scene scene = new Scene();
// ExEnd:CreateScene
```

## Шаг 4: Добавьте левый и правый узлы

Мы создадим два sibling‑узла: один без кручения (для сравнения) и один с кручением на 90 градусов.

```java
// ExStart:CreateNodes
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
// ExEnd:CreateNodes
```

## Шаг 5: Выполните линейную экструзию с кручением

Конструктор `LinearExtrusion` принимает профиль и длину экструзии.  
- `setTwist(0)` → без вращения (прямая экструзия).  
- `setTwist(90)` → полное вращение на 90° вдоль длины.  
Оба узла используют 100 срезов для плавной геометрии.

```java
// ExStart:LinearExtrusionWithTwist
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(0); setSlices(100); }});
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(90); setSlices(100); }});
// ExEnd:LinearExtrusionWithTwist
```

## Шаг 6: Сохраните 3D‑сцену как OBJ

Наконец, запишите сцену в OBJ‑файл, чтобы её можно было открыть в любом стандартном 3‑D‑просмотрщике.

```java
// ExStart:Save3DScene
scene.save(MyDir + "TwistInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
// ExEnd:Save3DScene
```

## Распространённые проблемы и советы

- **Ошибки пути к файлу:** Убедитесь, что `MyDir` заканчивается разделителем пути (`/` или `\\`), соответствующим вашей ОС.  
- **Слишком большой угол кручения:** Углы выше 360° могут вызвать наложение геометрии; держите их в диапазоне 0‑360° для предсказуемых результатов.  
- **Производительность:** Увеличение `setSlices` повышает гладкость, но может увеличить расход памяти; 100 срезов – хороший компромисс для большинства случаев.
