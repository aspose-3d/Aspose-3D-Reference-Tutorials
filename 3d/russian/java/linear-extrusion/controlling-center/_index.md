# Учебник по 3D графике Java – Центр в линейной экструзии

## Введение

Если вы создаёте 3D визуализации на Java, освоение техник экструзии является обязательным. Этот **java 3d graphics tutorial** проведёт вас через управление центром линейной экструзии с помощью Aspose.3D for Java, позволяя создавать точные, симметричные модели без дополнительной математики. К концу руководства вы поймёте, почему свойство `center` важно, как **установить радиус скругления**, и как **сохранить OBJ файл java**‑совместимый вывод.

## Быстрые ответы
- **Что делает свойство center?** Оно определяет, будет ли экструзия симметричной относительно начала профиля.  
- **Нужна ли лицензия для запуска кода?** Временная лицензия подходит для тестирования; полная лицензия требуется для продакшна.  
- **В каком формате сохраняется результат?** Сцена сохраняется в виде файла Wavefront OBJ.  
- **Можно ли изменить количество срезов?** Да, используйте `setSlices(int)` у объекта `LinearExtrusion`.  
- **Совместим ли Aspose.3D с Java 8+?** Абсолютно – поддерживает все современные версии Java.

## Пошаговое руководство

### Шаг 1: Создание базового профиля

Сначала создайте 2‑D форму, которую будете экструзировать. Здесь используется прямоугольник с **установленным радиусом скругления** 0.3, что сглаживает углы.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

### Шаг 2: Создание 3D сцены

Объект `Scene` служит контейнером для всех 3‑D узлов, источников света и камер.

```java
Scene scene = new Scene();
```

### Шаг 3: Добавление левого и правого узлов

Мы разместим два отдельных узла рядом, чтобы вы могли сравнить экструзию с центрированием и без него.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

### Шаг 4: Линейная экструзия – без центра (левый узел)

Создайте экструзию на левом узле, отключите центрирование и ограничьте сетку тремя срезами для низкополигонального предварительного просмотра.

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

### Шаг 5: Добавление плоской земли для ориентира (левый узел)

Тонкий короб служит визуальным полом, помогая увидеть ориентацию экструзии.

```java
left.createChildNode(new Box(0.01, 3, 3));
```

### Шаг 6: Линейная экструзия – центрировано (правый узел)

Теперь повторите экструзию, но включив `center`. Геометрия будет симметричной относительно начала профиля.

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

### Шаг 7: Добавление плоской земли для ориентира (правый узел)

Тот же самый пол для правой стороны, чтобы сравнение было очевидным.

```java
right.createChildNode(new Box(0.01, 3, 3));
```

### Шаг 8: Сохранение 3D сцены

Наконец, экспортируйте всю сцену в файл Wavefront OBJ – формат, который поддерживается большинством 3‑D редакторов.

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|----------|
| **Экструзия смещена** | `setCenter(false)` оставляет профиль привязанным к его углу. | Используйте `setCenter(true)` для симметричных результатов. |
| **OBJ файл пустой** | Неправильный путь к выходному каталогу или отсутствие прав записи. | Убедитесь, что `MyDir` указывает на существующую папку и приложение имеет права записи. |
| **Скруглённые углы выглядят острыми** | Радиус скругления слишком мал относительно размеров прямоугольника. | Увеличьте значение радиуса (например, `setRoundingRadius(0.5)`). |
