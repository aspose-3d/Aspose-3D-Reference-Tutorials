# Центр управления в линейной экструзии с Aspose.3D для Java

## Введение

Когда вы создаёте 3D‑сцены на Java, возможность **добавить плоскость пола** и точно **установить свойство center** у линейной экструзии может стать разницей между простым прототипом и полированным визуалом. В этом руководстве мы пройдём весь процесс управления центром экструзии и добавления плоскости пола с помощью Aspose.3D для Java. Вы узнаете, почему это важно, как настроить окружение и получите готовый пример кода, который можно адаптировать под свои проекты.

## Быстрые ответы
- **Что делает «add ground plane»?** Создаёт тонкую вспомогательную геометрию, помогающую увидеть положение экструзии относительно мировых осей.  
- **Как установить центр линейной экструзии?** Используйте метод `setCenter(boolean)` у объекта `LinearExtrusion`.  
- **Нужна ли лицензия для запуска примера?** Временная лицензия подходит для тестирования; полная лицензия требуется для продакшна.  
- **В каком формате сохраняется файл?** Пример сохраняет в формате Wavefront OBJ (`.obj`).  
- **Какая версия Java требуется?** Достаточно Java 8 или новее.

## Что такое «add ground plane» в 3D‑сцене?

Добавление плоскости пола означает вставку тонкой прямоугольной геометрии (часто коробки с минимальной толщиной), расположенной в плоскости X‑Z. Она служит визуальным полом, упрощая оценку высоты и выравнивания других объектов, особенно при работе с центрами экструзии.

## Почему стоит задавать свойство center у линейной экструзии?

По умолчанию линейная экструзия начинается из начала координат профиля. Установка свойства center (`setCenter(true)`) смещает профиль так, чтобы экструзия была центрирована вокруг начала координат, что удобно для симметричных дизайнов или когда требуется согласованное выравнивание нескольких объектов.

import com.aspose.threed.*;

## Шаг 1: Настройка базового профиля

Инициализируйте базовый профиль, который будет экструдирован. В этом примере мы используем прямоугольную форму с радиусом скругления 0.3.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

## Шаг 2: Создание 3D‑сцены

Постройте основу вашего 3D‑мира, создав сцену.

```java
Scene scene = new Scene();
```

## Шаг 3: Создание левых и правых узлов

Создайте левые и правые узлы внутри сцены. Эти узлы служат холстом для ваших 3D‑объектов.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## Шаг 4: Линейная экструзия со свойством Center (левый узел)

Выполните линейную экструзию в левом узле **без центрирования** и задайте количество срезов 3. Обратите внимание на вызов `setCenter(false)` – здесь мы **устанавливаем свойство center** в *false*.

```java
left.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(false); setSlices(3); }});
```

## Шаг 5: Добавление плоскости пола для справки (левый узел)

Улучшите визуальное представление, **добавив плоскость пола** к левому узлу. Тонкая коробка выступает в роли пола, позволяя чётко увидеть высоту экструзии.

```java
left.createChildNode(new Box(0.01, 3, 3));
```

## Шаг 6: Линейная экструзия со свойством Center (правый узел)

Теперь выполните линейную экструзию в правом узле, на этот раз **центрируя экструзию**. Вызов `setCenter(true)` демонстрирует, как **установить свойство center *true*.

```java
right.createChildNode(new LinearExtrusion(profile, 2) {{ setCenter(true); setSlices(3); }});
```

## Шаг 7: Добавление плоскости пола для справки (правый узел)

Точно так же, как и слева, добавьте плоскость пола к правому узлу для согласованного визуального базиса.

```java
right.createChildNode(new Box(0.01, 3, 3));
```

## Шаг 8: Сохранение 3D‑сцены

Сохраните вашу 3D‑сцену в формате Wavefront OBJ, чтобы её можно было просмотреть в любом стандартном 3D‑просмотрщике.

```java
scene.save(MyDir + "CenterInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|---------| Плоскость пола не видна | Толщина коробки слишком мала для масштаба просмотрщика. | Увеличьте толщину (первый параметр `Box`) или отдалитесь в просмотрщике. |
| Экструзия смещена | Значение `setCenter` задано неверно. | Проверьте, какой булевый параметр передаётся в `setCenter`. |
| Файл не сохраняется | Неправильный путь к каталогу или отсутствуют права записи. | Убедитесь, что `MyDir` указывает на существующую папку с правами записи. |
