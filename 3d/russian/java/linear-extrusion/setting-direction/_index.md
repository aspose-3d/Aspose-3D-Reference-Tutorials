# Создание 3D-сцены – Установка направления экструзии Aspose.3D Java

## Введение

В этом руководстве вы узнаете **как создавать 3d сцену** объектов и управлять направлением экструзии с помощью Aspose.3D для Java. Независимо от того, создаёте ли вы архитектурные визуализации, прототипы продуктов или игровые ассеты, освоение линейной экструзии дает гибкость для быстрого моделирования сложных форм. Мы пройдём каждый шаг, от добавления узлов в Java до **экспорта 3d модели в формате obj**, чтобы вы могли сразу увидеть результат.

## Быстрые ответы
- **Что означает «создать 3d сцену»?** Это инициализация объекта Aspose.3D `Scene`, который будет содержать всю геометрию, свет и камеры.  
- **Как задать направление экструзии?** Используйте метод `setDirection(Vector3)` у экземпляра `LinearExtrusion`.  
- **Какой формат использовать для экспорта?** Формат OBJ (`FileFormat.WAVEFRONTOBJ`) широко поддерживается в 3‑D рабочих процессах.  
- **Нужна ли лицензия для Aspose.3D?** Бесплатная пробная версия подходит для разработки; коммерческая лицензия требуется для продакшна.  
- **Можно ли добавить больше узлов в Java?** Да — используйте `scene.getRootNode().createChildNode()` для добавления необходимого количества объектов.

## Что такое процесс «создания 3d сцены»?

Процесс **создания 3d сцены** начинается с пустого объекта `Scene`, добавляет геометрию (например, экструдированные профили), позиционирует её с помощью трансформаций и в конце сохраняет сцену в файловый формат, такой как OBJ. Этот шаблон является основой большинства 3‑D приложений, построенных на Aspose.3D.

## Почему нужно задавать направление экструзии?

Задание направления экструзии позволяет наклонять, вращать или искажать форму во время экструдирования, предоставляя контроль над конечной геометрией без дополнительной пост‑обработки. Это особенно полезно для:
- Создания сужающихся колонн или труб нестандартной формы.  
- Выравнивания экструдированных элементов по нестандартным осям в механических деталях.  
- Генерации художественных форм для визуальных эффектов.

## Импорт пакетов

Сначала импортируйте необходимые классы Aspose.3D:

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## Шаг 1: Инициализация базового профиля

Создайте 2‑D профиль, который будет экструдирован. В этом примере мы используем скруглённый прямоугольник:

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

> **Совет:** Регулируйте радиус скругления, чтобы контролировать насколько острыми или плавными выглядят углы прямоугольника до экструдирования.

## Шаг 2: Создание сцены

Теперь мы **создаём 3d сцену**, которая будет содержать наши объекты:

```java
Scene scene = new Scene();
```

## Шаг 3: Добавление узлов Java – Позиционирование объектов

Мы добавим два дочерних узла (левый и правый) к корневому узлу сцены и немного сместим левый в сторону:

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## Шаг 4: Как выполнить экструдирование – Левый узел (направление по умолчанию)

Экструдируйте профиль на левом узле, используя направление оси Z по умолчанию. Мы также задаём полный поворот на 360° и увеличиваем количество срезов для плавности:

```java
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); }});
```

## Шаг 5: Как задать направление – Правый узел

Здесь мы **запрограммируем задание направления** предоставляя пользовательский `Vector3`. Это наклоняет экструдирование в сторону вектора (0.3, 0.2, 1):

```java
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); setDirection(new Vector3(0.3, 0.2, 1));}});
```

## Шаг 6: Сохранение OBJ‑файла – Экспорт 3D‑модели

Наконец, мы **сохраняем obj‑файл**, чтобы увидеть результат в любом 3‑D просмотрщике:

```java
scene.save(MyDir + "DirectionInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

Когда вы откроете сгенерированный OBJ‑файл, вы увидите два экструдированных прямоугольника: один с направлением по умолчанию, а другой наклонённый согласно заданному вектору.

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|----------|
| OBJ‑файл пустой | Сцена не сохранена или путь неверный | Убедитесь, что `MyDir` указывает на папку с правом записи, и имя файла заканчивается на `.obj`. |
| Экструзия выглядит плоской | `setSlices` слишком мало | Увеличьте `setSlices` (например, до 200) для более плавной геометрии. |
| Направление не изменилось | Вектор не нормализован | Используйте нормализованный `Vector3` или скорректируйте значения, чтобы отразить требуемый наклон. |
