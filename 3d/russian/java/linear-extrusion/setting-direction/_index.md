# Как задать направление при линейной экструзии с Aspose.3D для Java

## Введение

В этом подробном руководстве вы узнаете **как задать направление** при выполнении линейной экструзии с Aspose.3D для Java. Независимо от того, создаёте ли вы инструмент, похожий на CAD, или генерируете геометрию для игрового движка, управление направлением экструзии позволяет получить именно ту форму, которая вам нужна. Мы пройдём каждый шаг, от инициализации профиля до сохранения результата в файл OBJ, чтобы вы также могли **экспортировать 3d model obj** файлы напрямую из Java.

## Быстрые ответы
- **Какой основной класс для линейной экструзии?** `LinearExtrusion`
- **Какой метод задаёт направление экструзии?** `setDirection(Vector3 direction)`
- **Можно ли экспортировать результат как OBJ?** Да, используя `scene.save(..., FileFormat.WAVEFRONTOBJ)`
- **Нужна ли лицензия для разработки?** Доступна бесплатная пробная версия; для продакшн‑использования требуется лицензия.
- **Какая IDE для Java лучше всего?** Поддерживаются IntelliJ IDEA и Eclipse.

## Что такое линейная экструзия?

Линейная экструзия берёт 2‑D профиль (например, прямоугольник или круг) и растягивает его вдоль прямой линии, создавая 3‑D твёрдое тело. По умолчанию экструзия следует вдоль положительной оси Z, но Aspose.3D позволяет изменить этот путь с помощью свойства `setDirection`.

## Почему стоит задавать направление при линейной экструзии?

Задание собственного направления полезно, когда:
- Нужно выровнять геометрию с существующими объектами в сцене.
- Требуется создать наклонные или косые детали без дополнительных трансформаций.
- Экспортируются модели, которые должны соответствовать определённой системе координат (например, для 3‑D печати или игровых движков).

## Предварительные требования

Прежде чем приступить, убедитесь, что у вас есть:

- Базовые знания Java.
- Установленная библиотека Aspose.3D. Скачать её можно [здесь](https://releases.aspose.com/3d/java/).
- IDE, такая как Eclipse или IntelliJ IDEA.

## Импорт пакетов

Сначала импортируйте пространства имён, предоставляющие основные 3‑D классы и вспомогательные типы.

```java
import com.aspose.threed.*;


import java.io.IOException;
```

## Шаг 1: Инициализация базового профиля

Создайте форму, которую будете экструдировать. В этом примере мы используем `RectangleShape` с небольшим радиусом скругления, чтобы края выглядели плавно.

```java
// The path to the documents directory.
String MyDir = "Your Document Directory";
RectangleShape profile = new RectangleShape();
profile.setRoundingRadius(0.3);
```

## Шаг 2: Создание сцены

Объект `Scene` служит контейнером для всех 3‑D узлов, источников света, камер и материалов.

```java
Scene scene = new Scene();
```

## Шаг 3: Создание узлов

Добавьте два дочерних узла к корню сцены — один для левой экструзии и один для правой. Правый узел сдвинут, чтобы два объекта не перекрывались.

```java
Node left = scene.getRootNode().createChildNode();
Node right = scene.getRootNode().createChildNode();
left.getTransform().setTranslation(new Vector3(5, 0, 0));
```

## Шаг 4: Выполнение линейной экструзии на левом узле

Экструдируйте профиль на левом узле, используя направление оси Z по умолчанию. Мы также добавляем полное вращение на 360° и увеличиваем количество срезов для более гладкой сетки.

```java
left.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); }});
```

## Шаг 5: Выполнение линейной экструзии на правом узле с направлением

Здесь мы **задаём направление**. Передавая пользовательский `Vector3` в `setDirection`, экструзия следует вектору (0.3, 0.2, 1), создавая наклонную форму.

```java
right.createChildNode(new LinearExtrusion(profile, 10) {{ setTwist(360); setSlices(100); setDirection(new Vector3(0.3, 0.2, 1));}});
```

## Шаг 6: Сохранение 3D сцены

Наконец, экспортируйте сцену в формат Wavefront OBJ. Этот шаг демонстрирует, как **save obj file java** проекты и упрощает просмотр результата в любом 3‑D просмотрщике.

```java
scene.save(MyDir + "DirectionInLinearExtrusion.obj", FileFormat.WAVEFRONTOBJ);
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|---------|
| OBJ‑файл пустой | Профиль не был добавлен в узел | Убедитесь, что `createChildNode` вызывается у корректного узла |
| Направление не изменилось | `setDirection` был вызван после того, как экструзия уже построена | Задайте направление внутри инициализатора `LinearExtrusion`, как показано |
| Сетка низкого разрешения | Значение `setSlices` слишком мало | Увеличьте количество срезов (например, 100 и более) |
