# Создание примитивных 3D‑моделей с помощью Aspose.3D для Java

## Введение

Если вам когда‑нибудь нужно было **создать 3D‑цилиндр** (или любую другую базовую форму) непосредственно из кода Java, Aspose.3D делает эту задачу простой. В этом руководстве мы пройдем весь процесс — от настройки окружения до сохранения конечной сцены в файл FBX — чтобы вы могли сразу начать генерировать 3D‑геометрию программно.

## Быстрые ответы
- **Какая библиотека позволяет создать 3D‑цилиндр в Java?** Aspose.3D for Java.
- **В какой формат я могу экспортировать сцену?** FBX (ASCII 7500) с использованием `FileFormat.FBX7500ASCII`.
- **Нужна ли лицензия для разработки?** Бесплатная пробная версия подходит для тестирования; для продакшна требуется постоянная лицензия.
- **Какие основные примитивы поддерживаются?** Box, Cylinder, Sphere, Cone и другие.
- **Совместим ли код с Java 8 и новее?** Да, Aspose.3D ориентирован на JDK 8+.

## Что такое примитив 3D‑цилиндр?

Примитив цилиндра — это базовая геометрическая форма, определяемая радиусом и высотой. Во многих 3D‑конвейерах он служит строительным блоком для более сложных моделей, таких как трубы, колёса или архитектурные колонны. Aspose.3D предоставляет готовый класс `Cylinder`, так что вам не нужно вручную рассчитывать вершины.

## Почему стоит использовать Aspose.3D для Java?

- **Full‑featured 3D engine** – поддерживает импорт/экспорт популярных форматов (FBX, OBJ, STL и др.).
- **Pure Java API** – без нативных зависимостей, идеально для кросс‑платформенных проектов.
- **Robust scene graph** – позволяет организовывать объекты иерархически.
- **Easy licensing** – начните с бесплатной пробной версии, затем перейдите на постоянную лицензию.

## Требования

Прежде чем погрузиться в код, убедитесь, что у вас есть:

1. **Java Development Kit (JDK)** – JDK 8 или новее, установленный на вашем компьютере.  
2. **Aspose.3D for Java library** – скачайте и установите её со [страницы загрузки](https://releases.aspose.com/3d/java/).  

## Импорт пакетов

Начните с импорта основного пространства имён Aspose.3D. Это даст вам доступ к `Scene`, `Box`, `Cylinder` и константам форматов файлов.

```java
import com.aspose.threed.*;
```

Теперь, когда библиотека подключена, создадим сцену и добавим несколько примитивных геометрий.

## Как создать 3D‑цилиндр и другие примитивы в сцене

### Шаг 1: Инициализация объекта Scene

Сначала нам нужен контейнер `Scene`, который будет хранить все наши 3D‑узлы.

```java
// The path to the documents directory.
String myDir = "Your Document Directory";

// Initialize a Scene object
Scene scene = new Scene();
```

### Шаг 2: Создание 3D‑модели коробки

Примитив коробки полезен для проверки системы координат. Здесь мы добавляем его как дочерний элемент корневого узла сцены.

```java
// Create a Box model
scene.getRootNode().createChildNode("box", new Box());
```

### Шаг 3: Создание 3D‑модели цилиндра

Теперь мы действительно **создаём 3D‑цилиндр**. Класс `Cylinder` имеет разумные размеры по умолчанию, но при необходимости вы можете задать радиус и высоту через его конструктор.

```java
// Create a Cylinder model
scene.getRootNode().createChildNode("cylinder", new Cylinder());
```

### Шаг 4: Сохранение рисунка в формате FBX

После сборки сцены экспортируйте её, чтобы другие инструменты (например, Unity, Blender) могли её прочитать. Мы используем формат `FBX7500ASCII`, который широко поддерживается.

```java
// Save drawing in the FBX format
myDir = myDir + "test.fbx";
scene.save(myDir, FileFormat.FBX7500ASCII);
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|---------|
| **File not found** при сохранении | `myDir` указывает на несуществующую папку | Убедитесь, что каталог существует, или создайте его с помощью `new File(myDir).mkdirs();` |
| **Empty scene** после экспорта | Перед вызовом `save` не были добавлены узлы | Проверьте, что вызовы `createChildNode` выполнены (отладьте с помощью `scene.getRootNode().getChildCount()` ) |
| **License exception** | Запуск без действующей лицензии в продакшн | Примените временную или постоянную лицензию, используя `License license = new License(); license.setLicense("Aspose.3D.Java.lic");` |
