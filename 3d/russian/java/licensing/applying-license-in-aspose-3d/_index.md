# Как установить лицензию Aspose в Aspose.3D для Java

## Введение

В этом полном руководстве вы узнаете **как установить лицензию Aspose** для Aspose.3D в среде Java. Независимо от того, предпочитаете ли вы загрузку файла лицензии, потоковую передачу или использование лицензирования с измерением (metered licensing) с публичными и приватными ключами, мы пошагово пройдём каждый подход, чтобы вы могли быстро и уверенно разблокировать полный набор функций Aspose.3D.

## Быстрые ответы
- **Какой основной способ установить лицензию Aspose.3D?** Используйте класс `License` и вызовите `setLicense` с путем к файлу или потоком.  
- **Можно ли загрузить лицензию из потока?** Да — оберните файл `.lic` в `FileInputStream` и передайте его в `setLicense`.  
- **Что делать, если нужна лицензия с измерением?** Инициализируйте объект `Metered` и вызовите `setMeteredKey`, передав свои публичный и приватный ключи.  
- **Нужна ли лицензия для сборок разработки?** Требуется пробная или временная лицензия для любого сценария, не являющегося оценочным.  
- **Какие версии Java поддерживаются?** Aspose.3D работает с Java 6 и новее.

## Предварительные требования

Прежде чем начать, убедитесь, что у вас есть следующие предварительные требования:

- Базовое понимание программирования на Java.  
- Библиотека Aspose.3D установлена. Вы можете скачать её со [страницы релизов](https://releases.aspose.com/3d/java/).  

## Импорт пакетов

Чтобы начать, импортируйте необходимые пакеты в ваш Java‑проект. Убедитесь, что Aspose.3D добавлен в classpath. Ниже пример:

```java
import com.aspose.threed.License;
import com.aspose.threed.Metered;

import java.io.FileInputStream;
import java.io.IOException;
```

## Применение лицензии с использованием файла

### Шаг 1: Создайте объект License

Сначала создайте объект `License` в вашем Java‑коде.

```java
License license = new License();
```

### Шаг 2: Примените лицензию из файла

Укажите путь к вашему файлу лицензии и установите лицензию, используя следующий код. Это демонстрирует технику **применения лицензии из файла**.

```java
license.setLicense("Aspose._3D.lic");
```

## Применение лицензии с использованием объекта Stream

### Шаг 1: Создайте объект License

Аналогично, создайте объект `License` в вашем Java‑коде.

```java
License license = new License();
```

### Шаг 2: Установите лицензию из объекта Stream

Используйте `FileInputStream` для создания потока и установки лицензии:

```java
try (FileInputStream myStream = new FileInputStream("Aspose._3D.lic")) {
    license.setLicense(myStream);
}
```

## Использование публичного и приватного ключей для лицензирования с измерением

### Шаг 1: Инициализируйте объект Metered License

Инициализируйте объект лицензии `Metered`:

```java
Metered metered = new Metered();
```

### Шаг 2: Установите публичный и приватный ключи

Установите свои публичный и приватный ключи, чтобы включить лицензирование с измерением. Это покрывает сценарий **установки публичного и приватного ключей**.

```java
metered.setMeteredKey("your-public-key", "your-private-key");
```

## Почему важно установить лицензию

Применение правильной лицензии удаляет водяные знаки оценки, разблокирует премиум‑форматы файлов и обеспечивает соответствие модели лицензирования Aspose. Использование подходящего метода (файл, поток или metered) позволяет бесшовно интегрировать лицензирование в CI/CD конвейеры, облачные развертывания или настольные приложения.

## Распространённые проблемы и советы

- **File not found** – Убедитесь, что путь к файлу `.lic` корректен относительно рабочей директории или используйте абсолютный путь.  
- **Stream closed prematurely** – При использовании потока держите объект `License` живым на протяжении всего приложения; лицензия кэшируется после первого успешного вызова.  
- **Metered key mismatch** – Дважды проверьте, что публичный и приватный ключи соответствуют одной лицензии с измерением; опечатка вызовет исключение во время выполнения.  
- **Pro tip:** Храните файл лицензии в безопасном месте вне дерева исходного кода и загружайте его через переменную окружения, чтобы избежать его коммита в систему контроля версий.

## Часто задаваемые вопросы

**Q: Совместима ли Aspose.3D со всеми версиями Java?**  
A: Да, Aspose.3D совместима с Java 6 и более новыми версиями.

**Q: Где я могу найти дополнительную документацию?**  
A: Вы можете обратиться к документации [здесь](https://reference.aspose.com/3d/java/).

**Q: Могу ли я попробовать Aspose.3D перед покупкой?**  
A: Да, вы можете ознакомиться с бесплатной пробной версией [здесь](https://releases.aspose.com/).

**Q: Как получить поддержку по Aspose.3D?**  
A: Посетите [форум Aspose.3D](https://forum.aspose.com/c/3d/18) для получения поддержки.

**Q: Нужна ли временная лицензия для тестирования?**  
A: Да, получите временную лицензию [здесь](https://purchase.aspose.com/temporary-license/).

**Q: В чём разница между файловой лицензией и лицензией с измерением?**  
A: Файловая лицензия — это статический файл `.lic`, привязанный к конкретной версии продукта, тогда как лицензия с измерением проверяет использование через облачную службу измерения Aspose с использованием публичных/приватных ключей.

**Q: Можно ли встроить код загрузки лицензии в статический инициализатор?**  
A: Абсолютно — размещение инициализации `License` в статическом блоке гарантирует, что лицензия будет применена один раз при первой загрузке класса.
