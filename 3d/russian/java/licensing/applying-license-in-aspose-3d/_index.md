# Как установить лицензию в Aspose.3D для Java

## Введение

Добро пожаловать! В этом пошаговом руководстве вы узнаете **как установить лицензию** для Aspose.3D в Java‑приложении, а также научитесь **использовать публичные и приватные ключи** для метерного лицензирования. Aspose.3D — мощная Java‑библиотека, упрощающая работу с 3D‑форматами файлов, и действительная лицензия раскрывает весь набор функций. К концу этого руководства вы сможете бесшовно интегрировать лицензирование в любой Java‑проект.

## Быстрые ответы
- **Какой основной способ установить лицензию?** Используйте класс `License` и вызовите `setLicense`, передавая путь к файлу или поток.  
- **Можно ли загрузить лицензию из потока?** Да — `FileInputStream` работает идеально.  
- **Для чего нужны публичные/приватные ключи?** Они позволяют использовать метерное лицензирование через класс `Metered`.  
- **Нужна ли лицензия для разработки?** Временная или пробная лицензия достаточна для тестирования; полная лицензия требуется для продакшн.  
- **Какие версии Java поддерживаются?** Aspose.3D работает с Java 6 и более новыми версиями.

## Требования

Перед началом убедитесь, что у вас есть:

- Базовые знания программирования на Java.  
- Библиотека Aspose.3D, добавленная в ваш проект. Скачайте её со [страницы релизов](https://releases.aspose.com/3d/java/).  
- Действительный файл `.lic` или ваши публичные и приватные метерные ключи.

## Импорт пакетов

Добавьте необходимые импорты в ваш Java‑файл. Убедитесь, что JAR‑файл Aspose.3D находится в classpath.

```java
import com.aspose.threed.License;
import com.aspose.threed.Metered;

import java.io.FileInputStream;
import java.io.IOException;
```

## Как установить лицензию с помощью файла

### Шаг 1: Создайте объект лицензии

Создайте экземпляр класса `License` — этот объект будет хранить информацию о лицензии.

```java
License license = new License();
```

### Шаг 2: Установите лицензию из файла

Укажите относительный или абсолютный путь к вашему файлу `.lic` и примените его.

```java
license.setLicense("Aspose._3D.lic");
```

> **Совет:** Храните файл лицензии вне директории системы контроля версий, чтобы избежать случайного раскрытия.

## Как установить лицензию с помощью потока

### Шаг 1: Создайте объект лицензии

Как и ранее, начните с нового экземпляра `License`.

```java
License license = new License();
```

### Шаг 2: Установите лицензию из потока

Прочитайте файл лицензии в `FileInputStream` и передайте поток в `setLicense`. Блок try‑with‑resources гарантирует автоматическое закрытие потока.

```java
try (FileInputStream myStream = new FileInputStream("Aspose._3D.lic")) {
    license.setLicense(myStream);
}
```

## Как использовать публичные и приватные ключи для метерного лицензирования

### Шаг 1: Инициализируйте объект Metered

Создайте экземпляр класса `Metered`, который управляет метерным (pay‑as‑you‑go) лицензированием.

```java
Metered metered = new Metered();
```

### Шаг 2: Установите публичный и приватный ключи

Передайте ключи, полученные от Aspose. Эти ключи позволяют библиотеке отправлять информацию об использовании на сервер лицензирования.

```java
metered.setMeteredKey("your-public-key", "your-private-key");
```

> **Внимание:** Никогда не встраивайте ваш приватный ключ напрямую в публично распространяемый JAR. Рассмотрите возможность загрузки его из безопасного места или переменной окружения.

## Распространённые сценарии использования

- **Корпоративные конвейеры 3D‑рендеринга** — разблокируйте высокопроизводительные API после установки лицензии.  
- **Автоматизированные тестовые среды** — используйте временную лицензию (см. FAQ) для проверки функциональности без покупки полной лицензии.  
- **Метёрные SaaS‑решения** — интегрируйте публичные/приватные ключи для выставления счетов клиентам на основе реального использования.
