## Часто задаваемые вопросы

**Q:** *Можно ли использовать Aspose.3D в коммерческом Java приложении?*  
**A:** Да. Для использования в продакшн требуется действующая лицензия Aspose; бесплатная пробная версия доступна для оценки.

**Q:** *Какие форматы 3D файлов поддерживаются для загрузки и сохранения?*  
**A:** Aspose.3D поддерживает OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X и несколько других.

**Q:** *Как обновить материалы до PBR без переписывания всей сцены?*  
**A:** Используйте метод `MaterialConverter.convertToPbr()` из API — он обновляет все материалы одним вызовом.

**Q:** *Есть ли способ уменьшить размер файла при сохранении больших сцен?*  
**A:** Да. Настройте `SaveOptions` (например, включите сжатие, уменьшите точность мешей), чтобы оптимизировать размер вывода.

**Q:** *Нужны ли дополнительные зависимости для работы с Aspose.3D?*  
**A:** Требуется только JAR Aspose.3D for Java; все остальные зависимости включены.
