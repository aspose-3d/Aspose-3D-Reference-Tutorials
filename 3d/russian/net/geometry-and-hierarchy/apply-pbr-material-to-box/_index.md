## Что такое материал PBR?

Physically Based Rendering (PBR) — это модель освещения, имитирующая взаимодействие света с реальными поверхностями. Настраивая такие свойства, как коэффициенты metallic и roughness, можно достичь чрезвычайно реалистичных результатов без ручной настройки сложных шейдеров.

## Почему использовать Physically Based Rendering (PBR)?

- **Реализм:** Материалы реагируют на освещение так, как это происходит в реальной физике.  
- **Последовательность:** Один и тот же материал выглядит корректно при любом освещении.  
- **Эффективность:** Современные GPU оптимизированы для вычислений PBR, обеспечивая производительность без дополнительных затрат.

## Пошаговое руководство

### Шаг 1: Инициализировать сцену
Начните с создания пустой 3D‑сцены. Этот контейнер будет хранить всю геометрию, материалы и источники света, которые вы добавите позже.

```csharp
Scene scene = new Scene();
```

### Шаг 2: Создать материал PBR
Теперь мы **создаём pbr материал**, который придаст нашей коробке реалистичный вид. Класс `PbrMaterial` раскрывает все параметры, необходимые для физически‑основанного рендеринга.

```csharp
PbrMaterial mat = new PbrMaterial();
```

### Шаг 3: Установить свойства материала
Точно настройте свойства материала. В этом примере мы делаем поверхность почти металлической и очень грубой — идеально для коробки из полированного металла.

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

### Шаг 4: Создать сетку коробки
Создайте простую геометрию коробки. Это шаг **create box mesh**, на который ссылается основной ключевой запрос.

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

### Шаг 5: Применить материал PBR к коробке
Назначьте ранее настроенный **add pbr material** узлу коробки, который мы только что создали.

```csharp
boxNode.Material = mat;
```

### Шаг 6: Экспортировать STL ASCII (Как экспортировать STL)
Наконец, **export stl ascii**, чтобы модель могла быть открыта в любом стандартном 3D‑просмотрщике или слайсере. Использование `FileFormat.STLASCII` гарантирует человекочитаемый файл.

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```

## Распространённые ошибки и советы
- **Лицензия не найдена:** Убедитесь, что файл лицензии загружен *до* любых вызовов Aspose; иначе библиотека работает в режиме оценки.  
- **Неправильный путь к файлу:** Используйте `Path.Combine`, чтобы избежать отсутствия разделителей пути на разных операционных системах.  
- **Баланс Roughness и Metallic:** Установка обоих факторов слишком высокими может сделать поверхность плоской; экспериментируйте со значениями от `0.5‑0.9` для сбалансированного вида.  
- **Совет по производительности:** Повторно используйте один экземпляр `PbrMaterial`, если нужно применить один и тот же материал к нескольким сеткам; это уменьшит нагрузку на память.
