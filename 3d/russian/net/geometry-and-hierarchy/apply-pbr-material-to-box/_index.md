# Как применить PBR материал к коробке

## Введение

Добро пожаловать в увлекательный мир 3D‑графики! В этом пошаговом руководстве вы узнаете **how to apply pbr** материал к коробке с помощью Aspose.3D for .NET. Мы пройдем процесс создания PBR‑материала, его применения к мешу и, наконец, **exporting STL ASCII**, чтобы вы могли использовать модель в любой последующей цепочке обработки. Независимо от того, создаёте ли вы прототип игры или визуализацию продукта, освоение этого рабочего процесса открывает двери к реалистичному физически‑основанному рендерингу (PBR) в ваших .NET‑приложениях.

- **Нужна ли лицензия?** Да, для продакшна требуется временная или полная лицензия.

## Что такое PBR материал?
Physically Based Rendering (PBR) — это модель затенения, имитирующая взаимодействие света с реальными поверхностями. Настраивая такие свойства, как metallic и roughness, можно достичь чрезвычайно реалистичных результатов без ручной настройки сложных шейдеров.

## Почему стоит использовать Physically Based Rendering (PBR)?
- **Реализм:** Материалы реагируют на освещение согласно реальной физике.  
- **Последовательность:** Один и тот же материал выглядит правильно при любом освещении.  
- **Эффективность:** Современные GPU оптимизированы для вычислений PBR, предоставляя производительность «в подарок».

Сначала импортируйте необходимые пространства имён, чтобы воспользоваться возможностями Aspose.3D for .NET:

## Шаг 1: Инициализация сцены
Начните с инициализации 3D‑сцены, используя следующий фрагмент кода:

```csharp
Scene scene = new Scene();
```

## Шаг 2: Создание PBR материала
Теперь **create pbr material**, который придаст нашей коробке реалистичный вид:

```csharp
PbrMaterial mat = new PbrMaterial();
```

## Шаг 3: Настройка свойств материала
Точно настройте свойства материала, сделав его почти металлическим и очень грубым — идеально для коробки с матовой металлической отделкой:

```csharp
mat.MetallicFactor = 0.9;
mat.RoughnessFactor = 0.9;
```

## Шаг 4: Создание коробки
Создайте коробку, к которой будет применён PBR материал:

```csharp
var boxNode = scene.RootNode.CreateChildNode("box", new Box());
```

## Шаг 5: Применение PBR материала к коробке
Назначьте ранее сконфигурированный **add pbr material** узлу созданной коробки:

```csharp
boxNode.Material = mat;
```

## Шаг 6: Сохранение 3D‑сцены как STL ASCII
Наконец, **export stl ascii**, чтобы модель могла быть открыта в любом стандартном 3D‑просмотрщике или слайсере:

```csharp
scene.Save("Your Output Directory" + "PBR_Material_Box_Out.stl", FileFormat.STLASCII);
```

Поздравляем! Вы успешно применили PBR материал к коробке в 3D‑сцене с помощью Aspose.3D for .NET.

## Распространённые ошибки и советы
- **Лицензия не найдена:** Убедитесь, что файл лицензии загружен до любых вызовов Aspose; иначе библиотека работает в режиме оценки.  
- **Неправильный путь к файлу:** Используйте `Path.Combine`, чтобы избежать отсутствия разделителей пути на разных ОС.  
- **Roughness vs. Metallic:** Установка обоих факторов слишком высокими может сделать поверхность плоской; экспериментируйте со значениями от 0.5‑0.9 для сбалансированного результата.
