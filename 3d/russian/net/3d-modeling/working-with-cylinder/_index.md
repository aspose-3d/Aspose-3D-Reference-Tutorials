# Как создать цилиндр с наклонным основанием – Aspose.3D for .NET

## Введение
Если вы задаётесь вопросом **как создать цилиндр** с пользовательским наклонным основанием в среде .NET, вы попали в нужное место. В этом руководстве мы пройдём каждый шаг — от настройки 3‑D сцены до экспорта готовой модели в файл OBJ — чтобы вы могли улучшить свои навыки *начального 3D моделирования* с помощью **Aspose.3D for .NET**.

## Быстрые ответы
- **Какой основной класс для начала 3D модели?** `Scene` создаёт корневой контейнер для всей геометрии.  
- **Какой метод экспортирует модель в OBJ?** `scene.Save(..., FileFormat.WavefrontOBJ)`.  
- **Нужна ли лицензия для тестирования?** Доступна бесплатная пробная версия — см. ссылку на пробную версию в FAQ.  
- **Можно ли изменить угол наклона?** Да, измените `ShearBottom`, задав значение `Vector2`.  
- **Подходит ли это для новичков?** Абсолютно; API абстрагирует работу с низкоуровневой сеткой.

## Что такое 3D сцена?
*3D сцена* — это иерархический контейнер, содержащий все геометрические объекты, источники света, камеры и трансформации. В Aspose.3D класс `Scene` предоставляет удобный способ организации и последующего экспорта ваших моделей.

## Почему экспортировать в OBJ?
OBJ — это широко поддерживаемый текстовый формат, который могут импортировать многие 3‑D приложения (Blender, Maya, Unity). Экспорт в OBJ позволяет делиться вашими моделями цилиндров или дальше их редактировать вне .NET.

## Предварительные требования
- Базовые знания C# и разработки на .NET.  
- **Aspose.3D for .NET** установлен — вы можете скачать его **[здесь](https://releases.aspose.com/3d/net/)**.  
- IDE для .NET (Visual Studio, Rider или VS Code), готовая к написанию кода.

## Импорт пространств имён
Сначала подключите необходимые пространства имён, чтобы типы API были распознаны.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Utilities;
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;
using System.Threading.Tasks;
```

## Шаг 1: Создать 3D сцену
Объект `Scene` служит корнем иерархии вашей модели.

```csharp
Scene scene = new Scene();
```

## Шаг 2: Создать цилиндр 1
Мы создаём первый цилиндр с радиусом 2, высотой 10 и 20 радиальными сегментами.

```csharp
var cylinder1 = new Cylinder(2, 2, 10, 20, 1, false);
```

## Шаг 3: Настроить наклонное основание для цилиндра 1
Примените трансформацию сдвига, включите генерацию fan‑cylinder и настройте другие свойства, чтобы получить желаемую форму.

```csharp
// Shear 47.5deg in the xy plane (z-axis)
cylinder1.ShearBottom = new Vector2(0, 0.83); 

// Set GenerateFanCylinder to true
cylinder1.GenerateFanCylinder = true;
// Set ThetaLength
cylinder1.ThetaLength = MathUtils.ToRadian(270);

// Set OffsetTop
cylinder1.OffsetTop = new Vector3(5, 3, 0);
```

## Шаг 4: Добавить цилиндр 1 в сцену
Разместите первый цилиндр в удобном месте, используя трансформацию перемещения.

```csharp
scene.RootNode.CreateChildNode(cylinder1).Transform.Translation = new Vector3(10, 0, 0);
```

## Шаг 5: Создать цилиндр 2
Второй цилиндр создаётся с теми же базовыми размерами, но без пользовательского сдвига — идеально для сравнения бок о бок.

```csharp
var cylinder2 = new Cylinder(2, 2, 10, 20, 1, false);
```

## Шаг 6: Добавить цилиндр 2 в сцену
Мы просто присоединяем второй цилиндр к графу сцены.

```csharp
scene.RootNode.CreateChildNode(cylinder2);
```

## Шаг 7: Экспортировать сцену в файл OBJ
Наконец, сохраняем всю сцену в файл OBJ, чтобы её можно было открыть в любом стандартном 3‑D просмотрщике.

```csharp
scene.Save("Your Document Directory" + "CustomizedShearBottomCylinder.obj", FileFormat.WavefrontOBJ);
```

## Распространённые проблемы и решения
| Проблема | Почему происходит | Решение |
|----------|-------------------|---------|
| **Файл OBJ пустой** | Сцена не содержит прикреплённой геометрии. | Убедитесь, что оба цилиндра добавлены в `scene.RootNode`. |
| **Наклон выглядит неправильно** | `ShearBottom` ожидает тангенс угла. | Используйте `Math.Tan(angleInRadians)` или значение `0.83` для ~47.5°. |
| **Ошибки пути к файлу** | Неверный или отсутствующий каталог. | Используйте `Path.Combine(Environment.CurrentDirectory, "CustomizedShearBottomCylinder.obj")`. |

**Дополнительные вопросы и ответы**

**Q: Как экспортировать модель в другой формат, например STL?**  
**A:** Замените `FileFormat.WavefrontOBJ` на `FileFormat.STL` в вызове `scene.Save`.

**Q: Можно ли анимировать цилиндры после создания?**  
**A:** Да, вы можете добавить анимацию ключевых кадров к трансформациям узлов, используя классы `Animation`, предоставленные Aspose.3D.

**Q: Поддерживает ли API .NET Core?**  
**A:** Библиотека полностью совместима с .NET Core, .NET 5+ и .NET 6+.
