## Быстрые ответы
- **Что такое выдавливание?** Операция трехмерного моделирования, которая расширяет двухмерный профиль по траектории для создания объема.
- **Почему стоит выбрать Aspose.3D?** Полная поддержка .NET, отсутствие внешних зависимостей и высокопроизводительная обработка геометрии.
- **Сколько времени занимает базовая линейная экструзия?** Обычно менее секунды для простых профилей на современном оборудовании.
- **Нужна ли мне лицензия?** Доступна пробная версия, но для производственного использования требуется коммерческая лицензия.
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6/7.

## Часто задаваемые вопросы

**В: Могу ли я объединить несколько методов экструзии в одной модели?**
О: Да, Aspose.3D позволяет объединять операции центрирования, направления, скручивания и смещения для создания сложной геометрии.

**В: Есть ли ограничение на количество срезов или поворотов, которые я могу применить?**
О: Практически пределом являются память и вычислительная мощность; API эффективно обрабатывает большие наборы данных.

**В: Нужно ли вручную рассчитывать путь выдавливания?**
О: Нет, библиотека предоставляет вспомогательные методы для определения путей и автоматического создания геометрии.

**В: Как предварительно просмотреть экструзию перед экспортом?**
О: Используйте встроенный просмотрщик сцен или экспортируйте в распространенный формат, например OBJ/GLTF, для быстрого просмотра.

**В: Есть ли советы по производительности для больших экструзий?**
О: Используйте повторно геометрические объекты, ограничьте количество вершин на срез и включите многопоточную обработку, где это возможно.
