## Что такое Linear Extrusion?

Linear extrusion берёт плоский профиль (например, прямоугольник или круг) и вытягивает его вдоль прямой линии, при желании добавляя скручивания, масштабирование или смещения. В результате получается твёрдое тело, которое можно визуализировать, редактировать или экспортировать для использования в других 3‑D инструментах.

## Выполнение Linear Extrusion

Ниже представлен полный рабочий процесс. Каждый шаг объясняется простым языком перед соответствующим блоком кода, чтобы вы могли следовать без догадок о том, что делает код.

### Шаг 1: Инициализация базового профиля

Сначала создаём 2‑D форму, которая будет экструдирована. В этом примере мы используем прямоугольник с небольшим радиусом скругления.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

*Почему это важно:* Профиль определяет поперечное сечение конечного 3‑D объекта. Изменяйте `RoundingRadius`, ширину или высоту, чтобы получить разные силуэты.

### Шаг 2: Применение Linear Extrusion

Теперь вытягиваем профиль на 10 единиц вдоль оси Z, добавляя 100 срезов для плавности, центрируя геометрию и применяя полный скрут 360° со смещением.

```csharp
var extrusion = new LinearExtrusion(profile, 10) { Slices = 100, Center = true, Twist = 360, TwistOffset = new Vector3(10, 0, 0) };
```

*Совет:* Экспериментируйте с `Slices`, чтобы сбалансировать производительность и визуальное качество, и пробуйте `Twist` для спиральных эффектов.

### Шаг 3: Создание сцены

`Scene` выступает контейнером для всех 3‑D сущностей — своего рода холст.

```csharp
Scene scene = new Scene();
```

### Шаг 4: Добавление экструдирования в сцену

Привяжите экструдированную геометрию к корневому узлу сцены, чтобы она стала частью иерархии рендеринга.

```csharp
scene.RootNode.CreateChildNode(extrusion);
```

### Шаг 5: Сохранение 3‑D модели

Наконец, экспортируйте сцену в широко поддерживаемый файл OBJ. Это демонстрирует возможность **export 3d model obj** в Aspose.3D.

```csharp
scene.Save(RunExamples.GetOutputFilePath("LinearExtrusion.obj"), FileFormat.WavefrontOBJ);
```

Когда вы откроете полученный `LinearExtrusion.obj` в любом 3‑D просмотрщике, вы увидите скрученный прямоугольный призмат — точно то, что описывает код.

## Распространённые ошибки и советы

| Проблема | Почему происходит | Как исправить |
|----------|-------------------|---------------|
| **Profile not visible** | Забыл добавить экструдирование в сцену. | Убедитесь, что вызван `CreateChildNode`. |
| **Twist looks flat** | `Slices` слишком мало, геометрия грубая. | Увеличьте `Slices` (например, до 200) для более плавного скручивания. |
| **Export fails** | Папка вывода не существует или нет прав на запись. | Используйте `RunExamples.GetOutputFilePath` или создайте каталог вручную. |
| **Unexpected scaling** | `Center` установлен в `false`, смещая геометрию. | Установите `Center = true`, если только не нужен смещение. |
