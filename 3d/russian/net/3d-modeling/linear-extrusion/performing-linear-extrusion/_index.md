# Выполнение линейной экструзии

## Введение:

Отправьтесь в захватывающее путешествие по миру 3D‑графики с Aspose.3D for .NET, мощным инструментом, который поднимает ваш уровень разработки. В этом руководстве **вы узнаете, как создать экструзию** – увлекательную технику, превращающую 2‑D профиль в полноценную 3‑D сетку. Мы пройдём каждый шаг, от установки Aspose.3D до экспорта результата в файл Wavefront OBJ, чтобы вы могли **создавать 3D из 2D** фигур с уверенностью.

## Быстрые ответы
- **Что такое линейная экструзия?** Расширение 2‑D формы вдоль прямой траектории для получения 3‑D объекта.  
- **Какую библиотеку использует этот урок?** Aspose.3D for .NET.  
- **Как сохранить OBJ?** Используйте `scene.Save(..., FileFormat.WavefrontOBJ)`.  
- **Можно ли экспортировать Wavefront OBJ?** Да – формат полностью поддерживается.  
- **Нужна ли лицензия?** Для тестирования достаточно временной лицензии; для продакшна требуется коммерческая лицензия.

using Aspose.ThreeD;

Эти пространства имён закладывают основу вашего 3D‑кода, предоставляя доступ к инструментам для бесшовной интеграции функций Aspose.3D.

## Выполнение линейной экструзии:

Создадим завораживающий 3D‑объект с помощью линейной экструзии в Aspose.3D. Следуйте этим шагам:

### Как создать экструзию – Шаг 1: Инициализация базового профиля
```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

Этот шаг задаёт 2‑D профиль, который станет фундаментом нашего 3‑D шедевра. При необходимости скорректируйте параметры, чтобы получить желаемую форму.

### Как создать экструзию – Шаг 2: Линейная экструзия
```csharp
var extrusion = new LinearExtrusion(profile, 10) { Slices = 100, Center = true, Twist = 360, TwistOffset = new Vector3(10, 0, 0) };
```

Здесь выполняется линейная экструзия: 2‑D профиль расширяется в третье измерение. Поэкспериментируйте с параметрами **Slices**, **Twist** и **TwistOffset**, чтобы **генерировать варианты 3D‑сеток**, соответствующие вашему замыслу.

### Как создать экструзию – Шаг 3: Создание сцены
```csharp
Scene scene = new Scene();
```

Создаётся пустой холст – сцена, где ваш 3‑D объект оживёт.

### Как создать экструзию – Шаг 4: Добавление экструзии в сцену
```csharp
scene.RootNode.CreateChildNode(extrusion);
```

Ваше экструзированное творение добавляется в виде дочернего узла к сцене.

### Как создать экструзию – Шаг 5: Сохранение 3D‑сцены
```csharp
scene.Save(RunExamples.GetOutputFilePath("LinearExtrusion.obj"), FileFormat.WavefrontOBJ);
```

Наконец, **как сохранить OBJ** – сохраняем результат в популярный формат Wavefront OBJ, который открывается в большинстве 3‑D‑редакторов.

Вот и ваш 3D‑чудо‑объект!

## Почему это важно

Линейная экструзия – быстрый способ **создавать 3D из 2D** эскизов, идеальный для быстрого прототипирования, архитектурного моделирования или генерации печатаемых сеток. Овладев этой техникой, вы получаете универсальный рабочий процесс, экономящий время и уменьшающий необходимость в сложных инструментах моделирования.

## Распространённые ошибки и советы

- **Слишком большие значения Twist** могут вызвать самопересечения. Держите twist ниже 720° для большинства простых форм.  
- **Недостаточное количество slices** приводит к гранёному виду. Увеличьте свойство `Slices` для более гладкого результата.  
- **Не забудьте установить `Center = true`**, если хотите, чтобы экструзия была центрирована относительно начала профиля – это часто упрощает последующее позиционирование.
