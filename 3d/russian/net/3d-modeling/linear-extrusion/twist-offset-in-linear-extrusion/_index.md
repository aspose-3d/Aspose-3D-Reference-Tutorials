# Как добавить крутку в линейную экструзию с помощью Aspose.3D для .NET

## Введение

Если вы ищете понятное пошаговое руководство о **том, как добавить крутку** к линейной экструзии, вы попали по адресу. В этом учебнике мы пройдём весь процесс с Aspose.3D для .NET, показывая, **как использовать экструзию** для создания динамических 3D‑форм, идеально подходящих для сценариев *asp.net 3d modeling*. К концу вы получите готовый к запуску пример, демонстрирующий смещение крутки, количество срезов и сохранение результата в файл OBJ.

## Быстрые ответы
- **Что делает «twist offset»?** Он смещает начальную точку крутки вдоль оси экструзии.  
- **Нужна ли лицензия для запуска примера?** Временная лицензия подходит для тестирования; полная лицензия требуется для продакшна.  
- **Какие версии .NET поддерживаются?** .NET Framework 4.5+, .NET Core 3.1+, .NET 5/6+.  
- **Можно ли изменить количество срезов?** Да — измените свойство `Slices`, чтобы контролировать гладкость сетки.  
- **Ограничен ли формат вывода OBJ?** Нет, Aspose.3D поддерживает множество форматов; OBJ используется здесь для простоты.

## Что такое Twist Offset в линейной экструзии?

Twist offset определяет трансляционный сдвиг, применяемый к операции крутки. Вместо вращения вокруг точного начала экструзии, геометрия начинает вращаться из указанного вектора смещения, что даёт вам больше художественного контроля над конечной формой.

## Импорт пространств имён

Сначала импортируйте пространства имён, дающие доступ к основным 3D‑классам.

```csharp
using Aspose.ThreeD;
using Aspose.ThreeD.Entities;
using Aspose.ThreeD.Profiles;
using Aspose.ThreeD.Utilities;
```

Эти инструкции делают типы `Scene`, `LinearExtrusion`, `Vector3` и другие необходимые типы доступными в вашем коде.

## Пошаговое руководство

### Шаг 1: Инициализация базового профиля

Начинаем с простого прямоугольного профиля и задаём небольшое скругление радиуса, чтобы края не были идеально острыми.

```csharp
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
```

### Шаг 2: Создание сцены

`Scene` выступает контейнером для всех узлов, источников света, камер и геометрии.

```csharp
Scene scene = new Scene();
```

### Шаг 3: Создание узлов

К корню сцены добавляются два дочерних узла — один для обычной экструзии, другой для крутки. Левый узел смещён по оси X для визуального разделения.

```csharp
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(18, 0, 0);
```

### Шаг 4: Линейная экструзия на левом узле (без Twist Offset)

Здесь демонстрируется базовая экструзия с полной круткой 360° и 100 срезами для плавности.

```csharp
left.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100 });
```

### Шаг 5: Линейная экструзия на правом узле с Twist Offset

Теперь применяем смещение крутки `(3, 0, 0)`. Это перемещает начало крутки на три единицы вдоль оси X, создавая явно смещённую спираль.

```csharp
right.CreateChildNode(new LinearExtrusion(profile, 10) { Twist = 360, Slices = 100, TwistOffset = new Vector3(3, 0, 0) });
```

### Шаг 6: Сохранение 3D‑сцены

Наконец, записываем сцену в файл OBJ. При необходимости измените путь вывода под свою среду.

```csharp
scene.Save("Your Output Directory" + "TwistOffsetInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|---------|
| **Крутка выглядит плоской** | `Slices` установлен слишком низко, получаем грубую сетку. | Увеличьте `Slices` (например, до 200) для более плавного вращения. |
| **Объект смещён от центра** | `TwistOffset` использует мировые координаты; узел уже может быть смещён. | Примените смещение относительно локального трансформа узла или скорректируйте перевод узла. |
| **Файл не сохраняется** | Неправильный путь вывода или отсутствие прав записи. | Убедитесь, что каталог существует и приложение имеет права на запись. |
| **Исключение лицензии** | Запуск без действующей лицензии в сборке, не являющейся пробной. | Загрузите временную или постоянную лицензию перед созданием сцены. |
