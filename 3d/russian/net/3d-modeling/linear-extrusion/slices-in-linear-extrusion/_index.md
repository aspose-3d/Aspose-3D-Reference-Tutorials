# Создание 3D‑сцены с линейной экструзией и срезами

## Введение

Добро пожаловать в мир 3D‑дизайна с использованием Aspose.3D для .NET! В этом руководстве вы **создадите 3d сцену** объектов, примените линейную экструзию с пользовательским числом срезов и в конце **сохраните 3d модель** в файл Wavefront OBJ. Независимо от того, создаёте ли вы быстрый прототип или готовую к производству визуализацию, нижеописанные шаги покажут вам **как использовать Aspose** для генерации высококачественной геометрии непосредственно из C#.

## Быстрые ответы
- **Что означает «create 3d scene»?** Это создание объекта `Scene`, который содержит все 3D‑сущности (меши, источники света, камеры).  
- **Какой формат используется для экспорта?** Пример экспортирует в **Wavefront OBJ** (`export wavefront obj`).  
- **Сколько срезов можно задать?** Можно задать любое целое число; в демонстрации показаны 2 и 10 срезов.  
- **Нужна ли лицензия?** Для использования в продакшене требуется временная или полная лицензия.  
- **Можно ли запускать это на .NET Core?** Да, Aspose.3D поддерживает .NET Framework и .NET Core.

- Желание изучать 3D‑дизайн: Страсть к созданию визуально впечатляющих 3D‑моделей!

Чтобы начать путь 3D‑дизайна с Aspose.3D, вам нужно импортировать необходимые пространства имён. Это гарантирует, что ваш код получит доступ к требуемым классам и функциям.

using Aspose.ThreeD;

## Как создать 3d сцену с линейной экструзией

Ниже мы пройдём каждый шаг, необходимый для построения сцены, применения экструзии и **сохранения 3d модели** в файл OBJ. Объяснения написаны в разговорном тоне, чтобы вам было легко следовать.

### Шаг 1: Инициализация базового профиля

Сначала мы определяем 2‑D форму, которая будет экструдирована. В данном случае это прямоугольник со скруглёнными углами.

```csharp
// ExStart:InitializeBaseProfile
var profile = new RectangleShape()
{
    RoundingRadius = 0.3
};
// ExEnd:InitializeBaseProfile
```

### Шаг 2: Создание 3D‑сцены

Объект `Scene` является контейнером для всей геометрии, источников света и камер — ядром **создания 3d сцены**.

```csharp
// ExStart:Create3DScene
Scene scene = new Scene();
// ExEnd:Create3DScene
```

### Шаг 3: Создание левого и правого узлов

Мы добавляем два дочерних узла к корню сцены. Один будет использовать небольшое количество срезов, другой — большее, чтобы вы могли увидеть визуальную разницу.

```csharp
// ExStart:CreateLeftRightNodes
var left = scene.RootNode.CreateChildNode();
var right = scene.RootNode.CreateChildNode();
left.Transform.Translation = new Vector3(15, 0, 0);
// ExEnd:CreateLeftRightNodes
```

### Шаг 4: Выполнение линейной экструзии на левом узле

Здесь мы экструдируем прямоугольник с **2 срезами**. Меньшее количество срезов даёт более грубый вид.

```csharp
// ExStart:LinearExtrusionLeftNode
left.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 2 });
// ExEnd:LinearExtrusionLeftNode
```

### Шаг 5: Выполнение линейной экструзии на правом узле

Теперь мы экструдируем тот же профиль, но с **10 срезами**, получая более гладкую поверхность.

```csharp
// ExStart:LinearExtrusionRightNode
right.CreateChildNode(new LinearExtrusion(profile, 2) { Slices = 10 });
// ExEnd:LinearExtrusionRightNode
```

### Шаг 6: Сохранение 3D‑сцены

Наконец, мы экспортируем сцену в файл Wavefront OBJ. Это демонстрирует **как сохранить obj** и **экспортировать wavefront obj** с помощью Aspose.3D.

```csharp
// ExStart:Save3DScene
scene.Save("Your Output Directory" + "SlicesInLinearExtrusion.obj", FileFormat.WavefrontOBJ);
// ExEnd:Save3DScene
```

## Распространённые проблемы и решения

| Проблема | Почему происходит | Решение |
|----------|-------------------|---------|
| Файл OBJ пустой | Неправильный путь вывода или отсутствуют права записи. | Убедитесь, что каталог существует и приложение имеет права записи. |
| Срезы не влияют на гладкость | Значение `Slices` слишком мало для размера геометрии. | Увеличьте количество срезов или скорректируйте размеры профиля. |
| Исключение лицензии | Запуск без действующей лицензии в сборке без пробного периода. | Примените временную или постоянную лицензию, используя `License license = new License(); license.SetLicense("Aspose.3D.lic");` |
