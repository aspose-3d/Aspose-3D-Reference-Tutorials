# Создание примитивных 3D‑моделей

## Введение

Добро пожаловать в захватывающий мир 3D‑моделирования с Aspose.3D для .NET! В этом подробном руководстве мы покажем, **how to create box** примитивные 3D‑модели, пройдем шаги **how to save FBX** и дадим вам уверенность в **export 3D model FBX** файлов для использования в любой конвейерной системе. Независимо от того, являетесь ли вы опытным разработчиком или только начинаете, вы найдете чёткие, практические рекомендации, которые можно применить сразу.

## Быстрые ответы
- **Какова основная цель?** Научиться создавать примитивные 3D‑модели коробки с помощью Aspose.3D.  
- **Какой формат используется для экспорта?** Формат FBX (FileFormat.FBX7500ASCII).  
- **Нужна ли лицензия?** Доступна бесплатная пробная версия; для продакшна требуется лицензия.

- **Сколько времени займет?** Около 10‑15 минут для создания базовой сцены.

## Что такое примитивная 3D‑модель?
Примитивная 3D‑модель — это базовая геометрическая форма, такая как коробка, сфера или цилиндр, используемая как строительный блок для более сложных сцен. Aspose.3D предоставляет готовые классы, позволяющие создавать эти формы одной строкой кода.

## Почему стоит использовать Aspose.3D для .NET?
- **Отсутствие внешних зависимостей** — не требуется внешний графический движок.  
- **Широкая поддержка форматов** — прямой экспорт в FBX, OBJ, STL и другие.  
- **Полная интеграция с .NET** — работает с .NET Framework, .NET Core и .NET 5/6+.

## Как создать примитивную 3D‑модель коробки

### Шаг 1: Инициализация объекта Scene

```csharp
// Initialize a Scene object
Scene scene = new Scene();
```

Создайте новый объект сцены, который будет служить холстом для вашего 3D‑шедевра.

### Шаг 2: Создание модели коробки

```csharp
// Create a Box model
scene.RootNode.CreateChildNode("box", new Box());
```

Добавьте модель коробки в корень вашей сцены. Это ядро **how to create box** геометрии. При необходимости позже можно изменить её размеры.

### Шаг 3: Создание модели цилиндра

```csharp
// Create a Cylinder model
scene.RootNode.CreateChildNode("cylinder", new Cylinder());
```

Улучшите сцену, добавив модель цилиндра. Настройте её параметры для получения нужной формы и размеров.

### Шаг 4: Сохранение рисунка в формате FBX (How to Save FBX)

```csharp
// Save drawing in the FBX format
var output = "Your Output Directory" + "test.fbx";
scene.Save(output, FileFormat.FBX7500ASCII);
```

Здесь мы демонстрируем **how to save FBX** — сцена экспортируется в файл FBX, который можно импортировать в большинство 3D‑инструментов. Этот шаг также показывает, как **export 3D model FBX** для последующих рабочих процессов.

### Шаг 5: Вывод сообщения об успехе

```csharp
// Display success message
Console.WriteLine("\nBuilding a scene from primitive 3D models successfully.\nFile saved at " + output);
```

Поздравляем! Сцена успешно построена из примитивных 3D‑моделей, и файл сохранён.

## Распространённые проблемы и решения
- **уть вывода не найден** — Убедитесь, что указанный в `output` каталог существует, либо используйте `Path.Combine` с `Environment.CurrentDirectory`.  
- **Исключение лицензии** — Для продакшна требуется действующая лицензия Aspose.3D; бесплатная пробная версия подходит только для оценки.  
- **Неправильная версия FBX** — В коде используется `FBX7500ASCII`; при необходимости переключитесь на другое значение перечисления `FileFormat`.

## Часто задаваемые вопросы

**Q: Можно ли экспортировать сцену в форматы, отличные от FBX?**  
A: Да, Aspose.3D поддерживает OBJ, STL, 3MF и многие другие. Просто измените значение перечисления `FileFormat` при вызове `scene.Save()`.

**Q: Можно ли настроить размеры коробки?**  
A: Конечно. Используйте конструктор `Box(double width, double height, double depth)`, чтобы задать пользовательские размеры.

**Q: Требуется ли 64‑разрядная ОС для запуска экспортированного FBX‑файла?**  
A: Нет, FBX‑файл платформенно‑независим; любой современный 3D‑просмотрщик сможет его открыть.

**Q: Как добавить материалы или текстуры к примитивам?**  
A: Создайте объект `Material`, присвойте его свойству `Material` узла и, при необходимости, задайте карты текстур.
