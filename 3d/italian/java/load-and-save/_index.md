## Problemi Comuni e Soluzioni

MaterialConverter è una classe di utilità che converte definizioni di materiali legacy al formato Physically Based Rendering (PBR).

**Problema:** *I materiali perdono la texture dopo la conversione.*  
**Soluzione:** Usa `MaterialConverter.convertToPbr()` prima del salvataggio; preserva le mappe di texture e aggiorna automaticamente i parametri dello shader.

LoadOptions è una classe di configurazione che controlla come Aspose.3D analizza i file di input.

**Problema:** *File OBJ di grandi dimensioni causano OutOfMemoryError.*  
**Soluzione:** Abilita lo streaming tramite `LoadOptions.setEnableMemoryOptimization(true)` e aumenta la dimensione dell'heap JVM (`-Xmx4g`).

SaveOptions configura le impostazioni di output come formato, compressione e inclusione di funzionalità durante il salvataggio di una scena.

**Problema:** *I file FBX esportati mancano dei dati di animazione.*  
**Soluzione:** Imposta `SaveOptions.setExportAnimations(true)` e verifica che la scena di origine contenga tracce di animazione chiave.
