## Preguntas Frecuentes

**Q:** *¿Puedo usar Aspose.3D en una aplicación Java comercial?*  
**A:** Sí. Se requiere una licencia válida de Aspose para uso en producción; una prueba gratuita está disponible para evaluación.

**Q:** *¿Qué formatos de archivo 3D son compatibles para cargar y guardar?*  
**A:** Aspose.3D supports OBJ, STL, FBX, 3DS, glTF, PLY, U3D, X, and several others.

**Q:** *¿Cómo actualizo los materiales a PBR sin reescribir toda la escena?*  
**A:** Use the `MaterialConverter.convertToPbr()` method provided in the API – it updates all materials in one call.

**Q:** *¿Hay alguna forma de reducir el tamaño del archivo al guardar escenas grandes?*  
**A:** Yes. Configure `SaveOptions` (e.g., enable compression, reduce mesh precision) to optimize output size.

**Q:** *¿Necesito dependencias adicionales para trabajar con Aspose.3D?*  
**A:** Only the Aspose.3D for Java JAR is required; all other dependencies are bundled.
