## 常见问题

**Q: 使用 Aspose.3D for Java 是否需要显卡？**  
A: 不需要。Aspose.3D 在 CPU 上完成所有计算，因此可以在任何能够运行 Java 的机器上使用。

**Q: 我可以将 PBR 材质与自定义着色器结合使用吗？**  
A: 可以。您可以在使用 Aspose.3D 的 PBR 工作流的同时，将自定义着色器程序附加到网格上。

**Q: “how to concatenate quaternions” 如何提升动画？**  
A: 拼接四元数可以将多个旋转合并为一次平滑的变换，避免万向锁。

**Q: 是否支持导出为 glTF 或 OBJ？**  
A: Aspose.3D 可以将场景导出为 glTF、OBJ、FBX 以及其他多种常见的 3D 格式。

**Q: 我在哪里可以找到示例项目？**  
A: Aspose.3D 的 GitHub 仓库和官方文档站点提供了可直接运行的示例，涵盖上述所有教程。
