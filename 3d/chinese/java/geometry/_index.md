## 常见问题

**问：使用 Aspose.3D for Java 是否需要显卡？**  
答：不需要。Aspose.3D 所有计算均在 CPU 上完成，任何能够运行 Java 的机器都可使用。

**问：可以将 PBR 材质与自定义着色器结合使用吗？**  
答：可以。你可以在保持 Aspose.3D PBR 工作流的同时，为网格附加自定义着色器程序。

**问：“how to concatenate quaternions” 如何提升动画效果？**  
答：四元数拼接可以将多个旋转合并为一次平滑的变换，避免万向锁问题。

**问：是否支持导出为 glTF 或 OBJ？**  
答：Aspose.3D 可以将场景导出为 glTF、OBJ、FBX 等多种常见 3D 格式。

**问：在哪里可以找到示例项目？**  
答：Aspose.3D 的 GitHub 仓库以及官方文档站点提供了所有上述教程的可直接运行示例。
