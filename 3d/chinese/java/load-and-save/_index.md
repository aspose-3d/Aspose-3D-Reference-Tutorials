## 常见问题

**Q: 如何使用 Aspose.3D 开始一个新的 3D 项目？**  
A: 添加 Aspose.3D 的 Maven/Gradle 依赖，创建一个 `Scene` 对象，即可开始构建几何体、灯光和相机。

**Q: 我可以为特定格式自定义加载过程吗？**  
A: 可以，使用 `LoadOptions` 指定单位转换、材质处理以及是否导入动画。

**Q: 升级材质为 PBR 的最佳方式是什么？**  
A: 在加载后对每个材质调用 `Material.convertToPbr()`；API 会自动将纹理映射到 PBR 工作流。

**Q: 保存大型场景时如何减小文件大小？**  
A: 启用网格压缩，移除未使用的节点，并通过 `SaveOptions` 选择二进制格式。

**Q: 是否有办法以编程方式批量转换多个文件？**  
A: 遍历源目录，使用 `Scene.load()` 加载每个文件，然后使用 `save()` 并指定目标格式。
