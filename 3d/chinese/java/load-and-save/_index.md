## 常见问题

**Q:** *我可以在商业 Java 应用中使用 Aspose.3D 吗？*  
**A:** 是的。生产环境需要有效的 Aspose 许可证；免费试用可用于评估。

**Q:** *支持哪些 3D 文件格式用于加载和保存？*  
**A:** Aspose.3D 支持 OBJ、STL、FBX、3DS、glTF、PLY、U3D、X 等多种格式。

**Q:** *如何在不重写整个场景的情况下将材料升级为 PBR？*  
**A:** 使用 API 中提供的 `MaterialConverter.convertToPbr()` 方法——一次调用即可更新所有材料。

**Q:** *在保存大型场景时有没有办法减小文件大小？*  
**A:** 有。配置 `SaveOptions`（例如启用压缩、降低网格精度）即可优化输出大小。

**Q:** *使用 Aspose.3D 是否需要额外的依赖？*  
**A:** 只需 Aspose.3D for Java 的 JAR 包，所有其他依赖已打包。
