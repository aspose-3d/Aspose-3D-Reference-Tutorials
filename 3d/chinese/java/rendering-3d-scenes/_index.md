## 常见问题

**Q: Aspose.3D 在底层使用哪种渲染引擎？**  
A: Aspose.3D 利用高性能、硬件加速的渲染引擎，跨平台运行且无需额外的本地依赖。

**Q: 我可以在同一场景中组合多个渲染目标吗？**  
A: 可以，您可以通过配置不同的 `RenderOptions`，将同一场景渲染到多个目标（例如窗口和 `BufferedImage`）。

**Q: 能否将渲染的场景导出为 PNG/JPEG 之外的格式？**  
A: 完全可以。Aspose.3D 支持 BMP、TIFF，甚至原始像素缓冲区，以便进行自定义后处理。

**Q: 如何实现实时帧率？**  
A: 使用 SWT 或 JavaFX 集成，启用 GPU 加速，并保持场景复杂度低。仅在每帧更新变化的对象也有助于提升性能。

**Q: 渲染为图像时需要手动管理内存吗？**  
A: API 提供自动资源清理，但及时释放大型 `BufferedImage` 对象可提升长时间运行应用的性能。
