## 为什么这些主题重要

* **性能：** 生成切线并拆分网格可减少绘制调用，提升 GPU 利用率。  
* **文件大小：** Draco 压缩可将资源体积缩小至原来的 10 % 左右，加快终端用户的下载速度。  
* **视觉保真度：** 正确的切线确保法线贴图渲染准确，使模型呈现更真实的外观。
