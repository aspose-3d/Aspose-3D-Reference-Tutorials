## 常见问题

**Q: 我可以在同一流水线中将 Draco 压缩与网格数据生成结合使用吗？**  
A: 是的。典型的工作流程是先生成缺失的数据（法线、切线、双切线），然后对已丰富的网格应用 Draco 压缩。

**Q: 降低 3D 文件大小会影响运行时性能吗？**  
A: 降低文件大小通常会提升加载速度并减少内存占用。若结合适当的网格组织（例如材质拆分），还能降低绘制调用次数，提升运行时性能。

**Q: 对于可以使用 Draco 压缩的网格大小是否有任何限制？**  
A: Draco 能处理非常大的网格，但极高多边形模型可能需要调整压缩设置（例如量化位数），以在大小和质量之间取得平衡。

**Q: 解压 Draco 网格后需要重新生成切线吗？**  
A: 不需要。如果在压缩前已生成切线，Draco 会保留包括切线在内的顶点属性。

**Q: 生产使用是否需要商业许可证？**  
A: 是的。虽然免费试用可以让您体验功能，但在生产部署中需要有效的 Aspose.3D 许可证。
