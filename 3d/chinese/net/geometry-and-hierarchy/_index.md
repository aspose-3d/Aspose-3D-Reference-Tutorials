# 几何与层次结构

## 介绍

欢迎阅读使用 Aspose.3D for .NET 的 **geometric transformation 3d** 终极指南。无论您是刚入门还是已有多年构建 3D 应用的经验，这套动手教程将帮助您掌握从应用 PBR 材质到使用变换矩阵操作节点的全部内容。**您将学习如何应用 PBR——具体而言，如何将 pbr 材质应用于您的 3D 模型**。让我们深入了解，看看如何将概念转化为真实的 3D 场景。

## 快速回答
- **What is PBR?** 物理基础渲染 (PBR) 模拟光照下材料的真实响应。  
- **Why use Aspose.3D for PBR?** 它提供了简洁的 .NET API 用于材质创建和渲染。  
- **Do I need a license?** 提供免费试用；生产环境需要商业许可证。  
- **Which .NET versions are supported?** 支持 .NET Framework 4.5+、.NET Core 3.1+、.NET 5/6+。  
- **Can I combine PBR with custom shaders?** 是的，您可以使用自定义着色器代码扩展材质。

## 在 Aspose.3D 中应用 PBR 材质的方法
应用 PBR 材质是创建写实 3D 场景的基石。在本节中，我们将逐步讲解关键步骤——创建材质、将其分配给几何体并渲染结果。相同的方法在下面的许多教程中都会复用，掌握它将加速您探索的所有其他示例。

### 为什么使用 PBR？
- **Realism:** PBR 模拟光线在真实世界中与表面的交互。  
- **Consistency:** 材质在任何光照条件下都表现正确。  
- **Performance:** 优化的着色器使 PBR 适用于交互式应用。

### 前置条件
- 已安装 Aspose.3D for .NET（最新版本）。  
- .NET 开发环境（Visual Studio 2022 或更高版本）。

### 它如何与其他主题结合
一旦您了解 **how to apply pbr**，就可以轻松将其与其他变换结合使用，例如 **euler angle rotation**、四元数混合或直接矩阵操作——这些都在后续教程中涵盖。

## 常见问题

**Q: 如何开始一个使用 PBR 材质的新项目？**  
A: 安装 Aspose.3D NuGet 包，创建一个 `Scene`，并遵循 “Applying PBR Material to Box” 教程添加您的第一个 PBR 材质。

**Q: 这些教程可以在 .NET Core 上使用吗？**  
A: 是的，所有示例均可在 .NET Core 3.1 及更高版本，以及 .NET 5/6 上运行。

**Q: 查询节点层次结构的最佳方法是什么？**  
A: 使用在 “XPath‑Like Object Queries” 教程中演示的类 XPath 查询语法——这是查询层次结构的最简洁方法 **how to query hierarchy**。

**Q: 我的立方体出现不正确的光照——我应该检查什么？**  
A: 确认法线已正确设置（参见 “Setting Up Normals on Cube”），并确保您的 PBR 材质包含适当的金属度和粗糙度值。

**Q: 使用大量变换矩阵时是否有性能考虑？**  
A: 尽可能批量进行变换并复用矩阵对象以减少分配；“Transforming Node by Transformation Matrix” 指南包含最佳实践提示。
