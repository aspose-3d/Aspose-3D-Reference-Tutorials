### 1단계: 출력 디렉터리 설정

```csharp
// 문서 디렉터리의 경로입니다.
var output = "Your Output Directory" + "document.fbx";
```

### 2단계: 빈 3D 문서 만들기

```csharp
// ExStart:CreateEmpty3DDocument

// Scene 클래스의 객체 생성

// 3D 장면 문서를 FBX 형식으로 저장

// ExEnd:CreateEmpty3DDocument
```

### 3단계: 결과 표시
